主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月5日｜周三
2025-03-04｜SJTU,Meituan｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.02357v2https ://github.com/zzc-1998/Q-Eval
随着生成性人工智能的迅速发展，文本到视觉内容的生成在多个平台上日益普及。然而，评估这些生成内容的视觉质量和文本与视觉之间的对齐程度仍然是一个挑战。现有的评估模型往往依赖于人类标注的规模和质量。为了解决这一问题，研究者们提出了Q-Eval-100K数据集，这是迄今为止最大的文本到视觉评估数据集，包含100K实例（60K图像和40K视频）及960K人类标注的均值意见分数（MOS）。该数据集专注于视觉质量和对齐程度两个关键维度，旨在为文本到视觉模型的评估提供更为全面和准确的依据。
Q-Eval-100K的构建遵循几个基本原则，确保生成内容的多样性和标注质量。首先，研究者设计了涵盖实体生成、属性生成和交互能力的多样化提示，确保生成内容的丰富性。其次，采用“样本与审查”策略，确保标注的准确性，最终形成960K高质量的人类标注数据。为了提升评估的有效性，研究者提出了Q-Eval-Score模型，这是一种统一的评估框架，能够独立评估视觉质量和对齐程度。在训练过程中，数据被转化为上下文提示格式，以便更好地引导大型多模态模型（LMMs）进行评估。特别是针对长文本提示的对齐评估，采用了模糊到具体的策略，以提高评估的准确性。
实验部分展示了Q-Eval-100K和Q-Eval-Score模型的有效性。通过与现有的评估方法进行对比，Q-Eval-Score在视觉质量和对齐程度的评估中均表现出色，尤其是在图像模型层面，达到了超过0.94的性能，接近人类评估结果。此外，实验还探讨了长文本提示的处理效果，表明模糊到具体策略显著提升了评估准确性。通过对不同生成模型的性能分析，研究发现视觉质量普遍低于对齐评分，反映了视觉质量评估的复杂性和主观性。整体而言，Q-Eval-Score为文本到视觉模型的评估提供了强有力的支持，展示了其在实际应用中的潜力。
在这个研究中，作者们创建了一个叫做Q-Eval-100K的大型数据集，用来评估生成的图像和视频的质量。这个数据集包含了很多人类对图像和视频的评分，帮助我们理解这些生成内容的视觉效果和与文本描述的匹配程度。为了更好地评估这些内容，研究者们设计了一种新的评估方法，叫做Q-Eval-Score。这个方法可以同时评估生成内容的视觉质量和与文本的对齐情况。研究者还特别考虑了长文本描述的情况，提出了一种将复杂描述简化为核心信息的方法，这样可以更准确地评估生成内容是否符合描述。通过这些方法，研究者们希望能够提高生成内容的评估质量，推动生成性人工智能的发展。
2025-03-04｜XDU,Paris-SaclayU｜⭐️🟡
http ://arxiv.org/abs/2503.02206v1
本研究提出了一种新颖的多模态框架，旨在实现图像质量评估（IQA）和条件图像生成（CIG）任务中的语义与感知特征的解耦。当前流行的对比视觉语言模型（例如CLIP）在语义识别任务中表现出色，但往往将语义与感知特征混合在一起，限制了对图像质量的准确评估。为了解决这一问题，本文构建了一个新的I&2T数据集，其中每个图像配有两个不同的文本描述，一个关注于感知特征，另一个关注于语义特征。通过利用这些解耦的文本描述作为监督信号，研究者能够从CLIP的特征空间中提取出纯粹的感知表示，称为DeCLIP，从而提升模型在图像质量评估和条件图像生成任务中的表现。
本研究的方法主要包括三个步骤：首先，构建I&2T数据集，该数据集包含112,769对图像及其对应的感知和语义文本描述。其次，利用解耦的文本描述作为监督信号，进行图像的解耦表示学习。具体而言，使用CLIP的视觉编码器和文本编码器，分别对图像和文本进行编码，生成感知和语义的特征表示。最后，将解耦后的视觉表示应用于图像质量评估和条件图像生成任务中。为实现更精确的控制，研究者还引入了DP-Adapter，通过将解耦的视觉特征与文本描述相结合，增强了生成图像的质量和一致性。
在实验部分，研究者评估了DeCLIP在图像质量评估和条件图像生成任务中的性能。通过与传统的CLIP模型进行比较，DeCLIP在多个数据集上显示出显著的性能提升。特别是在图像技术质量评估（TQA）和图像美学质量评估（AQA）任务中，DeCLIP在真实和合成失真场景下均表现优异。此外，研究者还进行了跨数据集测试，验证了模型的泛化能力。通过用户研究，评估了生成图像的语义一致性和感知一致性，结果表明，DP-Adapter在控制生成图像的语义和感知特征方面具有更好的表现。
在这个研究中，科学家们想要更好地理解和生成图像。他们发现，现有的方法常常把图像的内容和感觉混在一起，这样很难准确评估图像的质量。为了解决这个问题，他们创建了一个新的数据集，里面的每张图像都有两个描述：一个描述图像的内容（比如是什么物体），另一个描述图像的感觉（比如颜色和光线）。接着，他们使用这两个描述来帮助计算机更好地理解图像的不同方面。最后，他们测试了这个新方法的效果，发现它在评估图像质量和生成新图像时表现得更好。总之，这项研究帮助我们更清晰地分辨图像的内容和感觉，从而提高图像的分析和生成能力。
2025-03-04｜ANU｜⭐️🟡
http ://arxiv.org/abs/2503.02883v1https ://github.com/Qinyu-Allen-Zhao/Arinar
本研究提出了一种双层自回归特征生成模型ARINAR（AR-in-AR），用于图像生成任务。传统的自回归（AR）模型通常采用逐个生成令牌的方式，这种方法在处理高维令牌分布时面临挑战。现有模型往往要么过于简单，无法有效拟合数据分布，要么生成速度较慢。ARINAR通过将生成过程分为外层和内层两个自回归模型，分别在令牌和特征层面进行建模，从而提高了生成效率和质量。在ImageNet256×256图像生成任务中，ARINAR-B模型以213M参数实现了2.75的FrechetInceptionDistance（FID）分数，表现与当前最先进的MAR-B模型相当，但速度快了五倍。这表明该方法在图像生成领域具有良好的潜力。
ARINAR模型的架构由两个自回归层组成。首先，使用自编码器将图像转换为256个16维的连续令牌。外层自回归模型接收之前生成的令牌作为输入，并预测下一个令牌的条件向量z。内层自回归模型则基于条件向量z，逐步生成下一个令牌的特征。具体步骤如下：
外层生成条件向量z，基于之前的令牌。
内层接收条件向量z，生成下一个令牌的特征，并利用高斯混合模型（GMM）来建模特征分布。
生成的特征被反馈至内层，以预测后续特征，直到所有特征生成完成。
最终生成的令牌序列被输入至分词器，形成完整图像。模型通过负对数似然损失进行训练，确保生成的图像质量。
在实验部分，ARINAR-B模型在ImageNet数据集上进行了测试，使用了213M参数并训练了400个周期。模型的性能通过FID、InceptionScore（IS）、精确度和召回率等指标进行评估。在没有分类引导的情况下，ARINAR-B的FID为9.17，而在使用分类引导后，FID提升至2.75，这表明其生成质量显著提高。此外，ARINAR-B模型的生成速度也得到了验证，平均每张图像的生成时间为11.57秒，显著快于其他对比模型。这些结果表明，ARINAR在生成高质量图像方面具备良好的效率和效果。
ARINAR模型的工作原理可以简单理解为分两步进行图像生成。首先，模型将输入的图像转换成多个小块，这些小块包含了图像的不同特征。接着，模型的外层会根据已经生成的小块，预测下一个小块的特征。这就像在拼图时，先根据已拼好的部分，猜测接下来应该放哪一块。内层则会根据外层给出的信息，逐步生成每个小块的具体内容。通过这种方法，ARINAR能够更快速、更准确地生成高质量的图像，避免了传统方法中逐个处理每个小块的低效问题。
2025-03-04｜USTC,ETHZurich｜⭐️
http ://arxiv.org/abs/2503.02508v1https ://github.com/xinding-sys/Quant-Cache
在高效图像生成领域，量化与缓存机制通常被单独应用于扩散变换器（DiTs），各自展现了显著的加速潜力。然而，将这两种机制结合使用以提升生成效率的效果尚未得到充分探索。本文通过实证研究发现，量化与缓存机制的结合并不简单，且在生成过程中面临两个关键挑战：一是后训练量化（PTQ）中标定数据集的样本有效性被缓存操作显著削弱；二是这两种机制的结合导致了采样分布中的曝光偏差加剧，从而在图像生成过程中引发了更严重的误差积累。为了解决这些问题，本文提出了一种混合加速方法，通过引入时间感知并行聚类（TAP）和分布方差补偿（VC），旨在提升DiTs的效率，同时保持其优良的生成能力。经过广泛实验验证，所提方法在保持竞争生成功能的同时，实现了12.7倍的加速效果。
本文提出的混合加速方法主要包含两个核心技术：时间感知并行聚类（TAP）和分布方差补偿（VC）。
时间感知并行聚类（TAP）：该方法通过动态选择来自大规模数据集中的信息量大且独特的样本，优化PTQ中的标定数据集。TAP利用并行子采样技术，降低计算复杂度，并通过构建空间和时间相似性矩阵，增强样本选择的有效性。
分布方差补偿（VC）：该方法旨在减轻曝光偏差，通过在每个时间步中校正特征方差来实现。VC利用少量中间样本计算重构因子，从而在生成过程中自适应调整方差，显著提高生成模型的整体性能。
通过这两种方法的结合，本文成功解决了量化与缓存结合使用时的主要挑战，从而提升了生成质量和效率。
在实验设置中，本文采用ImageNet数据集，使用预训练的DiT-XL/2模型进行评估，图像分辨率设置为256×256和512×512。实验中应用了250个采样步骤的DDPM求解器，并进行了100和50个采样步骤的额外评估，以测试所提方法的鲁棒性。实验结果显示，所提方法在多个评估指标上均表现优异，尤其是在FID、sFID、InceptionScore和Precision方面，均显著优于传统方法。具体而言，所提方法在8位量化下，生成质量与原始模型相近，同时实现了12.7倍的计算效率提升。此外，消融实验表明，TAP和VC的引入对生成质量的提升起到了重要作用，验证了本文提出的混合加速方法的有效性。
在这项研究中，研究人员提出了一种新方法来提高图像生成的速度和质量。他们使用了两种主要技术。首先是“时间感知并行聚类（TAP）”，这就像从一大堆照片中快速挑选出最有用的几张，而不是随便选择。这种方法通过并行处理来加快选择过程，并确保选择的照片能代表整体的样子。其次是“分布方差补偿（VC）”，这个方法可以帮助模型在生成图像时减少错误。它通过调整生成过程中每一步的样本特征，来减轻模型在学习时产生的偏差。通过这两种技术的结合，研究人员不仅提高了生成图像的速度，还保持了图像的质量，使得图像生成变得更高效。
2025-02-25｜UIUC,AWSAILabs｜⭐️
http ://arxiv.org/abs/2503.01872v1
随着文本到图像扩散模型的广泛应用，生成偏见问题日益受到关注。尤其是在生成特定职业或性别的图像时，模型往往会表现出对某些人群的偏好，导致生成结果的不公正性。为了解决这一问题，本文提出了FairGen，一种通过自适应潜在引导机制来控制生成过程中敏感属性的模型。FairGen旨在在保证生成质量的同时，减少生成偏见。它通过引入一个动态调整的潜在引导模块和一个用于跟踪生成统计数据的记忆模块，来实现对目标属性的精确控制。此外，本文还提出了一种全面的偏见评估基准HBE，以更好地评估扩散模型中的偏见表现。实验结果表明，FairGen在偏见减少和生成质量方面均优于现有方法，展示了其在多属性场景中的灵活性和有效性。
FairGen的核心由两个主要模块组成：潜在引导模块和指示引导模块。潜在引导模块负责计算有效的潜在引导方向，以便在生成过程中引导特定属性的分布。具体步骤如下：
潜在引导方向计算：通过一个训练好的引导网络，依据输入提示和当前潜在状态，生成针对特定属性的引导方向。
指示引导模块：该模块生成标量引导方向，指示当前生成的目标属性值（如性别），以确保生成符合用户设定的公平分布。
记忆模块：用于存储生成统计数据，确保生成过程中的属性分布与目标公平分布相一致。记忆模块通过比较新输入的特征表示与存储的集群，决定如何调整生成策略，以实现属性的平衡。
在实验部分，FairGen被评估于HBE和StableBias两个数据集，以验证其在偏见减少和生成质量方面的效果。通过与多种基线方法（如提示干预、微调和潜在干预方法）进行比较，FairGen在多个敏感属性（如性别、种族、年龄）的生成中consistentlyachievedthelowestbiasscores,indicatingitssuperiorcapabilitytomitigatemulti-attributebias.实验结果显示，FairGen在保持生成质量的同时，显著降低了生成的偏见。例如，在性别属性的控制中，FairGen在不同的文本到图像扩散模型上都表现出了较低的偏见得分，证明了其在复杂场景下的有效性和稳定性。此外，FairGen还展现了在不同目标生成分布下的灵活性，能够适应用户的具体需求。
FairGen的工作原理可以简单理解为一个智能助手，它通过两个主要部分来帮助生成公平的图像。首先，潜在引导模块就像一个导航系统，帮助模型在生成图像时朝着特定方向前进，比如生成更多女性程序员的图像。其次，指示引导模块则负责确定具体的目标，比如“我想要看到一位女性程序员”。最后，记忆模块就像一个记忆库，记录之前生成的图像数据，以便在新图像生成时参考，确保生成的图像符合公平的标准。通过这种方式，FairGen不仅可以生成高质量的图像，还能确保不同性别、种族等属性的生成是平衡的，避免了以往模型中常见的偏见问题。
