313-A2｜文生视频质量评估，视频帧插值与预测，基于KAN的音频隐式表示

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月5日｜周三
2025-03-04｜THUSIGS,CUHK,HKUST｜⭐️🟡
http ://arxiv.org/abs/2503.02341v1
在文本到视频生成（T2V）的领域，尽管技术取得了显著进展，但有效评估生成视频的质量仍然是一个重大挑战。现有的自动化评估指标缺乏对视频内容的深层语义理解与推理能力，导致其评估结果难以解释且不够准确。为了解决这一问题，研究团队构建了GRADEO-Instruct，这是一个多维度的T2V评估指令调优数据集，包含来自十个以上现有视频生成模型的3300个视频和经过16000条人工注释转换而来的多步骤推理评估。基于此数据集，研究者们提出了GRADEO模型，该模型专门设计用于视频评估，通过多步骤推理为AI生成的视频提供可解释的评分和评估。实验结果表明，GRADEO在与人类评估的一致性方面优于现有方法，并揭示了当前视频生成模型在生成符合人类推理和复杂现实场景内容方面的不足。
GRADEO的评估方法包括多个关键步骤。首先，定义了七个评估维度，包括质量、美学、一致性、对齐、合理性、安全性和创造力。每个维度都包含具体的评估标准，以确保全面的评估。其次，构建了GRADEO-Instruct数据集，通过收集来自人类评估者的评分和理据，形成视频、推理和评分的三元组。接着，采用多步骤推理流程，模型在评估过程中依次进行概述、描述、分析和评估。具体而言，概述阶段规划推理过程，描述阶段详细说明视频内容，分析阶段进行深度推理并结合人类常识，最后在评估阶段生成评分。通过这种方式，GRADEO不仅提供分数，还能给出评估的理据，从而提高评估的可解释性和准确性。
实验部分主要通过与现有模型的比较来验证GRADEO的有效性。研究者们使用了多种数据集和评估指标，包括Spearman和Pearson相关系数，以及平均绝对误差（MAE），以评估模型的预测与人类评估之间的相关性。通过在GRADEO-Instruct数据集上进行实验，提取340个样本作为测试集，比较GRADEO与其他自动化评估方法的表现。结果表明，GRADEO在多个维度上与人类评估的一致性显著高于其他模型。此外，研究还对多个最新的T2V生成模型进行了基准测试，以揭示它们在各评估维度上的表现，并为视频生成领域的发展提供了新的见解和方向。
GRADEO的评估方法可以简单理解为一个四步走的过程。首先，模型会对要评估的视频进行一个总体的概述，帮助理解视频的主题和内容。接着，它会详细描述视频中的各个元素，比如画面中的颜色、构图等，这样可以更好地分析视频的质量。然后，模型会结合常识进行深度分析，考虑视频是否符合现实逻辑，比如角色的行为是否合理。最后，模型会给出一个评分，并解释这个评分的理由。通过这种多步骤的推理，GRADEO不仅仅是给出一个数字评分，而是让我们知道这个评分是如何得出的，这样就能更好地理解和信任评估结果。
2025-03-04｜SamsungElectronics｜ICASSP2025｜⭐️🟡
http ://arxiv.org/abs/2503.02316v1https ://github.com/srcn-ivl/uniVIP
视频帧插值和预测是计算机视觉领域的重要任务，旨在合成现有帧之间及其后续帧。尽管这两项任务密切相关，传统上却采用不同的模型架构进行研究，或者使用相同架构但分别训练其权重。近年来，任意时间插值的研究取得了显著进展，但对于任意时间预测的潜力却未受到足够重视。为此，本文提出了uniVIP（统一任意时间视频插值与预测模型），该模型通过扩展插值专用网络，实现对任意时间的插值和预测。uniVIP在共同的三帧输入上进行训练，能够在任意时间点（例如t=1.4或-0.6）合成帧，提供了在视频插值方面的竞争力结果，并在视频预测中超越了现有的最先进技术。
uniVIP模型的设计基于UPR-Net，这是一种前向扭曲的视频插值模型。其主要步骤包括：
光流估计：从输入帧中估计光流，以便进行前向扭曲，合成任意时间目标帧。
融合扭曲帧：将扭曲后的输入帧融合，以合成目标帧，考虑目标时间位置作为合成的先验信息。
任务通道添加：为了使合成模块能够区分插值和预测任务，uniVIP引入了一个特殊的任务通道。该通道的设计使得模型能够根据任务类型自适应地处理扭曲帧中的伪影模式，从而提高合成质量。通过这些步骤，uniVIP不仅支持任意时间的插值，还能进行任意时间的预测，展现了其在多任务学习中的优势。
在实验部分，uniVIP在多个数据集上进行了评估，包括Vimeo90K和SNU-FILM，测试其在视频插值和预测任务中的表现。实验结果表明，uniVIP在插值任务中与最先进的插值方法表现相当，而在预测任务中则显著优于现有模型。特别是在任意时间预测上，uniVIP能够有效处理浮动索引的帧，而传统方法只能处理整数索引的帧。此外，实验还通过消融实验验证了任务通道和单向预测在训练过程中的有效性。综合各项指标，uniVIP展示了出色的性能和广泛的应用潜力。
在uniVIP模型中，研究者们采用了一个简单而有效的方式来处理视频帧的插值和预测。首先，他们从输入的两帧视频中计算出运动信息（光流），这就像是找到视频中物体的移动路径。接下来，他们将这些运动信息应用于输入帧，生成一个新的帧，位于这两帧之间或者之后。为了让模型更聪明，他们还增加了一个“任务通道”，这个通道可以告诉模型当前是要进行插值还是预测，这样模型就能根据不同的任务类型处理视频中的伪影问题。通过这种方法，uniVIP不仅可以生成任意时间点的帧，还能在视频质量上取得很好的效果，适用于各种实际应用场景。
2025-03-04｜JagiellonianU,WUST｜⭐️
http ://arxiv.org/abs/2503.02585v1https ://github.com/gmum/fewsound.git
本研究提出了一种基于Kolmogorov-Arnold网络（KAN）的新颖架构，旨在改进音频信号的隐式神经表示（INR）技术。尽管KAN在低维数据处理上表现出色，但在高维音频数据的应用中仍面临挑战。研究表明，KAN在音频信号建模方面优于以往的INR方法，具体表现为在1.5秒音频样本中达到最低的对数谱距离（Log-SpectralDistance,LSD）为1.29，以及最高的感知语音质量评估（PESQ）分数为3.57。为了扩展KAN的应用范围，研究还引入了FewSound，一种基于超网络的架构，能够有效生成和更新KAN的参数。FewSound在多个性能指标上超越了现有的HyperSound模型，展现出更优的适应性和可扩展性。
本研究的核心方法包括两个主要部分：KAN模型和FewSound超网络架构。具体步骤如下：
KAN模型：使用可学习的激活函数，KAN通过对输入信号进行加权求和，形成音频信号的隐式表示。该模型采用样条函数作为激活函数，能够更好地捕捉音频信号中的细节。
FewSound架构：该架构结合了超网络和元学习的理念，能够从原始音频数据中提取信息，并生成针对特定任务的参数更新。FewSound的训练过程包括：
输入音频数据通过编码网络转换为低维表示。
通过超网络生成更新参数，用于调整目标网络的权重。
应用模型无关的元学习（MAML）算法，快速适应不同的音频任务。
实验部分通过对比不同INR架构的性能，评估KAN和FewSound的有效性。使用VCTK数据集进行训练，分别处理1.5秒和3秒的音频样本。实验结果表明，KAN在多个评价指标上均表现优异，包括均方误差（MSE）、对数谱距离（LSD）、信噪比（SI-SNR）和感知质量（PESQ）。具体来说，FewSound在MSE上比HyperSound减少了33.3%，在SI-SNR上提高了60.87%。此外，实验还分析了不同编码器架构、超网络参数和样条函数的影响，进一步验证了KAN的鲁棒性和适应性。
在这项研究中，研究人员开发了一种新方法，帮助计算机更好地理解和处理声音。首先，他们使用了一种叫做KAN的技术，这种技术可以让计算机通过学习声音的特征来生成更准确的声音表示。接着，研究人员引入了一个名为FewSound的系统，它利用超网络的力量来快速调整和优化声音模型。简单来说，FewSound可以从原始声音中提取信息，并根据不同的任务需求来更新模型。通过一系列实验，研究人员发现这种新方法在处理音频时比旧方法更有效，能够更好地捕捉声音的细节和质量，最终使计算机在识别和生成声音方面表现得更出色。
