313-A1.1｜无需训练的高分辨图像生成，基于多智能体的故事可视化，基于文本的图像编辑，真实手部生成，少样本字体生成

2025-03-04｜HKUST(GZ),KuaishouTechnology,ZJU,HKUST｜⭐️🟡
http ://arxiv.org/abs/2503.02537v1https ://zhenyangcs.github.io/RectifiedHR-Diffusion/
在高分辨率图像生成领域，Diffusion模型取得了显著的进展，但在生成超过训练分辨率的图像时，性能显著下降。为了解决这一问题，本文提出了一种名为RectifiedHR的方法，它是一种高效且简单的无训练高分辨率图像生成方案。该方法通过引入噪声刷新策略和能量整治策略，显著提高了生成效率和图像质量。通过对比多种基线方法，RectifiedHR在图像生成任务中展现出优越的效果，尤其是在处理高分辨率图像时，克服了传统方法的复杂性和低效率的问题。
RectifiedHR方法主要由两个模块组成：噪声刷新和能量整治。具体而言：
噪声刷新：在特定的采样步骤中增强“预测图像”的分辨率，并引入新噪声以与当前步骤对齐。通过逐步应用噪声刷新，该方法实现了粗到细的高分辨率图像生成，同时保持原始采样步骤以确保高效性能。
能量整治：通过分析潜在能量的变化，观察到在高分辨率生成过程中存在能量衰减现象。为此，增加分类器无关引导的超参数值，能够有效控制能量变化，从而达到整治效果。这一策略的实施使得生成的图像在细节表现上更为突出，显著减少了模糊现象。
在实验部分，研究者们对RectifiedHR进行了全面的性能评估，比较了其与多种现有方法的效果。实验中使用了多个不同分辨率的图像生成任务，结果表明，RectifiedHR在生成速度和图像质量上均优于其他方法。具体而言，使用噪声刷新和能量整治后，生成的图像在细节和结构上得到了明显改善。此外，实验还分析了不同超参数对生成效果的影响，展示了RectifiedHR在高分辨率图像生成中的广泛适用性和有效性。
RectifiedHR的方法可以简单理解为一种提升图像质量的技巧。首先，它使用“噪声刷新”，就像在画画时不断地调整和细化画面，确保每一步都能产生更清晰的图像。其次，它还引入了“能量整治”，这就像在制作食物时，调整配料的比例，使得最后的菜肴更加美味。通过这两个步骤，RectifiedHR能够在不重新训练模型的情况下，快速生成高分辨率的图像，并且图像的细节和清晰度都有所提升。这种方法简单易行，且效果显著，适合广泛应用于图像生成的各个领域。
2025-03-04｜SNU,NAVERWEBTOONAI｜ICASSP2025｜⭐️⭐️
http ://arxiv.org/abs/2503.02399v1
随着故事可视化技术的不断发展，将叙事内容转化为视觉表现的挑战日益突出，尤其是在保持叙事的精髓与细节方面。现有的研究多集中于生成独立场景，然而这些生成的图像往往无法准确捕捉叙事的深层含义与细微差别。为了解决这一问题，本文提出了VisAgent，一个训练自由的多智能体框架，旨在理解和可视化给定故事中的关键场景。VisAgent通过考虑故事的提炼、语义一致性和上下文连贯性，采用了智能化的工作流程，确保生成的图像不仅具备视觉吸引力，同时也忠实于原始叙事内容。
VisAgent框架由两个核心模块组成：故事模块和图像模块。故事模块负责从输入的故事文本中提炼出关键事件和角色属性，生成适合于图像生成模型的分层提示。具体流程如下：
叙事结构分析：通过分析故事的三幕结构，提取重要场景和角色信息。
角色描述提取：识别并提取角色的外观、动作等详细描述。
用户反馈整合：允许用户对生成的提示进行确认或修改，以减少生成过程中的偏差。图像模块则由多个智能体协同工作，生成符合叙事的故事图像。主要步骤包括：
场景元素生成：根据分层提示生成前景和背景元素的图像。
场景定位：确定前景元素在背景中的合理位置，确保场景的和谐性。
图像渲染：结合前景和背景图像，生成最终的故事场景图像，确保语义一致性。
为了评估VisAgent的有效性，我们进行了多项实验。首先，在故事模块方面，我们的模型成功提取了关键叙事元素，生成的分层提示在保持叙事流畅性方面表现优越。相较于基线方法，VisAgent在场景生成的连贯性和角色一致性上均有显著提升。其次，在图像模块的实验中，我们通过对比分析了生成的故事图像。实验结果显示，VisAgent在图像质量和叙事保真度上均优于现有的训练自由方法，尤其是在多场景生成时，能够有效整合各元素，提升整体视觉效果。最后，通过人类评估，我们的模型在叙事提炼和语义一致性方面获得了高分，验证了其在故事可视化中的应用潜力。
VisAgent的工作流程可以想象成一个团队合作的过程。首先，故事模块像一个编辑，仔细阅读输入的故事，然后挑选出最重要的情节和角色，制作出一个详细的“剧本”。这个剧本包含了每个场景需要的描述，比如角色的外貌和背景环境。接下来，图像模块就像一组艺术家，他们根据这个剧本分别绘制角色和背景，然后将这些元素组合在一起，确保它们在视觉上和谐统一。为了确保每个角色和场景都能忠实于故事，艺术家们会不断地检查和调整他们的作品，甚至可以得到观众的反馈，来进一步完善图像。通过这样的方式，VisAgent不仅能生成美丽的图像，还能确保这些图像能够传达出故事的核心情感和意义。
2025-03-04｜DeakinU｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.02187v1https ://github.com/nktoan/h-edit
本文提出了一种新的图像编辑方法h-Edit，该方法基于扩散模型，并通过Doob的h变换构建了理论框架。h-Edit将图像编辑视为反向时间桥接建模问题，旨在同时实现文本引导和奖励模型基础的编辑。该方法通过对预训练的扩散模型的向后过程进行修改，构建出一个在时间0时收敛到目标分布的桥接模型。h-Edit的创新之处在于其将更新过程分解为“重构”项和“编辑”项，使得编辑过程更加灵活，能够处理复杂的编辑任务。通过大量定量和定性实验，h-Edit在编辑效果和内容保真度方面超越了现有的最先进基线，展示了其在图像编辑领域的强大性能。
h-Edit的核心方法论是将图像编辑视为反向时间桥接建模问题。具体步骤包括：
构建桥接模型：通过修改预训练扩散模型的向后过程，利用Doob的h变换来创建一个收敛到目标属性的桥接模型。
更新分解：将编辑样本的更新过程分解为“重构”项和“编辑”项，分别表示编辑的保真度和效果。重构项通过现有的反演技术计算，而编辑项则可以根据不同的任务进行自定义。
MonteCarlo采样：使用LangevinMonteCarlo方法从桥接模型中进行采样，确保生成的图像不仅具有目标属性，还保持真实感。
灵活性设计：h-Edit允许通过不同的h函数结合多种编辑目标，支持文本引导和外部奖励模型的编辑任务。
在实验部分，本文对h-Edit进行了广泛的评估，涵盖了文本引导编辑、风格编辑和面部交换等多种任务。通过使用PIE-Bench数据集，评估了编辑效果和保真度的多项指标。实验结果表明，h-Edit在多个编辑任务中都显著优于现有的基线方法。在文本引导编辑中，h-Edit-D和h-Edit-R在编辑效果和保真度方面都取得了最佳结果。面部交换实验验证了h-Edit在奖励模型基础编辑中的能力，显示出其在身份转移和属性保留方面的优势。综合评估结果表明，h-Edit不仅在性能上达到最先进水平，还具备较强的灵活性和适应性。
h-Edit是一种新型的图像编辑工具，它的工作原理可以想象成一条桥，连接着原始图像和我们想要的编辑效果。首先，它会对原始图像进行分析，找出需要修改的部分。然后，h-Edit将这个修改过程分成两部分：一部分是确保图像的基本结构和内容不被破坏，另一部分则是进行实际的编辑，比如改变颜色或添加新元素。为了实现这些目标，h-Edit使用了一种叫做LangevinMonteCarlo的方法，像在进行高难度的跳跃，确保每次编辑都能达到预期效果。总之，h-Edit不仅能够灵活地处理不同类型的编辑需求，还能保持图像的真实感，让编辑后的图像看起来自然又美观。
2025-03-03｜QMUL,UCambridge｜⭐️🟡
http ://arxiv.org/abs/2503.02127v1
在手势生成领域，传统的扩散模型虽然在文本到图像的生成上表现出色，但在生成真实手势时仍面临诸多挑战，例如手指数量错误或姿势不自然等问题。为了解决这些问题，本文提出了HanDrawer模块，该模块旨在通过学习空间信息来提升生成手势的质量。HanDrawer通过利用图卷积层提取手势的内在空间结构和物理约束，结合手部区域与背景环境的对齐与融合，改进了单阶段扩散模型的去噪过程。该方法通过定义包含手势和环境信息的数据集，克服了现有数据集在手势生成任务中的不足，从而实现了更高质量的手势生成。
HanDrawer的核心方法包括以下几个步骤：首先，通过手势标签、手部深度图以及3DMANO网格顶点的输入，HanDrawer提取手势的内在空间结构和上下文信息。其次，利用图卷积层从MANO网格中提取手部的空间特征，并通过交叉注意力机制将这些特征与其他模态进行融合。接着，提出了位置保持零填充（PPZP）策略，以确保提取的手部特征在扩散模型的特定层中得到有效融合。最后，通过重建损失来引导模型关注手部区域，从而提高生成手势的准确性和真实感。这种方法的优越性在于它能够在单阶段中直接生成高质量的手势，而无需多阶段的复杂处理。
在实验部分，作者首先对广泛使用的HaGRID手势数据集进行了清洗和重新标注，以生成包含文本、深度图和3DMANO网格顶点的高质量多模态本地上下文数据集。通过与其他方法（如HandRefiner和RealisHuman）进行比较，HanDrawer在多项定量评估指标上表现出色，显示出其在手势生成质量上的显著提升。实验结果表明，HanDrawer不仅在生成手势的真实感和准确性方面超越了现有技术，还在处理复杂手势的空间结构方面展现了强大的能力。此外，定性分析也证实了该方法在生成高质量手势方面的有效性。
HanDrawer的工作原理可以简单理解为一个智能的手势生成助手。首先，它会从不同的输入信息中提取手势的特征，比如手势的类型、手部的深度信息以及手的三维模型。接着，它会将这些特征结合起来，利用一种叫做“交叉注意力”的方法，确保生成的手势既准确又自然。为了确保生成的手势与背景环境协调，HanDrawer还采用了一种特殊的“零填充”技术，这样手部特征就能在生成过程中得到重点关注。最后，HanDrawer通过一个重建过程来提升手势的质量，确保生成的手势看起来真实可信。总的来说，HanDrawer通过学习和整合多种信息，帮助计算机更好地理解和生成自然的手势。
2025-03-04｜BILIBILIInc.｜ICASSP2025｜⭐️🟡
http ://arxiv.org/abs/2503.02799v1https ://github.com/stephensun11/MXFontpp
本文提出了一种新颖的少量样本字体生成方法MX-Font++，旨在通过有限的参考字形创建新的字体库。尽管现有方法在视觉效果上表现良好，但在低资源语言的未见字符转换中仍面临显著挑战，尤其是当字体字形在训练集中差异较大时。MX-Font++采用了一种混合专家（MoE）方法，从局部组件的角度考虑字符内容，以适应性地提取组件，从而改善字符生成的质量。为了解决内容和风格特征的有效解耦问题，MX-Font++引入了异构聚合专家（HAE）和新的内容-风格同质损失函数，以增强特征提取能力。通过在多个数据集上的广泛实验，MX-Font++展示了在视觉效果和性能上超越现有最先进方法的能力，尤其是在复杂脚本和跨语言生成任务中。
MX-Font++的核心在于其混合异构聚合专家（HAE）架构，旨在提高特征提取能力。具体方法包括：
HAE结构：采用Transformer编码器结构，结合异构聚合注意力（HAA）模块，有效聚合图像的通道和空间信息，以便更好地分离内容和风格特征。
内容-风格同质损失：引入新的损失函数，通过增加内容和风格特征在潜在空间中的异质性，优化特征解耦过程。
多专家模型：通过多个专家对不同组件进行建模，增强模型对新语言系统的适应性，利用自注意力机制生成通道和空间的重要性映射，确保生成的字体在风格和内容上都具备良好的表现。
在实验部分，作者对MX-Font++进行了全面评估，主要针对中文和多语言字体生成。实验首先介绍了使用的数据集和评估指标，包括L1损失、RMSE、SSIM、LPIPS和FID等。对于中文生成，使用了包含400种字体和3346个字符的数据集，其中385种字体用于训练，剩余的15种字体用于测试。多语言生成则应用了中文模型进行跨语言生成，测试了低资源语言的识别准确率。实验结果表明，MX-Font++在所有评估指标上均优于现有方法，尤其是在生成复杂字符和低资源语言的字体时，显示出显著的性能提升。此外，用户研究进一步证实了MX-Font++在视觉质量和用户体验上的优势。
MX-Font++的工作原理可以简单理解为一个智能的字体设计助手。首先，它使用一种叫做“混合专家”的方法，这意味着它有多个小专家，每个专家专注于不同的字体样式和内容。这样，当我们给它一些字体样本时，它能更好地理解和模仿这些样式。其次，MX-Font++还引入了一种新方法来确保字体的内容和风格能够被清晰地区分开来，这就像在制作蛋糕时，确保巧克力和奶油不会混在一起。通过这些技术，MX-Font++不仅能够生成高质量的字体，而且在处理不同语言时表现也很出色，尤其是在那些字体资源稀缺的语言中。这使得它在现代数字排版和设计中非常有用。
