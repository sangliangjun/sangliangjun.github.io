317-A2.1.2｜全能视频创作与编辑，视频实例添加，关系视频定制化；视频风格迁移，视频风格变形

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月11日｜周二
2025-03-10｜AlibabaGroup｜⭐️⭐️
http ://arxiv.org/abs/2503.07598v1https ://ali-vilab.github.io/VACE-Page/
VACE（全能视频创作与编辑框架）是一种创新的模型，旨在整合多种视频生成和编辑任务。它通过统一的接口，结合了参考视频生成、视频编辑和掩膜视频编辑等功能。VACE的核心在于其强大的DiffusionTransformer架构，使其能够灵活应对视频合成中的时空一致性挑战。该模型不仅支持单一任务，还允许用户通过组合不同的任务来实现更复杂的创作场景。通过广泛的实验，VACE在多种子任务上表现出与专用模型相当的性能，展现了其在视频内容创作领域的巨大潜力。
VACE的设计基于多模态输入，整合文本、图像、视频和掩膜等多种数据类型，以满足不同视频生成和编辑任务的需求。模型的关键组件包括：
多模态输入：将视频任务分为四类，涵盖文本、图像、视频和掩膜输入。
视频条件单元（VCU）：统一处理不同输入条件，支持文本提示和视频帧序列的组合，确保空间和时间的一致性。
概念解耦策略：允许模型区分需要保留和修改的内容，提升生成效果的灵活性。
上下文适配器：通过注入不同任务的概念，增强模型的适应能力，使其能够处理多种视频合成任务。
此外，VACE的训练方法包括全量微调和上下文适配器调优，以提高模型的训练效率和效果。
在实验部分，VACE的性能通过与现有的专用模型进行比较来验证。我们构建了一个包含480个样本的数据集，以评估VACE在多种任务上的表现。实验结果表明，VACE在视频质量、时间一致性等方面的表现优于许多现有模型。评估方法包括自动评分和用户研究，分别测量视频的美学质量、背景一致性、动态程度等指标。通过这些综合评估，VACE展示了其在视频生成和编辑领域的竞争力，能够满足用户在创作过程中的多样化需求。
VACE模型的工作方式可以简单理解为一个强大的视频编辑工具，它能够处理多种输入形式，比如文本、图片和视频。首先，用户可以通过输入简单的文本指令或参考图像来生成新的视频。接着，VACE会将这些信息整合在一起，生成出符合用户需求的视频。为了确保生成的视频在时间和空间上保持一致，VACE使用了一种叫做“视频条件单元”的技术来处理不同类型的数据。此外，VACE还具备灵活的编辑功能，用户可以轻松修改视频中的特定部分，比如添加新元素或改变颜色。总的来说，VACE就像一个多功能的创作平台，让用户能够轻松地进行视频创作和编辑。
2025-03-08｜SJTU,TencentWeChat,USTC,SIAT,ShanghaiAILab,ZJU｜⭐️🟡
http ://arxiv.org/abs/2503.06268v1https ://zhuangshaobin.github.io/GetInVideo-project/
随着视频编辑需求的不断增长，用户希望能够将特定的现实实例融入现有视频中。然而，现有的编辑方法主要依赖文本提示或预定义操作，无法有效捕捉目标实例的独特视觉特征。为了解决这一问题，研究者提出了“Get-In-VideoEditing”这一新概念，允许用户通过提供参考图像来准确指定希望融入视频的视觉元素。该研究的核心贡献包括创建了GetIn-1M数据集、提出了GetInVideo这一新型端到端框架，并建立了GetInBench基准测试，旨在评估这一新编辑范式的性能。该方法的实施不仅提高了视频编辑的可及性，还显著提升了个性化视频编辑的能力。
本研究提出的GetInVideo框架采用了一种基于扩散变换器的架构，旨在解决视频对象插入中的三个关键挑战：保持时间一致性、保留视觉身份以及确保自然场景交互。具体方法包括以下几个步骤：
数据集构建：通过自动化的“识别-跟踪-删除”流程生成GetIn-1M数据集，包含一百万个视频编辑对，提供了丰富的文本注释和参考图像。
模型输入：GetInVideo同时处理五种输入，包括描述实例的文本提示、提供视觉外观的参考图像、指示对象位置的时间掩码（可选）、去除实例的条件视频以及原始目标视频。
模型架构：使用3D全注意力机制，确保所有输入和条件信息通过注意力机制有效交互，提高了模型的性能和适应性。
为了评估GetInVideo的有效性，研究者建立了GetInBench基准，包含100个测试用例，涵盖单对象和多对象的参考场景。通过与现有方法的比较，GetInVideo在视觉质量和操作效率上表现出色。实验结果显示，该方法在多个关键指标上均优于其他主流视频编辑方法，如FID、FVD和CLIP-I等。通过综合定性和定量评估，研究表明GetInVideo能够高效、准确地将参考对象融入到视频中，同时保持时间一致性和内容的真实感。这一成果为个性化视频编辑提供了新的可能性。
在这个研究中，科学家们提出了一种新方法，叫做GetInVideo，帮助用户轻松地将特定的图像或物体添加到视频中。这个方法的关键在于用户可以提供一张参考图片，系统会根据这张图片自动将相应的对象插入到视频里。具体来说，系统会同时接收几种信息，包括描述这个对象的文字、参考图片、视频中对象的位置（如果有的话），以及去掉对象后的原始视频。然后，系统会利用一种特殊的算法来处理这些信息，确保添加的对象在视频中看起来自然且一致。这种方法大大简化了视频编辑的过程，让普通用户也能轻松进行个性化的编辑。
2025-03-10｜FDU,AlibabaGroup,AntGroup,NTU,ZJU｜⭐️🟡
http ://arxiv.org/abs/2503.07602v1https ://dreamrelation.github.io
在视频定制领域，关系视频定制是一项关键任务，旨在生成描绘用户指定关系的个性化视频。现有方法虽然能够实现主体外观和单一对象动作的定制，但在复杂关系的建模和跨领域的泛化能力上仍显不足。为了解决这个问题，研究者们提出了DreamRelation，这是一种新颖的关系视频定制方法。该方法通过少量示例视频个性化关系，利用两个关键组件：关系解耦学习和关系动态增强。通过将关系信息与主体外观分离，DreamRelation能够更好地捕捉复杂的空间安排和时间动态，从而生成符合用户需求的高质量视频。
DreamRelation的核心在于两个主要部分：关系解耦学习和关系动态增强。首先，在关系解耦学习中，研究者引入了关系LoRA三元组和混合掩码训练策略，以有效分离关系信息与主体外观。具体而言，关系LoRA用于捕获关系信息，而主体LoRA则聚焦于主体的外观。其次，在关系动态增强过程中，提出了一种新颖的时空关系对比损失，旨在强调关系动态的重要性，减少对细节外观的依赖。通过这种方式，模型能够在训练过程中更好地捕捉到主体之间的互动和动态变化，最终生成符合文本提示的定制视频。
在实验部分，研究者们在NTURGB+D动作识别数据集上进行了广泛的评估，选取了26种人类关系进行测试。为了验证DreamRelation的有效性，研究者设计了多种基线方法进行比较，包括直接LoRA微调和改编的关系图像定制方法。实验结果显示，DreamRelation在关系准确性、文本对齐和视频质量等多项指标上均优于现有方法。此外，用户研究表明，DreamRelation生成的视频在关系对齐和整体质量方面获得了用户的高度认可，进一步证明了该方法的实用性和创新性。
DreamRelation的工作原理可以简单理解为两个主要步骤。首先，研究者们通过一种叫做”关系解耦学习”的方法，将视频中的关系信息和主体外观分开处理。就像在拍电影时，导演会分别关注演员的表演和场景的布置一样。这样做的好处是，模型可以更清晰地理解视频中发生的关系，比如“拥抱”或“握手”。其次，在“关系动态增强”阶段，研究者们引入了一种新的损失函数，专注于捕捉主体之间的互动，而不是过多关注外观细节。这就像在训练运动员时，教练会强调团队配合，而不是单个运动员的外观。通过这两个步骤，DreamRelation能够生成更自然、更符合用户要求的视频。
2025-03-10｜NJU｜⭐️🟡
http ://arxiv.org/abs/2503.07363v1
视频风格迁移技术旨在改变视频的风格，同时保持其内容完整性。传统方法常常面临内容泄漏和风格不一致的问题，特别是在使用图像驱动的方法时。为了解决这些问题，本文提出了一种新颖的方法——轨迹重置注意力控制（TRAC），旨在实现高质量的风格迁移并保持内容的完整性。TRAC通过重置去噪轨迹和强化注意力控制，提升了内容一致性，并显著降低了计算成本。此外，本文还引入了“风格媒介”的概念，以桥接内容和风格之间的差距，从而实现更精准、和谐的风格元素迁移。实验结果表明，所提出的框架能够处理多种风格化输出，从精确的内容保留到视觉上引人注目的结果。
本文的核心方法包括两个主要部分：风格媒介的生成和轨迹重置注意力控制（TRAC）。首先，风格媒介（I_sm）是通过选择合适的风格参考图像（I_s）与内容视频（V）结合生成的，旨在有效防止内容泄漏。其次，TRAC通过重置每个时间步的去噪潜在向量（x_t），确保生成的视频在内容和布局上保持一致。具体步骤包括：1）利用多模态大型语言模型（MLLM）生成视频描述，指导风格媒介的生成；2）在去噪过程中，TRAC替换查询矩阵（Q），以保留内容信息；3）引入辅助路径以传递内容信息，并在每个时间步重置潜在向量，以确保其与理想轨迹对齐。这种方法不仅提高了风格迁移的质量，还显著降低了计算复杂性。
在实验部分，本文使用稳定扩散（StableDiffusion）作为基础模型，结合MLLM进行风格迁移的评估。通过对比不同方法的性能，采用了多种指标（如LPIPS、FID和ArtFID）来量化风格和内容的保留效果。实验结果显示，所提出的方法在内容一致性和风格匹配上优于现有技术，同时减少了内容泄漏的现象。此外，还进行了用户研究，评估参与者对生成视频的风格迁移质量的偏好。结果表明，本文的方法在用户满意度和整体偏好上均表现出色，进一步验证了其在实际应用中的有效性和灵活性。
在我们的研究中，我们提出了一种新方法，让视频的风格转变更加自然，同时保持视频内容不变。首先，我们创建了一种叫做“风格媒介”的新概念，它把想要的风格和视频内容结合在一起，确保风格应用得当，而不会混入不必要的内容。接下来，我们引入了“轨迹重置注意力控制”技术，这是一种在视频处理过程中不断调整和优化内容的方法。具体来说，我们会在每一个处理步骤中对潜在信息进行重置，确保它与理想的结果保持一致。这种方法不仅能让视频在视觉上更加吸引人，还能减少处理所需的计算资源。通过这些创新，我们的技术在实际应用中表现得更加灵活和高效，能够满足用户对视频风格转换的需求。
2025-03-10｜ZJU,UESTC｜⭐️🟡
http ://arxiv.org/abs/2503.06998v1
SOYO是一种创新的视频风格变换框架，旨在实现无调优的平滑风格过渡。传统的视频风格转移方法往往面临结构一致性不足、风格转换突兀和内容保留困难等问题。SOYO通过结合自注意力机制和自适应实例归一化（AdaIN），在扩散模型中实现了多风格的平滑过渡。该方法有效地保持了视频帧的结构一致性，同时在多个风格之间实现了流畅的视觉过渡。SOYO的核心在于引入了一种新的自适应采样调度器，以应对不同风格之间的平衡，确保视频风格变换的自然性和连贯性。实验结果表明，SOYO在开放领域的视频风格变换任务中超越了现有技术，能够更好地保留视频帧的结构连贯性，并实现稳定的风格过渡。
SOYO的方法主要包括以下几个部分：
DDIM逆变换：对源视频和风格图像进行DDIM逆变换，以提取其潜在表示和自注意力特征。
风格特征插值：通过线性插值方法，将两个风格图像的统计特征（均值和标准差）进行动态混合，调制视频内容。
自注意力机制：在去噪过程中，注入源视频的查询特征，以保持结构信息，并通过插值的关键和数值特征实现风格的平滑过渡。
自适应风格距离映射：为了解决风格转换不平衡的问题，采用自适应风格距离映射（ASDM），根据风格距离动态调整插值权重，以实现视觉上的平衡和流畅过渡。这些方法共同作用，确保了视频帧在风格变换过程中的结构一致性和视觉连贯性。
在实验中，SOYO通过构建SOYO-Test基准来评估其在视频风格变换任务中的表现。该基准包含25个视频序列，涵盖多种现实场景，并与40种不同艺术风格配对。实验通过生成500个风格变换序列来全面评估SOYO的时间一致性、结构保留和风格插值的平滑性。结果显示，SOYO在多个指标上优于现有方法，特别是在结构完整性和风格适应性方面。此外，通过消融实验，验证了各个组件（如互注意力和双风格AdaIN）的有效性及其对整体性能的贡献。通过定量和定性分析，SOYO展现出在复杂风格转换场景中的优越表现，进一步推动了视频风格变换技术的发展。
SOYO的方法可以简单理解为一个三步走的过程。首先，我们从原始视频和风格图像中提取出它们的特征，就像为每个视频和风格建立一个“档案”。接着，我们通过一种“混合”技巧，把两个风格的特征按比例结合在一起，这样可以在视频中逐渐引入新的风格，而不是一下子就改变。最后，在视频生成的每一步，我们会确保保留原始视频的结构，比如人物和背景的形状，同时逐步加入新风格的颜色和纹理。这种方法让视频在风格转换时，看起来更自然、更流畅，避免了突兀的变化，就像在看一幅美丽的画作，颜色和形状都和谐统一。
