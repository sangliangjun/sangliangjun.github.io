2025-03-07｜Hedra,PKU,HKUST(GZ),HKU｜⭐️🟡
http ://arxiv.org/abs/2503.05978v1https ://magicinfinite.github.io/
MagicInfinite是一种创新的扩散变换器（DiT）框架，旨在克服传统肖像动画的局限性，实现高保真度的动画生成，适用于多种角色类型，包括真实人类、全身人物和风格化的动漫角色。该框架能够支持多样的面部姿势，包括背面视图，并能在多角色场景中通过输入掩码精确指定发言者。MagicInfinite的核心在于其三项创新：首先，采用3D全注意力机制与滑动窗口去噪策略，确保在不同角色风格中生成具有时间一致性和视觉质量的无限视频；其次，实施双阶段课程学习方案，结合音频进行口型同步、文本提供情感动态，以及参考图像保持身份一致，增强对长序列的灵活多模态控制；最后，通过区域特定掩码和自适应损失函数，平衡全局文本控制与局部音频指导，支持特定发言者的动画生成。该模型在新基准上表现出色，展示了在音频口型同步、身份保持和运动自然性方面的优势。
MagicInfinite的核心方法是将静态肖像图像与驱动音频和文本提示结合，通过一个预训练的文本到视频（T2V）生成模型生成动画视频。具体步骤包括：首先，利用3D全注意力机制处理输入的静态图像、音频和文本提示，确保视频内容与文本之间的强关联；其次，引入跨注意力层，以便将音频特征注入到去噪网络中，从而实现精确的口型同步；最后，采用两阶段课程学习策略，首先训练模型仅使用肖像图像和文本提示，随后加入音频，通过面部区域掩码和自适应损失函数强化音频控制，确保模型在生成过程中能够更好地学习音频与口型动作之间的关系。此外，模型还利用DMD2算法优化推理速度，减少计算资源消耗，实现高效的长视频生成。
在实验部分，研究团队开发了一个名为MagicInfinite-Benchmark的新基准，包含25个多样化的驱动音频片段和20个文本提示，涵盖不同的说话场景，如唱歌、演讲和说唱。基准还包括多种风格的肖像图像，涵盖不同的面部朝向和大小。MagicInfinite在这一基准上进行了全面评估，展示了其在肖像图像和动作场景中的强大表现。通过与传统的肖像动画方法进行比较，MagicInfinite在视频质量、动作准确性和口型同步方面均表现优异。用户研究显示，参与者认为MagicInfinite生成的视频在真实感和生动性上优于其他最先进的方法，进一步验证了该模型的有效性和应用潜力。
MagicInfinite的工作原理可以简单理解为一种智能动画生成工具。首先，它通过分析一张静态肖像图像、音频和文本信息，来创造出一个动态视频。具体来说，模型会先学习如何将肖像图像与文本内容相结合，确保生成的视频在视觉上与输入的肖像匹配。接着，音频信息被用来控制视频中的口型同步，确保嘴巴的动作与说话的声音相符合。为了让模型更好地学习这些信息，研究团队采用了分阶段的训练方法，先让模型熟悉肖像和文本，再加入音频，帮助它更精准地生成动画。此外，MagicInfinite还利用先进的技术加快了视频生成的速度，使得制作长视频变得更高效。这种方法使得生成的动画不仅质量高，而且能在不同场景中表现出自然的动作和表情。
2025-03-10｜HKUST,Dartmouth｜⭐️⭐️
http ://arxiv.org/abs/2503.07217v1
ReelWave是一个旨在专业电影声音生成的多代理框架，旨在通过丰富的上下文信息来提升音频生成的质量。该框架主要聚焦于从短小的情节摘要生成与视觉内容相匹配的音效、对话和音乐。尽管近年来在音频生成领域取得了一定进展，但现有技术仍未能满足电影和电视行业对音频设计的高要求。ReelWave通过捕捉与屏幕同步的音效，结合多场景的音频生成，旨在为影视制作提供更加细腻和丰富的声音设计工具。
ReelWave的核心是一个多代理框架，主要包括以下几个部分：
场景检测与理解：使用V2A-SceneDetector将视频划分为多个场景，并借助视频理解模型提取关键事件描述。
音频生成代理：包括剧本编写者、Foley艺术家、作曲家、配音演员和混音师，各自负责不同的音频生成任务。
剧本编写者：将视频摘要转化为结构化剧本，包含场景描述和对话。
Foley艺术家：生成与屏幕上动作同步的音效，并规划环境音。
作曲家：根据剧本生成音乐设计计划。
配音演员：为角色生成对话音轨。
混音师：负责将不同音轨混合，以确保音量和时间的协调。
声音导演：监督整个生成过程，提供反馈和指导，确保各个音频组件的协调一致。
在实验部分，ReelWave使用AVSync15数据集进行训练和评估，重点考察其在音频质量和时间同步方面的表现。通过多种评估指标，包括Frechet距离和均值KLD等，比较ReelWave与其他基线模型的性能。结果表明，ReelWave在语义对齐和音频质量方面均表现优异，尤其是在音频生成的时间结构预测上，显示出其在多场景处理中的优势。此外，尽管训练数据集相对较小，ReelWave依然能够产生专业水准的音频效果，展示了其潜在的应用价值。
ReelWave是一个为电影制作设计的智能音频生成系统。它的工作原理就像一个电影制作团队，每个角色都负责不同的任务。首先，系统会分析视频，识别出不同的场景和关键事件。接下来，剧本编写者会根据视频内容编写出详细的剧本，包括角色的对话和场景描述。然后，Foley艺术家会为视频中的动作添加音效，比如爆炸声或脚步声。作曲家则会根据场景的情感创作背景音乐，配音演员会为角色录制对话。最后，混音师会将所有音轨结合在一起，确保声音的平衡和协调。整个过程由声音导演监督，实时调整和优化生成的音频，确保最终效果符合电影的情感和节奏。
2025-03-10｜PolyU,AlibabaTongyiLab｜⭐️🟡
http ://arxiv.org/abs/2503.06984v1https ://wjc2830.github.io/MelQCD/
本研究聚焦于视频到音频生成（V2A），旨在为静默视频合成高质量音轨。通过提取视频中的关键信号，本文提出了一种新的方法，称为Mel量化-连续分解（Mel-QCD），以平衡梅尔频谱图的完整性与复杂性。Mel-QCD将梅尔频谱分解为三种不同类型的信号：语义向量、能量向量和标准偏差向量。通过量化和连续处理这些信号，研究团队开发了视频到所有（V2X）预测器，以从视频中有效地预测这些信号。随后，生成的信号被重新组合，并与文本反演设计结合，控制音频生成过程。该方法在多个评估指标上表现出色，显示出其在音频质量、同步性和语义一致性方面的优势。
本研究的核心方法是Mel量化-连续分解（Mel-QCD），其步骤如下：
梅尔信号分解：将梅尔频谱图分解为三种信号：能量信号（E）、语义信号（S）和标准偏差信号（D）。能量信号代表音频的整体响度，语义信号则反映音频的语义内容。
语义向量量化：对语义信号进行量化，构建语义向量量化（SVQ）代码本，以减少复杂性并保留关键信息。
信号预测器设计：开发视频到所有（V2X）预测器，通过视觉编码器从视频中提取特征并生成梅尔频谱图。
生成控制：利用ControlNet和文本反演技术，将生成的信号与文本提示结合，从而实现更精确的音频生成控制。
实验部分主要通过在VGGSound数据集上进行评估，比较Mel-QCD方法与现有的主流V2A生成模型的性能。研究团队采用了多个评估指标，包括生成质量（FID、MKL）、同步性（OnsetACC）和语义一致性（IB-AA、IB-AV）。通过将生成的音频与真实音频进行对比，分析了不同控制信号对生成音频的完整性和复杂性的影响。此外，进行了消融实验，以验证Mel-QCD和文本反演模块的有效性。结果显示，Mel-QCD在大多数指标上均表现优越，尤其在音频的质量和语义一致性方面，证明了其有效性。
在这项研究中，科学家们提出了一种新方法，用于从视频中生成与之匹配的音频。他们的核心思想是将音频信号分解为更简单的部分，以便更好地理解和生成。首先，他们把音频信号分成三部分：一种表示声音的强度，另一种表示声音的语义内容，最后一种则反映声音的变化程度。接着，研究者们对语义部分进行了简化处理，让计算变得更容易。然后，他们设计了一个系统，可以从视频中提取这些信息，并生成相应的音频。最后，他们把生成的音频与视频结合，确保二者在内容和时间上都能很好地同步。这种方法不仅提高了音频的质量，还能确保生成的音频与视频内容相吻合。
2025-03-09｜HMS,XJTU,NJU,WFUSM｜⭐️🟡
http ://arxiv.org/abs/2503.06588v1
本研究提出了一种新颖的知识增强条件变分自编码器（KE-CVAE），用于从动态磁共振成像（MRI）序列生成语音音频。随着动态MRI技术的发展，能够实时捕捉发音器官的运动变形变得愈发重要。然而，MRI采集过程中常常会出现数据丢失、噪声污染及音频文件损坏等问题，这使得从图像生成高质量音频变得极具挑战性。现有的生成方法在音频真实感和可推广性上存在局限。我们的方法通过整合未标记的MRI数据以增强知识，结合变分推理架构，显著提升了生成模型的能力。实验结果表明，KE-CVAE在生成自然语音波形方面表现优异，且优于传统的深度学习合成方法。
KE-CVAE的核心在于其两个主要创新：知识增强和变分推理。首先，我们采用自监督学习策略，通过教师-学生网络结构，从大规模的MRI数据中学习潜在变量。具体步骤包括：
知识增强：利用教师网络和学生网络，分别提取MRI图像中的特征，通过一致性损失、重建损失和KoLeo正则化损失来优化特征学习。
变分推理框架：模型包括后验编码器、先验编码器和解码器。后验编码器从输入音频中提取潜在表示，先验编码器则根据MRI序列生成潜在变量的条件分布。解码器利用潜在变量重建音频波形，结合对抗训练以提升生成质量。
我们在一个开放的动态MRI语音数据集上评估了KE-CVAE的性能。实验包括以下几个方面：
数据集构建：收集并整理了来自13个公共MRI数据集的头颈部图像，形成了一个大规模的声道MRI数据集（LaSVoM），涵盖多种发音情境和说话者。
评估指标：使用2D皮尔逊相关系数（Corr2D）、感知语音质量评估（PESQ）和主观均值意见分数（MOS）等指标来评估生成的音频质量。
结果分析：实验结果显示，KE-CVAE在多个评估指标上均优于传统的CNN和变换器模型，表明其在语音合成中的有效性和潜力。
在这项研究中，我们开发了一种新方法，帮助从MRI图像中生成语音。首先，我们利用大量的MRI图像数据进行训练，学习如何提取与发音相关的特征。我们构建了两个网络，一个负责提取特征（教师网络），另一个则学习如何生成音频（学生网络）。接着，我们通过一种叫做变分推理的技术，将这些特征转化为实际的音频波形。这样做的好处在于可以更好地捕捉语音的细微变化，使生成的音频更自然。此外，我们还加入了对抗训练的策略，进一步提升了音频的质量。通过这些步骤，我们的模型能够有效地将MRI图像转换为清晰的语音，具有很高的实用价值。
2025-03-09｜THU,THU-SIGS｜⭐️
http ://arxiv.org/abs/2503.06397v1
近年来，基于扩散模型的口型同步生成技术取得了显著进展，能够生成高度同步的说话面孔视频，用于视觉配音。然而，这些模型在生成图像时往往难以保持细致的面部特征，尤其是在处理未见过的自然视频时，出现了“口型平均”现象。这种现象使得生成的嘴唇细节模糊，无法准确反映说话者的独特面貌。为了解决这一问题，本文提出了一种名为UnAvgLip的新方法，该方法通过提取参考视频中的身份嵌入，生成高保真的面部序列，同时保持准确的口型同步。实验结果表明，UnAvgLip在保持身份一致性和视觉质量方面显著优于传统方法，展现了其在视觉配音领域的潜力。
UnAvgLip的核心由两个主要组件构成：身份感知模块（IdentityPerceiver）和身份交叉注意模块（ID-CrossAttn）。身份感知模块使用基于Transformer的架构，将身份嵌入映射到与音频特征相同维度的特征序列。这个过程包括以下步骤：
身份嵌入投影：将高维身份嵌入通过线性投影转换为与音频特征相同维度的身份查询令牌。
迭代注意机制：交替应用交叉注意模块和Transformer解码层，以动态提取和整合与身份相关的信息。
身份交叉注意：在原UNet模型中添加新的交叉注意层，将身份查询令牌嵌入到音频-视觉交叉注意机制中，从而增强模型对身份细节的控制。
通过这种方式，UnAvgLip能够在高效性和准确性之间取得平衡，确保生成的面孔保留说话者的独特特征，同时实现精准的口型同步。
在实验部分，研究团队在两个公开数据集上评估了UnAvgLip的性能：LipReadingintheWild（LRW）和HDTF数据集。通过对比多种先进的口型同步生成方法，UnAvgLip在视觉质量和身份一致性指标上均表现出色。实验中，使用了重建损失、感知损失和口型同步损失等多种损失函数，以确保生成图像的质量和同步效果。此外，还进行了消融实验，以验证各模块的有效性。例如，去除ID-CrossAttn模块导致身份一致性显著下降，而不使用同步损失则使得生成的嘴唇动作无法与音频同步。最终，UnAvgLip在多个指标上超越了现有的最先进方法，证明了其在口型同步生成中的优越性。
UnAvgLip的工作原理可以简单理解为一个高科技的“面部配音助手”。首先，它通过分析一段视频中的面孔，提取出与说话者相关的独特特征，比如嘴唇的形状、牙齿的轮廓等。然后，这些特征会被转化为一种“身份标签”，就像给每个人贴上个性化的标签。接下来，UnAvgLip会把这些标签与音频信号结合起来，确保生成的视频中的嘴唇动作和声音是同步的。通过这种方法，即使在处理新的视频时，UnAvgLip也能保持每个人的独特面貌，避免出现模糊或泛化的效果。最终，UnAvgLip不仅能生成更真实的说话面孔，还能确保这些面孔在视觉上与说话者的身份保持一致。
