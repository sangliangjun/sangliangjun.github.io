317-A2.1.1｜基于多智能体CoT的自动电影生成，故事视频叙事，情节发展与相机运动，视频生成的相机运动模拟

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月11日｜周二
2025-03-10｜NUS｜⭐️⭐️
http ://arxiv.org/abs/2503.07314v1https ://github.com/showlab/MovieAgent
本研究提出了一种名为MovieAgent的自动电影生成框架，旨在通过多代理链式思维（CoT）规划来简化长视频的制作过程。当前大多数视频生成框架仍依赖于手动输入故事情节、场景和角色互动，导致成本高昂且效率低下。MovieAgent通过自动化的方式生成多场景、多镜头的长视频，确保叙事的一致性和角色的持续性，同时提供同步字幕和稳定的音频。该框架的两个主要优势在于：首先，它定义了自动化电影生成的新范式，可以在短时间内生成高质量的长视频；其次，MovieAgent采用层次化的CoT推理过程，显著减少了人工干预的需求。实验结果表明，MovieAgent在剧本忠实度、角色一致性和叙事连贯性等方面达到了新的性能水平，为完全自动化的电影生成提供了新的见解。
MovieAgent的核心方法是通过多代理系统来模拟电影制作中的关键角色，包括导演、场景规划师和镜头规划师。具体步骤如下：1)导演代理负责高层次的叙事结构，分析剧本并将其分解为多个子剧本，确保叙事的逻辑进展和角色的一致性。2)场景规划代理将这些子剧本细化为具体的场景组成，定义每个场景的情感基调、视觉风格和摄影笔记。3)镜头规划代理则专注于细化每个场景的镜头细节，包括镜头类型、运动和光照效果。4)最后，MovieAgent利用定制化的视频和音频生成技术，生成最终的视频和音频内容。通过这种结构化的推理过程，MovieAgent能够有效地将抽象的叙事和电影需求转化为详细的可执行计划，确保生成的内容具备连贯性和一致性。
在实验部分，我们采用了多种评估指标来验证MovieAgent的性能，包括自动化指标和人工投票。实验设置中使用了两台A6000GPU进行所有实验。我们与现有的方法（如DreamFactory、StoryAgent和StoryDiffusion）进行了比较，发现这些方法在自动化电影生成任务上存在不足，特别是在多角色一致性和剧本规划方面。MovieAgent通过分解任务为多个组件，结合了基于大语言模型的剧本处理、图像生成和视频生成技术，提升了生成质量和一致性。实验结果表明，MovieAgent在剧本到关键帧/视频生成的各项指标上均超越了现有模型，尤其在角色一致性、背景一致性和动作流畅性等方面表现卓越，展示了其在自动化电影生成领域的潜力。
MovieAgent是一种新型的自动电影制作工具，它通过模拟电影制作团队中的不同角色（如导演、场景设计师和镜头设计师），来帮助快速生成完整的长视频。首先，导演角色会分析剧本，把故事分成几个小部分，确保每个部分的内容连贯且角色形象一致。接下来，场景设计师会把这些小部分转化为具体的场景，决定每个场景的情感和视觉效果。然后，镜头设计师会设定每个场景的拍摄细节，比如镜头的类型和运动方式。最后，MovieAgent会将这些信息转化为视频和音频，生成最终的电影。这个过程不仅快速而且高效，减少了人力成本，同时保证了故事的完整性和角色的一致性。
2025-03-08｜UMD,DolbyLab｜⭐️🟡
http ://arxiv.org/abs/2503.06310v1
在视频创作领域，从文本生成连贯的长视频序列是一项重要且具有挑战性的任务。尽管基于扩散模型的技术在短视频合成上取得了显著进展，但它们在生成长视频时常常面临时间一致性、语义保持和动作连续性等问题。本文提出了一种新颖的叙事方法，通过引入双向时间加权潜在混合策略，确保生成的视频在不同片段之间保持时间一致性。我们的方法扩展了用于图像生成的布莱克-斯科尔斯算法，以实现对视频生成的控制，并通过结构化的文本条件来增强动作的连续性。实验结果表明，该方法在生成具有连贯叙事的长视频方面显著优于现有技术。
本研究提出的方法主要包括三个关键技术，以解决长视频生成中的主要挑战。首先，提示平滑（PromptSmoothing）技术通过扩展布莱克-斯科尔斯算法，确保文本提示之间的平滑过渡，从而提高视频片段的生成质量。其次，时间加权混合（Temporally-AwareBlending）利用潜在空间混合策略，在生成每个视频片段时保持一致性。该策略通过动态调整混合权重，确保在时间上保持一致的运动和视觉效果。最后，语义动作表示（SemanticActionRepresentation）机制通过对高层次动作语义的编码，进一步增强动作的连续性，确保视频片段中的动作与文本描述保持一致。这三种技术的结合，使得生成的长视频在语义和视觉上都能保持一致性和连贯性。
在实验部分，我们采用Mochi-1模型作为基础进行所有实验，设置视频分辨率为848×480，使用64步去噪以确保生成质量。我们构建了一个包含404个提示的多样化数据集，涵盖多个场景和角色，以评估我们方法的有效性。通过与现有的Mochi和Vlogger等基线方法进行比较，我们使用多个指标评估生成视频的质量，包括文本对齐、视觉真实感和故事一致性。实验结果表明，我们的方法在CLIP和BLIP评分上显著优于基线，且在LPIPS值上表现出更好的故事一致性，证明了其在长视频生成中的有效性和优越性。
在我们的方法中，首先，我们使用一种叫做“提示平滑”的技术来确保不同视频片段之间的过渡自然顺畅。这就像在讲故事时，前后情节要连贯，不会让人觉得突兀。接着，我们引入了一种“时间加权混合”的方法，它可以根据前面的画面来调整新画面的生成，确保动作和场景保持一致。最后，我们使用“语义动作表示”，这意味着我们会把每个动作的意义编码进来，以确保视频中的角色按照文本描述的方式行动。这样，我们生成的长视频不仅看起来连贯，而且故事情节也能很好地传达。通过这些技术，我们能够制作出更自然、更具吸引力的视频。
2025-03-08｜IEIT,NKU,THU｜⭐️⭐️
http ://arxiv.org/abs/2503.06053v1https ://dropletx.github.io
本研究关注视频生成中的时空一致性问题，提出了“整体时空一致性”的概念，强调情节发展与摄像机技术之间的相互作用。以往的研究往往只关注时间或空间一致性，缺乏对摄像机运动对情节的影响的深入探讨。为了解决这一问题，研究团队构建了一个包含1000万段视频的DropletVideo-10M数据集，视频中包含动态摄像机运动和物体动作，平均每段视频配有206字的详细注释。这些注释不仅描述了摄像机运动，还涵盖了情节发展。随后，团队开发了DropletVideo模型，旨在生成视频时保持时空一致性。该模型在生成过程中能够有效地整合摄像机运动与情节进展，从而实现更复杂的多情节叙事。
本研究的方法分为几个关键步骤：
数据集构建：创建DropletVideo-10M数据集，包含1000万段视频，采用动态摄像机运动和物体动作。每段视频都配有详细的注释，确保捕捉到时空动态。
视频处理：视频从多种来源收集后，进行分段和筛选，确保每段视频都包含摄像机运动。使用光流估计技术来检测摄像机运动，并提取出符合条件的片段。
视频注释：采用视频到文本模型生成高质量的注释，确保涵盖摄像机运动、场景变化和情节发展等细节。通过微调多模态模型，生成详细且一致的描述。
模型训练：基于DropletVideo-10M数据集，训练DropletVideo模型，采用潜在扩散模型（LDM）架构，以保持时空一致性。模型能够在生成过程中灵活控制摄像机运动和情节变化。
在实验阶段，研究团队对DropletVideo模型进行了全面评估，以验证其在时空一致性生成方面的有效性。实验使用DropletVideo-10M数据集，进行多个实验以测试模型在不同情境下的表现。结果表明，该模型能够有效保持视频生成中的空间一致性和时间连贯性，尤其是在复杂情节和摄像机运动的情况下。实验还比较了DropletVideo与现有模型的性能，显示出其在生成质量和一致性方面的显著优势。为进一步验证模型的实用性，研究团队还进行了用户评估，收集反馈以优化生成效果，确保模型在实际应用中的可行性和可靠性。
在本研究中，我们的目标是让生成的视频看起来更自然和连贯。为此，我们提出了一个新方法，结合了摄像机运动和故事情节的变化。首先，我们创建了一个非常大的数据集，里面有1000万段视频，每段视频都详细描述了摄像机的移动和情节的发展。接着，我们设计了一种智能程序，能够自动从这些视频中找到合适的片段，并生成详细的文字描述，帮助模型理解视频内容。最后，我们训练了一个新的视频生成模型，确保它在生成新视频时能够同时考虑到摄像机的运动和故事情节的变化。这样，我们希望生成出更加生动、有趣的视频，能够吸引观众的注意力。
2025-03-09｜XMU｜⭐️🟡
http ://arxiv.org/abs/2503.06508v1https ://github.com/QuanjianSong/LightMotion
本文提出了一种名为LightMotion的新方法，旨在解决现有视频生成技术中与相机运动相关的计算瓶颈。传统方法通常需要耗时的微调和复杂的推理过程，而LightMotion通过在潜在空间中直接操作，消除了这些额外的需求。该方法的创新之处在于其能够模拟多种相机运动（如平移、缩放和旋转），而无需额外的深度估计或图像修复。具体来说，LightMotion的核心贡献包括：通过潜在空间的排列操作有效模拟相机运动；结合背景感知的重采样和跨帧对齐策略，精确填补新视角并保持帧之间的一致性；提出的潜在空间校正机制能够减轻由于排列和重采样引起的信噪比（SNR）变化，从而提升视频生成的质量。通过大量实验，LightMotion在定量和定性评估中均优于现有方法。
LightMotion的整体流程包括几个关键步骤。首先，系统从高斯噪声开始进行去噪处理，直到达到预设的时间步。接下来，通过潜在空间的排列和重采样更新潜在变量，以模拟所需的相机运动。具体方法如下：
潜在空间排列：通过调整不同帧之间像素的相对顺序，模拟平移、缩放和旋转等相机运动。例如，在平移中，像素将沿着平移方向移动，而在缩放中，则通过插值调整潜在变量。
潜在空间重采样：在更新后的潜在空间中，由于某些位置缺乏对应值，采用背景感知的重采样策略，确保新视角中的像素来自原始潜在空间的背景区域，避免重复生成。
潜在空间校正：在去噪过程中，重新引入噪声以纠正潜在空间中的SNR变化，从而提升生成质量。通过这些步骤，LightMotion实现了有效的相机运动模拟。
在实验部分，LightMotion的性能通过多个指标进行评估，包括生成质量、相机可控性和用户研究。首先，使用Fréchet视频距离（FVD）、CLIP文本-图像相似度（CLIP-T）和CLIP帧间相似度（CLIP-F）等指标评估生成视频的质量。结果显示，LightMotion在所有指标上均优于其他基线方法，显示出其在视觉质量和语义一致性方面的显著优势。此外，通过用户研究收集的反馈表明，参与者对LightMotion生成的视频质量和可控性给予了较高评价。实验还包括对比LightMotion与其他几种相机运动控制方法的定量比较，结果进一步验证了LightMotion在相机运动模拟中的出色表现。
LightMotion的工作原理可以简单理解为一个视频生成的“魔法师”。首先，它从一堆随机噪声开始，就像从一张白纸出发。然后，它通过调整图像中像素的位置来模拟相机的运动，比如向左或向右移动、放大或缩小。这就像在拍摄视频时，摄像机在不同的位置拍摄同一个场景。接下来，LightMotion会从原始图像中提取背景部分，确保在新的视角中不会重复出现同样的物体。最后，它会进行一些小调整，确保生成的视频看起来更加自然和流畅。通过这种方式，LightMotion能够快速、有效地生成高质量的视频，而不需要繁琐的设置或调整。
