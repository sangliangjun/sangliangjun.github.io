2025-03-08｜BUAA,SenseTime,PBVR｜⭐️🟡
http ://arxiv.org/abs/2503.06136v1
本研究提出了一种名为GSV3D的方法，旨在通过单幅图像生成高质量的三维对象。该方法结合了二维扩散模型的多样性和三维结构的一致性，解决了当前技术在生成三维模型时面临的挑战。传统的三维扩散模型受限于数据集的稀缺性，而二维扩散模型在几何一致性方面存在不足。GSV3D通过引入高斯散射解码器，增强了三维模型的几何一致性，并利用稳定视频扩散技术生成360度的多视角图像。实验结果显示，该方法在生成多视角一致的图像和准确的三维模型方面表现优异，具有广泛的应用潜力。
GSV3D的核心方法包括以下几个步骤：
多视角扩散模型：采用SV3D模型生成360度的多视角图像，利用单幅图像作为条件输入。
双重编码策略：通过预训练的VAE编码器和CLIP编码器，分别提取几何信息和语义特征，确保生成过程中的低层次和高层次信息均被有效利用。
高斯散射解码器：将多视角潜在表示转化为显式的三维结构，确保生成的图像在几何上保持一致。
几何蒸馏过程：在训练阶段，通过引入三维约束对多视角潜在表示进行蒸馏，增强模型对空间关系和多视角一致性的理解。
本研究在多个数据集上进行了广泛的实验，以验证GSV3D的有效性。训练过程中使用了来自Objaverse的数据集，生成了高质量的RGB图像和深度图。实验结果表明，GSV3D在外观质量和几何质量评估中均优于现有的先进方法。通过对比分析，GSV3D在生成多视角一致的三维模型方面表现出更高的保真度和多样性。此外，用户研究表明，GSV3D在外观和几何质量方面获得了最高评分，证明了其在实际应用中的潜力。
GSV3D是一种新方法，可以从一张图片生成三维物体。它的工作原理是先生成多个不同角度的图片，这些图片围绕着物体旋转。为了确保这些图片在形状和外观上是一致的，GSV3D使用了一种叫做高斯散射解码器的技术。这种技术可以把生成的图片转化成一个清晰的三维模型。同时，GSV3D还利用了两种不同的编码器来提取图片的细节和整体意思，确保生成的三维模型既真实又丰富。通过这些步骤，GSV3D能够生成高质量的三维物体，广泛应用于游戏、虚拟现实等领域。
2025-03-10｜CUHKMMLab,CPIIunderInnoHK｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.06900v1
本文提出了一种名为DirectTriGS的新框架，旨在利用高斯点云（GaussianSplatting,GS）进行3D物体生成。尽管GS在3D内容渲染中日益受到关注，但目前关于直接生成3D高斯的研究仍然有限，主要挑战在于GS复杂的数据结构及其多通道离散点云表示。为了解决这一问题，作者提出了一种三平面（triplane）表示方法，该方法将GS内容编码为图像般的连续场。此表示不仅有效地编码了几何和纹理信息，还能够通过TriRenderer将其平滑转换回高斯点云，并渲染成图像。通过使用仅基于2D的监督信号，作者展示了该框架在文本到3D任务中的高效性和优越性，能够生成高质量的3D物体几何和渲染结果。
DirectTriGS的生成方法主要分为四个步骤：
三平面表示：首先，将3D对象编码为三平面表示。这种表示方式允许将高斯点云转换为连续的场，便于后续处理。
训练三平面变分自编码器（VAE）：接着，训练一个VAE以压缩三平面数据到潜在空间。这一过程有助于捕捉高层次特征，为后续模块提供有效的信息。
基于潜在扩散模型的生成：然后，利用潜在扩散模型生成三平面潜在编码，通过解码器将其转换回相应的高斯点云。
可选的SDS后处理：最后，使用基于得分蒸馏（SDS）的后处理步骤进一步提升生成3D对象的纹理和细节，使得最终结果更加精致。
这些步骤的设计确保了生成过程的高效性和灵活性，能够在较短时间内完成高质量的3D物体生成。
实验部分采用OmniObject3D和Objaverse数据集进行评估，重点考察DirectTriGS在文本到3D任务中的表现。首先，作者对200K个对象进行采样，训练对应的三平面和TriRenderer。随后，训练VAE和扩散模型以实现3DGS生成。实验结果表明，DirectTriGS在几何和渲染质量上均优于当前主流方法，如Shap-E和Direct3D。用户研究显示，参与者对生成对象的真实感和细节水平给予了较高评分。此外，使用SDS进行纹理提升的实验也证明了该方法在细节和一致性上的优势。整体而言，实验结果表明DirectTriGS在直接生成3DGS方面具有显著的竞争力。
DirectTriGS的生成过程可以简单理解为四个主要步骤。首先，我们把3D物体的形状和颜色信息转换成一种叫做三平面的格式，这种格式就像是把物体的各个面展开成平面图。接着，我们使用一种叫做变分自编码器的技术，将这些平面数据压缩成更小、更易处理的形式。然后，我们利用潜在扩散模型来生成新的物体信息，最后再把这些信息转换回3D形状。如果生成的物体看起来不够完美，我们还可以用一种叫做得分蒸馏的技巧来进一步改善它的外观。通过这些步骤，DirectTriGS能够快速而有效地生成高质量的3D物体。
2025-03-09｜LeidenU,BMWGroup,TUMunich｜⭐️🟡
http ://arxiv.org/abs/2503.06485v1
在3D形状生成领域，生成高质量的3D形状一直是深度生成建模社区面临的主要挑战之一。传统的方法通常依赖于训练好的自编码器来获取源形状的连续隐式表示，但这种方法计算开销较大。本文提出了一种新颖的框架——谱域扩散建模（SpoDify），通过奇异值分解（SVD）进行形状编码，将复杂网格高效地编码为连续隐式表示。该方法能够将一个15K顶点的网格压缩为512维的潜在代码，并在样本有限或GPU资源紧张的情况下表现出显著优势。SpoDify在生成的3D形状质量上与当前最先进的技术相媲美，甚至在某些情况下更为出色，展示了其在生成高维形状方面的潜力。
SpoDify的核心方法是利用奇异值分解（SVD）对网格进行谱表示。具体步骤如下：
聚类：通过扩散映射对可用网格进行聚类，确保从每个聚类中选择一个代表性网格，以增强训练集的多样性。
隐式表示：使用带符号距离函数（SDF）来表示网格几何形状，SDF具有良好的可微性和光滑性。
数据分解：对SDF进行离散小波变换（DWT），提取特征后应用SVD，将其压缩为低维谱特征，并保留基础信息以便后续重建。
扩散模型：采用去噪扩散概率模型（DDPM），训练模型从噪声输入中预测谱特征，以生成新的形状特征。
生成新形状：通过生成的谱特征与预存的基进行组合，重建出新的低频系数，最终通过逆小波变换生成新的3D网格。
在实验部分，研究团队利用ShapeNet数据集对SpoDify进行了评估，特别关注飞机和椅子这两个类别。每个数据集选取1000个样本进行训练，设置潜在维度为512。训练过程中，使用单个NVIDIAA10GPU，批量大小为32，学习率为5×10^-4，持续100k步。研究还进行了消融实验，探讨了不同损失函数和限制函数对模型性能的影响。结果表明，SpoDify在多个评估指标上表现优越，尤其在重建精度和生成多样性方面。与其他最先进的方法相比，SpoDify在训练速度和内存使用上均展现出显著优势，能够在短时间内高效生成高质量的3D形状。
SpoDify的工作原理可以简单理解为一个高效的“形状生成工厂”。首先，它通过对已有的3D形状进行智能分类，找到一些典型的代表形状。接着，使用一种特殊的数学工具（称为SDF），将这些形状转化为一种更容易处理的形式。然后，SpoDify运用另一种技术（小波变换）来提取形状的关键特征，并通过奇异值分解将其压缩成更小的格式，方便存储和传输。最后，SpoDify使用一个智能模型来学习如何从噪声中生成新的形状。当它需要创造新的形状时，只需将这些特征与之前存储的基础信息结合，就能快速生成出新的3D形状。这种方法不仅节省了计算资源，还能在样本较少的情况下依然生成高质量的形状。
2025-03-09｜HFIPS,USTC,HFUT,SJTU,ByteDance｜⭐️
http ://arxiv.org/abs/2503.06677v1
在日常生活中，关节物体的三维表示至关重要。然而，现有的方法在实现高保真纹理表面重建和动态生成方面面临许多挑战。为了解决这些问题，本文提出了REArtGS框架，该框架通过引入几何和运动约束，利用3D高斯原语实现高质量的纹理表面重建和未见状态的生成。具体而言，REArtGS采用多视角RGB图像，首先通过无偏的有符号距离场（SDF）引导来规范化高斯不透明度场，以增强几何约束并改善表面重建质量。接着，构建了受关节物体运动结构约束的可变形场，实现未见状态的无监督生成。通过在合成和真实数据集上的广泛实验，REArtGS在关节物体的表面重建和生成任务中表现出色，显示出其优越性。
REArtGS的核心方法包括以下几个步骤：首先，采用无偏的SDF引导来增强3D高斯原语的几何约束。通过引入一个贝尔形函数，REArtGS能够更好地保留高斯原语的几何特征，从而提高重建质量。其次，框架建立了受关节物体运动结构约束的时间连续可变形场。通过学习旋转轴和角度的参数，REArtGS能够推导出在任何状态下的高斯原语变形位置。此外，对于平移关节，框架通过线性插值获取变形位置。最后，通过对动态部分的无监督分割，REArtGS能够有效地优化可变形场，从而实现高质量的动态表面生成。整体上，这些技术的结合使得REArtGS在处理关节物体时具备了更强的灵活性和准确性。
在实验部分，REArtGS在PartNet-Mobility和AKB-48数据集上进行了全面评估，验证了其在表面重建和生成任务中的性能。实验结果显示，REArtGS在多个指标上超越了现有的最先进方法。具体而言，在表面重建任务中，REArtGS在Chamfer距离、F1分数和地球移动者距离等指标上表现优异，展示了其在处理复杂几何形状和纹理时的能力。此外，REArtGS在动态生成任务中也展现了较好的效果，能够在未见状态下生成高质量的纹理网格。通过对比实验，REArtGS的表现不仅在合成数据上出色，在真实世界数据的应用中也显示出强大的泛化能力，证明了该方法的有效性。
REArtGS的方法可以简单理解为一个三步走的过程。第一步，利用一种叫做有符号距离场的技术，帮助我们更准确地表示物体的形状。这就像在画图时，先用铅笔勾勒出物体的轮廓，再用颜色填充。第二步，REArtGS通过分析物体的运动方式，计算出物体在不同状态下的形状变化。这类似于我们观察一个玩具车在不同位置时的样子，记录下它的每个姿势。第三步，REArtGS会将动态部分和静态部分分开，确保在生成新形状时，能够保持物体的真实感。这样一来，REArtGS不仅能重建物体的外观，还能让它在不同状态间流畅地变化，展现出更真实的效果。

本研究聚焦于可控的三维户外场景生成，提出了一种基于场景图的方法。三维场景生成在计算机视觉领域具有重要意义，广泛应用于自动驾驶、游戏以及元宇宙等领域。现有方法往往缺乏用户控制，或者依赖于模糊且不直观的条件描述。为解决这一问题，本文引入场景图作为一种更直观的控制格式，开发了一种交互式系统，使用户能够通过场景图生成和编辑密集的三维场景。通过将稀疏的场景图转换为密集的鸟瞩图嵌入图（BEV），并利用条件扩散模型生成与场景图描述相匹配的三维场景，实验结果表明该方法在生成高质量三维城市场景方面表现出色。
本方法的核心在于利用图神经网络（GNN）和扩散模型的结合，具体步骤如下：
场景图的构建：首先，定义场景图为一组节点和边，节点表示场景中的对象，边表示对象之间的关系。通过图神经网络提取节点的上下文信息，生成节点嵌入。
BEV嵌入图生成：使用一个新的分配模块将稀疏场景图转换为鸟瞩图嵌入图（BEM），该图为后续的三维场景生成提供条件输入。
条件扩散模型：通过训练的条件扩散模型，将BEM作为输入，生成最终的三维场景。该模型在生成过程中考虑了场景图的结构信息，确保生成场景的真实性和一致性。
交互式系统：开发用户友好的交互系统，允许用户通过手动编辑和文本输入方式灵活创建和修改场景图，从而实现对三维场景的精准控制。
为验证所提方法的有效性，研究团队创建了一套大规模的数据集，包含配对的三维场景和场景图。通过一系列定量和定性的实验，评估生成场景的质量和控制能力。实验结果显示，该方法在语义一致性和对象数量控制方面优于现有基线方法。使用均值交并比（mIoU）和平均准确率（MA）等指标评估场景质量，结果表明所生成的场景在语义上与输入的场景图高度一致。此外，通过用户研究评估生成场景的主观质量，结果显示用户对生成场景的满意度较高，进一步证明了该方法的实用性和有效性。
在这项研究中，我们提出了一种新颖的方法来创建三维户外场景，用户可以通过一种叫做“场景图”的工具来控制生成的内容。首先，用户可以定义场景图，这就像给场景中的每个对象（如建筑、车辆和行人）贴上标签，并描述它们之间的关系。接着，我们利用一个智能算法（图神经网络）来理解这些信息，并将其转化为一个鸟瞩图，这是一种从上方俯视场景的视图。然后，我们使用另一个模型（扩散模型）来根据这个鸟瞩图生成详细的三维场景。最后，我们还开发了一个简单易用的系统，让用户能够轻松地创建和调整场景图，确保生成的三维场景符合他们的想法和需求。这样，用户不仅可以看到生成的场景，还能对其进行精确控制。

本文提出了一种名为SRM-Hair的新方法，旨在通过单张图像重建3D头部网格，特别是独立的3D头发网格。传统的3D可变模型（3DMMs）在头发重建方面面临挑战，因为它们难以在不同发型之间保持一致的语义意义。SRM-Hair通过引入语义一致的光线建模技术，使得头发的3D形态可以通过系数进行控制和变形。研究团队收集了超过250个高保真头发网格扫描数据，并与3D面部数据配对，为头发重建提供了丰富的先验信息。实验结果表明，SRM-Hair在3D网格重建方面的性能超越了现有的最先进方法，展示了其在虚拟化身创建和高保真动画中的应用潜力。
SRM-Hair的核心方法包括三个主要步骤：数据准备、建模和重建。首先，数据准备阶段涉及收集和标注超过250个真实头发网格扫描，与3D面部数据配对。其次，在建模阶段，SRM-Hair利用语义一致的光线建模技术，生成一个有序的3D头发表面顶点序列。具体而言，模型通过定义固定的多光线方向模板并在头皮表面对称放置光线集，来提取头发的几何信息。最后，在重建阶段，SRM-Hair从输入图像中预测3D头发网格的系数，利用这些系数生成最终的头发网格。该方法不仅支持发型的叠加和适应性，还可以通过调整系数实现发型的翻转和厚度修改，确保生成的头发网格在形态上具有高度的灵活性和适应性。
为了验证SRM-Hair的有效性，研究团队进行了大量定量和定性实验。实验使用了来自FaceVerse和NPHM的3D数据集，以及超过10,000张来自网络的真实图像。通过与现有的最先进方法进行比较，SRM-Hair在头发区域和整体头部区域的重建精度上均表现出色，NormalizedRootMeanSquareError（NRMSE）和召回率分别达到了0.01637和0.78923。定性结果显示，SRM-Hair能够准确捕捉到头部的细节，头发重建与面部重建相互独立，增强了其在实际应用中的适用性。此外，消融实验进一步验证了各个组件对模型性能的影响，表明语义一致的光线建模和3D可变头发在独立头发网格重建中的重要性。
SRM-Hair的工作原理可以简单理解为三个步骤：首先，研究人员收集了大量真实的头发和面部数据，用于训练模型。接下来，模型通过一种叫做“语义一致的光线建模”的方法，像发射光线一样从头皮上提取头发的形状信息。这些光线帮助模型理解头发的结构，并生成一个有序的头发顶点序列。最后，模型从一张普通的照片中预测出头发的3D形状，就像在图像上绘制出头发的轮廓一样。这种方法使得生成的头发不仅可以适应不同的发型，还能根据需要调整发型的厚度或方向，极大地提高了头发重建的灵活性和真实感。
