2025-03-10｜ANU,USYD,Tencent,McGillU,JD,UTS(Sydney),MBZUAI,ZJU,Google｜⭐️⭐️
http ://arxiv.org/abs/2503.06955v1https ://steve-zeyu-zhang.github.io/MotionAnything
MotionAnything是一种创新的多模态人类运动生成框架，旨在通过结合文本、音乐等多种输入条件，生成高质量、可控的人类运动。尽管条件运动生成在计算机视觉中得到了广泛研究，但仍面临两个主要挑战：第一，现有的掩蔽自回归模型在动态帧和身体部位的优先级处理上存在不足；第二，不同条件模态的集成往往缺乏有效性，导致生成运动的控制和一致性不足。为了解决这些问题，MotionAnything引入了一种基于注意力的掩蔽建模方法，能够在空间和时间维度上细致控制关键帧和动作。此外，研究团队还推出了新的Text-Music-Dance(TMD)数据集，包含2153对文本、音乐和舞蹈的样本，填补了当前研究中的重要空白。
MotionAnything采用了一种任何到运动的生成方法，能够适应多种输入条件。具体来说，方法包括以下几个关键步骤：
条件编码：通过文本和音频编码器对不同模态的条件进行编码，以指导后续的掩蔽和运动生成过程。
基于注意力的掩蔽：该方法通过选择高关注度的部分，识别与给定条件相关的运动序列中的重要帧和动作，从而进行掩蔽。
**时间自适应变换器(TAT)**：该模块根据条件的模态动态调整注意力计算，确保运动的关键帧与文本关键词或音乐节拍对齐。
**空间对齐变换器(SAT)**：此模块将条件和运动嵌入重新排列，以便在空间维度上恢复关键动作，确保动作与条件的有效对齐。这些方法的结合使得MotionAnything在生成运动时更加灵活和精准，提升了生成的可控性和一致性。
在实验部分，研究团队针对多个标准基准数据集进行了广泛的测试，以验证MotionAnything的有效性。首先，使用新创建的TMD数据集进行文本、音乐和舞蹈的生成评估，结果显示该方法在运动质量和多样性上均优于现有的最先进技术。其次，在HumanML3D和AIST++等数据集上进行的比较实验表明，MotionAnything在多个指标上均表现出色，尤其在生成的运动与输入条件之间的对齐度上，显著提升了生成的连贯性和控制力。最后，通过消融实验验证了基于注意力的掩蔽策略在提升运动生成质量方面的有效性，进一步证明了该方法的优越性和创新性。
MotionAnything的工作原理可以简单理解为一个“智能舞者”。这个智能舞者可以根据你给它的指令（比如文字描述或音乐）来生成相应的舞蹈动作。首先，它会把这些指令转化为机器能理解的格式。接着，它会根据这些指令来决定哪些动作是最重要的，就像在舞会上选择最吸引人的舞步一样。然后，它会根据这些选择来生成完整的舞蹈动作，确保动作流畅且与指令相符合。通过这种方法，MotionAnything不仅能让舞蹈看起来更自然，而且还能根据不同的音乐或文字内容，灵活地调整舞蹈风格和动作，真是一个多才多艺的“舞者”！
2025-03-10｜UBirmingham,KETI,DankookU,KwangwoonU｜⭐️🟡
http ://arxiv.org/abs/2503.07390v1http ://boeun-kim.github.io/page-PersonaBooth
本文提出了一种新的任务——运动个性化（MotionPersonalization），旨在根据文本描述生成与特定个性相匹配的运动。这一创新任务的基础是一个新的大规模运动数据集PerMo（PersonaMotion），该数据集捕捉了多个演员的独特个性。为了解决在个性化运动生成中的主要挑战，研究者们提出了名为PersonaBooth的多模态微调方法。该方法解决了两个主要问题：首先，PerMo数据集与现有的缺乏个性特征的预训练数据集之间存在显著的分布差异；其次，如何从不同内容（动作类型）中捕捉一致的个性。本研究通过引入个性化标记和对比学习技术，增强了个性特征的捕捉能力，并提出了上下文感知融合机制，以优化多输入运动的整合效果。
PersonaBooth采用了一种多模态微调方法，旨在有效地调整预训练的运动扩散模型。其主要步骤包括：
个性提取器（PersonaExtractor）：该模块从输入的多个原子运动中提取视觉个性特征和个性标记。这些特征有助于后续的运动生成。
文本和视觉适配：引入个性化文本编码器，结合个性标记，生成适应性文本特征。通过对比学习损失，确保同一个性的样本在特征空间中更为接近。
训练过程：使用微调的运动扩散模型进行训练，结合个性化特征和文本描述，优化生成的运动质量。
上下文感知融合（CAF）：在多个输入运动的情况下，通过评估输入与文本描述的相关性，选择最合适的运动特征进行融合，避免不自然的运动混合。
实验部分评估了PersonaBooth在运动个性化和运动风格转移任务中的性能。首先，使用PerMo和100Style数据集进行比较，设定了单输入和多输入两种评估设置。通过对比FrechetInceptionDistance（FID）、R-Precision和多样性等指标，结果显示PersonaBooth在生成运动的质量、与文本的对齐程度以及多样性方面均优于现有方法。此外，进行消融研究，验证了个性提取、文本适配和CAF等关键组件对性能的贡献。实验结果表明，结合个性化特征和上下文感知的多模态微调方法显著提高了生成运动的自然性和个性化程度。
在这个研究中，科学家们开发了一种新的技术，叫做PersonaBooth，旨在帮助计算机根据文本描述生成个性化的运动。想象一下，如果你告诉计算机“一个人正在跳舞”，它不仅能理解这个动作，还能根据你的个性来生成独特的舞蹈动作。为了实现这一点，研究者们首先设计了一个个性提取器，能够从你提供的运动中提取出个性特征。接着，他们将这些特征与文本结合，通过一种叫做对比学习的方法，确保相似个性的动作在一起。最后，当计算机接收到多个运动时，它会通过上下文感知融合技术选择最相关的运动特征，避免生成看起来不自然的混合动作。这样，生成的运动不仅符合文本描述，还能展现出你独特的风格。
2025-03-10｜SZU｜⭐️
http ://arxiv.org/abs/2503.06897v1
文本到运动生成（Text-to-MotionGeneration）是一个快速发展的研究领域，位于多模态学习和计算机图形学的交汇处。其目标是根据自然语言描述生成相应的人类运动序列，广泛应用于游戏、动画、机器人和虚拟现实等领域。现有的方法通常依赖于简单的时空堆叠，导致特征冗余，并且从空间角度忽视了细微的关节级细节。为了解决这些问题，本文提出了一种新颖的HiSTFMamba框架。该框架由三个关键模块组成：双空间Mamba（Dual-SpatialMamba）、双时间Mamba（Bi-TemporalMamba）和动态时空融合模块（DynamicSpatiotemporalFusionModule,DSFM）。通过结合“基于部分+整体”的并行建模策略，HiSTFMamba能够有效捕捉全身协调与细粒度关节动态，同时通过双向扫描策略编码短期运动细节和长期依赖关系。实验结果表明，HiSTFMamba在多个评估指标上均实现了最先进的性能，尤其是在HumanML3D数据集上，FID分数从0.283降低至0.189，减少近30%。
HiSTFMamba框架的设计旨在生成由文本输入引导的人类运动。其主要组成部分包括双空间Mamba、双时间Mamba和动态时空融合模块。双空间Mamba采用“部分+整体”的并行学习策略，确保模型能够捕捉细粒度的关节细节，同时保持整体身体协调性。双时间Mamba则通过双向扫描机制，在时间维度上前后捕获运动序列的上下文信息，从而生成更加协调和连贯的运动序列。动态时空融合模块（DSFM）旨在有效整合从双时间Mamba学习到的时序特征和双空间Mamba输出的空间特征，去除冗余信息并提取互补特征，最终构建出更高效的时空特征表示。这种设计不仅显著提升了运动序列的真实感和平滑度，还确保了生成的运动在语义上与输入文本的高度一致性。通过这些模块的协同作用，HiSTFMamba在细粒度关节动态和全局运动模式的建模上取得了显著进展。
为评估HiSTFMamba的有效性，本文在两个广泛使用的文本到运动数据集上进行了实验：HumanML3D和KIT-ML。HumanML3D数据集包含3911个运动序列和6278个自然语言注释，采用FrechetInceptionDistance（FID）、R-Precision、MultiModalDistance（MMDist）、多样性（Diversity）和多模态性（Multimodality）等评估指标。实验结果显示，HiSTFMamba在HumanML3D数据集上的FID指标显著降低至0.189，相较于当前最先进的MotionMamba提升超过30%。此外，HiSTFMamba在R-Precision和MMDist等其他指标上也表现优异，验证了其在生成高保真且语义一致的运动序列方面的优势。通过消融实验，进一步分析了模型中各组件的重要性，结果表明，双空间Mamba和动态时空融合模块的有效融合是提升生成质量的关键因素。
在HiSTFMamba的工作原理中，首先，模型将输入的文本信息转化为运动序列。它的核心在于三个主要模块。第一个模块叫做双空间Mamba，它可以同时关注整个身体的运动和每个关节的细微动作，就像在画一幅完整的画时，同时注意每一笔的细节。第二个模块是双时间Mamba，它利用一种双向扫描的方法，既向前看也向后看，帮助模型理解动作的前因后果。最后，动态时空融合模块（DSFM）则像一个调和器，帮助整合这些信息，去掉多余的部分，同时保留重要的细节。通过这样的设计，HiSTFMamba能够生成更加自然和流畅的动作，确保这些动作与输入的文本描述高度一致。这种方法不仅提高了生成运动的质量，也使得训练过程更加高效。

本文提出了一种名为BioMoDiffuse的创新性生物力学导向运动生成框架，旨在解决现有方法在生成物理上可信的人类运动时的局限性。尽管近年来的自回归和扩散模型在视觉质量上有所提升，但它们往往忽视了关键的生物动力学特征，如肌肉激活模式和关节协调性，从而导致生成的运动不符合物理法则或缺乏可控性。BioMoDiffuse通过将欧拉-拉格朗日动力学嵌入到生成过程中，生成符合物理规律的运动轨迹，显著提高了运动的真实性。此外，本文还提出了一套综合评估协议，结合传统指标与新的生物力学标准，以便更全面地评估生成运动的质量。
BioMoDiffuse的核心方法包括几个创新点。首先，构建了一个轻量级的生物动力学网络，结合了肌电图（EMG）信号和运动学特征，并引入加速度约束，以确保生成的运动符合刚体动力学。其次，采用了物理引导的扩散过程，通过修改的欧拉-拉格朗日方程实现实时的生物力学验证，从而确保生成运动的物理合理性。最后，设计了一个解耦控制机制，允许独立调节运动的速度和语义内容。这一系列的创新保证了生成运动的平滑性和真实性，同时减少了常见的运动伪影，如脚滑、身体穿透和运动不连续性。
在实验部分，BioMoDiffuse在多个数据集上进行了评估，包括HumanML3D和KIT-ML，展示了其在生成质量和物理合理性方面的优势。通过与现有的运动生成方法进行对比，BioMoDiffuse在多个传统指标（如FID、R-precision等）和新提出的生物力学指标（如平滑度、脚滑率等）上均表现出色。此外，实验还通过用户研究验证了生成运动的质量、平滑性、物理合理性和文本对齐度，结果表明，BioMoDiffuse在这些方面的表现优于当前的最先进方法，并且在某些示例中，其生成结果与真实运动相媲美。
BioMoDiffuse是一种新型的运动生成系统，它通过结合生物力学原理，能够生成更真实的人类运动。简单来说，这个系统像是在给运动加上了一层“物理法则”的保护，确保生成的动作不仅看起来好，而且在物理上也合理。它的工作方式包括几个步骤：首先，它使用一种轻便的网络来分析肌肉如何运动，并确保这些运动符合自然规律；接着，它利用一个物理引导的过程来实时检查生成的动作是否符合这些规律；最后，它允许用户独立调节动作的速度和内容。这种方法解决了以前生成运动时常见的问题，比如动作不连贯或不自然，使得生成的运动更加平滑和真实。
2025-03-10｜THU,IDEA,JHU,UChicago,HKUST,HKU｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.07597v1https ://zhangyuhong01.github.io/HumanMM
在这项研究中，我们提出了HumanMM，一个新颖的框架，用于从多镜头视频中恢复人类运动，特别是在世界坐标系中。现有的方法通常侧重于单镜头视频，而忽视了多镜头场景中由于镜头切换、部分遮挡和动态背景带来的挑战。HumanMM通过集成增强的相机姿态估计与人类运动恢复（HMR），解决了这些问题。我们的框架利用定制的运动整合器，有效减轻了脚滑问题，并确保了运动的时间一致性。通过在公共3D人类运动数据集上进行广泛评估，我们证明了该方法在重建现实人类运动方面的鲁棒性，尤其是在复杂的多镜头视频场景中。
HumanMM的核心方法分为几个关键模块。首先，镜头切换检测器用于识别视频中的镜头切换帧。该模块结合了场景变化检测、边界框跟踪和人类关键点跟踪，以提高检测精度。其次，相机运动和人类姿态估计模块为每个镜头初始化相机姿态和人类参数，采用MaskedLEAP-VO算法来排除动态人类对相机估计的干扰。接下来，人类运动对齐模块确保不同镜头间的人类姿态和方向一致性，通过立体校准和训练的多镜头HMR编码器来实现。最后，后处理模块通过双向LSTM轨迹预测器和轨迹精炼器，平滑整个人类运动序列，减轻脚滑现象，增强整体运动一致性。
为了评估HumanMM的性能，我们构建了一个名为ms-Motion的多镜头3D人类运动数据集，并与现有的HMR方法进行比较。实验结果显示，HumanMM在多个基准测试中表现优异，尤其是在PA-MPJPE、WA-MPJPE和脚滑指标上，均优于其他方法。我们还进行了消融实验，以分析各个模块对整体性能的贡献。结果表明，采用多镜头HMR模块和方向对齐模块是提升运动恢复准确性的关键因素。此外，我们在不同的镜头数量下进行了评估，验证了我们方法在多种场景下的有效性和鲁棒性。
HumanMM的方法可以简单理解为一个多步骤的过程，首先是镜头切换检测，就像在看电影时识别不同镜头的变化。接着，我们通过一种智能算法来估计相机的位置和人类的姿势，确保在不同镜头之间的运动不会出现不连贯的情况。然后，对齐模块会帮助我们调整人类的姿势，使其在镜头切换时保持一致。最后，后处理模块会平滑整个运动，减少脚滑等问题。通过这些步骤，HumanMM能更准确地重建出人类在复杂视频中的运动，确保观众看到的动作流畅自然。
2025-03-09｜RUC,Kuaishou,THU,BUAA｜⭐️
http ://arxiv.org/abs/2503.06499v1
在数字人类动画领域，音频驱动的手势合成是一项关键技术，广泛应用于虚拟化身、人机交互和创意内容生成。尽管现有方法在生成逼真和同步手势方面取得了显著进展，但仍存在手势粗糙、缺乏表现力及未能完全与音频语义对齐等问题。为了解决这些挑战，本文提出了一种新颖的检索增强扩散框架ExGes。该框架由三个核心模块组成：运动基础构建模块、运动检索模块和精确控制模块，旨在生成更具表现力和语义对齐的手势。实验结果表明，ExGes在BEAT2数据集上显著提升了手势的表现力和多样性，相较于现有方法，Frechet手势距离降低了6.2%，运动多样性提高了5.3%。用户研究显示，71.3%的参与者更偏爱ExGes生成的自然性和语义相关性。
ExGes框架的设计围绕三个主要模块展开。首先，运动基础构建模块负责建立一个丰富的手势库，利用训练数据集捕捉多样化和细致的运动模式。其次，运动检索模块通过对比学习和动量蒸馏技术，从手势库中检索与输入音频语义相关的手势。这一模块确保生成的手势在语义上与音频内容相符。最后，精确控制模块通过部分掩蔽和随机掩蔽技术，允许对生成手势进行灵活、细致的控制，从而提高生成手势的表现力和准确性。通过这三个模块的结合，ExGes实现了对手势生成的全新视角，强调辅助指导和精确控制在克服表现力和语义对齐挑战中的重要性。
在实验部分，ExGes在BEAT2数据集上进行了广泛的评估，重点考察其在生成质量和控制精度方面的表现。通过使用Frechet手势距离、节拍一致性和多样性等指标，评估生成手势的准确性、音频对齐和可变性。此外，使用人类姿态估计指标，如每关节位置误差（MPJPE），来评估生成手势的保真度和精确度。实验结果显示，ExGes在各项指标上均优于现有强基线，尤其在表现力和多样性方面表现突出。通过消融研究，进一步验证了运动检索模块和精确控制模块对整体性能的重要性，结果表明这两个模块的缺失都会显著降低生成手势的质量。
ExGes的工作原理可以简单理解为三个步骤。首先，系统创建一个包含多种手势的数据库，这些手势是基于大量训练数据构建的，目的是为了捕捉不同的动作和表情。接下来，当用户提供一段音频时，系统会从这个数据库中检索出与音频内容相匹配的手势，这样生成的手势就能更好地表达音频所传达的意思。最后，系统允许用户对生成的手势进行精细调整，比如改变手势的方向或速度，以确保生成的动作既自然又符合语境。这种方法结合了丰富的手势库和灵活的控制机制，使得生成的手势更加生动和具有表现力。
