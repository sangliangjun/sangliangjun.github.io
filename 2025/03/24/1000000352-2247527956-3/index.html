主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月11日｜周二
2025-03-10｜Bytedance,UCAS｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.07418v1https ://github.com/iva-mzsun/AR-Diffusion
在视频生成领域，创建视觉上真实且时间上连贯的视频帧是一项挑战。现有的方法主要依赖于异步自回归模型和同步扩散模型。异步自回归模型在训练与推理阶段之间存在不一致性，导致错误累积等问题；而同步扩散模型则受限于固定的序列长度。为了解决这些问题，本文提出了一种新颖的视频生成模型——自回归扩散（AR-Diffusion）。该模型结合了自回归和扩散模型的优点，实现灵活的异步视频生成。通过逐步破坏视频帧并在训练和推理阶段减少差异，AR-Diffusion能够生成具有时间一致性的多样化视频。此外，本文还设计了两种专门的时间步调度器，以支持不同的生成需求，实验结果显示该方法在多个基准测试中取得了优异的性能。
AR-Diffusion的核心方法包括几个关键组件。首先，提出了自回归视频自编码器（AR-VAE），它通过时间无关的视频编码器提取特征，并通过时间因果解码器重建帧，从而保持时间一致性。其次，在AR-Diffusion框架中，采用了逐步破坏视频帧的扩散过程，确保训练和推理阶段的一致性。此外，引入了非递减时间步约束，确保早期帧的清晰度高于后期帧，从而支持灵活的生成。为了优化训练过程，采用了帧导向概率传播（FoPP）时间步调度器，确保时间步的均匀采样和帧特定时间步的平衡。在推理阶段，采用自适应差异（AD）时间步调度器，允许相邻帧之间的时间步差异适应变化。这些设计使得AR-Diffusion在生成不同长度的视频时，能够保持时间连贯性和视觉质量。
在实验部分，研究者们对AR-Diffusion进行了广泛的性能评估，使用了四个具有挑战性的基准数据集，包括FaceForensics、Sky-Timelapse、Taichi-HD和UCF-101。通过与现有的生成模型进行比较，AR-Diffusion在视频质量和时间一致性方面表现出明显的优势。实验结果表明，AR-Diffusion在多个数据集上均取得了竞争性和最先进的结果，尤其是在UCF-101数据集上，其FVD得分比之前的最优异的异步视频扩散模型高出60.1%。此外，研究还探讨了时间步差异对生成性能的影响，结果显示适当的时间步差异能够显著提升生成效果。这些实验验证了AR-Diffusion在视频生成任务中的有效性和高效性。
AR-Diffusion是一种新的视频生成技术，它的工作原理可以简单理解为“逐步画画”。首先，它通过一种特殊的编码器把视频的每一帧压缩成简洁的特征，就像把复杂的图画变成简单的线条。然后，在生成视频时，它会逐步“添加细节”，确保每一帧都与前一帧保持一致。为了让生成的视频看起来更自然，AR-Diffusion特别设计了时间步调度器，这就像是给每一帧设置了不同的“画笔”，让早期的帧更加清晰，而后面的帧则逐渐增加细节。通过这种方式，AR-Diffusion能够灵活地生成不同长度的视频，同时保持视频的流畅性和视觉效果。
2025-03-09｜UCL｜⭐️🟡
http ://arxiv.org/abs/2503.06364v1
本研究提出了一种新颖的生成视频模型，名为视频双流（VideoBi-flow），旨在通过稳健地学习时间变化来优化视频生成过程。该模型采用神经常微分方程（ODE）流的框架，结合了两种主要的生成策略：直接从过去的帧生成未来的帧，以及在训练过程中通过引入噪声来消除累积的误差。与传统的条件扩散模型相比，视频双流能够实现更高效的无条件视频生成，并具备流式生成的能力，使其适用于交互式应用场景，如计算机游戏。通过对比实验，视频双流在视频质量和生成速度上均表现出优越性，展示了其在生成视频时的高效性和稳定性。
视频双流的核心在于构建一个联合目标，通过以下几个关键步骤实现视频生成：
流匹配：定义从视频帧的过去分布到未来分布的流场，利用流匹配方法来回归目标流场。
联合训练：在训练过程中，通过在每个帧之间添加噪声来提高训练空间的覆盖率，从而增强模型对时间变化的适应能力。
双流场设计：模型同时预测视频生成方向和去噪方向，通过这种双重流场设计来稳定生成过程。
流式生成：基于当前帧直接生成下一帧，避免从噪声开始的高计算成本，提升生成速度和质量。
联合采样策略：在推理阶段，模型能够在时间和去噪的维度上同时进行求解，从而有效地生成连续的视频帧。
我们在多个视频数据集上对视频双流进行了系统评估，包括Minecraft游戏视频、汽车驾驶视频、天际线延时视频等。实验中采用了不同的生成方法作为基线进行对比，主要包括条件扩散模型和传统的流模型。评估指标包括生成视频的质量（通过Fréchet视频距离FVD评估）和计算速度。结果显示，视频双流在生成质量上与条件扩散模型相当，但所需的计算步骤显著减少，显示出更高的效率。此外，通过调整噪声水平，视频双流在稳定性与生成质量之间达到了良好的平衡，证明了其在长视频生成中的优势。
视频双流模型的工作原理可以简单理解为，它通过学习如何从一帧视频生成下一帧来制作视频。首先，它会在训练时引入一些噪声，以帮助模型更好地理解视频帧之间的变化。接下来，模型会同时学习如何生成新帧和如何修复之前生成帧的错误。这样，在生成视频时，它可以直接从当前帧出发，快速生成下一帧，而不需要从随机噪声开始，这样可以大大提高生成速度。最后，模型在生成每一帧时，还能根据需要调整噪声的程度，以确保视频的质量和流畅性。这种方法不仅提高了生成效率，还允许实时视频生成，非常适合用于游戏等互动场景。
2025-03-09｜SJTU,MGTV｜⭐️🟡
http ://arxiv.org/abs/2503.06545v1https ://github.com/JunyiWuCode/QuantCache
本研究提出了一种名为QuantCache的创新框架，旨在提高视频生成过程中的推理效率。随着扩散变换器（DiTs）在生成建模领域的广泛应用，尽管其性能优越，但其计算和内存需求显著增加，限制了在资源受限设备上的应用。QuantCache通过联合优化层次潜在缓存、适应性重要性引导量化和结构冗余感知剪枝等技术，成功实现了6.72倍的推理加速，同时保持了高生成质量。通过在多个视频生成基准上的广泛实验，QuantCache展现了其在高效推理方面的有效性，设置了新的标准。
QuantCache的核心方法包括三个主要部分：
层次潜在缓存（HLC）：该方法通过动态调整缓存特征的刷新频率，基于特征间的变化来减少冗余计算。HLC计算每个时间步的特征差异，决定何时需要重新计算特征，从而优化计算资源的使用。
适应性重要性引导量化（AIGQ）：AIGQ根据特征的敏感性动态调整权重和激活的位宽。关键计算保持较高的位宽，而冗余计算则使用较低的位宽，从而在保证生成质量的同时减少计算负担。
结构冗余感知剪枝（SRAP）：SRAP通过分析同一时间步内层之间的相似性，选择性地剪枝冗余层，进一步降低计算成本。该方法在不同时间步中动态评估层的重要性，以减少不必要的计算。
在实验部分，QuantCache在Open-Sora模型上进行了评估，探索了不同位宽配置和加速策略的效果。实验结果显示，QuantCache在8/8和4/6位宽配置下均表现出显著的性能提升，尤其是在生成质量和速度方面。通过使用VBench基准测试，QuantCache在运动平滑性、背景一致性和美学质量等多个维度上均优于现有的最先进方法。此外，QuantCache的CUDA实现通过优化的GEMM内核，结合量化和缓存机制，达到了6.72倍的推理加速，证明了其在实际应用中的有效性和可行性。
QuantCache的工作原理可以简单理解为三大步骤。首先，层次潜在缓存（HLC）就像一个智能的记忆系统，它会根据生成视频的内容变化，决定何时需要重新计算一些特征，以避免重复劳动。其次，适应性重要性引导量化（AIGQ）就像是根据重要性分配资源，关键的细节使用更多的计算能力，而那些不太重要的部分则节省资源。最后，结构冗余感知剪枝（SRAP）则是通过观察不同层之间的相似性，决定哪些层可以被省略，从而减少计算负担。通过这三步，QuantCache能够在不牺牲生成质量的情况下，大幅提升视频生成的速度。
2025-03-09｜UCLA,Google｜⭐️🟡
http ://arxiv.org/abs/2503.06800v1https ://videophy2.github.io/
VIDEOPHY-2是一个新推出的基准数据集，旨在评估现代视频生成模型在生成视频时对物理常识的遵循程度。尽管大型视频生成模型在创建各种视觉概念的逼真视频方面取得了显著进展，但它们在真实世界动作的物理常识遵循方面的能力仍然不明确。现有的评估基准存在样本量小、缺乏人类评估、仿真与现实之间的差距以及缺乏精细的物理规则分析等问题。为了解决这些问题，研究者们整理了197个多样化的动作，并为视频合成生成了3940个详细提示。通过人类评估，研究揭示了视频生成模型在物理常识和语义一致性方面的重大不足，尤其是在质量守恒等物理法则的遵循上。最终，研究团队还训练了VIDEOPHY-2-AUTOEVAL，一个自动评估工具，以支持快速、可靠的评估。
VIDEOPHY-2的数据构建过程分为几个关键阶段。首先，研究团队从多个流行的视频数据集中筛选出600多个与物理常识评估相关的动作，并最终确定197个动作，这些动作涵盖了体育活动和物体交互等多种场景。接下来，利用大型语言模型（LLM）为每个动作生成20个详细提示，总计3940个提示，确保这些提示专注于可视的物理交互。第三阶段是生成与这些视频相关的候选物理规则和法律，使用视频描述生成模型来确保生成的规则与视频内容相符。最后，为了增加数据集的难度，研究者们创建了一个困难子集，包含60个特别难以生成符合提示和物理常识的视频的动作，进一步提高了评估的挑战性。
在实验中，研究者们评估了多种最新的文本到视频生成模型在VIDEOPHY-2数据集上的表现。通过人类评估，发现即使是表现最好的模型Wan2.1-14B，在整体数据集和困难子集上的联合表现得分也仅为32.6%和21.9%。此外，研究还揭示了模型在遵循物理法则方面的主要问题，尤其是在质量和动量守恒方面的表现不佳。为了提高评估的效率，研究团队开发了自动评估模型VIDEOPHY-2-AUTOEVAL，能够在多个任务上表现出色，尤其是在语义一致性和物理常识的评分上，相较于其他基线模型有显著的提升。这些实验结果展示了VIDEOPHY-2作为一个高质量数据集的潜力，并为未来的物理模拟研究奠定了基础。
在VIDEOPHY-2的研究中，研究者们采用了一种分阶段的方法来构建数据集。首先，他们从多个视频数据集中挑选出与物理常识相关的动作，例如打网球或做后空翻。然后，利用先进的语言模型为每个动作生成详细的描述，这些描述会在生成视频时作为提示。接下来，研究者们还会为这些视频生成一些物理规则，比如“网球应该在重力作用下呈抛物线飞行”。最后，为了让模型面临更高的挑战，他们特意选择了一些特别复杂的动作，这些动作在生成视频时往往难以遵循这些物理规则。这种方法不仅帮助他们评估现有视频生成模型的能力，也为未来的研究提供了新的方向。
2025-03-09｜GDUT,GUFS｜⭐️
http ://arxiv.org/abs/2503.06624v1
在人工智能生成内容（AIGC）迅速发展的背景下，AI生成视频的检测成为了一项重要的研究任务。虽然已有大量研究集中在AI生成文本和图像的识别上，但针对视频的检测研究仍显不足。现有的AI生成视频检测数据集在多样性、复杂性和现实性方面存在局限，无法全面满足真实世界取证的需求。为了解决这些问题，本文提出了一个新的数据集——Chameleon，专注于AI生成视频的检测。Chameleon数据集通过多种生成工具和真实视频源生成视频，保留了视频的真实世界复杂性，包括场景切换和动态视角变化，超越了以面部为中心的检测方法，涵盖了人类动作和环境生成。该研究为AI生成视频的检测提供了一个重要的基准，帮助应对不断演变的AI生成内容威胁。
本研究的核心在于构建Chameleon数据集，以提高AI生成视频的检测能力。具体方法分为以下几个步骤：
原始视频选择：从新闻、演讲和产品推荐等多个公共视频平台中选择视频，确保涵盖多样化的场景和动态元素。
视频生成策略：采用多种AI生成技术，随机将真实视频分成两部分，一部分用于生成合成视频，另一部分保持原样，以增强模型的泛化能力。
数据内容构建：根据场景切换和环境变化等标准，手动筛选出符合要求的视频片段，并生成相应的结构化元数据，以指导后续视频生成过程。
通过这些步骤，Chameleon数据集不仅在场景多样性和领域多样性方面进行了全面的模拟，还为AI生成视频的检测提供了丰富的基础。
在实验部分，我们评估了不同检测方法在Chameleon数据集上的性能。实验主要分为三个阶段：训练、验证和测试。我们构建了一个包含600个真实视频和600个AI生成视频的总数据集，共1200个样本。数据集按照6:2:2的比例划分为训练集、验证集和测试集。我们使用深度学习方法和大型视觉模型进行比较，评估它们在不同置信阈值下的准确性。结果表明，深度学习方法如NPR、FreqNet和BNet在检测AI生成视频方面表现优异，尤其是在低置信度条件下。与此同时，尽管大型视觉模型如Claude3.5和GPT-4V在某些场景中有一定的识别能力，但整体性能仍低于深度学习模型。这些实验结果为后续的研究提供了宝贵的参考。
在Chameleon数据集的构建过程中，研究团队首先从多个视频平台上选择真实的视频，比如新闻、演讲和产品推荐。然后，他们将这些视频分成两部分：一部分用于生成新的AI视频，另一部分则保持原样。这样做的目的是为了让AI视频看起来更真实，并且帮助模型更好地学习如何分辨AI生成的视频和真实的视频。接着，团队手动挑选出包含明显场景变化和多样化元素的视频片段，以确保数据集的丰富性。最后，他们通过这些步骤创建了一个包含1200个视频的综合数据集，既有真实视频也有AI生成的视频。这种方法不仅提高了检测的准确性，还为未来的研究提供了一个强大的基础。
