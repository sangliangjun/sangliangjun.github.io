2025-03-07｜HKUST,HuaweiNoah’sArkLab｜CVPR2025
http ://arxiv.org/abs/2503.05082v1https ://zhongyingji.github.io/guidevd-3dgs/
在近年来的3D场景建模中，3DGaussianSplatting（3DGS）技术取得了显著进展，然而，处理稀疏输入的场景建模仍然面临诸多挑战。本文提出了一种新方法，通过利用视频扩散模型的重建生成管道，有效解决了稀疏输入建模中的外推和遮挡问题。我们引入了一种场景引导的生成策略，旨在生成一致的图像序列，从而改善3DGS模型的整体表现。具体而言，我们的研究聚焦于如何通过优化生成的图像序列来增强场景的完整性，并通过实验验证了该方法在多个基准数据集上的优越性，达到了最先进的性能。
本研究的核心方法包括三个主要部分：场景引导指导、轨迹初始化策略和基于生成序列的3DGS优化。首先，场景引导指导通过对生成的图像序列施加一致性约束，确保生成的图像在时间上的连贯性。其次，轨迹初始化策略用于识别场景中未被稀疏输入覆盖的区域，帮助生成更全面的视图。最后，我们设计了一种优化方案，将生成的序列与3DGS模型相结合，以提升模型的表现。具体步骤包括：利用视频扩散模型生成多视角序列，进行场景引导指导以优化生成的序列，并使用这些序列来优化3DGS模型的参数，从而有效地解决外推和遮挡问题。
在实验部分，我们在两个具有挑战性的室内数据集上验证了所提出方法的有效性，分别为Replica和ScanNet++。实验结果表明，使用我们的方法相比于基线模型在PSNR、SSIM和LPIPS等指标上均表现出显著提高。在Replica数据集上，我们的方法在PSNR上超过了3.5dB的提升，而在ScanNet++数据集上也实现了2.5dB的提高。此外，定性结果显示，我们的方法在处理遮挡和外推问题时，能够生成更高质量的图像，减少黑影和伪影的出现，从而显著提升图像的整体质量和细节表现。
在这项研究中，我们提出了一种新方法来改善3D场景的建模，特别是在视角稀疏的情况下。首先，我们使用一种叫做视频扩散模型的技术来生成多个视角的图像序列，这些序列可以帮助我们更好地理解和重建场景。然后，我们引入了一种“引导”机制，确保这些生成的图像在时间上是连贯的，这样就不会出现不一致的画面。接着，我们设计了一种轨迹初始化策略，帮助模型识别那些未被观察到的区域，以便生成更全面的图像。最后，我们将生成的图像序列与3D模型结合起来，优化模型的表现，从而更好地处理遮挡和外推问题。通过这些步骤，我们的方法在多个数据集上都取得了显著的成功。

CoMoGaussian是一种新颖的框架，旨在从摄像机运动模糊图像中重建精确的3D场景。运动模糊通常是由于摄像机在曝光期间的移动所引起的，这会对3D重建的准确性造成影响。该研究通过利用连续运动感知的高斯点云表示来解决这一问题。CoMoGaussian通过整合神经常微分方程（ODE）技术，精确建模连续摄像机轨迹，从而提高了重建的准确度。研究表明，该方法在多种运动模糊场景下都能保持实时渲染速度，并在基准数据集上展示了优越的性能。
CoMoGaussian的核心方法包括以下几个步骤：
连续摄像机运动建模：使用神经ODE来预测摄像机在曝光期间的连续轨迹。每个摄像机姿态由刚体变换表示，保持物体的形状和大小不变。
连续运动优化：引入连续运动优化（CMR）变换，通过学习可调参数来改善刚体变换，确保更精确的运动模糊轨迹近似。
图像渲染与合成：通过Mip-Splatting方法渲染从每个摄像机姿态生成的图像，并应用像素加权策略来合成最终的模糊图像。该过程涉及对每个像素的权重计算，以确保最终输出的图像能够有效反映场景的真实状态。
在实验部分，CoMoGaussian在多个数据集上进行了评估，包括Deblur-NeRF合成数据集和ExbluRF真实场景数据集。实验结果显示，该方法在峰值信噪比（PSNR）、结构相似性指数（SSIM）和学习感知图像补丁相似性（LPIPS）等指标上均超越了现有的最先进方法。此外，通过消融实验验证了各个组件的有效性，特别是CMR变换在提高性能方面的显著作用。实验还表明，CoMoGaussian在处理不同程度的运动模糊时表现出色，能够在极端和中等模糊场景中保持高质量的重建效果。
CoMoGaussian的工作流程可以简单理解为：首先，它通过一种叫做神经ODE的数学方法，模拟摄像机在拍照时的连续移动轨迹。这种模拟帮助我们了解在拍照过程中，摄像机是如何移动的。接下来，它使用刚体变换来保持拍摄对象的形状和大小不变，同时引入一种新的技术，称为连续运动优化，这样可以更好地处理运动模糊。最后，系统会把从不同角度拍摄的图像进行组合，计算每个像素的权重，生成一张最终的模糊图像。这个过程确保了即使在模糊的情况下，我们也能得到尽可能清晰和真实的3D场景重建效果。

本文提出了一种名为MGSR的2D/3D互助增强高斯点云表面重建方法，旨在解决在不同光照条件下的高保真表面重建和新视图合成问题。传统的基于高斯点云的方法通常在渲染质量与重建准确性之间存在权衡，而MGSR通过引入两个相互促进的分支（2D-GS和3D-GS），有效地提升了渲染质量和重建精度。2D-GS分支专注于表面重建，提供精确的几何信息，而3D-GS分支则利用该几何信息进行光照分解，从而实现更真实的渲染效果。通过交替优化和互助监督，MGSR在优化过程中实现了两个分支的协同发展，克服了以往方法在复杂光照条件下的不足。实验结果表明，MGSR在合成新视图和表面重建任务中均表现出色，优于现有的基线方法。
MGSR方法的核心在于其2D/3D互助增强框架，包含两个主要分支：2D-GS和3D-GS。首先，每个分支都经历独立的预热阶段，以便为后续的交替优化做好准备。在3D-GS分支中，采用了一种几何引导的光照分解模块，利用来自2D-GS分支的深度信息来改善渲染性能，特别是在复杂光照条件下。具体步骤包括：
光照分解：3D-GS分支引入反射相关参数，分离反射和透射成分，以提升渲染效果。
表面重建：2D-GS分支使用扁平化的2D高斯原语来捕捉场景的几何信息，并生成初步的深度图。
交替优化：在交替优化阶段，3D-GS分支通过2D-GS生成的深度图进行监督，同时3D-GS渲染的无反射图像为2D-GS分支提供了反向监督。
损失函数设计：采用加权损失函数来平衡两个分支的优化目标，确保在不同光照条件下的表面重建和渲染质量均得到提升。
在实验部分，MGSR在多个合成和真实世界数据集上进行了评估，包括DTU和OmniObject3D等。实验采用了SSIM和PSNR等指标来评估渲染质量，同时使用法线一致性（NC）和Chamfer距离（CD）来验证重建准确性。实验结果显示，MGSR在表面重建和新视图合成的性能上均优于现有的基线方法。例如，在ShinyBlender和OmniObject3D数据集上，MGSR在PSNR和NC指标上均取得了显著的提升。此外，MGSR的方法在处理复杂反射和光照条件下的场景时，能够有效重建出高质量的表面网格，且优化时间相对较短，显示出良好的实用性和效率。
MGSR方法的工作原理可以简单理解为一个团队合作的过程。它有两个主要成员，一个专注于获取物体的形状（2D-GS），另一个则负责处理光线如何影响物体的外观（3D-GS）。首先，这两个成员各自进行热身，准备好后就开始合作。2D-GS会告诉3D-GS物体的准确形状，而3D-GS则利用这个形状来更好地理解光线的反射和透射。这样，3D-GS可以生成更真实的图像，而2D-GS也能在光线变化时更准确地重建物体表面。通过这种交替合作，MGSR能够在不同光照条件下都表现出色，确保我们看到的图像既真实又清晰。

本文提出了一种新颖的框架——StereoAnyVideo，旨在解决视频立体匹配中的时序一致性问题。该方法能够在没有依赖相机位姿或光流等辅助信息的情况下，准确估计空间上连贯的视差。通过有效整合单目视频深度模型的丰富先验信息与卷积特征，StereoAnyVideo显著提升了视差估计的稳定性和准确性。关键的架构创新包括全对全配对相关性模块和时间凸上采样机制，这些设计确保了匹配成本体积的平滑性与鲁棒性，从而实现了更高的时序一致性。实验结果表明，该方法在多个数据集上均达到了最先进的性能，尤其在动态场景下表现出色，展示了其在自动驾驶、机器人导航和虚拟现实等应用中的广泛潜力。
StereoAnyVideo的方法框架主要包括三个核心组件：特征提取、相关性计算和时序成本聚合。首先，在特征提取阶段，我们采用了一个集成了单目视频深度模型的卷积网络，以确保提取到的特征在时间维度上保持稳定。其次，在相关性计算阶段，我们引入了全对全配对相关性机制，通过计算每一对潜在匹配点的相似性，增强了匹配的平滑性和一致性。最后，在时序成本聚合阶段，采用了时间凸上采样方法，通过对比前后帧的视差信息，迭代优化每一帧的视差估计。这一系列步骤共同作用，确保了最终输出的视差图在时间上具有一致性和准确性，克服了传统方法在动态场景下的局限性。
在实验部分，我们使用了多个数据集评估StereoAnyVideo的性能，包括合成数据集和真实世界的视频数据。实验结果显示，该方法在零-shot设置下的性能超过了现有的多种立体匹配算法，尤其是在动态场景中，StereoAnyVideo显示出了卓越的时序一致性。我们通过定量指标（如端点误差EPE和时间端点误差TEPE）对比了不同方法的表现，结果表明我们的模型在多个基准测试中均实现了至少15%的性能提升。此外，进行了一系列消融实验，以验证各个组件对最终性能的贡献，结果进一步证明了特征提取、相关性计算和时序聚合的有效性。
在我们的研究中，我们设计了一种新的视频立体匹配方法，叫做StereoAnyVideo。这个方法的工作流程可以简单理解为三个步骤。首先，我们会从视频中提取出图像特征，这些特征可以帮助我们理解场景的深度信息。为了确保这些特征在时间上保持一致，我们使用了一种叫做单目视频深度模型的技术。接下来，我们会计算图像中每一对像素点之间的相似性，确保在不同帧之间的匹配是流畅的，而不是突然跳动。最后，我们通过一种称为时间凸上采样的方法，来优化每一帧的视差估计，确保最终的结果在时间上是连贯的。这样，我们就能在动态场景中获得准确且一致的深度信息，而不需要依赖额外的相机运动数据。

http ://arxiv.org/abs/2503.05578v1https ://github.com/CNJianLiu/SinRef-6D
本文提出了一种新颖的基于单一参考视图的6D物体姿态估计方法，称为SinRef-6D。现有的6D姿态估计方法通常依赖于CAD模型或密集参考视图，这些方法在获取上存在困难。相较之下，使用单一参考视图的方法具有更好的可扩展性，但面临着姿态差异大和几何及空间信息有限的挑战。SinRef-6D通过迭代建立相机坐标系中的点对齐，利用状态空间模型（SSMs）来解决这些问题。该方法在经过合成数据预训练后，可以仅用单一参考视图来估计新物体的6D姿态，而无需重训练或CAD模型。大量实验表明，SinRef-6D在六个流行数据集和现实机器人场景中表现出与传统方法相当的性能，证明了其在单一参考视图条件下的有效性。
SinRef-6D方法主要由四个模块组成：初始化、点聚焦、SSM特征提取和点对齐与姿态求解。具体步骤如下：
初始化：从输入的RGB-D图像中分割出目标物体，并将深度图反投影为点云，生成参考视图和查询视图的点云。
点聚焦：将参考和查询点云转换到相同的相机坐标系中，以确保几何一致性，便于后续的点对齐。
SSM特征提取：通过RGB和点云的状态空间模型提取丰富的空间特征，形成点对齐所需的特征表示。
点对齐与姿态求解：利用提取的特征建立点对齐关系，采用加权奇异值分解算法求解物体的6D姿态，并通过迭代反馈机制不断优化对齐结果。
这种方法能够有效处理姿态差异和空间信息不足的问题，提升了6D姿态估计的准确性和鲁棒性。
为了验证SinRef-6D的有效性，研究者在六个公开数据集（如LineMod、YCB-V等）和真实世界场景中进行了大量实验。评估指标包括平均点距离的召回率（ADD）和面积曲线下的ADD（AUC）。实验结果显示，SinRef-6D在多个数据集上均超越了现有的手动参考视图方法，并且在CAD模型基础的方法中表现出相当的性能。此外，研究者还进行了消融实验，验证了各个模块对整体性能的贡献。结果表明，SinRef-6D在仅使用单一参考视图的情况下，依然能够实现高效、准确的6D物体姿态估计，展示了其在实际应用中的潜力。
SinRef-6D的工作原理可以简单理解为通过一个参考图像来判断新物体的位置和方向。首先，我们从一张图像中提取出物体的形状和颜色信息。接着，我们将这个物体的点云（即物体的三维表示）转换到一个统一的坐标系统中，这样我们就可以直接比较不同视角下物体的样子。然后，使用一种特别的模型，我们提取出这些点云和图像中的特征，以便更好地理解它们的关系。最后，通过不断调整和优化这些特征，我们能够准确地计算出物体在新视角下的姿态。这个过程就像在不断调整一个拼图，直到所有的部分都完美契合。这样，我们只需一张参考图像，就能有效地识别和定位新物体。

EvolvingGS是一种新颖的动态场景重建方法，旨在通过演变的3D高斯表示来实现高保真、可流式传输的体积视频。该方法克服了传统技术在处理长时间复杂动态场景时的局限性，尤其是在快速运动和细致服装变形方面。EvolvingGS采用了一种两阶段的策略，首先通过变形高斯模型粗略对齐目标帧，然后通过最小的点添加和删除进行精细调整。该方法不仅确保了高质量的渲染和时间一致性，还实现了超过50倍的压缩率，同时保持了视觉质量。这使得在带宽受限的情况下，实时流媒体传输成为可能。
EvolvingGS的方法分为三个主要阶段：
变形阶段：该阶段通过光流引导的稀疏控制点生成变形场，将初始高斯模型调整为与后续帧的外观相匹配。控制点的位置信息、旋转和位移都被记录，以便进行局部区域的仿射变换。
细化阶段：在这一阶段，优化高斯模型以适应新出现或消失的对象。通过添加新高斯点和删除不再可见的点，确保了动态场景的准确性。此阶段中，外观相关特征保持不变，以增强时间一致性。
压缩方案：利用时间一致性进行有效的数据压缩，特别是通过编码参考高斯点和新生成高斯点的特征差异，显著减少存储需求。该方案确保在保持重建质量的同时，实现高效的存储管理。
在多个公共基准和自定义数据集上进行的实验表明，EvolvingGS在动态场景重建中表现优越。使用的ActorsHQ数据集包含八位演员的表演，捕捉了复杂的运动和服装变化。实验评估了不同方法在重建质量、存储需求和流媒体性能方面的表现。结果显示，EvolvingGS在PSNR、SSIM和LPIPS等指标上均优于其他现有技术，尤其是在长时间序列处理时，展现出更强的稳定性和一致性。此外，通过对比不同的GoP长度，EvolvingGS证明了其在延长时间窗口内的优越表现，能够有效处理复杂的动态场景。
EvolvingGS的方法可以简单理解为一个两步走的过程。首先，它通过分析连续帧之间的变化，使用一些关键点来粗略调整场景中的物体位置。这就像在画画时，先用铅笔打好轮廓，然后再用颜料细致上色。接下来，它会进行更精细的调整，比如添加新出现的物体或删除不再可见的物体。这一步确保了画面中的每个细节都能准确呈现。最后，EvolvingGS还会通过聪明的编码方式来减少存储空间，就像把一大堆书整理成电子书，既节省空间又方便携带。这样，EvolvingGS不仅能高效重建动态场景，还能在网络条件不佳的情况下流畅播放视频。
2025-03-07｜BIT,PengchengLab
http ://arxiv.org/abs/2503.05196v1
本研究提出了一种名为选择性训练高斯头部头像（STGA）的方法，旨在增强动态高斯头像的细节表现。通过对面部表情变化显著的区域进行优化，STGA能够在高效的训练时间内生成更具真实感的头像。与网络基础方法相比，该方法不仅缩短了训练时间，还在相同时间内提供了更高的细节表现。此外，通过消融实验验证了STGA在细节质量提升方面的有效性。这项研究为高保真3D头像重建提供了新的思路，解决了传统方法在面部动态表现中的不足，推动了计算机图形学和计算机视觉领域的发展。
STGA方法的核心在于选择性优化动态高斯头像的训练过程。具体步骤如下：
高斯点嵌入：在FLAME模型的三角面上嵌入高斯点，建立局部坐标系。
区域选择：通过比较每帧的FLAME模型与其中性模型，识别出面部表情变化显著的三角区域。
优化策略：在每帧训练中，仅优化这些关键区域的高斯点，而其他高斯点的参数保持不变，确保训练聚焦于细节。
全局一致性：每20次迭代进行一次全局优化，以维护模型的整体一致性。
批量训练：采用批量加载图像的方式，控制GPU内存使用，确保训练效率。通过这些步骤，STGA能够在较短的时间内达到更高的细节表现，从而生成更真实的头像。
本研究使用了NeRSemble数据集进行实验，数据集包含多视角的面部表情视频及相应的FLAME模型。实验中，使用NVIDIARTX3090显卡进行训练，记录了不同方法的训练时间和效果。与现有的网络驱动高斯头像和网格驱动高斯头像方法进行比较，STGA在270分钟内完成训练，显著缩短了训练时间。实验结果表明，STGA在细节表现上优于其他方法，尤其在面部细节如眼睛和牙齿的表现上，呈现出更高的真实感。此外，通过对比PSNR、SSIM和LPIPS等指标，STGA在图像质量上也表现出色，验证了其选择性训练策略的有效性。
STGA方法的核心是让我们在训练3D头像时，专注于那些面部表情变化最大的部分。首先，我们把高斯点放到头像的每个小面上，这样它们就能随着面部的动作一起移动。接着，我们通过比较每帧的头像与静态模型，找出哪些地方变化最大，比如嘴巴、眼睛等。然后，在训练时，我们只对这些重要的部分进行调整，而不去动其他地方，这样可以更快地提高细节表现。同时，每隔一段时间，我们会对整个模型进行一次大调整，以确保所有部分看起来协调一致。最后，我们采用分批加载图像的方式，避免占用过多的计算资源，从而提高训练效率。这样做的结果是，我们能在短时间内训练出更真实的头像。
