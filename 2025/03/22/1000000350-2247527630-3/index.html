316-A3｜3D与文本潜在空间对齐，草图到CAD重建，MLLM复杂场景对象放置；火柴人生成3D人类动作，探讨舞伴间的隐性交互

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月10日｜周一
2025-03-07｜Ecolepolytechnique,SapienzaURome,UNIMIB,UTAustin｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.05283v1
本研究聚焦于3D编码器与文本编码器之间的潜在空间对齐，探讨如何有效地将这两种不同模态的表示进行关联。尽管已有的多模态学习模型（如CLIP）在图像与文本之间建立了明确的对齐，然而3D编码器的作用尚未得到充分探讨。我们发现，简单的后期训练对齐方法在3D与文本编码器的结合中效果有限，表明需要更精细的对齐策略。为此，研究者们提取了特征空间的子空间，通过将学习到的表示投影到低维子空间中，显著提升了对齐的质量。这一发现不仅为3D与文本的跨模态任务提供了新的视角，也为后续研究奠定了基础，强调了3D数据在表示学习中的独特性和潜力。
本研究提出了一种新颖的潜在空间对齐方法，结合了典型的相关性分析工具和现有的对齐技术。具体步骤包括：
初步准备：通过选择一组锚点对（即3D形状与其文本描述），为后续的对齐提供基础。
中心化核对齐（CKA）：使用CKA度量不同编码器的特征相似性，识别出高相关性的子空间。
子空间投影：将3D和文本的潜在表示通过线性投影映射到共享的低维子空间，以便在此空间中进行有效的对齐。
仿射变换：利用仿射变换方法进一步优化对齐过程，确保在低维子空间中实现最佳的表示匹配。
局部CKA匹配：通过局部CKA计算不同查询对之间的相似性，以提高检索任务的准确性。
在实验部分，研究者使用了Objaverse数据集进行预训练，评估了不同对齐方法在匹配和检索任务中的表现。实验设计包括多个步骤：
数据集准备：选择Objaverse中的3D形状和对应文本描述，构建训练和测试集。
编码器选择：比较多模态与单模态的3D和文本编码器，包括PointBERT和MinkowskiNet等。
对齐性能评估：使用匹配准确率和Top-5检索率等指标，分析不同对齐方法的有效性。
结果对比：将研究结果与现有的对齐技术进行比较，验证提出方法的优越性。
可扩展性测试：考察锚点数量对对齐性能的影响，确保方法在大规模数据集上的有效性。
在我们的研究中，我们尝试解决一个有趣的问题：如何让3D模型和文本之间更好地理解和关联。我们的方法可以简单分为几个步骤。首先，我们从3D模型和文本描述中挑选一些代表性的样本，作为“锚点”，帮助我们找到它们之间的联系。接着，我们使用一种叫做中心化核对齐的方法，比较这些样本在不同编码器中的相似性，找出它们的共同点。然后，我们将这些样本的表示投影到一个较小的空间中，这样可以更容易地进行比较和匹配。最后，我们还使用了仿射变换来进一步优化这个过程，确保我们的对齐效果最佳。通过这些步骤，我们发现3D模型和文本之间的联系变得更加紧密，从而提高了检索和匹配的准确性。
2025-03-07｜HubeiElectric,BUPT｜⭐️🟡
http ://arxiv.org/abs/2503.05161v1
在计算机视觉领域，自动从光栅CAD草图中重建3D计算机辅助设计（CAD）模型是一项具有挑战性的任务。传统方法通常依赖于矢量CAD草图和3D真实数据进行监督，这在工业应用中往往难以获得，并且对噪声输入非常敏感。为了解决这些问题，本文提出了一种名为GaussianCAD的新方法，将CAD重建视为稀疏视图重建的特定实例。通过将CAD草图转换为类似自然图像的表示，并提取相应的掩模，本文的框架能够在没有3D真实数据的情况下进行自监督学习，从而提高重建的鲁棒性。实验结果表明，GaussianCAD在CAD重建性能上显著优于现有方法，并且对噪声输入表现出强大的鲁棒性。
GaussianCAD方法分为三个主要步骤。首先，草图过滤模块对多视角光栅CAD草图进行处理，检测并移除表示不可见线条的虚线，保留可见边缘。其次，使用外部边缘确定每个视图的前景，并进行上色，以便后续处理。最后，手动计算正投影视图的相机姿态，确保在3D坐标系中的准确对齐。通过这些步骤，GaussianCAD能够将光栅CAD草图转换为自然图像风格的表示，进而利用3D高斯点云重建方法进行3D模型的生成。该方法通过自监督学习框架消除了对矢量CAD草图和3D真实数据的依赖，从而提升了模型的适用性和灵活性。
在Sub-Fusion360数据集上进行的实验验证了GaussianCAD的有效性。实验评估了该方法在不同CAD草图上的重建性能，并与其他现有方法进行了比较。通过Chamfer距离、Hausdorff距离和地球移动者距离等评估指标，结果显示GaussianCAD在所有指标上均表现出色，达到了最新的研究水平。实验还探讨了GaussianCAD在面对噪声输入时的鲁棒性，结果表明该方法能够成功重建复杂模型，而传统方法在噪声影响下则表现不佳。此外，消融实验进一步分析了各个模块的贡献，证实了草图过滤和相机姿态定位模块在重建过程中的重要性。
GaussianCAD的方法可以简单理解为一个三步走的过程。第一步是对CAD草图进行“清理”，这就像是把一张乱糟糟的图画整理好，去掉那些不必要的线条，保留主要的边缘。第二步是给草图上色，帮助我们更好地识别前景和背景，就像给一幅画添加色彩，使其更加生动。最后一步是手动确定相机的位置和角度，这样我们就能确保从不同的视角观察草图时，它们在三维空间中是准确对齐的。通过这三个步骤，GaussianCAD能够将这些经过处理的草图转变为高质量的3D模型，且不需要依赖于昂贵的真实数据，这使得它在实际应用中非常灵活和高效。
2025-03-06｜Stanford,GoogleDeepMind｜⭐️⭐️
http ://arxiv.org/abs/2503.04919v1
FirePlace是一种新颖的框架，旨在利用多模态大语言模型（MLLMs）在复杂的三维场景中进行对象放置。该方法结合了高层次的语义理解与低层次的几何推理，以解决传统方法在处理复杂几何时的局限性。FirePlace的核心在于其对三维场景的精细几何推理，能够从输入的三维场景、待放置对象及语言提示中提取相关几何信息，并构建相应的几何约束。通过这一过程，FirePlace不仅能确保放置的对象符合几何约束，还能满足常识推理的要求，从而生成更为合理和美观的三维布局。
FirePlace的工作流程分为几个主要步骤。首先，输入包括三维场景、待放置对象和描述性语言提示。接着，系统生成约束草图，列出与放置任务相关的几何约束。然后，FirePlace执行三维推理，解析出具体的表面参考和几何参数，以确保所生成的放置方案符合这些约束。第三步是通过“批量视觉选择”方法，逐步筛选出最合适的放置选项，减少MLLM在选择时的负担。最后，进行可行性修剪，确保最终的放置方案不仅在几何上可行，还符合美学、功能性和可达性等常识要求。整个过程中，FirePlace充分利用了MLLM的语言理解能力与几何处理工具的精确性。
为验证FirePlace的有效性，研究团队在50个专家设计的逼真三维场景中进行了实验，创建了266个放置任务。通过与现有方法（如Holodeck和LayoutGPT）进行比较，FirePlace在多个评估指标上表现优越，包括最小L2误差、可见性得分和合理性得分。人类评估结果显示，FirePlace生成的放置方案在物理合理性、语义一致性和常识方面均优于其他基线方法，显示出其在复杂三维场景中进行对象放置的潜力和优势。这些实验结果表明，FirePlace不仅在几何约束方面表现出色，同时也能生成符合人类审美的三维布局。
FirePlace的工作原理可以简单理解为一个智能助手，它帮助用户将新物体放置到已有的三维场景中。首先，用户提供一个三维场景、要放置的物体和一个描述性指令，比如“把书放在书架上”。FirePlace会分析这个指令，找出书和书架的具体位置和形状。接下来，系统会生成一些规则，确保书放置在书架上是合理的，比如书的底部必须与书架的表面接触。然后，FirePlace会通过“批量选择”的方式，逐步缩小选择范围，找到最合适的放置位置。最后，它会检查这些位置是否符合我们的常识，比如书是否能被轻松拿到。通过这种方式，FirePlace不仅确保书放置得当，还能让整个场景看起来更自然和美观。
2025-03-05｜BUPT,USTC,NEU,ChinaMobile(Suzhou),NUS,THU,TeleAI,NWPU｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.04829v1
本文介绍了一种名为StickMotion的新型3D人类动作生成方法，该方法利用用户绘制的简笔画和文本描述来生成自然的人类动作序列。传统的文本到动作生成技术面临着准确捕捉用户想象中复杂动作的挑战，而StickMotion通过引入简笔画条件，显著提高了生成的灵活性和自然性。该方法从数据生成、多条件融合和动态监督三个方面解决了用户友好的简笔画带来的挑战。研究表明，StickMotion能够有效减少用户在生成动作时所需的时间，并且在多个条件组合下生成的动作序列具有更高的质量和一致性。
StickMotion的核心方法包括以下几个步骤：
简笔画生成算法：开发了一种StickmanGenerationAlgorithm（SGA），自动生成手绘的简笔画，以便在不同数据集格式中进行训练和测试。用户只需提供简笔画的大致位置，系统将自动调整其索引。
多条件模块（MCM）：该模块集成了文本描述和简笔画条件，通过扩散过程实现多条件的融合，减少了计算复杂性并提高了生成性能。MCM通过对输入数据的编码和解码，有效地处理了多种条件组合。
动态监督策略：StickMotion能够根据用户指定的简笔画位置动态调整生成的动作，确保生成的动作更自然、更符合用户的期望。该策略利用训练过程中生成的索引分数，优化了简笔画在动作序列中的位置。
在实验部分，研究者对StickMotion在KIT-ML和HumanML3D两个数据集上的性能进行了评估。通过与现有的文本到动作生成方法进行比较，StickMotion展示了在生成质量、准确性和用户满意度方面的优势。使用了多种评估指标，包括FID、RPrecision和StickmanSimilarity等，以量化生成动作的自然性和与用户意图的一致性。此外，进行了用户研究，评估了用户在生成动作时的时间成本和满意度，结果显示StickMotion能够显著减少生成时间，提升用户体验。
StickMotion的方法可以简单理解为三个步骤。首先，用户只需画出一个简单的简笔画，表示他们想要的动作，比如“一个人跳起来”。然后，系统会利用这个简笔画和用户的文字描述，结合多种条件，智能地生成出符合这些描述的动作。最后，StickMotion会根据用户的指示，自动调整简笔画在动作中的位置，确保生成的动作看起来更自然。这种方法不仅让用户更容易表达他们的想法，还能节省大量的时间，快速生成出他们想要的3D动作。
2025-03-04｜UAmsterdam,GeorgiaTech,NEU,LBNL｜⭐️

本研究探讨了舞蹈中伴侣之间的隐性交互，旨在揭示舞者在表演中如何相互影响。虽然舞蹈艺术家在伴舞时对彼此的动态有深刻的身体理解，但这些微妙的关系往往未被充分记录。为填补这一空白，研究团队采用图神经网络（GNN）技术，分析两位舞者之间的复杂联系。通过将舞蹈视频转换为三维姿态数据，研究者能够追踪舞者的身体关节运动，并利用这些数据来识别并解释舞者之间的相互作用。研究结果表明，图结构学习和自监督学习方法可以有效捕捉伴舞中潜在的交互模式，为舞蹈艺术家提供新的创作视角和实践策略。
本研究的方法主要包括以下几个步骤：
视频到3D姿态提取：首先，研究团队使用视频记录的舞蹈表演，通过专门的姿态提取系统将二维视频转换为三维身体姿态。这一过程确保了舞者运动的准确捕捉。
图神经网络（GNN）构建：将舞者的每个关节视为图中的节点，节点之间的连接则代表舞者之间的相互作用。GNN用于传播信息，捕捉舞者间的动态关系。
时间序列处理：采用递归神经网络（RNN）处理舞蹈的时间序列特性，以更好地捕捉舞者动作的演变。
自监督学习：由于标注舞者之间的影响关系存在主观性，研究者采用自监督学习的方法，通过优化序列重构和发现新模式，避免强制设定特定交互关系。
在实验部分，研究团队使用两种数据集进行模型评估：一个是已知交互图的粒子数据集，另一个是包含舞蹈表演的三维数据集。通过对模型的训练，研究者观察到在不同输入序列长度和边类型配置下，模型在重构精度上表现出色。具体而言，模型在重构舞者的运动轨迹时，能够有效捕捉到舞者之间的相互影响。实验结果显示，重构均方误差（MSE）在不同任务和模型配置下有显著的降低，表明模型能够在捕捉舞蹈动态方面取得良好的效果。此外，模型还展示了对舞者之间隐性连接的敏感性，为进一步理解舞蹈中的交互提供了有力支持。
在这项研究中，科学家们使用了一种名为图神经网络的技术，来揭示舞者在表演时是如何相互影响的。首先，他们将舞蹈视频转换成三维的身体姿态，这样可以更清晰地看到每个舞者的动作。接着，他们把舞者的每个身体关节看作一个网络中的节点，节点之间的连接代表舞者之间的互动。为了捕捉舞蹈的时间变化，他们还使用了一种叫做递归神经网络的技术。由于判断舞者之间影响的方式比较主观，研究者选择了一种自我学习的方法，允许模型在没有明确规则的情况下发现新的舞蹈模式。这种方法不仅帮助舞者理解彼此的动作，而且为他们提供了新的创作灵感。
