2025-03-05｜KAIST,summaryAI｜ICLR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.04835v1https ://github.com/aailab-kaist/DDiF
在深度学习模型的训练中，使用大规模数据集是提升性能的关键，但同时也带来了显著的计算和存储成本。为了解决这些问题，数据集蒸馏作为一种有效的解决方案应运而生，它通过将大规模数据集压缩为一个较小的合成数据集，保留了训练所需的关键信息。本文提出了一种新颖的参数化框架，称为“将数据集蒸馏到神经场”（DDiF），该框架利用神经场来存储大规模数据集的必要信息。神经场的独特性质使其能够以坐标为输入，输出相应的量，从而有效地保持信息并生成多种形状的数据。理论分析表明，DDiF在相同预算下比以往文献中提出的方法表现出更强的表达能力。通过广泛的实验，DDiF在多个基准数据集上表现出色，涵盖了图像、视频、音频和3D体素等多个领域。
DDiF框架的核心在于将真实数据集的信息存储在合成神经场中，适应有限的存储预算。该方法的主要组成部分包括坐标集和合成神经场。首先，坐标集定义了合成函数的输入空间，确保每个合成实例的结构与真实实例相同。然后，DDiF利用神经场作为合成实例的参数化工具，通过输入坐标集生成合成实例。神经场的优势在于其可以独立于数据维度进行编码，能够高效地处理高维数据。此外，DDiF还具有分辨率适应性，通过调整输入坐标集，可以生成不同分辨率的数据，从而满足下游任务的需求。最后，DDiF的优化目标是通过数据集蒸馏损失函数来最小化训练误差，从而实现高效的模型学习。
本文通过多项实验验证了DDiF方法的有效性，主要集中在高分辨率图像数据集上，例如ImageNet-Subset。实验结果显示，DDiF在所有实验设置中均表现出最佳性能，尤其是在预算有限的情况下，表现出显著的性能提升。此外，DDiF在不同网络架构上的一致性表现也得到了验证，显示出其优越的通用性。通过使用不同的蒸馏损失函数，DDiF在多种数据集上均实现了高准确率，证明了其广泛适应性。实验还涵盖了视频、音频和3D体素等多种模态，结果表明DDiF在这些领域也展现了卓越的性能，进一步支持了其作为数据集蒸馏方法的潜力。
DDiF方法的核心思想是将大量数据的关键信息压缩到一个小的数据集中，这样可以减少存储和计算的需求。具体来说，DDiF使用一种叫做“神经场”的技术，神经场就像一个能够根据坐标生成数据的函数。首先，DDiF会定义一个坐标集，这些坐标就像是地图上的点，指向数据的不同部分。然后，DDiF利用这些坐标来生成合成的数据实例。这个过程非常灵活，因为我们可以根据需要调整坐标来生成不同分辨率的数据，适应各种任务。最后，DDiF通过不断优化来减少模型的训练误差，从而确保生成的数据能够有效帮助模型学习。这种方法不仅提高了数据处理的效率，还能在多种应用场景中保持良好的表现。

本研究提出了一种名为Pi-GPS的新框架，旨在通过充分利用图示信息来增强几何问题求解的能力。近年来，几何问题求解（GPS）逐渐受到关注，尤其是在智能教育领域的潜在应用上。现有的研究多集中于文本解析，而忽视了图示所能提供的清晰信息。Pi-GPS的核心在于其微模块设计，包括一个用于消歧义的整流器和一个验证器。整流器利用多模态大型语言模型（MLLMs）从图示中提取信息，以消除文本中的模糊性，而验证器则确保输出结果符合几何规则，从而减少模型的错误生成。通过这种方式，Pi-GPS不仅提升了几何问题的求解效率，还在实验中显示出显著的性能提升，超越了现有的最先进模型。
Pi-GPS框架由两个主要组成部分构成：文本解析和推理。文本解析模块分为文本解析器和图示解析器。文本解析器通过正则表达式识别问题文本中的基本元素和数值关系，并生成一组命题。而图示解析器则使用PGDPNet模型高效提取几何图形中的基本元素及其逻辑关系，进而生成形式化命题。接下来，文本消歧义模块通过整流器和验证器来消除模糊性。整流器利用MLLMs分析文本和图示，识别不明确的点、形状和区域，并生成更精确的描述。验证器则应用图示启发式规则，确保整流器输出的一致性和正确性。最后，推理部分包括定理预测器，利用LLM生成适当的定理顺序，以便更有效地解决问题。
为了验证Pi-GPS的有效性，研究在Geometry3K和PGPS9K数据集上进行了实验。Geometry3K包含3,002道几何问题，涵盖多种几何形状，并附有图示和解析标注。实验结果显示，Pi-GPS在完成度和选择准确率上均显著优于其他基线模型，尤其在面积问题类型上表现突出，取得了近10%的性能提升。此外，研究还进行了消融实验，分析了文本消歧义模块和定理预测器对整体性能的影响，结果表明，整流器和验证器的结合显著提高了求解的准确性和可解释性。通过这些实验，Pi-GPS展示了其在几何问题求解中的强大潜力。
在Pi-GPS的工作中，我们首先需要理解几何问题的文本描述和图示。这个框架的核心在于两个部分：一个是用来解析文本和图示的信息，另一个是用来推理出答案。文本解析器负责从问题中找到关键的数字和关系，而图示解析器则从图形中提取出形状和点的信息。接下来，整流器会利用这些信息来消除文本中的模糊之处，比如哪些点和形状是重要的。验证器则像一个检查员，确保整流器的输出符合几何的规则。最后，推理部分会根据消歧义后的信息来决定使用哪些定理来求解问题。通过这种方式，Pi-GPS能够更准确地解决几何问题，让我们在面对复杂的数学问题时，能够更清晰地找到答案。

http ://arxiv.org/abs/2503.05255v1
在复杂的多图像理解任务中，现有的多模态慢思考方法表现出色，但在处理多图像场景时却面临显著的局限性。这种局限主要源于它们过度依赖基于文本的推理过程。人类在分析复杂多图像时，通常会进行两种互补的认知操作：一是通过区域匹配进行持续的跨图像视觉比较，二是动态记忆重要的视觉概念。为了解决这一挑战，我们提出了CMMCoT（复杂多模态链思维）框架，该框架模仿人类的“慢思考”过程，旨在提高多图像理解的性能。CMMCoT结合了两项关键创新：构建交错的多模态多步骤推理链，通过提取关键视觉区域标记作为监督信号，以及引入实时记忆增强模块，以在推理过程中扩展模型的推理能力。这些创新不仅提升了跨模态理解的全面性，还增强了模型的可解释性。
CMMCoT框架的核心在于其独特的多步骤推理机制，旨在增强复杂多图像任务的处理能力。具体方法包括：
交错的多模态推理链构建：该方法通过提取关键视觉区域标记来指导推理过程，确保模型在不同图像之间能够有效进行比较和分析。
实时记忆增强模块（RIFREM）：该模块在推理过程中存储视觉特征的关键-值对，以便在需要时快速检索，增强模型的推理能力。
多阶段训练策略：分为两阶段，第一阶段专注于多图像任务的训练，第二阶段则结合通用数据集，以减轻因多图像任务造成的灾难性遗忘。
多模态序列表示：通过特殊的图像索引标记，模型能够在推理过程中有效地处理和对比不同图像间的关联性。这些方法的结合，使得CMMCoT在处理复杂的多图像理解任务时，能够显著提升推理的准确性和效率。
为验证CMMCoT的有效性，我们在多个基准测试上进行了广泛的实验。首先，我们构建了一个名为CMMCoT-260K的新数据集，专门用于复杂的多图像慢思考任务，包含260,000个数据实例，涵盖了图像描述、共指消解、比较和推理等多种任务类型。实验结果显示，在多个多图像基准测试中，CMMCoT的表现超过了现有的最先进模型，尤其在NLVR2和Mantis-Eval等任务中，表现出了显著的优势。此外，我们还进行了单图像任务的对比实验，结果表明CMMCoT在这些任务中同样展现了卓越的性能。通过对不同模型的系统比较，我们不仅评估了CMMCoT的推理能力，还分析了其在多图像场景中的表现，进一步确认了其有效性和适用性。
CMMCoT方法的核心在于帮助计算机更好地理解多张图片之间的关系。简单来说，它就像是一个聪明的助手，能够在看多张图片时，记住每张图片中重要的细节，并将这些细节进行比较。方法分为几个步骤：首先，它会建立一个“思维链”，在这个链中，计算机会逐步分析每张图片，提取出关键的视觉信息。接着，CMMCoT会利用一个“记忆库”，在处理图片时快速找到之前提取的信息，确保不会遗漏任何重要的细节。最后，通过两阶段的训练，计算机不仅能在多图像任务中表现出色，还能在处理单张图片时也保持良好的表现。这样一来，CMMCoT就能更准确地回答复杂问题，比如“如果有风，哪张图片中的物体最容易倒下？”

在近年来，随着大规模视觉语言模型（LVLMs）的迅速发展，它们在图像与文本理解任务中展现出卓越的性能。然而，确保模型生成与人类偏好一致的能力仍然面临重大挑战，尤其是在缺乏多样化和高成本的人类标注数据的情况下。为了解决这一问题，本文提出了一种新的自我监督框架SHAPE（Self-ImprovedHolisticPreferenceAlignment），旨在通过自动构建竞争性偏好对来提高LVLM的偏好对齐性能。SHAPE的设计理念在于利用现有的文本-图像配对数据，生成更具整体性的偏好三元组，从而有效降低对人类标注的依赖。该框架通过迭代自我改进的方式，推动模型在生成文本时更好地与视觉输入相匹配，进而提升其在实际应用中的可靠性。
SHAPE的核心方法包括几个关键步骤。首先，通过对标准的监督文本-图像对进行转换，生成竞争性偏好对。具体而言，SHAPE引入多种视觉增强技术，生成多个图像的变体，并将这些变体与相应的文本总结相结合，形成“赢家”文本，而原始文本则被视为“输家”文本。其次，SHAPE利用偏好优化机制，通过比较生成的赢家和输家文本的质量，来引导模型的学习过程。偏好对齐的优化过程采用了基于Bradley-Terry模型的方式，确保赢家文本在整体质量上优于输家。最后，SHAPE通过自我监督的偏好优化，不断更新和改进模型，以实现更高效的偏好对齐，减少对人工标注的需求。
在实验部分，SHAPE在多个基准测试上进行了全面评估，包括MMVet、MMBench和POPE等。实验结果表明，SHAPE在不同模型架构和规模上均取得了显著的性能提升。例如，在MMVet基准测试中，SHAPE相较于基线模型提高了11.3%的得分，而在MMBench和POPE测试中也分别实现了1.4%和8.0%的提升。通过对比分析，SHAPE展示了其在偏好对齐、鲁棒性和任务特定能力方面的优势。此外，定性分析也表明，SHAPE生成的文本在视觉细节的关注度和与人类偏好的对齐程度上都有所增强。这些结果进一步验证了SHAPE在提升LVLM性能方面的有效性。
SHAPE的工作原理可以简单理解为一种智能学习机制。首先，它会从已有的图像和文本数据中提取信息，并利用这些信息生成多个图像的不同版本。接着，SHAPE会把这些新生成的图像与文本结合起来，形成“赢家”和“输家”文本。赢家文本是指那些质量更高、内容更丰富的文本，而输家文本则是原始的文本。通过这种方式，SHAPE可以自动学习哪些文本更符合人类的偏好，而不需要依赖人工标注。最后，SHAPE会不断迭代更新，使得模型在理解图像和生成文本方面越来越精准。这种方法不仅提高了模型的表现，还大大减少了人工干预的需要。

http ://arxiv.org/abs/2503.05584v1https ://github.com/libozhu03/QArtSR
QArtSR是一种新颖的图像超分辨率（SR）量化方法，旨在优化基于单步扩散的图像超分辨率模型（OSDSR）。尽管现有的OSDSR模型已在降噪步骤上实现了显著的简化，并且可以量化到8位以降低计算成本，但仍然有潜力进一步降低量化位数。QArtSR通过引入时步重训练量化（TRQ）和反向模块量化（RPQ）策略，探索了在超低位（如2位和4位）量化下的性能提升。该方法不仅显著减少了模型参数和计算操作，还在保持图像质量的同时加速了推理过程。实验结果表明，QArtSR在4位和2位量化设置下的表现优于现有主流方法，接近全精度模型的性能。
QArtSR的核心方法包括三个主要步骤：首先，进行时步重训练量化（TRQ），该步骤通过选择最佳的时步值来降低量化误差，确保模型在量化之前具有更好的初始状态。其次，实施反向模块量化（RPQ），其特点是将量化过程的顺序反向，使得模块在量化时能够更有效地进行联合优化。RPQ通过逐步引入量化模块，缓解了性能下降的问题，确保每个模块在量化过程中都得到充分更新。最后，进行扩展训练（ET），以确保所有模块在量化后得到充分微调，从而进一步提升模型性能。该方法充分考虑了图像和模块损失的共同优化，确保在超低位设置下仍能实现高质量的图像重建。
在实验中，研究团队构建了多个数据集，包括RealSR、Urban100和DIV2K，以测试QArtSR的性能。通过与多种量化方法（如MaxMin、LSQ、Q-DM等）进行比较，QArtSR在4位和2位设置下均展示了优越的性能。在4位设置中，QArtSR的表现接近全精度模型，且在多个评估指标上超过了其他方法。对于2位设置，尽管大多数模型表现不佳，QArtSR依然保持了较小的性能下降，展现出其在极限量化条件下的鲁棒性。此外，实验结果还表明，结合TRQ与RPQ的扩展训练显著提升了模型的整体表现，进一步验证了QArtSR方法的有效性。
QArtSR是一种新的图像处理技术，旨在提高低分辨率图像的质量。它通过一系列步骤来实现这一目标。首先，研究人员选择最佳的处理时机，以减少在图像转换过程中可能出现的错误。接着，他们采用了一种独特的方式来逐个处理图像的不同部分，这样可以更有效地优化每个部分的质量。最后，经过一段时间的额外训练，确保所有部分都能达到最佳状态。这样的处理方式使得即使在降低图像质量的情况下，QArtSR仍然能够生成清晰、细致的高分辨率图像。这种方法不仅提高了图像的视觉效果，还显著减少了计算成本和时间，使其在实际应用中更具优势。

本研究提出了一种新的视频表示方法，称为D2GV（Deformable2DGaussianSplatting），旨在提高视频内容的表示效率和质量。D2GV结合了可变形的二维高斯点云和差异化光栅化技术，能够以超过400帧每秒的速度进行高质量渲染，且在训练和推理过程中展现出优越的效率。相较于现有的隐式神经表示（INRs），D2GV不仅提高了可解释性和可扩展性，还增强了对下游任务的适应性，如视频插值、修复和去噪。通过将视频序列划分为固定长度的图像组（GoP），D2GV实现了并行训练，显著降低了计算复杂度。实验结果表明，D2GV在多个视频数据集上表现出色，能够在较短的训练时间内达到与最先进的INRs相媲美的效果。
D2GV的核心方法包括以下几个关键步骤：
视频分组：将视频序列分为固定长度的图像组（GoP），以实现并行训练和线性可扩展性。
二维高斯表示：每个GoP中的视频帧通过可变形的二维高斯点云进行表示，利用共享的标准空间进行动态变形，以适应不同的时间戳。
差异化光栅化：采用高效的CUDA光栅化技术，在渲染过程中将二维高斯转换为视频帧，确保快速且高质量的输出。
可学习的剪枝与量化：引入可学习的剪枝和量化策略，优化D2GV的内存占用和计算效率，使其在保持质量的同时，减少模型的存储需求。
在实验部分，D2GV在三个广泛使用的视频数据集上进行了评估，包括BigBuckBunny、UVG和DAVIS。实验结果表明，D2GV在训练时间和渲染速度上均优于当前主流的方法，尤其是在处理高动态场景时表现突出。通过与INRs和现有的高斯点云方法进行比较，D2GV在多个指标上取得了显著的优势，包括峰值信噪比（PSNR）和多尺度结构相似性指数（MS-SSIM）。此外，D2GV还展示了其在视频插值、修复和去噪等下游任务中的强大适应性，验证了其作为视频表示方法的有效性和实用性。
D2GV方法的核心在于如何高效地处理视频数据。首先，我们将视频分成小段，每段包含若干帧，这样可以同时处理多个小段，提高效率。接着，我们用二维高斯模型来表示每一帧，这种模型能灵活地变形，从而适应不同的时间点。通过一种叫做CUDA的技术，我们能快速地将这些高斯模型转换成清晰的图像。最后，为了让D2GV更节省空间，我们引入了一些智能的剪枝和量化方法，这样可以在不损失太多质量的情况下，减少存储和计算的需求。总之，D2GV通过这些步骤，不仅能快速生成高质量的视频内容，还能在其他视频处理任务中表现出色。

在视频检索领域，文本与视频的有效对应是一个重要的研究方向。当前的方法往往依赖于视频级别的描述，但这些描述无法充分捕捉视频中的丰富语义和时间变化。针对这一问题，本文提出了一个新框架——NarratingtheVideo（NarVid），它通过全面利用帧级描述（即叙述）来提升文本与视频的检索效果。NarVid框架的核心在于充分挖掘帧级描述中蕴含的时间变化信息，并通过多种方式增强特征提取和匹配的精度。通过对比实验，NarVid在多个基准数据集上表现出色，超越了现有的最先进模型，展示了其在文本视频检索中的巨大潜力。
NarVid框架的设计围绕四个主要模块展开，以优化文本视频检索的效果。首先，通过生成帧级描述（叙述），为每个视频帧提供详细的语义信息。其次，采用跨模态交互技术，利用视频帧与其对应的描述进行特征增强。这一过程通过共注意力机制实现，能够有效捕捉到帧与描述之间的细微关系。第三，实施查询感知自适应过滤，以剔除与查询不相关或错误的信息，确保检索的准确性。最后，通过定义新的相似性度量和硬负样本损失，提升模型的判别能力。这些方法的结合使得NarVid能够在复杂的检索任务中充分发挥帧级描述的优势。
在实验部分，NarVid框架在四个广泛使用的基准数据集上进行了评估，包括MSR-VTT、MSVD、VATEX和DiDeMo。通过对比实验，NarVid在各项检索指标上均表现优异，尤其是在R@1、R@5和R@10等指标上，展示了其在文本视频检索中的有效性。实验结果表明，利用帧级描述的叙述显著提高了检索性能，尤其是在处理具有丰富时间变化的视频时。此外，通过消融实验，进一步验证了各个模块对整体性能的贡献，强调了跨模态交互和查询感知过滤的重要性。这些实验结果不仅证明了NarVid的有效性，也为未来的研究提供了新的思路。
NarVid的工作原理可以简单理解为一个视频检索的“翻译器”。首先，它会为每个视频的每一帧生成一个详细的描述，就像为每一张图片写一个小故事。接着，NarVid会比较这个故事和用户输入的查询，比如“两个男人在打招呼”，然后找出视频中最相关的片段。为了确保找到的片段是准确的，NarVid会过滤掉那些不相关或错误的信息，就像在筛选合适的答案。最后，它还会通过一些特别的算法，确保找到的结果是最具代表性的。这种方法使得NarVid在处理复杂视频时，能够更好地理解和匹配用户的需求，从而提升检索的准确性和效率。
