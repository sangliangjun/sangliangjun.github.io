2025-03-10｜DuXiaoman｜⭐️🟡
http ://arxiv.org/abs/2503.07493v1https ://github.com/zhangguiwei610/V2Flow
V2Flow是一种新颖的视觉标记化器，旨在将视觉信息与大型语言模型（LLMs）的词汇空间统一起来，以实现自回归图像生成。当前的自回归模型在处理视觉生成时面临着结构和潜在分布的不一致性，V2Flow通过将视觉数据压缩为离散标记，确保与LLMs的词汇结构对齐，从而克服这些挑战。该方法通过将视觉标记化视为流匹配问题，学习从标准正态分布到连续图像分布的映射，这种映射基于嵌入在LLMs词汇空间中的标记序列。V2Flow的有效性主要来源于两个核心设计：视觉词汇重采样器和掩码自回归修正流解码器，这两者的结合使得高保真图像重构和自回归视觉生成成为可能。
V2Flow的设计包括两个关键部分。首先，视觉词汇重采样器负责将输入图像压缩为紧凑的一维标记序列，每个标记通过Gumbel-softmax技术映射到LLMs的词汇空间。该过程确保了视觉标记的结构和潜在分布与LLMs的词汇一致。其次，掩码自回归修正流解码器使用掩码变换器架构来增强标记的上下文信息，从而提高重构质量。在解码过程中，输入图像被划分为多个非重叠的补丁，并随机选择一部分进行重构。通过逐步重建掩码补丁，解码器能够生成高保真的图像，同时保持与文本信息的语义一致性。此外，采用自回归“下一个标记预测”策略，使得模型在不同序列长度下依然保持高效重构性能。
在实验部分，V2Flow在ImageNet-1K数据集上进行了全面评估，采用256×256和512×512两种分辨率进行图像重构。实验结果表明，V2Flow在多个关键指标上超越了现有的主流VQ基础标记化器，显示出其在图像重构中的高效性和准确性。具体而言，V2Flow在PSNR和SSIM等评估指标上均表现出色，证明了其在保持视觉细节方面的优越性。此外，通过对不同标记长度的重构性能进行对比，V2Flow展现了其在标记化灵活性上的优势，能够在较少的标记数量下实现更高的重构质量。这些实验结果验证了V2Flow在自回归视觉生成任务中的有效性和潜力。
V2Flow的工作原理可以简单理解为将图像转换为一串数字标记，这些数字可以被大型语言模型理解和处理。首先，V2Flow将图像压缩成简短的数字序列，每个数字代表图像的一部分，这样做的目的是使这些数字能够与语言模型所使用的词汇相匹配。接着，V2Flow使用一种特殊的解码器来逐步重建图像。在这个过程中，模型会根据已有的信息预测下一部分图像，确保生成的图像不仅清晰，而且与输入的文字描述相符。通过这种方式，V2Flow实现了高质量的图像生成，让计算机能够更好地理解和创造视觉内容。
2025-03-10｜CAS-SIA,UCAS,HKU｜⭐️🟡
http ://arxiv.org/abs/2503.07334v1
本文提出了一种新的训练框架——自回归表示对齐（ARRA），旨在提升自回归大语言模型（LLMs）在文本到图像生成任务中的表现。不同于以往需要复杂架构修改的方法，ARRA通过重新设计训练目标，结合外部视觉表示，解决了跨模态生成中的全局一致性问题。该框架通过引入混合标记（），在局部和全局层面上施加约束，使得模型能够在保持自回归原有架构的同时，学习到空间和上下文的一致性。实验结果表明，ARRA在多个数据集上实现了显著的FID（FréchetInceptionDistance）降低，展现出其在医学成像和自然场景合成等领域的强大适应性。
ARRA框架的核心在于通过引入混合标记，实现局部和全局的约束学习。具体方法如下：
混合标记定义：不仅依赖于传统的自回归损失（ARloss），还结合了全局视觉对齐损失（GVAloss），使得模型在生成过程中能够同时考虑局部和全局信息。
视觉表示提取：利用预训练的视觉基础模型（如BioMedCLIP）提取图像的全局视觉表示，并将其与自回归模型的隐藏状态进行对齐。
损失函数设计：引入复合损失函数，将自回归损失与全局视觉对齐损失结合，以优化模型在生成图像时的表现。
训练过程：在训练阶段，模型通过对的优化，逐步学习到更丰富的空间关系和上下文信息，而在推理阶段则保持原有的自回归生成机制。
实验部分通过多个数据集验证了ARRA的有效性和灵活性。具体包括：
数据集选择：使用MIMIC-CXR、DeepEyeNet和ImageNet等数据集进行训练和评估，涵盖医学成像和自然图像生成任务。
性能评估指标：采用FID、MS-SSIM和CLIP-Score等定量指标，全面评估生成图像的质量和语义一致性。
对比实验：与其他先进的生成模型（如StableDiffusion和LLM-CXR）进行对比，结果显示ARRA在FID和CLIP-Score上均有显著提升，特别是在医学图像生成任务中表现突出。
组件分析：通过对不同组件的影响进行分析，探讨视觉编码器的选择、对齐机制及特征聚合策略对生成性能的影响，进一步验证了ARRA框架的有效性。
ARRA框架的工作方式可以简单理解为在生成图像的过程中，模型不仅关注当前生成的单个部分，还会参考整体的视觉信息。具体来说，ARRA使用了一种叫做混合标记的方式，这个标记可以同时考虑当前生成的内容和整体的视觉效果。想象一下，当你画一幅画时，不仅要关注每一笔的细节，还要考虑整幅画的和谐与美感。ARRA通过这种方式，让模型在生成图像时，能更好地理解各个部分之间的关系，从而生成出更连贯、更加真实的图像。此外，ARRA还利用外部的视觉信息来指导模型的学习，使得生成的图像在视觉上更加一致和自然。
2025-03-10｜NWPU,TeleAI,BUAA｜⭐️🟡
http ://arxiv.org/abs/2503.07076v1
Next-FrequencyImageGeneration(NFIG)是一个新颖的自回归图像生成框架，旨在解决传统自回归模型在图像生成任务中遇到的长距离依赖捕捉和计算成本高等问题。该方法通过将图像生成过程分解为多个频率引导的阶段，首先生成低频成分以建立全局结构，然后逐步添加高频细节，从而遵循自然图像的频谱层次结构。NFIG通过更好地捕捉图像成分之间的因果关系，显著提高了生成图像的质量，同时在推理过程中显著减少了计算开销。实验结果表明，NFIG在ImageNet-256基准测试上达到了最先进的性能，FID分数为2.81，并且在计算效率上相较于VAR-d20提升了1.25倍。
NFIG的核心在于其频率引导的自回归生成过程，主要包括以下几个步骤：
频率引导的残差量化变分自编码器（FR-VAE）：该方法通过快速傅里叶变换（FFT）将图像分解为不同频率成分，低频成分捕捉全局结构，而高频成分则保留细节。
频率引导的重建：使用FR-VAE，低频成分以较少的图像标记生成，随后在此基础上逐步添加高频成分，从而实现图像的逐步生成。
自回归生成：采用解码器仅的变换器框架，通过自适应层归一化和块状因果注意力机制，逐步生成低频到高频的图像成分，形成一个“粗到细”的生成策略。
向量量化：通过简单的向量量化方法，将连续特征转换为离散标记，以便于生成图像。该方法有效减少了信息损失，并提高了生成质量。
在实验部分，研究团队使用了ImageNet数据集的ILSVRC2012子集，进行了全面的对比实验。评估指标包括FID、InceptionScore、精确度和召回率。NFIG与多种最先进的图像生成模型进行了比较，包括GAN、扩散模型和其他自回归模型。结果显示，NFIG在生成质量和计算效率上表现优异，FID分数为2.81，InceptionScore达到332.42，显著优于其他自回归模型。实验还包括消融研究，验证了频率引导残差量化和DINO判别器在提升生成质量方面的重要性。此外，NFIG还在推理速度上表现出色，相较于VAR-d20提升了25%的速度。
NFIG的方法可以简单理解为分阶段的图像生成过程。首先，系统会先生成图像的基础部分，也就是低频成分，这部分主要决定图像的大致形状和结构。接下来，系统会逐步添加更细致的高频成分，比如纹理和细节。这个过程就像是在画一幅画，首先描绘出轮廓，然后再逐渐添加颜色和细节。为了让这个过程更高效，NFIG采用了一种聪明的方式来处理信息，通过频率分析来指导生成，使得每一步都能更好地捕捉到图像的特征。这种方法不仅提升了生成图像的质量，还让整个生成过程变得更快、更省力。
2025-03-09｜SYSU,HuaweiNoah’sArkLab｜⭐️🟡
http ://arxiv.org/abs/2503.06764v1
本文提出了一种名为SemHiTok的统一图像标记器，利用语义引导的层次化代码簿，为多模态理解和生成任务提供一致的离散特征表示。近年来，统一多模态大模型（MLLMs）的研究逐渐兴起，然而现有的方法在同时提取语义特征和纹理特征方面面临挑战，尤其是在特征优先级不同的情况下。SemHiTok通过构建基于预训练语义代码簿的纹理子代码簿，有效地解决了这一问题。该设计将语义重建和像素重建的训练解耦，使得标记器在提取低级纹理特征的同时，不降低高层语义特征的提取能力。实验结果表明，SemHiTok在256×256分辨率下的rFID得分达到了最先进的水平，并在多模态理解和生成任务中展现出竞争力的表现。
SemHiTok的核心是建立一个统一的图像标记器，能够同时处理理解和生成任务。其方法主要包括以下几个步骤：
语义优先代码簿（SPC）训练：使用冻结的文本对齐图像编码器（如CLIP）提取连续的语义特征，并将其量化为离散特征集。
纹理特征提取：通过像素编码器提取图像的连续纹理特征，随后根据SPC的量化结果选择相应的纹理子代码簿进行量化。
特征融合：将量化后的语义特征和纹理特征进行拼接，为后续的像素解码器提供输入，重建原始图像。
训练过程：采用端到端的训练方式，通过最小化重建损失、量化损失以及感知损失来优化模型。
通过这种方法，SemHiTok能够有效地整合语义和纹理特征，同时保持理解能力，避免了传统联合优化方法中的性能下降问题。
在实验中，SemHiTok的性能在多个多模态任务中得到了验证。首先，在统一图像标记器的重建性能上，SemHiTok在ImageNet-50k验证集上表现出色，rFID得分为1.10，优于大多数像素专家模型。其次，在多模态理解任务中，SemHiTok利用离散特征在LLaVA-1.5设置下超越了其他基于下一个标记的模型，显示出其在理解任务中的有效性。此外，在文本到图像生成任务中，SemHiTok在GenEval和MJHQ30K数据集上均展现出强大的生成能力，生成的图像在细节和现实感上都表现优异。通过多轮实验，SemHiTok不仅在重建和生成性能上达到了最先进的水平，还在理解任务中保持了良好的性能表现。
在SemHiTok的开发中，我们采用了一种简单而有效的方法来处理图像信息。首先，我们使用一种叫做“语义优先代码簿”的工具来提取图像中的重要信息，这相当于给图像打标签，帮助我们理解它的内容。接着，我们还会提取图像的细节，比如纹理和颜色等，这就像是为图像添加更多的视觉元素。然后，我们将这些信息结合在一起，形成一个完整的图像描述。最后，我们通过不断训练这个系统，使其能够更好地理解和生成图像。在这个过程中，我们确保了系统在理解图像的内容时不会丢失细节，这样它不仅能生成漂亮的图像，还能准确地反映出图像的真实含义。
2025-03-08｜AlibabaGroup｜⭐️🟡
http ://arxiv.org/abs/2503.06132v1https ://github.com/cxxgtxy/USP
在过去十年中，预训练-微调的范式在图像识别领域取得了显著成功。本文提出了一种名为统一自监督预训练（USP）的新框架，该框架旨在解决图像理解和生成之间的关系。通过将自监督学习与扩散模型相结合，USP能够有效地提升生成模型的收敛速度和生成质量，同时在理解任务上也表现出色。传统的预训练方法在处理大规模数据集时面临挑战，尤其是当视觉模型和扩散模型之间存在输入不匹配时。USP通过在变分自编码器（VAE）的潜在空间中进行掩蔽潜在建模，克服了这些限制，从而实现了更高效的特征学习和更快的训练过程。
USP框架的核心在于其两步策略。首先，利用预训练的VAE对输入图像进行编码，将其映射到潜在空间。然后，在该潜在空间中执行掩蔽潜在建模，以提高特征表示的质量。具体方法包括：
输入匹配：通过对图像进行降采样，并使用VAE处理潜在特征，解决图像识别模型和扩散模型之间的输入不匹配问题。
模型架构调整：采用修改后的视觉变换器（ViT）架构，以确保与预训练模型的兼容性，同时进行必要的归一化调整。
掩蔽特征建模：在预训练阶段，随机掩蔽部分潜在特征，使用未掩蔽部分进行重建，促进模型学习鲁棒的特征表示。
权重初始化：将预训练模型的权重用于下游任务，避免额外的训练开销和内存消耗。
为了验证USP的有效性，本文在多个扩散模型上进行了实验。实验结果显示，USP在不同规模的DiT模型上均显著提升了性能。例如，使用DiT-B/2模型进行预训练后，在仅400K步训练中，FID（FrechetInceptionDistance）得分从42.62降至28.26，显示出优越的生成质量。此外，USP在SiT框架下也表现出色，证明了其在加速收敛和提高生成质量方面的有效性。通过与其他加速方法进行比较，USP在计算资源的消耗上具有明显优势，且在多个下游任务中均实现了优异的性能。
USP的方法可以简单理解为一种“学习方法”。首先，我们把图像转化为一种更简单的形式，叫做“潜在特征”，就像把复杂的事情简化为几个关键点。接着，我们会随机遮住一些特征，训练模型只用剩下的部分去猜测被遮住的内容，这样可以让模型学会更强的识别能力。然后，我们把学习到的知识直接应用到其他任务上，比如分类或生成新图像，而不需要重新训练。这样做的好处是节省了很多时间和资源，同时提高了模型的表现。通过这种方法，USP能够有效地将图像理解和生成结合起来，达到更高的效率和效果。
2025-03-10｜RUC,AntGroup｜⭐️🟡
http ://arxiv.org/abs/2503.07197v1
本文提出了一种名为eMIGM的高效且有效的掩蔽图像生成模型，旨在统一掩蔽图像生成模型和掩蔽扩散模型的设计框架。研究表明，尽管掩蔽扩散模型在文本生成领域表现出色，但在图像生成方面的应用仍有待探索。eMIGM通过系统性地分析训练和采样策略，识别出影响模型性能和效率的关键因素。实验结果显示，eMIGM在ImageNet数据集上表现优异，尤其是在256×256和512×512分辨率下，其生成质量在相似的模型参数和功能评估数量下超越了现有的最先进模型VAR。随着模型参数和功能评估数量的增加，eMIGM的性能逐渐接近最先进的扩散模型，展示了其在图像生成任务中的强大潜力。
eMIGM的设计方法包括以下几个关键步骤：首先，统一框架整合了掩蔽图像生成和掩蔽扩散模型的优势，允许对训练和采样策略进行系统探索。其次，在训练过程中，研究发现图像的高冗余性使得掩蔽比率的提高能够显著提高生成质量。为此，采用了一种简单的加权函数，灵感来源于MaskGIT和MAE。此外，引入了无条件生成的CFG与掩蔽策略，通过替换假类标记为掩蔽标记来进一步增强性能。在采样阶段，通过在早期阶段预测较少的标记来改善结果，同时采用时间间隔策略对无条件引导进行调整，以降低早期引导的方差，保持生成质量。最后，eMIGM在训练和采样效率上表现出色，能够在减少计算成本的同时，生成高质量的图像。
在实验部分，eMIGM在ImageNet256×256和512×512上进行了评估，使用FrechetInceptionDistance(FID)作为主要指标。实验结果表明，eMIGM在功能评估数量显著减少的情况下，仍能实现与最先进扩散模型相媲美的性能。例如，eMIGM-L在仅使用60%的功能评估数量下，达到了FID为1.77的结果，超越了其他需要更多计算资源的模型。通过对不同模型规模的比较，发现随着模型规模的增大，eMIGM表现出更高的训练和采样效率，能够在相似的计算成本下生成更高质量的样本。最终，eMIGM在多个实验中均表现出优越的生成质量和高效的采样速度，验证了其在掩蔽图像生成领域的有效性。
eMIGM的工作原理可以简单理解为一种新型的图像生成方法，它通过掩蔽部分图像信息来训练模型。首先，模型会隐藏一些图像的部分内容，然后根据剩余的部分来预测被隐藏的内容。为了提高生成的图像质量，eMIGM采用了一种智能的训练方式，允许模型在训练时使用更少的计算资源而仍然保持高效。具体来说，模型在生成图像的早期阶段会先预测少量的内容，这样可以减少错误的积累。为了进一步提升效果，eMIGM还引入了一种新的引导策略，确保在生成过程中始终保持高质量的输出。总之，eMIGM不仅能快速生成高质量的图像，还能在计算上更加节省资源，非常适合需要快速处理图像的应用场景。
