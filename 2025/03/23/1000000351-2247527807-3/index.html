2025-03-10｜TiamatAI,ShanghaiTech,NUS,LiblibAI｜⭐️⭐️
http ://arxiv.org/abs/2503.07027v1https ://github.com/Xiaojiu-z/EasyControl
在图像生成领域，基于扩散模型的系统经历了显著的架构演变，尤其是从UNet向Transformer架构的转变。尽管现有的控制机制在空间和主体控制方面取得了进展，DiffusionTransformer（DiT）架构仍面临着高效和灵活控制的挑战。为了解决这些问题，本文提出了EasyControl，一个新颖的框架，旨在统一条件引导的扩散Transformer，提供高效性和灵活性。EasyControl通过三个核心创新实现了显著的性能提升：轻量级的条件注入LoRA模块、位置感知训练范式和因果注意力机制结合KV缓存技术。这些创新不仅提高了图像生成的速度和质量，还支持多条件生成，展示了卓越的零-shot多条件泛化能力。
EasyControl的框架包含多个关键组成部分，首先是条件注入LoRA模块。该模块在不修改基础模型权重的情况下，将条件信号有效地注入到模型中。其次，提出了位置感知训练范式，通过将输入条件标准化为固定分辨率，优化了计算效率，支持生成任意长宽比和灵活分辨率的图像。最后，EasyControl实现了因果注意力机制，并结合KV缓存技术，显著降低了推理延迟。具体而言，条件信号通过独立的条件分支注入，确保了与文本和噪声分支的无缝兼容，同时通过低秩适应（LoRA）增强条件表示。这种设计使得模型能够灵活地集成多种条件信息，保持了生成图像的高保真度。
在实验部分，研究团队对EasyControl进行了全面的评估，包括定量和定性分析。首先，使用多种条件生成任务评估了模型的控制能力、生成质量和文本一致性。实验结果表明，EasyControl在单条件和多条件生成任务中均表现出色，尤其在生成时间和模型参数数量上显示出显著优势。通过与ControlNet、OmniControl等基线方法进行比较，EasyControl在多个指标上均优于对手，尤其是在生成图像的质量和一致性方面。此外，消融实验进一步验证了各个模块对整体性能的贡献，强调了条件注入LoRA模块和位置感知训练范式在提高效率和灵活性方面的关键作用。
EasyControl的工作原理可以简单理解为一个智能助手，它帮助图像生成模型更高效地处理各种输入条件。首先，它使用一个轻巧的模块来接收和处理这些条件，就像一个翻译器，把不同的指令转换成模型可以理解的语言。接着，EasyControl确保这些指令在训练时都被标准化，这样模型就能在不同的图像尺寸和比例下灵活工作。最后，它利用一种聪明的注意力机制来节省计算资源，使生成图像的速度更快。通过这些方法，EasyControl不仅能生成高质量的图像，还能在处理多种条件时保持一致性，就像一个优秀的多任务工作者。
2025-03-09｜FDU,TencentYou,HKUST,WesternU,UCAS｜⭐️⭐️
http ://arxiv.org/abs/2503.06684v1https ://hithqd.github.io/projects/PixelPonder/
在文本到图像生成领域，PixelPonder提出了一种创新的框架，旨在有效处理多种视觉条件下的图像生成问题。传统的控制方法在处理多条件输入时，往往面临着信息冲突和结构失真等挑战，导致生成图像的质量和一致性不足。PixelPonder通过引入一种动态补丁适应机制，能够根据不同的空间区域优先选择相关的控制信号，从而在保持图像的语义一致性的同时，提升视觉质量。此外，时间感知的控制注入策略被引入，以在去噪过程中逐步调整条件的影响，确保从结构保留到纹理细化的平滑过渡。通过一系列实验，PixelPonder在多个基准数据集上均表现出优于现有方法的生成效果，证明了其在多条件图像生成中的有效性和优越性。
PixelPonder的核心方法包括两个主要组件：补丁级适应条件选择机制和时间感知控制注入策略。具体而言：
补丁适应模块（PAM）：该模块通过对不同视觉条件下的图像进行补丁级别的重组，动态调整每个补丁的控制条件。PAM在每个时间步骤初始化一个统一的控制信号，并通过迭代更新过程选择最相关的补丁，确保生成图像的局部特征得到精确指导。
时间感知控制注入：为了充分利用不同时间步骤下视觉条件的控制效果，该策略在每个时间步骤引入时间信息，指导补丁选择的倾向。这种方式确保了在整个去噪过程中，补丁选择的一致性和有效性，从而提升了生成图像的质量。
在实验部分，PixelPonder在MultiGen-20M和Subject-200K数据集上进行了广泛的验证，评估其在多条件图像生成中的表现。实验结果显示，PixelPonder在控制能力和生成质量上均显著优于其他先进方法，如Uni-ControlNet和T2I-Adapter。具体而言，PixelPonder在结构相似性指数（SSIM）和生成图像的FID分数上都取得了显著的改善，证明了其在处理复杂视觉条件时的有效性。同时，实验还探讨了不同视觉条件组合对生成结果的影响，结果表明，PixelPonder能够在多条件设置下保持高一致性和优质的视觉效果。
PixelPonder的工作原理可以理解为一种智能的图像生成助手。它不仅仅是根据文本描述生成图像，还能根据多种视觉信息（如深度图、轮廓图等）进行细致的调整。首先，它会将这些视觉信息分成小块（补丁），然后根据每个补丁的特点来选择最相关的信息，这样就能在保留整体结构的同时，增强细节。其次，PixelPonder还会随着生成过程的推进，逐步调整这些信息的影响，确保最终生成的图像在外观上既美观又符合描述。通过这种方式，PixelPonder能够生成更高质量、更具一致性的图像，帮助用户实现他们的创意想法。
2025-03-09｜HKUST,NUS,HuaweiNoah’sArkLab｜⭐️🟡
http ://arxiv.org/abs/2503.06652v1
本研究提出了一种新颖的方法——联合分布匹配（JDM），旨在高效地将额外控制信号整合到一步扩散生成模型中。传统的扩散模型在生成过程中通常需要多次网络函数评估（NFE），这导致了计算效率低下。通过最小化图像条件联合分布之间的逆Kullback-Leibler（KL）散度，JDM能够在保持高生成质量的同时，显著减少所需的计算资源。实验结果表明，JDM在许多情况下超越了现有的多步控制方法，展现出在单步文本到图像合成中的优越性能。该方法不仅提升了生成图像的质量，还改善了模型对新控制信号的适应能力，展现了在实际应用中的广泛潜力。
JDM方法的核心在于通过以下几个步骤实现高效的控制信号整合：
逆KL散度最小化：JDM通过最小化图像条件联合分布的逆KL散度，解耦了生成图像的保真度学习与条件学习，从而使得生成模型能够处理教师模型未设计的控制信号。
非对称蒸馏策略：该方法采用非对称的蒸馏策略，使得一步生成模型能够在已知条件下学习新控制信号的能力，增加了生成的灵活性。
人类反馈学习的整合：通过将人类反馈学习(HFL)纳入训练过程，JDM不仅提高了生成质量，还增强了模型对用户偏好的适应能力。
条件生成模型的建立：通过引入控制网络（ControlNet），实现了对生成模型的条件化，使得生成的图像更符合特定的控制条件。
在实验部分，研究团队通过多种控制条件（如Canny边缘检测、HED、深度图等）对比了JDM与现有方法的性能。所有模型均在内部收集的数据集上进行训练，使用FID（FréchetInceptionDistance）和一致性度量指标评估生成图像的质量与控制信号的遵循度。实验结果显示，JDM在生成效率上显著优于传统的多步控制模型，从50NFE降至1NFE，同时在多个控制任务中保持或提升了FID得分和一致性度量。此外，JDM在生成高质量图像的同时，能够更好地遵循控制条件，展现了其在实际应用中的有效性和潜力。
我们的方法可以简单理解为一种新型的图像生成技术，旨在让计算机生成的图像更符合人们的需求。首先，我们通过一种叫做“联合分布匹配”的方法，帮助计算机更好地理解图像与其相关信息之间的关系。接下来，我们采用了一种特别的学习方式，让计算机不仅能生成图像，还能在生成过程中遵循特定的指令，比如边缘检测或深度信息。我们还引入了人类的反馈，确保生成的图像更符合用户的喜好。最后，我们通过实验证明，这种方法比传统的生成方式更快且质量更高，能够在更少的计算时间内生成出更好的图像。
2025-03-10｜AlibabaGroup｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.07026v1
本研究提出了一种新颖的图像修复技术，称为EraseDiffusion（EraDiff），旨在高效地移除图像中的目标物体，同时保持周围内容的视觉一致性。尽管现有的基于扩散的方法在图像修复领域取得了显著进展，但在物体去除的过程中，仍然存在生成意外物体或伪影的问题。EraDiff通过引入链式修正优化（CRO）和自我修正注意机制（SRA），重新设计了扩散路径，以更好地符合物体去除的目标。CRO建立了一种新型的扩散过渡路径，模拟物体逐渐消失的过程，从而提高了去除效果的准确性和一致性。通过这些创新，EraDiff在OpenImagesV5数据集上实现了最先进的性能，尤其在复杂的真实场景中表现出显著优势。
EraDiff的方法论主要由两个关键部分组成：链式修正优化（CRO）和自我修正注意机制（SRA）。具体而言，CRO通过以下步骤实现优化：
动态图像合成：生成一系列动态图像，以模拟物体逐渐消失的过程，帮助模型理解从物体到背景的平滑过渡。
新型扩散路径：建立从噪声到背景的新扩散路径，以确保去除目标物体的过程中不会生成意外元素。
优化目标：设计新的损失函数，最小化模型预测的潜在状态与真实状态之间的距离，从而引导模型逐步调整其参数，以适应新的分布变化。
SRA机制则通过调整自我注意的激活，增强背景特征，减少伪影的影响，确保生成内容的连贯性和一致性。通过这两种机制的结合，EraDiff在去除物体的同时，能有效维持图像的整体美观。
在实验部分，我们对EraDiff进行了全面的评估，使用了公开的OpenImagesV5数据集。实验设置包括与多种基线模型的比较，如SD2-Inpaint和LaMa等。我们采用了两种主要的评估指标：FrechetInceptionDistance（FID）和LearnedPerceptualImagePatchSimilarity（LPIPS），以衡量生成图像的视觉一致性和美学质量。此外，我们还引入了LocalFID指标，以更细致地评估局部视觉保真度。实验结果表明，EraDiff在去除目标物体方面显著优于其他模型，尤其是在处理复杂场景时，表现出更高的视觉一致性和更低的伪影生成。此外，用户研究也显示，EraDiff在图像去除和美学评估中得分较高，进一步验证了其有效性和实用性。
EraDiff的工作原理可以简单理解为两个主要部分：链式修正优化（CRO）和自我修正注意机制（SRA）。首先，CRO通过创建一系列动态图像，模拟物体逐渐消失的过程，帮助模型更好地理解如何从物体过渡到背景。其次，SRA则是通过调整模型关注点，确保它更关注背景而不是被去除的物体，从而减少生成伪影的风险。这种方法使得EraDiff能够在去除物体的同时，保持图像的整体美观和一致性，最终生成更加自然的修复效果。
2025-03-10｜JasperResearch｜⭐️🟡
http ://arxiv.org/abs/2503.07535v1https ://github.com/gojasper/LBM
本文提出了一种新颖的图像到图像转换方法，称为潜在桥匹配（LatentBridgeMatching，LBM），旨在高效地将图像从源域映射到目标域。该方法通过在潜在空间中进行桥匹配，能够在一次推断步骤中实现快速的图像转换。研究表明，LBM在多种图像翻译任务中均能达到或超过最先进的性能，特别是在对象移除、深度和表面估计以及对象重光照等任务中表现优异。此外，LBM还引入了条件框架，能够针对可控的图像重光照和阴影生成进行有效处理。研究团队提供了LBM的开源实现，以便其他研究者使用和验证其效果。
LBM的核心思想是通过桥匹配在潜在空间中寻找源和目标分布之间的传输映射。具体方法步骤如下：
桥匹配：给定源分布和目标分布，构建一个随机插值模型，利用布朗运动的条件分布来实现样本间的平滑过渡。
潜在空间嵌入：使用预训练的变分自编码器（VAE）将源图像和目标图像的样本嵌入到潜在空间中，以降低高维数据处理的复杂性。
训练过程：通过对样本对进行编码，利用神经网络学习潜在空间中的漂移函数，并通过标准的随机微分方程（SDE）求解生成样本。
条件设置：在特定任务中引入额外的条件变量（如光照图），以指导生成过程，提高生成结果的可控性。
为了验证LBM的有效性，研究团队在六个不同的图像到图像任务上进行了广泛的实验，包括对象移除、深度和表面估计、对象重光照和阴影生成等。实验使用了RORD数据集，包含约52,000对有对象和无对象的图像。通过与LAMA、SDXL-Inpainting等现有方法进行比较，LBM在多个评估指标（如FID、SSIM、PSNR等）上均表现出色。实验结果显示，LBM在使用单次神经函数评估时，依然能够超越其他方法，证明了其高效性和强大的性能。此外，研究团队还进行了消融研究，分析了不同组件对模型性能的影响，进一步验证了LBM的有效性。
潜在桥匹配（LBM）是一种新方法，用于快速转换图像。想象一下，我们有两幅图像，一幅是源图像，另一幅是目标图像。LBM的工作原理就像在这两幅图像之间架起一座桥，帮助我们从源图像“走”到目标图像。首先，LBM会把这两幅图像转化为一种简化的表示方式，称为潜在空间，这样可以更容易地处理。接着，LBM通过学习如何在这个潜在空间中移动，找到从源图像到目标图像的最佳路径。这个过程就像在地图上寻找最短的行车路线。最后，LBM还可以根据需要调整图像的某些特征，比如光照和阴影，让生成的图像看起来更自然。这样，LBM不仅快速，而且灵活，能够满足各种图像转换的需求。
2025-03-09｜BIT｜⭐️🟡
http ://arxiv.org/abs/2503.06419v1
本文提出了一种新颖的图像布局编辑方法，旨在解决当前图像编辑技术在处理真实图像布局时的局限性。尽管大规模的文本到图像扩散模型在图像生成和编辑方面取得了显著成功，但现有方法在调整真实图像的布局时仍面临挑战。为此，本文的方法不仅能够根据用户指定的目标布局重新排列图像，还能保持编辑前后物体的视觉一致性。具体而言，提出的框架包括两个主要组件：多概念学习方案和基于扩散模型中间特征的语义一致性利用。这种方法通过直接将物体的外观信息投影到目标区域，确保了在布局调整过程中物体外观的一致性。大量实验表明，该方法在布局对齐和视觉一致性方面均优于现有技术。
本文的方法分为两个阶段，旨在有效地进行图像布局编辑。第一阶段采用多概念学习（MCL），从单一图像中提取多个物体的概念，以保持编辑后的视觉一致性。具体步骤包括：
多概念学习：通过对原始图像进行处理，提取出不同物体的特征，确保各物体在布局调整时保持其独特性。
布局指导：在每个去噪步骤中，计算每个物体的区域损失，以引导其向目标布局调整。
外观投影：利用扩散模型的中间特征，进行外观信息的投影，确保物体在目标区域的外观与原始图像的一致性。
初始噪声设计：采用布局友好的初始化噪声，增强布局调整的效果。通过将物体从原始图像中裁剪并粘贴到空白背景中，形成初始布局，从而提高编辑的效率和效果。
为评估所提方法的有效性，本文进行了多项实验。首先，构建了一个名为Layout-Bench的公开数据集，包含25幅高质量图像，每幅图像涉及多个场景和布局。实验中，本文的方法与几种基线方法进行了比较，包括图像级操作、DesignEdit和CLED。定量评估采用视觉相似性、布局对齐和图像质量等指标，结果显示，所提方法在布局对齐和图像质量方面显著优于其他方法。主观评价方面，通过用户研究收集参与者对不同方法编辑结果的反馈，结果表明，本文的方法在物体一致性、布局对齐和图像质量等方面均表现优异。此外，进行的消融实验进一步验证了各个组件对最终效果的贡献，证明了多概念学习和外观投影在保持物体外观一致性中的关键作用。
本文的方法可以简单理解为一种智能的图像编辑工具，旨在帮助用户重新排列图像中的物体，同时确保这些物体在外观上保持一致。首先，这个工具会分析原始图像，识别出其中的各个物体，并学习它们的特征。接下来，用户可以指定想要的布局，比如将某个物体移动到新的位置。工具会在编辑过程中，确保这些物体在颜色、形状和纹理等方面与原始状态保持一致。为了实现这一点，工具还会在编辑开始前设计一个合适的“初始噪声”，这就像为编辑过程设置一个良好的起点。最终，用户可以看到一个新的图像，其中的物体按照新的布局排列，但看起来依然自然、和谐。这种方法不仅提高了图像编辑的灵活性，还确保了编辑后的图像看起来真实且美观。
