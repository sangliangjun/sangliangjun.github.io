2025-03-09｜ByteDance｜⭐️🟡
http ://arxiv.org/abs/2503.06790v1https ://icandle.github.io/gendr_page
GenDR（GenerativeDetailRestorator）是一种新型的图像超分辨率（SR）模型，旨在通过生成细节恢复技术提高低分辨率图像的质量。传统的文本到图像（T2I）扩散模型在处理超分辨率任务时，面临着生成速度与细节保留之间的矛盾。GenDR通过一系列创新方法，利用更大的潜在空间和改进的训练策略，成功地解决了这一问题。该模型不仅在图像质量上达到了新的高度，还在推理速度上显著提升，展示出在多种基准测试中的优越表现。通过将生成对抗网络（GAN）与扩散模型相结合，GenDR能够在保留图像细节的同时，减少生成过程中的计算复杂性。
GenDR的核心方法包括以下几个方面：
SD2.1-VAE16模型构建：采用16通道变分自编码器（VAE），为超分辨率任务提供更大的潜在空间。这一设计允许模型在不显著增加计算负担的情况下，保留更多细节。
一致性得分身份蒸馏（CiD）：通过将超分辨率任务的特定损失引入得分蒸馏，GenDR在生成过程中保持了一致性与稳定性。这种方法有效减少了生成与目标图像之间的差异。
简化推理管道：去除复杂的调度器和条件模块，使用固定的文本嵌入，简化了模型的结构，使得推理过程更加高效。
对抗学习与表示对齐：结合对抗学习策略，进一步优化生成过程，确保生成的图像在质量上更接近真实图像。
在实验设置中，GenDR在多个真实世界和合成数据集上进行评估，包括Laion-5B和DiffusionDB。训练过程中，使用了8个NVIDIAA100GPU，采用AdamW优化器进行100,000次迭代。实验结果表明，GenDR在多个标准图像质量评估指标上超越了现有的最先进方法，尤其是在PSNR、SSIM和LIQE等指标上表现优异。此外，用户偏好研究也显示，GenDR生成的图像在视觉质量和细节恢复方面获得了更高的认可率。整体而言，GenDR在图像超分辨率任务中展现出卓越的性能和效率，成为该领域的一项重要进展。
GenDR的工作原理可以简单理解为利用一种聪明的方法来“修复”模糊的低分辨率图片。它使用了一种叫做变分自编码器的技术，这就像给模型提供了一个更大的“画布”，让它可以在上面绘制更清晰的细节。为了让这个过程更高效，GenDR还引入了一种新的蒸馏方法，确保生成的图像与真实的高质量图像尽量一致。此外，它还简化了操作流程，就像把复杂的步骤变得简单明了，这样可以更快地得到结果。总之，GenDR就像是一个聪明的艺术家，能够迅速将模糊的图像变得生动清晰，让人眼前一亮。
2025-03-09｜USTC,AHU,HuaweiNoah’sArkLab,Chang’anU,HKUST,THU｜⭐️🟡
http ://arxiv.org/abs/2503.06617v1https ://github.com/peylnog/ContinuousSR
在本研究中，我们提出了一种新颖的连续超分辨率框架（ContinuousSR），旨在通过像素到高斯的建模方法，实现高效且高质量的任意尺度超分辨率（ASSR）。传统的超分辨率方法通常依赖于多次上采样和解码步骤，导致效率低下和重建质量受限。我们的ContinuousSR方法通过直接从低分辨率（LR）图像重建二维连续高分辨率（HR）信号，显著提高了性能和效率。我们引入了一种深度高斯先验（DGP）和动态优化的协方差加权机制，结合自适应位置漂移策略，进一步提升了重建质量。实验结果显示，在多个基准测试中，我们的方法在超分辨率质量和速度上均优于现有的最先进技术，达到了19.5倍的速度提升和0.90dB的PSNR改进。
我们的方法主要包括以下几个关键创新：首先，采用高斯建模直接重建连续HR信号。通过高斯混合模型（GMM），我们能够将复杂的连续函数表示为多个高斯函数的组合。其次，提出了深度高斯先验（DGP），用于构建预定义的高斯核，从而简化协方差参数的优化过程。我们还引入了DGP驱动的协方差加权机制，动态调整高斯核的权重，以适应输入图像的特征。最后，采用自适应位置漂移（APD）策略，根据图像内容动态调整高斯核的位置，确保在纹理丰富的区域有更密集的核分布。这些创新的结合使我们的ContinuousSR在重建高质量图像的同时，保持了高效的计算性能。
在实验部分，我们使用了多个常用的高分辨率图像数据集进行评估，包括DF2K、Set5、Set14等。我们通过双线性下采样生成低分辨率图像，并采用多个评价指标如PSNR、SSIM、FID和DISTS来评估重建质量。实验结果显示，我们的方法在所有基准测试中均表现出色，特别是在Urban100数据集上，超越了当前的最先进方法CiaoSR，PSNR提升了0.80dB。此外，我们还进行了消融实验，验证了DGP驱动的协方差加权和自适应位置漂移策略对模型性能的显著贡献，进一步证明了我们方法的有效性和优越性。
我们的方法可以看作是一个高效的图像“放大器”，它通过一种新颖的方式来处理图像。首先，我们使用高斯函数来表示图像中的每一个部分，这就像用不同颜色的水彩笔在画布上绘制出图像的细节。然后，我们通过分析大量的自然图像，找出这些高斯函数的最佳组合，确保它们能准确地反映真实世界中的图像特征。接下来，我们引入了一种智能机制，让这些高斯函数根据图像内容自动调整位置和权重，这样在复杂的区域可以放置更多的高斯函数，从而获得更清晰的细节。最后，我们的技术不仅能在短时间内生成高质量的图像，还能在不同的放大倍数下保持一致的效果。这样一来，无论是放大多少倍，我们都能得到更清晰、更真实的图像。
2025-03-10｜NJU｜CVPR2025｜⭐️
http ://arxiv.org/abs/2503.06896v1https ://github.com/EquationWalker/CATANet
在图像超分辨率（SR）领域，基于Transformer的方法展现了显著的性能，但其计算复杂度随空间分辨率的增加而呈平方增长。为了解决这一问题，许多研究尝试将低分辨率图像划分为局部窗口、轴向条纹或扩张窗口。然而，这些方法通常只关注内容无关的局部区域，限制了其捕捉长程依赖的能力。针对这些挑战，本文提出了一种轻量级的内容感知令牌聚合网络（CATANet），其核心是高效的内容感知令牌聚合模块，旨在聚合长程内容相似的令牌。该模块在训练阶段共享令牌中心，并利用组内自注意力机制实现长程信息交互。此外，设计了组间交叉注意力机制以进一步增强全局信息交互。实验结果表明，CATANet在性能上超越了现有的基于聚类的方法SPIN，最高PSNR提升0.33dB，并且推理速度接近翻倍。
CATANet的网络结构包含三个主要模块：浅层特征提取、深层特征提取和图像重建。首先，通过3×3卷积层进行浅层特征提取，将输入图像映射到高维特征空间。接着，使用多个残差组（RG）进行深层特征提取，每个残差组由令牌聚合块（TAB）、局部区域自注意力（LRSA）和3×3卷积组成。在每个RG中，首先通过TAB模块进行令牌聚合，以捕捉长程依赖；随后使用LRSA增强局部区域的依赖关系。最后，图像重建模块结合全局残差信息，生成高分辨率图像。此外，CATANet的内容感知令牌聚合（CATA）模块通过共享令牌中心并使用指数移动平均（EMA）更新，提升了模型的推理效率。
在实验中，CATANet在多个基准数据集上进行了评估，包括Set5、Set14、B100、Urban100和Manga109。使用DIV2K数据集进行训练，并通过PSNR和SSIM指标评估模型性能。实验结果显示，CATANet在所有数据集上均超越了其他轻量级SR模型，尤其在Urban100数据集中表现突出。此外，CATANet在参数量上相较于SPIN减少了15K，并且实现了接近两倍的推理速度提升。通过对比分析，CATANet的长程信息交互能力显著增强，能够更好地恢复图像细节，尤其是在边缘和纹理的重建上，展现出较强的优势。
CATANet的工作原理可以简单理解为一个智能的图像处理系统，它通过分析图像中的不同部分来提升图像的清晰度。首先，它会用一个小的卷积层提取图像的基本特征，然后通过多个处理层来进一步分析这些特征。每个处理层都像一个小团队，专注于不同的任务，比如聚集相似的图像区域、增强局部细节和重建图像。特别的是，CATANet会在处理时共享一些重要的信息点，这样可以避免在每次处理时都重新计算，从而加快速度。最终，CATANet能够在保持高质量图像的同时，显著提高处理速度，使得它在资源有限的情况下也能有效工作。
2025-03-07｜MIT,MAHE｜ICLR2025｜⭐️
http ://arxiv.org/abs/2503.05974v1
在图像到图像转换（I2IT）领域，图像对比度增强是提升视觉质量的重要技术，尤其是在处理低光和过曝图像时。然而，许多现有方法在保留细节方面存在困难，常常导致低级特征的丢失。为此，本文提出了一种新颖的对比度增强方法——LapLoss，基于拉普拉斯金字塔网络。该方法通过多尺度损失计算，结合多个判别器架构，能够在不同分辨率下捕捉高层特征，同时保持低层细节和纹理。通过在不同的金字塔层次上进行损失计算，LapLoss在重建精度和感知质量之间实现了良好的平衡，显著提升了图像生成的质量。实验结果表明，LapLoss在SICE数据集上表现出色，能够在不同光照条件下稳定地超越现有的对比度增强技术。
LapLoss的核心方法包括以下几个方面：
拉普拉斯金字塔构建：首先从原始图像创建高斯金字塔，通过下采样获得不同层级的图像，并计算原始图像与平滑版本之间的差异，以捕捉细微结构。
多尺度判别器架构：采用多个判别器，每个判别器对应金字塔的一个层级，旨在识别生成图像在该层级的真实性。通过这种方式，模型可以在不同尺度上优化生成结果。
损失函数设计：结合像素级均方误差（MSE）损失与对抗性损失，分别在各个金字塔层级计算损失，并通过加权平均的方式综合不同层级的损失，以优化生成器的输出。
轻量级网络架构：使用轻量级网络结构以提高推理速度，同时保持与其他先进模型相当的性能。
在实验部分，研究团队使用了SICE数据集进行训练和测试，该数据集包含589个高分辨率的多曝光序列，提供了丰富的对比度变化样本。通过对比不同的超参数设置，团队发现最佳的对抗损失与重建损失的比例为4.5×10^3，学习率设置为10^-3，以确保模型的稳定收敛。实验结果表明，LapLoss在不同光照条件下的表现优于其他对比度增强模型，尤其在结构相似性指数（SSIM）和峰值信噪比（PSNR）方面取得了显著提升。此外，通过对比不同的损失函数和网络结构，验证了LapLoss的有效性和鲁棒性。
LapLoss的方法可以简单理解为一个“图像修复工具”，它通过分层的方式来处理图像。首先，它把图像分成多个层次，就像把一个蛋糕切成几层，每一层都包含不同的细节。然后，针对每一层，LapLoss都会用一个“检查员”来判断这一层的图像是好还是坏。这个检查员会告诉我们哪一层的图像需要改进。最后，LapLoss结合了每一层的信息，通过权重来决定哪些层次更重要，从而生成更清晰、更自然的图像。这样，即使在光线不好的情况下，LapLoss也能帮助我们得到更好的图像效果。
2025-03-09｜SJTU,HarvardU,AdobeResearch｜⭐️
http ://arxiv.org/abs/2503.06537v1https ://github.com/xyLiu339/OSDD
本研究提出了一种新颖的单步扩散模型（OSDD），用于图像运动去模糊。现有基于卷积神经网络（CNN）和变换器的方法在去模糊性能上表现良好，但在面对复杂模糊时常常存在感知限制和泛化能力不足的问题。虽然扩散模型在一定程度上可以缓解这些问题，但其多步骤去噪过程限制了实际应用的效率。我们的OSDD通过将去噪过程简化为单一步骤，显著提高了推理效率，同时保持了高保真度。此外，我们引入了一种增强的变分自编码器（eVAE），以改善结构恢复，解决扩散模型中的保真度损失问题。通过构建高质量的合成去模糊数据集，我们进一步减轻了感知崩溃，并设计了动态双适配器（DDA），增强了感知质量。
本研究的核心方法包括以下几个方面：
增强变分自编码器（eVAE）：传统变分自编码器在图像恢复任务中存在信息损失问题。我们通过引入跳跃连接机制，在编码器和解码器之间建立中间连接，帮助模型保留更多细节，从而提高重建质量。
单步扩散模型（OSDD）：我们将扩散过程简化为单步，通过直接预测清晰的潜在变量，消除了逐步去噪的需求。这种方法提高了推理速度和效率。
动态双适配器（DDA）：该机制在推理阶段动态融合来自不同数据集的预训练模型，利用手动可调的权重因子，灵活控制合成和真实数据集之间的学习影响，确保在去模糊任务中同时提高保真度和感知质量。
高质量合成去模糊数据集：为了有效训练扩散模型，我们构建了一个高质量合成数据集，通过随机合成模糊核生成图像对，增强模型的泛化能力。
在实验部分，我们使用GoPro和RealBlur数据集评估了OSDD的性能。实验设置包括两个阶段的训练：第一阶段在固定分辨率下进行，第二阶段则采用逐步增加分辨率的策略，以增强模型的学习能力。我们通过多种全参考和无参考指标（如LPIPS和DISTS）对模型进行评估，结果显示OSDD在多项指标上均优于现有基线模型，尤其在感知质量方面表现出色。此外，我们还进行了消融实验，验证了eVAE和DDA在提升模型性能方面的有效性。结果表明，OSDD在处理复杂模糊场景时具有较强的鲁棒性和优越的视觉效果。
在这项研究中，我们开发了一种新的图像去模糊方法，旨在快速有效地将模糊的图像恢复为清晰的图像。我们使用了一种叫做增强变分自编码器的技术，这种技术可以帮助我们更好地保留图像中的细节。我们还设计了一种单步扩散模型，这意味着我们只需一步就能完成去模糊，而不是像传统方法那样需要多次处理，这样可以大大加快速度。另外，我们创建了一个高质量的合成数据集，帮助我们的模型学习如何更好地处理模糊图像。最后，我们的动态双适配器可以让模型在处理不同类型的模糊时灵活调整，从而保持图像的清晰度和质量。通过这些创新，我们的模型在多项指标上都超过了现有的去模糊技术。
2025-03-09｜BostonU,Runway｜⭐️
http ://arxiv.org/abs/2503.06698v1
在本研究中，我们探讨了如何利用扩散模型的潜在空间来实现领域泛化（DomainGeneralization）。领域泛化的目标是开发能够适应未见过的数据分布的模型。我们提出了一种新方法，旨在通过分析模型架构和预训练目标对特征丰富性的影响，来有效利用这些特征。具体而言，我们首先在无监督的情况下发现潜在的领域结构（伪领域），这些结构能够捕捉领域特定的变化。接着，我们通过将现有分类器与这些伪领域表示相结合，增强了分类器的适应性，使其更容易处理多样化的未见测试领域。我们的实证研究表明，扩散模型生成的特征在没有明确领域标签的情况下，能够有效地分离不同领域，从而提高模型的泛化能力。
我们的方法包括两个主要步骤。首先，我们从冻结的预训练特征空间中学习伪领域表示。具体步骤如下：
特征提取：使用预训练的特征提取器（如扩散模型）对训练样本进行特征提取。
聚类：采用K-Means++聚类算法，将提取的特征分为多个伪领域，得到每个伪领域的中心点作为其代表。
特征增强：在训练期间，我们通过将每个样本的特征与其对应的伪领域表示进行拼接，来增强分类器的输入特征。这一过程通过一个变换函数，将伪领域中心点的特征映射到输入特征空间中。
分类器训练：使用增强后的特征训练一个通用分类器，使其能够更好地适应未见领域。
通过这些步骤，我们的框架能够在没有领域标签的情况下，利用潜在的领域结构来提升模型的泛化能力。
我们在多个数据集上进行了实验，以验证我们方法的有效性。使用的主要数据集包括PACS、VLCS、TerraIncognita、OfficeHome和DomainNet。实验步骤如下：
数据集准备：从不同领域构建训练和测试集，确保测试集包含未见领域。
模型训练：使用预训练的扩散模型提取特征，并应用我们的伪领域发现方法进行特征增强。
性能评估：通过交叉验证评估模型在未见领域上的表现，比较我们的框架与其他领域泛化方法的准确性。
结果分析：通过计算领域特定的归一化互信息（NMI）得分，分析不同特征提取器在捕捉领域特征方面的表现。
实验结果表明，我们的方法在多个数据集上均显著提高了模型的泛化能力，相较于基线方法，准确性提升超过4%。
我们的方法可以简单理解为通过两步来帮助机器学习模型更好地适应新情况。第一步是从已经训练好的模型中提取特征，这些特征就像是模型的“记忆”，包含了它在学习过程中获得的信息。接下来，我们将这些特征进行分组，形成所谓的“伪领域”，就像把相似的记忆放在一起。第二步是将每个样本的特征与其对应的伪领域特征结合在一起，这样模型在学习时就能更全面地理解不同的情况。最终，我们训练一个新的分类器，利用这些丰富的特征来帮助模型在面对新数据时做出更好的判断。通过这种方式，即使在没有具体领域标签的情况下，我们的模型依然能够有效地进行预测。
