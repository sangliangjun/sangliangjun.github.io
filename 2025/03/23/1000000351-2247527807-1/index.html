主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月11日｜周二
2025-03-08｜OPPO,THU｜⭐️⭐️
http ://arxiv.org/abs/2503.06134v1https ://github.com/OPPO-Mente-Lab/X2I
在当前的人工智能领域，文本到图像（T2I）模型和多模态大语言模型（MLLMs）在图像生成和多模态理解方面展现了巨大的潜力。然而，现有的框架尚未有效地将MLLM的多模态理解能力转移到T2I模型上。为此，本文提出了X2I框架，旨在实现DiffusionTransformer（DiT）模型与多模态输入的无缝整合。X2I框架能够处理多种输入形式，包括多语言文本、图像、视频和音频，并通过仅使用100K的英语语料库和160GPU小时的训练，显著提升了图像生成的多模态理解能力。通过引入AlignNet结构和创新的注意力蒸馏方法，X2I在性能上仅有不到1%的下降，同时增强了多模态理解能力，满足了行业对图像生成的多样化需求。
X2I的构建包含几个关键步骤。首先，利用DiffusionTransformer模型作为教师模型，采用知识蒸馏方法提取其推理能力。具体方法如下：
AlignNet结构设计：构建一个轻量级的AlignNet作为中介桥梁，替换教师模型的文本编码器，以便更好地处理多模态输入。
注意力蒸馏：通过注意力蒸馏方法将教师模型的推理能力传递给学生模型，优化多模态特征的融合。
LightControl模块：设计LightControl以增强图像编辑任务中的图像指令生成能力，提升图像的保真度。
训练过程：X2I在仅使用文本数据的情况下进行训练，AlignNet的参数在训练过程中可调，而教师模型的其他参数则被冻结，以实现高效的知识转移。
为验证X2I的有效性，进行了多项实验。首先，从Laion2B数据集中随机选择了100K对图像-文本对，用于训练X2I。接着，构建了包含400K图像的训练数据集，涵盖五种艺术风格，以评估X2I的图像生成能力。实验结果显示，X2I在多个评估基准上表现优异，尤其是在复杂构图和多语言生成任务中，X2I的性能与教师模型Flux.1相当。此外，定量和定性分析表明，X2I在多模态输入支持下，能够生成高保真度的图像，并在图像指令编辑任务中表现出色，显示出其在实际应用中的广泛适用性。
X2I框架的核心方法可以简单理解为一个“老师-学生”教学过程。首先，X2I的“老师”是一个强大的图像生成模型，它能够处理多种输入，比如文本、图片和声音。然后，我们设计了一个“学生”模型，通过学习“老师”的知识来提升自己的能力。具体来说，学生模型通过一个叫做AlignNet的小工具来吸收老师模型的知识，尤其是如何理解和生成图像。接下来，使用LightControl模块来帮助学生模型更好地理解图像编辑的指令。整个训练过程只需要文本数据，这样可以让学生模型快速学习，而不需要耗费大量的计算资源。这样一来，X2I不仅能生成高质量的图像，还能处理多种输入形式，满足不同的需求。
2025-03-08｜PKU｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.06186v1
本研究提出了一种名为PTDiffusion的新方法，旨在生成光学幻觉隐藏图像。该方法基于现有的文本到图像（T2I）扩散模型，利用一种无训练的图像到图像（I2I）翻译框架，能够将参考图像无缝融入由文本提示描述的场景中。PTDiffusion通过动态转移扩散特征的相位谱，实现了参考图像的结构信息与文本语义的和谐融合。该方法特别适用于生成光学幻觉隐藏图像，这种图像在视觉上巧妙地将一幅图像隐藏在另一幅图像的细节中，使得隐藏的内容不易被察觉。通过广泛的定性和定量实验，PTDiffusion在图像质量、文本一致性、视觉可辨性和上下文自然性等方面超越了相关方法，展示了其有效性和优势。
PTDiffusion的核心是一个模块化的相位转移机制，能够在图像生成过程中灵活地控制隐藏内容的可辨性。其主要步骤包括：
反演轨迹：使用DDIM反演方法将参考图像投影到高斯噪声空间，以提取初始特征。
重建轨迹：从反演得到的噪声中重构出参考图像，同时保持其结构信息。
采样轨迹：在随机噪声的引导下，基于文本提示生成最终的幻觉图像。
相位转移模块（PTM）：通过2D快速傅里叶变换提取特征的幅度和相位，动态融合相位信息，从而在生成过程中平滑地将参考图像的结构信息与文本语义结合。
异步相位转移：允许在不同时间步之间进行相位转移，以灵活控制结构渗透强度，优化生成的图像质量。
在实验部分，研究者对PTDiffusion进行了定性和定量评估。定性结果显示，PTDiffusion能够生成高质量的光学幻觉图像，且这些图像在语义上与文本提示一致，同时能清晰地展现参考图像的结构信息。定量评估采用了美学评分、CLIP得分和LPIPS等指标，结果表明PTDiffusion在这些方面均优于其他相关方法。此外，研究还进行了消融实验，验证了相位转移强度和重建阶段对生成图像质量的影响。通过改变异步距离参数，研究者能够有效控制隐藏内容的可辨性，进一步提升了生成结果的多样性和自然性。
PTDiffusion方法的核心是如何将一幅图像巧妙地隐藏在另一幅图像中，同时又能确保隐藏的内容在视觉上不那么明显。首先，研究者利用一种叫做“反演”的技术，将输入的参考图像转换成一种特殊的噪声格式。接着，他们会重构出这幅图像，并在生成的过程中，使用文本提示来指导图像的风格和内容。最重要的是，PTDiffusion通过一个相位转移模块，动态地将参考图像的结构信息与文本描述结合起来，使得最终生成的图像既美观又具有深意。此外，研究者还引入了异步相位转移的概念，允许他们在不同的生成步骤中灵活调整隐藏内容的显现程度，从而创造出更具吸引力和多样性的幻觉图像。
2025-03-10｜SJTU,SIAT,CAS,ShanghaiAILab,ZJU,TencentWechat｜⭐️🟡
http ://arxiv.org/abs/2503.07416v1
本研究提出了一种名为TimeStepMaster(TSM)的新方法，旨在提高视觉生成中的扩散模型的效率和灵活性。扩散模型在图像和视频生成方面取得了显著进展，但在应用于下游任务时，因大规模模型的微调成本高而面临挑战。传统的低秩适应（LoRA）方法在不同时间步的微调中能力有限，无法充分捕捉不同噪声水平。为了解决这个问题，TSM采用了两个关键的微调阶段：培养阶段和组装阶段。通过在不同时间步应用不同的LoRA模块，TSM能有效捕捉不同噪声级别，并通过不对称的专家协作提升模型的多样性和适应性。
TSM方法包含两个主要阶段：专家培养和组装。在培养阶段，首先将时间步划分为多个区间，并为每个区间引入独立的LoRA模块。这种设计使得每个LoRA能够针对特定的噪声水平进行微调，从而形成不同的TimeStepLoRA专家。在组装阶段，TSM通过不对称的混合方式，将各个时间步的专家进行组合。具体来说，在每个时间步中，选取最小区间内的TimeStepLoRA专家作为核心专家，而其他区间的专家则作为上下文专家，通过时间依赖的门控机制进行协调。这种设计不仅提高了模型的噪声建模能力，还增强了其在不同任务中的泛化能力。
为验证TSM的有效性，研究团队在三个典型的扩散模型相关任务上进行了广泛的实验，包括领域适应、后预训练和模型蒸馏。实验结果表明，TSM在所有任务中均实现了最先进的性能，超越了传统的LoRA方法。具体而言，在T2I-CompBench基准测试中，TSM在颜色、形状、纹理等多个指标上表现优异，尤其是在复杂领域任务中，TSM能够有效提升模型的生成质量。此外，TSM在后预训练和模型蒸馏任务中也显示出显著的性能提升，表明其具有更好的适应性和效率。
在TSM的工作原理中，首先将整个生成过程分成多个时间段，每个时间段都有一个专门的“专家”来处理特定类型的噪声。想象一下，这就像在一个乐队中，每个乐器在不同的乐段中负责不同的旋律。通过这样的方法，TSM能够在每个时间段内使用最合适的专家来处理噪声，从而提高生成的图像或视频的质量。接着，在合成阶段，TSM将这些专家的能力结合起来，形成一个更强大的团队。这样一来，TSM不仅能更好地理解复杂的图像内容，还能在不同的生成任务中表现得更加灵活高效。
2025-03-10｜ETHZürich｜⭐️🟡
http ://arxiv.org/abs/2503.07274v1
本文提出了一种名为适配器引导蒸馏（AGD）的方法，旨在高效地实现无分类器引导（CFG）的功能。传统的CFG方法在每个推理步骤中需要进行两次前向传播，导致计算成本较高，尤其是在处理大规模扩散模型时。AGD通过将CFG集成到基础扩散模型中，能够在单次前向传播中模拟CFG，从而实现推理速度的显著提升，达到原有速度的两倍。该方法仅需训练约2%的额外参数，保持基础模型的权重不变，确保了资源的高效利用。此外，AGD还能够与其他基于同一基础模型的检查点无缝集成，极大地扩展了模型的应用范围。
AGD的核心在于两个主要组件：首先是基于CFG引导轨迹的训练，而非传统的扩散轨迹。通过在CFG指导下生成的轨迹进行训练，可以减少训练与推理过程之间的差异，提升模型性能。其次，AGD利用轻量级适配器进行蒸馏，而不是对整个模型进行微调。具体步骤包括：1）收集CFG引导的轨迹数据，生成每个时间步的预测；2）引入适配器模块，仅优化适配器的参数，同时保持基础模型权重不变；3）在推理时，结合训练好的适配器与基础模型进行一次前向传播，从而实现高效的图像生成。通过这种方式，AGD不仅能保留CFG的优点，还能显著降低计算资源的需求。
在实验部分，研究团队对AGD进行了广泛的评估，包括与传统CFG和其他引导蒸馏方法的比较。实验使用了多个模型，如DiffusionTransformer和StableDiffusion系列，均在单一RTX4090GPU上进行。结果表明，AGD在生成质量和多样性上与CFG相当，甚至在某些情况下优于CFG，同时推理速度却提升了两倍。此外，AGD在训练过程中显示出更高的资源效率，能够在较低的内存占用下进行大规模模型的蒸馏。通过对不同引导轨迹的训练，AGD展现了其在不同任务上的强大适应性和鲁棒性。
AGD是一种新技术，用于加速图像生成过程。传统的方法在每次生成图片时需要进行两次计算，耗时又耗力。AGD通过一种聪明的方式，只需要一次计算就能达到同样的效果。具体来说，AGD首先用已有的高质量图像生成方法生成一些参考图像，然后训练一种小型的“助手”程序，这个助手可以帮助主程序更快地生成图像。这样，主程序的核心部分不需要改变，只需加上这个小助手，就能在更短的时间内生成更好的图像。通过这种方式，AGD不仅节省了计算资源，还让更多的人能够使用这些强大的图像生成技术。
2025-03-09｜HKUST,NUS,HuaweiNoah‘sArkLab,HKUST(GZ)｜⭐️🟡
http ://arxiv.org/abs/2503.06674v1https ://tdm-t2x.github.io/
在当今的人工智能生成内容（AIGC）领域，加速扩散模型的采样过程至关重要。现有的扩散蒸馏技术虽然通过分布匹配和轨迹匹配的方法来减少采样步骤，但在处理复杂任务（如文本到图像生成）时仍然面临挑战。为了解决这一问题，本文提出了一种新的学习框架——轨迹分布匹配（TDM），它将分布匹配和轨迹匹配的优势结合起来。这种方法通过对学生模型的轨迹与教师模型的分布进行对齐，显著提高了生成质量和训练效率。具体而言，TDM在仅用500次迭代和2A800小时的训练时间内，成功地将PixArt-α蒸馏为一个4步生成器，并在用户偏好上超越了其教师模型。
TDM方法的核心在于通过轨迹分布匹配实现高效的知识转移。其主要步骤包括：
分布级轨迹匹配：通过最小化教师模型与学生模型在不同时间步的分布差异，使得学生模型能够有效学习到教师模型的生成轨迹。
采样步骤感知目标：该目标允许在不同的采样步骤中灵活调整学习目标，支持更高效的确定性采样。
代理训练目标：引入一致性模型的灵感，以增强生成器的优化过程，从而提高整体性能。通过这些步骤，TDM不仅避免了传统方法中常见的数值误差，还大幅降低了训练成本，使得生成模型在保持高质量的同时，能够快速收敛。
实验部分主要通过在JourneyDB数据集上进行蒸馏，评估TDM的性能。我们比较了两种不同的TDM变体：一种是在蒸馏前对SD-v1.5进行微调的TDM-unify-SFT，另一种是在蒸馏过程中使用GAN的TDM-unify-GAN。结果显示，TDM-unify-SFT的表现明显优于TDM-unify-GAN，尤其在4步生成任务中，HPS得分达到31.31。进一步的用户研究也表明，TDM在图像生成质量和文本图像对齐方面均优于其他竞争方法。此外，TDM在训练成本上表现出色，仅需2A800天的训练时间，远低于其他方法的需求，显示出其在生成模型训练中的高效性。
TDM方法的核心思想是通过一种新颖的方式来训练生成模型，使其在生成图像时更快且更准确。具体来说，TDM有三个主要步骤。首先，它通过比较生成的图像与理想图像之间的差距，来帮助模型学习如何生成更好的图像。其次，TDM允许模型在生成图像时选择不同的步骤，这样可以根据需要调整生成过程。最后，它使用一种特殊的训练方式，确保模型在学习的过程中能够更快地改进。这种方法的好处在于，模型在生成图像时不仅速度快，而且生成的图像质量高，能够更好地满足用户的需求。
