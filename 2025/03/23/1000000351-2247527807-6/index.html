317-A1.3｜评估系列（真实图像质量与美学，复杂语义理解与世界知识整合，生成特定数量对象，触觉图像生成）

AIGCResearch
主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月11日｜周二
2025-03-08｜AlibabaAmap,BUPT｜⭐️🟡
http ://arxiv.org/abs/2503.06141v1
随着移动互联网的快速发展，用户生成内容（UGC）图像的数量急剧增加，如何有效评估这些图像的质量和美学变得尤为重要。为了应对这一挑战，研究者们提出了一个名为“RealisticImageQualityandAesthetic”（RealQA）数据集，该数据集包含14,715张UGC图像，并为每张图像标注了10个细粒度的属性。这些属性分为低级（如图像清晰度）、中级（如主题完整性）和高级（如构图）三个层次。尽管多模态大型语言模型（MLLMs）在图像质量评估（IQA）和美学评估（IAA）方面取得了一定进展，但传统方法往往无法准确反映人类的层次感知。因此，本文提出了一种新方法，通过预测两个额外的有效数字，利用下一个标记的范式来提高评分的准确性和可解释性。
本研究的核心方法包括几个关键步骤。首先，构建RealQA数据集，通过多层次属性训练来提升MLLMs对细粒度属性的提取能力。具体步骤如下：
数据收集：从不同的UGC来源（如旅游景点、餐馆等）收集图像，并经过处理和标注，确保数据的多样性和代表性。
属性选择：将图像质量和美学分为高、中、低三个层次，分别评估图像的吸引力、构图、主题完整性、背景杂乱度等。
下一个标记预测：通过MLLMs预测数值评分，采用数值连续性度量（NCM）来解决数值模糊性问题，确保模型理解数值的整体性，而不仅仅是记忆不同位置的标记。
训练策略：利用冷启动策略和链式思维（CoT）相结合的方式进行模型训练，整合细粒度属性和数值评分，以提高模型的表现。
在实验部分，研究者们对提出的方法进行了全面评估。首先，使用RealQA数据集进行模型训练，并与现有的基准方法进行比较。通过在多个公共数据集（如Koniq-10k和AVA）上进行测试，结果表明，所提方法在多项指标上超越了现有的最佳性能，尤其是在跨域评估中表现优异。此外，研究还进行了消融实验，以探讨不同训练策略和属性选择对模型性能的影响。通过分析模型在零-shot场景下的表现，验证了其在视频质量评估任务中的强泛化能力，进一步证明了所提方法的有效性和实用性。
在这项研究中，科学家们开发了一种新方法来评估图片的质量和美感。首先，他们收集了大量的用户上传的图片，并为每张图片打上了多个标签，这些标签帮助他们从不同角度分析图片，比如清晰度、构图等。接着，他们使用一种叫做多模态大型语言模型的技术，这种技术可以理解图片和文字之间的关系。他们的创新之处在于，不仅仅给图片一个简单的评分，而是通过预测更多的数字来提高评分的准确性。最后，实验结果显示，这种方法在多个测试中表现优越，能够更好地反映人们对图片的真实感受。简单来说，这项研究让计算机在评价图片时更像是人类的思考方式。
2025-03-10｜PKU,CQU,PengChengLab,RabbitpreAI｜⭐️🟡
http ://arxiv.org/abs/2503.07265v1https ://github.com/PKU-YuanGroup/WISE
本文提出了WISE（WorldKnowledge-InformedSemanticEvaluation），一个旨在全面评估文本到图像生成（T2I）模型的基准，特别关注模型在复杂语义理解和世界知识整合方面的能力。传统的T2I评估主要集中于图像的现实性和表面文本-图像对齐，忽视了模型在处理复杂语义信息时的表现。WISE通过设计1000个精心构建的提示，涵盖文化常识、时空推理和自然科学等25个子领域，挑战模型在生成图像时的世界知识应用能力。研究发现，现有模型在有效整合和应用世界知识方面存在显著局限，强调了未来模型改进的关键路径。
WISE基准的构建分为几个关键步骤：首先，构建了涵盖自然科学、时空推理和文化常识三个主要领域的1000个评估提示。这些提示要求模型不仅理解文本描述，还需调用内部世界知识。其次，引入了WiScore，一个新颖的综合评估指标，强调生成图像中对象和实体的准确描绘。WiScore由三个方面的加权平均组成：一致性、现实性和美学质量，以全面评估生成图像的质量。此外，评估框架采用了多维度评分机制，确保对生成内容进行严格评估，特别关注模型在复杂语义理解和世界知识整合中的表现。
在实验中，研究团队评估了20个T2I模型，包括10个专用的T2I模型和10个统一的多模态模型。实验结果表明，所有模型在生成图像时有效整合世界知识的能力普遍不足。通过WISE基准的评估，发现专用T2I模型通常优于统一多模态模型，表明尽管后者在语言理解方面具有优势，但在图像生成中未能充分发挥其潜力。具体而言，文化常识相关的提示表现较好，而自然科学和时空推理的提示则暴露了模型在复杂推理和知识整合方面的不足。这些发现突显了T2I领域在知识应用和语义理解方面的改进需求。
在WISE方法中，我们创建了一个新的评估系统，旨在测试计算机生成图像的能力。首先，我们设计了许多问题，这些问题不仅要求计算机理解文字，还需要它们拥有一些关于世界的知识。例如，如果问题是“画一个在母亲节送的花”，计算机需要知道这通常是“康乃馨”。接下来，我们使用了一个叫做WiScore的评分系统，这个系统会给每张生成的图像打分，主要看三方面：图像是否与问题一致、图像看起来是否真实、以及图像是否美观。通过这种方式，我们能够更全面地了解计算机在处理复杂问题时的表现，特别是在理解和应用世界知识方面的能力。
2025-03-10｜ZJU,GUET,UArizona,HKU,UW-Midison,SITC,UMinnesota｜⭐️🟡
http ://arxiv.org/abs/2503.06884v1
在当今人工智能领域，生成模型被视为一项关键问题，文本到图像生成技术在实际应用中产生了深远影响。尽管扩散模型在文本到图像生成中表现出色，但它们在遵循用户指令中的数值限制方面存在显著缺陷，尤其是在生成特定数量对象时。为了解决这一问题，研究者们提出了T2ICountBench，这是一种专门设计的基准，用于系统评估当前最先进的文本到图像扩散模型的计数能力。该基准不仅涵盖了多种生成模型，还通过人类评估确保了评估的可靠性。研究结果表明，所有模型在生成正确数量的对象时表现不佳，随着对象数量的增加，准确率显著下降。同时，简单的提示优化并未改善计数准确性，突显了扩散模型在数值理解方面的固有挑战。
T2ICountBench的设计包括几个关键部分。首先，我们选择了15个最新的文本到图像扩散模型，确保覆盖广泛的生成AI研究和应用。其次，生成提示的设计至关重要，采用了一个简单有效的模板，直接测试模型的计数能力，范围从1到15的数字，涵盖六种常见对象类型。为确保评估的严谨性，采用全面的人类评估流程，五名具备AI和视觉感知专业知识的研究生对生成的图像进行标记。最后，评估协议明确了准确率作为主要评估指标，每个独特的对象、场景和风格组合都被视为单独的任务，从而提供了更为可靠的评估结果。
在实验部分，使用T2ICountBench对各种模型的计数性能进行了全面评估。结果显示，所有模型在生成对象数量的准确性方面均未达到50%的平均准确率，且随着对象数量的增加，准确率显著下降。具体来说，简单的计数任务（如1到5个对象）表现相对较好，而在复杂任务（如11到15个对象）中，几乎所有模型的准确率都低于10%。此外，模型在不同场景中的表现差异显著，复杂场景（如城市环境）导致计数性能下降。通过对影响计数能力的各种因素进行消融研究，发现背景复杂性和对象数量是影响模型表现的主要因素，进一步强调了扩散模型在处理数值约束时的局限性。
在我们的研究中，我们使用了一种新的测试方法，叫做T2ICountBench，来评估不同的文本到图像生成模型是否能准确生成指定数量的物体。我们设计了一些简单的提示，像“生成5个苹果”，并观察模型生成的图像是否符合要求。为了确保结果的可靠性，我们请了几位专业的学生评估这些图像，看看它们是否正确。通过这种方法，我们发现无论是哪个模型，在生成较多物体时都难以做到准确，尤其是当要求生成的物体数量超过10个时，几乎所有模型的表现都很差。此外，我们还发现，背景复杂的场景会使模型更难以准确计数。因此，我们的研究表明，当前的生成模型在处理数量要求时仍存在很大的挑战。
2025-03-10｜UPitt｜⭐️🟡
http ://arxiv.org/abs/2503.06860v1
本研究聚焦于触觉图像生成的普适性，特别是在避免数据泄漏的环境下进行评估。触觉感知是人类理解世界的重要途径，广泛应用于计算机视觉、机器人技术和多模态学习。由于真实触觉数据的获取成本高且稀缺，生成合成触觉图像成为一种可扩展的解决方案。然而，确保生成触觉图像的鲁棒性和泛化能力仍然面临挑战。研究发现，当前常用数据集中训练和测试样本的重叠会导致性能指标的虚假膨胀，掩盖了触觉模型的真实泛化能力。为了解决这一问题，本文提出了一种无泄漏评估协议，并引入了一系列新颖的无参考度量标准，旨在更准确地评估触觉图像生成的质量和多样性。
本研究提出了一种基于潜在扩散的触觉图像生成方法，结合视觉输入和文本描述以增强生成质量。具体方法如下：
潜在编码器：首先，使用预训练的VQ-GAN编码器将触觉图像编码为潜在表示，以减少计算成本并保留关键信息。
扩散过程：在生成过程中，随机噪声被引入潜在空间，随后通过条件U-Net模型进行去噪，逐步生成清晰的触觉图像。
文本引导：在训练阶段，结合材料特定的文本描述，以指导模型关注重要的触觉特征，过滤掉不相关的视觉细节。
评估指标：引入一系列新的无参考度量，包括TMMD、I-TMMD、CI-TMMD和D-TMMD，以评估生成触觉图像的质量和一致性，确保评估结果更具可信度。
在实验部分，研究者对两种流行的视觉触觉数据集（TouchandGo和HCT）进行了测试，评估生成模型的性能。实验步骤如下：
数据集准备：针对原始数据集中存在的泄漏问题，创建了新的无泄漏数据集（如TaG-NoLeak和HCT-NoLeak），确保训练和测试样本之间没有重叠。
性能评估：使用新引入的无参考度量标准对生成的触觉图像进行评估，同时与传统的评估指标（如SSIM和FID）进行比较。
结果分析：通过对比不同方法的生成结果，分析数据泄漏对性能的影响，并验证所提出方法在生成触觉图像方面的优势。
在这项研究中，科学家们想要解决一个问题：如何从视觉信息中生成触觉图像，而不需要真实的触觉数据。他们的方法是使用一种叫做“潜在扩散”的技术。首先，他们将触觉图像转化为一种更简单的形式，这样计算起来更快。接着，他们向这个简单的图像中加入一些随机噪声，然后通过一个模型一步步去除噪声，最终生成清晰的触觉图像。而且，为了让模型更好地理解材料的特性，他们在训练时还加入了材料的文本描述。这样，模型就能专注于重要的触觉特征，而不是背景中的不相关信息。最后，研究者们还提出了一些新的评估方法，以确保生成的触觉图像质量更高，能够更好地反映真实世界的触觉特性。
