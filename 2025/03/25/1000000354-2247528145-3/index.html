2025-03-09｜NKU,DIIS,USTC,WHU,ShanghaiAILab
http ://arxiv.org/abs/2503.06542v1https ://armor.github.io
在多模态理解与生成领域，ARMORv0.1框架通过引入不对称的编码器-解码器架构，提升了现有多模态大语言模型（MLLMs）的能力。该模型旨在高效地处理文本与图像的交互生成，克服了传统统一模型（UniMs）在生成交错文本-图像时的高计算需求和资源消耗问题。ARMOR通过对现有MLLMs的微调，整合了文本与视觉模态的嵌入空间，允许模型以更少的计算资源实现自然的文本-图像交互生成。该框架的设计包括三方面的扩展：模型架构、训练数据和训练算法。实验结果表明，ARMOR在多模态理解与生成任务中均表现出优异的性能，尤其在图像生成方面，展示了其在保持理解能力的同时，显著提升生成能力。
ARMOR的核心方法包括三个主要部分：不对称编码器-解码器架构、精心策划的训练数据集和分阶段的训练算法。具体而言：
模型架构：采用不对称的编码器-解码器设计，结合前向切换机制，允许模型在处理不同模态时动态选择适合的响应方式。此设计确保了文本和图像的嵌入能够有效融合。
训练数据：收集并整理了高质量的交错文本-图像数据集，以支持模型的微调。这些数据集包含多种类型的任务，确保模型能够学习到丰富的模态交互。
训练算法：提出了“什么生成、如何生成和如何更好回答”的三阶段训练算法，逐步强化模型的生成能力。在每个阶段中，模型针对不同的任务目标进行优化，从而避免灾难性遗忘，并确保生成能力的提升。
为了验证ARMOR框架的有效性，研究者们在九个基准测试上进行了广泛的实验，包括多模态理解和视觉生成能力的评估。实验结果显示，ARMOR在多模态理解任务中显著优于现有的统一模型，且在图像生成任务中表现出色。具体来说，ARMOR在MMB基准测试中得分为78.8，相较于其他模型有明显提升。同时，ARMOR仅增加了约7%的可训练参数，相比于需要从头开始训练的模型，展现了更高的资源效率。这些结果不仅证明了ARMOR在理解和生成任务中的潜力，也展示了通过升级现有MLLMs构建统一模型的可行性。
ARMOR框架的工作原理可以简单理解为一个智能助手，它可以同时理解文字和图片，并根据需要生成相应的内容。首先，ARMOR使用一种特殊的结构，像一个双向的翻译器，能在文字和图片之间快速切换。其次，它从大量的高质量数据中学习，包括文本和图像的组合，确保它能在不同情况下做出正确的反应。最后，ARMOR的训练过程分为三个步骤：首先让它学会识别问题的类型；然后教它如何生成合适的图像；最后帮助它更好地结合文本和图像，提升整体的回答质量。通过这种方式，ARMOR不仅能理解复杂的多模态信息，还能生成自然流畅的文本和图像，满足用户的需求。
2025-03-10｜FDU
http ://arxiv.org/abs/2503.06940v1https ://jianxgao.github.io/CineBrain
CineBrain是一个开创性的多模态大规模脑数据集，首次同时记录了在动态视听刺激下的人类EEG和fMRI信号。该数据集包含来自六名参与者的约六小时的叙事驱动内容，主要选取了受欢迎的电视剧《生活大爆炸》的片段。这项研究利用EEG的高时间分辨率和fMRI的深层空间覆盖能力，旨在探讨复杂的脑动态和多模态神经解码。通过CineSync这一新颖的多模态解码框架，研究者能够有效融合EEG和fMRI信号，显著提升复杂视听刺激的重建质量。此外，研究还引入了Cine-Benchmark评估协议，以全面评估视频和音频重建的效果，确保研究的严谨性和可重复性。
CineSync框架的设计包含两个主要组成部分：多模态融合编码器（MFE）和神经潜在解码器（NLD）。具体方法如下：
多模态融合编码器（MFE）：该模块使用双变换器架构，分别从fMRI和EEG信号中提取特征，并通过对比损失进行语义对齐。这一过程确保了不同模态之间的信息能够有效融合。
神经潜在解码器（NLD）：在融合的特征基础上，NLD利用扩散模型重建对应的视听刺激。通过引入噪声，模型能够逐步去噪，生成高质量的重建结果。
数据处理：在数据处理阶段，fMRI信号会进行z-score标准化，并考虑BOLD反应的延迟。EEG信号则经过多步去噪处理，以确保数据的准确性和有效性。这种方法的创新之处在于有效结合了EEG和fMRI的优势，提升了对复杂动态刺激的重建能力。
在实验设计中，CineBrain数据集为六名参与者提供了共计540分钟的视听刺激，参与者在3TMRI扫描仪中观看视频和听音频，同时记录EEG和fMRI信号。每位参与者观看了不同季节的《生活大爆炸》剧集，以增强刺激的多样性。实验采用了标准化的18分钟观看时间，确保数据处理的高效性。为了评估CineSync框架的有效性，研究者与基于单一模态的基线模型（CineSync-fMRI和CineSync-EEG）进行了比较。实验结果表明，CineSync在视频和音频重建任务中均表现出色，显著超越了现有的基线模型，验证了其在多模态脑信号解码中的潜力。
CineSync的工作方式可以简单理解为一个智能的“翻译器”，它把大脑的信号转化为我们可以看到和听到的内容。首先，它会从大脑的两种信号——EEG（电生理信号）和fMRI（血流变化信号）中提取信息。EEG能快速捕捉到大脑的即时活动，而fMRI则能够提供更详细的脑部位置信息。接着，这些信息会被送入一个融合模块，这个模块就像是一个拼图，把来自两种信号的碎片组合在一起。最后，经过一系列的处理，这些组合的信息被送入另一个模块，生成我们可以理解的视听内容。这样，CineSync不仅能帮助我们理解大脑如何处理复杂的信息，还能在未来的研究中提供更准确的脑信号解码。
2025-03-09｜SNU,BNL
http ://arxiv.org/abs/2503.06437v1
本文提出了一种新的评估指标SEED（SemanticEvaluationforVisualBrainDecoding），旨在提高视觉脑解码模型的语义解码性能评估的准确性。SEED整合了三种互为补充的评估指标，分别捕捉图像之间语义相似性的不同方面。通过精心收集的众包人类判断数据，研究表明SEED在与人类评估的一致性上超越了其他广泛使用的评估指标。尽管现有的视觉脑解码模型在传统评估指标上得分接近完美，但它们往往无法准确重建原始图像中的关键语义元素。为推动进一步研究，本文还开放源代码，提供人类判断数据，鼓励开发更先进的评估方法。
SEED的设计围绕三个主要组成部分展开：ObjectF1、Cap-Sim和EffNet。具体方法如下：
ObjectF1：利用MM-Grounding-DINO模型自动检测图像中的关键对象，并量化重建图像与真实图像之间的对象重合情况。通过计算对象召回率和对象精确率，得出ObjectF1分数。
Cap-Sim：通过生成图像的描述性文本，计算真实图像与重建图像之间的文本相似度。这一方法强调了图像的语义特征，而不仅仅是视觉特征。
EffNet：使用ImageNet预训练模型评估图像特征之间的相似性，补充ObjectF1和Cap-Sim的评估。EffNet提供了对图像风格和全球场景信息的额外洞察。
最终，SEED的计算为这三种指标的平均值，旨在提供更全面和一致的语义相似性评估。
在实验部分，研究者使用了来自NaturalScenesDataset(NSD)的数据集，进行了一系列评估以验证SEED的有效性。通过对1,000对真实图像和重建图像进行人类评估，比较了SEED与现有评估指标之间的一致性。结果显示，SEED在与人类判断的相关性方面表现最佳，显著优于其他指标。此外，研究还探讨了SEED的各个组成部分在不同场景下的表现，揭示了当前解码模型在重建关键对象方面的不足。实验结果表明，SEED不仅能够提供更准确的语义评估，还能够为未来的研究提供有价值的基准。
在SEED的评估方法中，研究者采用了三种不同的方式来判断图像之间的相似性。首先，ObjectF1就像一个自动的侦探，检查重建的图像中是否包含真实图像中的关键对象，并计算这些对象的匹配度。其次，Cap-Sim通过生成图像的描述性文字，比较这些文字的相似性，帮助我们理解图像的整体内容，而不仅仅是视觉元素。最后，EffNet则是利用一个预先训练的模型来评估图像的特征相似性，提供对图像风格和场景的更深刻理解。最终，SEED通过将这三种方法的结果结合起来，得出一个综合的评分，旨在更准确地反映这些图像在语义上的相似性。这样，我们就能更好地评估脑解码模型的表现。
2025-03-08｜TongjiU,M,XDU
http ://arxiv.org/abs/2503.06084v1
在本研究中，提出了一种名为可解释视觉提示调优（IVPT）的新框架，旨在提升视觉提示调优的可解释性。视觉提示调优是一种高效的技术，用于将预训练的视觉基础模型适应特定任务，然而，当前的研究在可解释性方面存在不足。本研究通过引入层次化概念原型，将视觉提示与人类可理解的语义概念关联，从而改善这一问题。IVPT通过将视觉提示与特定图像区域相结合，生成可解释的提示，并在不同的网络层次上进行结构化解释。通过在细粒度分类基准上进行全面的定性和定量评估，IVPT展现出比传统视觉提示调优方法和现有可解释方法更优越的可解释性和性能。
IVPT的核心方法包括三个主要部分：首先，概念原型发现模块（ConceptRegionDiscovery,CRD）将概念原型与特定图像区域关联，生成概念区域图。该模块通过学习每个图像区域与其对应的概念之间的关系，形成可解释的视觉提示。其次，内部区域特征聚合模块（Intra-regionFeatureAggregation,IFA）负责聚合与概念区域相关的特征，生成与每个概念原型对应的提示嵌入。最后，通过细到粗的层次化提示融合，IVPT在不同层次上整合细粒度和粗粒度的提示，确保在模型的多个层次上保持语义一致性和可解释性。这种方法不仅提高了模型的可解释性，还保持了分类性能。
在实验部分，IVPT在多个细粒度分类基准上进行了评估，包括CUB-200-2011数据集。研究采用了不同的视觉模型作为基础，评估了IVPT在可解释性和分类准确性方面的表现。实验结果表明，IVPT在一致性分数和稳定性分数上均优于传统的部分原型网络和视觉提示调优方法。此外，IVPT在分类准确性上也取得了显著提升，展示了其在保持高性能的同时提升可解释性的能力。通过对比分析，研究还验证了各个模块对整体性能的贡献，进一步证明了IVPT的有效性。
IVPT是一种新方法，旨在让计算机更好地理解和解释它们在图像中看到的内容。首先，它通过一个叫做概念区域发现的步骤，帮助计算机识别图像中的特定区域，并将这些区域与人类可以理解的概念相联系。接着，内部区域特征聚合模块会收集这些区域的特征，形成更具体的信息。最后，IVPT会将不同层次的信息进行融合，从细节到整体，确保计算机在做出判断时，能够清晰地解释它的思路。这种方法不仅帮助我们理解计算机的决策过程，还能提升它们在分类任务中的表现。通过这样的方式，我们能够更好地信任并利用人工智能的分析能力。
2025-03-10｜JHU,HRI-US
http ://arxiv.org/abs/2503.07591v1https ://bardisafa.github.io/PreSel
本研究提出了一种新的数据选择范式，称为预指令数据选择（PreSel），旨在优化视觉指令调优（VIT）过程中的数据处理效率。传统的VIT方法往往需要大量的图像-指令对，生成这些指令的成本高昂且耗时。为了解决这一问题，PreSel直接从未标记的图像中选择最有价值的样本，并仅为这些选定的图像生成指令，从而显著降低了指令生成的成本和VIT的运行时间。通过对任务重要性进行评估，PreSel能够有效地分配资源，确保在有限的计算预算内实现与全数据集相媲美的性能。实验结果表明，PreSel在LLaVA-1.5和Vision-Flan数据集上表现优异，展示了其在多任务VIT数据集创建中的应用潜力。
PreSel方法分为两个主要阶段：任务重要性评估和基于任务的聚类选择。首先，任务重要性评估通过随机从每个视觉任务中选择小部分未标记图像，生成相应的指令，并计算指令相关性分数（IRS），以确定每个任务的相对重要性。这一阶段的目标是合理分配样本预算，以确保在最终选择的子集中，各任务的样本比例符合其重要性。接下来，使用轻量级视觉编码器（如DINOv2）提取未标记图像的特征，并在每个任务内进行聚类，选择最具代表性的图像。最终，PreSel只为选定的图像生成指令，完成后续的LVLM微调。这种方法不仅提高了数据选择的效率，还降低了指令生成的资源消耗。
在实验部分，我们在两个大型VIT数据集上评估PreSel的有效性：LLaVA-1.5和Vision-Flan。我们设置了15%的采样比例，仅为选定的图像生成指令。通过与多种基线方法（如随机选择、CLIP评分、Self-Filter等）进行比较，PreSel在多个多模态基准测试中表现出色，达到了与全数据集微调相似的性能。实验结果显示，PreSel的指令生成成本仅为其他方法的15%，而且在不同任务的多样性下保持了稳定的性能。此外，PreSel在不同采样比例下的表现也表明其在处理任务多样性时的鲁棒性，进一步证明了其在实际应用中的潜力。
我们的研究提出了一种新的方法，叫做预指令数据选择（PreSel），它可以帮助计算机更高效地学习如何理解和响应图像。传统方法需要大量的图像和对应的文字描述，生成这些描述非常耗费时间和金钱。PreSel的工作原理是先从大量没有说明的图像中挑选出最有用的部分，然后只为这些选中的图像生成描述。这个过程分为两步：第一步是确定哪些图像最重要，第二步是从这些重要图像中选择最具代表性的。通过这种方式，PreSel不仅减少了生成描述的时间和成本，还保证了计算机在学习时能够得到最有效的信息。这使得在创建新的学习数据集时变得更加简单和高效。
