2025-03-09｜EPFL
http ://arxiv.org/abs/2503.06740v1
本研究提出了一种名为D3DR的方法，旨在解决在3D高斯斑点（3DGS）场景中自然插入对象时，光照一致性的问题。尽管已有多种方法尝试解决这一挑战，但大多数依赖于复杂的物理渲染，导致效率低下且效果不理想。D3DR利用了基于扩散模型的技术，能够自动调整插入对象的光照和阴影，使其与新场景的光照条件相匹配，从而实现更自然的插入效果。该方法通过优化扩散模型的DeltaDenoisingScore（DDS）目标，确保在插入对象后，能够有效修正其外观，保持与环境的一致性。研究表明，D3DR在重光照质量上显著优于现有方法，提升了0.5PSNR和0.15SSIM的指标。
D3DR方法的核心在于两个主要步骤：扩散模型个性化和DDS优化。首先，在扩散模型个性化阶段，使用DreamBooth技术对对象进行训练，以适应不同的光照条件。此阶段的输入包括对象图像、场景描述和光照方向，目的是生成多样化的对象图像，增加模型的泛化能力。其次，在DDS优化阶段，通过引入改进的DDS损失函数，调整对象的3D高斯参数。这一过程包括以下几个步骤：1）随机采样视角并渲染两个视图；2）生成高斯噪声并选择时间步；3）计算损失梯度并反向传播更新参数。使用这种两步优化策略，D3DR能够有效减少图像中的噪声和伪影，确保插入对象在新场景中的自然外观。
在实验部分，研究团队构建了一个多样化的数据集，包含不同的3DGS场景和对象，并进行了广泛的基准测试。通过与现有的TIP-Editor和R3DG方法进行比较，D3DR在多个性能指标上表现出色。在评估过程中，研究者们重点关注了PSNR和SSIM等指标，以量化插入对象的视觉一致性和自然性。此外，实验还探讨了IC-Light和两步DDS的影响，结果显示，使用这些技术显著提高了对象的光照质量和整体插入效果。研究表明，D3DR不仅在效率上优于传统方法，还在视觉效果上实现了显著提升，为3D对象插入技术的发展提供了新的思路。
D3DR方法的工作原理可以简单理解为两个步骤。首先，研究人员使用一种叫做“DreamBooth”的技术来训练一个模型，让它能够识别和适应不同光照下的对象。这个过程就像给模型提供了一本包含许多不同场景和光照条件的图画书，以帮助它学习如何在这些条件下正确显示对象。接着，模型会通过一种叫做“DDS”的方法来调整对象的外观，确保它在新场景中看起来自然。这个调整过程包括随机选择视角、生成噪声和计算损失，然后通过反向传播来优化参数。简单来说，D3DR就像一个聪明的艺术家，能够根据周围的光线和环境，灵活地调整对象的外观，让它们看起来更真实。
2025-03-10｜NTU,HarvardU,MIT,UCAS｜CVPR2025
http ://arxiv.org/abs/2503.07476v1
本研究提出了一种创新的锚点基础的三维高斯点云渲染技术，称为SOGS（Second-OrderAnchorforAdvanced3DGaussianSplatting）。该方法旨在提高渲染质量的同时，降低模型的复杂性和锚点特征的维度。SOGS通过引入基于协方差的二阶统计量，捕捉锚点特征维度之间的相关性，从而增强锚点特征的表现力，进而提升纹理和几何结构的渲染效果。与现有的Scaffold-GS模型相比，SOGS在保持较小锚点特征维度的情况下，显著提升了新视角合成的渲染质量，并减少了模型的整体大小。通过在多个广泛使用的基准数据集上的实验，结果表明SOGS在新视角合成中表现出色，能够有效解决大锚点特征与小锚点特征之间的矛盾。
SOGS的核心在于引入二阶锚点和选择性梯度损失，以实现更高质量的渲染。具体方法包括：
二阶锚点：通过协方差矩阵计算锚点特征之间的相关性，提取共变关系，从而进行特征增强。这使得SOGS能够捕捉到更复杂的纹理和结构模式。
选择性梯度损失：该损失函数通过比较渲染图像的梯度图和真实图像的梯度图，动态调整模型关注于难以渲染的区域。这一机制确保了模型在训练过程中能够专注于细节丰富的区域，提升渲染质量。
特征增强：结合二阶锚点和选择性梯度损失，SOGS在保持锚点特征维度较小的同时，有效地增强了特征表达能力，确保更高的渲染质量。
在实验部分，本研究采用了多个数据集来验证SOGS的有效性，包括Mip-NeRF360、Tanks&Temples和DeepBlending等。实验结果显示，SOGS在新视角合成任务中，均在PSNR、SSIM和LPIPS等指标上超越了Scaffold-GS，尤其是在锚点特征维度减小至16或12的情况下，依然保持了高质量的渲染效果。通过对比不同模型的渲染质量和模型大小，SOGS在减少模型复杂度的同时，显著提升了渲染效果。此外，消融实验进一步验证了二阶锚点和选择性梯度损失对于提升渲染质量的重要性，表明这两者的有效结合是SOGS成功的关键因素。
SOGS方法可以简单理解为一种通过“聪明的锚点”来提升三维图像渲染质量的技术。首先，它使用了一种新的锚点设计，能够更好地捕捉图像细节。想象一下，如果你在画画，使用不同颜色的铅笔来描绘细节，SOGS通过分析不同铅笔颜色之间的关系，帮助你更好地选择和混合颜色。其次，SOGS还引入了一种“选择性梯度损失”方法，类似于在画画时专注于那些难画的地方，比如复杂的纹理和边缘，这样能确保这些部分更加生动。通过这两种方法，SOGS不仅能在更小的模型下提供更高质量的图像，还能减少计算的复杂性，使得渲染速度更快。
2025-03-09｜MurdochU,IMTNordEurope
http ://arxiv.org/abs/2503.06762v1https ://grbfnet.github.io/
近年来，神经场（NeuralFields）在多维信号表示方面取得了显著进展，尤其是在RGB图像、3D几何体和辐射场的重建与合成中。然而，传统的深度神经网络在训练和推理过程中通常速度较慢且内存消耗高。尽管有一些加速技术被提出，但往往在速度和内存之间存在权衡。本文提出了一种新颖的神经表示方法——高斯径向基函数网络（GaussianRBFNet），旨在实现快速且高效的表示和重建神经场。通过将传统的多层感知机（MLP）中的神经元替换为高斯径向基函数，研究表明只需一层即可实现对2D、3D及5D信号的高精度表示。该方法不仅训练速度快（3D几何体重建不到15秒），而且在推理时可实现每秒超过60帧的渲染速度，且不牺牲质量。
本文的方法主要包括以下几个部分：首先，采用径向基函数（RBF）作为神经元，构建一个单层网络以解码低分辨率特征网格。相比于传统的多层感知机，RBF网络能够以更少的参数实现高效的多维信号表示。其次，训练过程中使用分层特征网格进行输入编码，通过三线性插值获取周围网格的特征。最后，采用联合优化的方式，网络权重和RBF参数在训练时共同更新，从而提高模型的表现力和训练效率。通过这种高效的结构，模型能够在较短时间内完成复杂信号的训练，并在推理阶段展现出优异的性能。
为验证所提出方法的有效性，本文进行了多项实验，包括3D几何体、RGB图像和辐射场的重建。实验结果显示，所提出的高斯RBF网络在Chamfer距离、法线一致性和交并比等指标上均优于现有主流方法，如NeuRBF和InstantNGP。此外，模型的训练时间显著减少，3D几何体重建仅需不到15秒，且在推理阶段能够实现每秒超过60帧的渲染速度。通过对比不同模型的表现，本文的高斯RBF网络在保持优质重建效果的同时，展现了更好的内存效率和计算速度。
在这项研究中，科学家们提出了一种新方法，名为高斯RBF网络，来更快更高效地处理和重建图像和三维形状。传统方法需要使用多层的神经网络，这样会消耗很多时间和内存。高斯RBF网络的核心思想是使用一种叫做径向基函数的简单计算单元，只需一层就能处理复杂的数据。首先，它通过一种特殊的方式（叫做特征网格）来获取输入数据的特征，然后在训练时同时调整网络的参数，以便更好地学习数据。最后，这种方法不仅能快速训练，还能在生成新图像时保持高质量，表现出色。简而言之，这项研究使得处理复杂图像和三维形状变得更加高效和快速。
2025-03-08｜AjouU,M&C
http ://arxiv.org/abs/2503.06179v1https ://forestsplats.github.io/
本文提出了一种名为ForestSplats的新方法，旨在从不受约束的2D图像集中有效地分离静态场景和瞬态元素。传统的3D高斯喷溅（3D-GS）技术在处理静态场景时表现良好，但在面对现实世界中的瞬态对象、光照变化和遮挡时性能显著下降。ForestSplats通过引入可变形瞬态场和超像素感知掩码，解决了现有方法在处理动态元素时的局限性。该方法无需依赖预训练的视觉基础模型（VFM），有效地捕捉瞬态元素，同时保持高质量的渲染效果。通过对多个基准数据集的广泛实验，ForestSplats展示了在瞬态元素表示方面的显著内存效率和优越性能。
ForestSplats的核心方法包括三个主要部分：1)可变形瞬态场：该部分通过动态调整瞬态元素的表示，捕捉每个视图中的瞬态特征，以提高渲染效果并减少内存消耗。2)超像素感知掩码：采用此掩码来清晰界定静态场景与瞬态元素的边界，特别是在存在遮挡物的情况下。该掩码结合了光度误差和超像素信息，确保了更精确的分离。3)不确定性感知稠密化：该策略在优化过程中避免在遮挡物边界生成高斯体，从而提高了渲染质量。整个方法通过多阶段训练方案进行联合优化，确保静态和瞬态场的有效分离，并提升整体性能。
在实验部分，ForestSplats在多个数据集上进行了评估，包括NeRFOn-the-go和PhotoTourism数据集。研究表明，该方法在不同遮挡水平下均表现出色，优于现有的最先进技术。通过使用PSNR、SSIM和LPIPS等指标进行量化评估，ForestSplats显示出较高的图像重建质量和内存效率。实验结果表明，采用可变形瞬态场和超像素感知掩码的组合显著改善了瞬态元素的捕捉能力。相较于依赖VFM的传统方法，ForestSplats在实时渲染和训练速度方面也表现出了优势，证明了其在动态场景重建中的有效性。
ForestSplats的方法可以简单理解为一个智能的图像处理工具，它能从一堆照片中分离出静态背景和动态元素，比如行人或飘动的树叶。首先，它利用一种灵活的方式来表示这些动态元素，使得在不同角度下都能捕捉到它们的变化。接着，ForestSplats使用一种叫做超像素的技术，帮助它更清晰地识别哪些部分是静态的，哪些是动态的，这样即使在复杂的场景中也能准确分离。最后，它还引入了一种“不确定性感知”的方法，确保在处理图像时不会在错误的地方生成多余的动态元素，从而保证图像的质量。整体而言，这个方法不仅高效，而且能够在不需要额外复杂模型的情况下，提供高质量的图像合成效果。
2025-03-10｜HUST,USC｜AAAI2025
http ://arxiv.org/abs/2503.07000v1https ://github.com/whoiszzj/FDS-GS
本研究提出了一种名为FDS-GS的新方法，旨在通过重新参数化和频率感知的密度控制策略，改进3D高斯点云的渲染质量。传统的3D高斯点云渲染在处理高频信息时，往往由于缺乏密度与规模之间的明确约束，导致使用不适当的高斯规模，从而影响了细节的准确呈现。FDS-GS通过建立密度与高斯规模之间的直接关系，确保在高频区域生成更多小规模的高斯点，从而更好地表达场景的细节变化。此外，采用动态阈值的密度控制策略，结合基于规模的过滤方法，进一步提高了渲染质量，并减少了所需的高斯数量。实验结果表明，FDS-GS在多个数据集上均超越了现有的最先进方法，展现了其在高质量渲染中的有效性。
FDS-GS方法分为几个关键步骤，以实现高质量的3D渲染。首先，通过对每个高斯的局部密度进行量化，建立密度与规模之间的直接关系。具体而言，密度是通过计算每个高斯周围单位体积内的高斯数量来估算的。其次，重新参数化高斯的规模，将其分解为绝对规模标量和相对规模向量，以便在高频区域使用小规模高斯。接下来，采用动态阈值策略进行密度控制，实时调整密度以响应场景的频率变化。最后，应用基于规模的置信度过滤器，筛选出不合适规模的高斯，从而确保密度与规模的一致性。这种方法不仅能有效捕捉场景的频率变化，还能减少高斯数量，提高渲染效率。
在实验部分，FDS-GS方法与多种现有的最先进方法进行了比较，包括Instant-NGP、Mip-NeRF360和3DGS等。使用多个数据集（如Mip-NeRF360、Tanks&Temples和DeepBlending），通过PSNR、SSIM和LPIPS等指标对渲染质量进行定量评估。实验结果显示，FDS-GS在所有测试中均取得了显著的性能提升，特别是在细节恢复方面。此外，进行了消融实验，验证了密度与规模链接、动态阈值和规模置信度过滤器的有效性。结果表明，FDS-GS不仅在渲染质量上优于其他方法，而且在高斯数量控制上也表现出更好的效率，进一步证明了其在复杂场景渲染中的优势。
在FDS-GS方法中，我们主要通过控制高斯的数量和大小来改善3D图像的渲染效果。首先，我们计算每个高斯点周围有多少其他高斯点，这帮助我们了解密度。接着，我们将每个高斯的大小分为绝对大小和相对大小，这样在细节丰富的地方，我们可以使用更小的高斯点来捕捉更多的细节。然后，我们设置一个动态的阈值，来决定在什么情况下需要增加或减少高斯点的数量，确保在重要的细节区域有足够的小高斯点。最后，我们还会根据高斯的大小来判断它们的有效性，删除那些不合适的高斯点。通过这些步骤，FDS-GS能够在保持高质量渲染的同时，减少需要处理的高斯点数量，从而提高效率。
2025-03-10｜UCSD,UMinnesota
http ://arxiv.org/abs/2503.06859v1
在计算机视觉领域，图像选择对于高效的场景渲染至关重要。传统的高斯点云渲染方法（GaussianSplatting,GS）通常依赖于被动选择的图像集，这些图像往往冗余且密集，导致计算效率低下和存储需求增加。本文提出了一种新颖的框架，称为“ActiveInitSplat”，旨在通过主动选择训练图像来优化GS的初始化和训练过程。该方法通过分析3D场景的密度和占用率，确保所选图像来自多样化的视角，从而提高场景覆盖率和高质量3D结构的对齐。通过数值测试，ActiveInitSplat在多个模拟和现实环境中显著提升了GS的渲染性能，相较于传统的被动选择方法，表现出更好的效果。
ActiveInitSplat的核心在于通过主动选择最具信息量的训练图像来优化GS的性能。其方法步骤包括：
多样化视角选择：通过评估3D场景的密度和占用率，选择来自不同视角的图像，以确保场景的全面覆盖。
优化目标函数：设计一个优化公式，将密度和占用率作为目标函数，指导图像选择过程。该函数基于所选图像生成的3D点云质量进行评估。
高斯过程建模：利用高斯过程作为黑箱优化工具，动态评估不同相机视角的效果，通过历史观测数据更新模型。
迭代选择：在每次迭代中，选择最优相机视角，以最大化目标函数的值，确保所选图像能够提升GS的初始化质量。
为验证ActiveInitSplat的有效性，研究者在多个真实和模拟数据集上进行了实验。实验中，主动选择的图像与两种被动选择方法进行了对比：一种是标准的被动选择，另一种是随机选择。通过LPIPS、SSIM和PSNR等指标评估渲染性能。结果显示，ActiveInitSplat在所有测试场景中均表现出更高的r值，表明其生成的点云在密度和场景覆盖方面优于被动选择方法。此外，ActiveInitSplat在每次迭代中都能更快地收敛到最佳渲染性能，显示出其在实际应用中的潜在优势。
ActiveInitSplat的工作原理可以简单理解为“智能选择最佳拍摄角度”。它通过分析3D场景的特征，主动挑选在不同角度拍摄的图像，以确保捕捉到尽可能多的细节和信息。这个过程就像是在拍摄一部电影时，导演会选择最能展现故事的镜头，而不是随意选择。通过使用一种叫做“高斯过程”的数学工具，ActiveInitSplat能够根据已有的拍摄数据，不断调整并选择下一个最佳拍摄角度，从而提高最终图像的质量。这样的方法不仅提高了渲染的效率，还能节省计算资源，确保生成的3D场景更加真实。
2025-03-09｜SZU
http ://arxiv.org/abs/2503.06587v1
在现代计算机视觉领域，2D高斯点云（2DGaussianSplatting,2DGS）技术被广泛应用于高精度表面重建。然而，该方法在处理光泽表面时存在明显的局限性，尤其是在高光区域，常常导致重建结果中出现明显的孔洞。本文提出了一种新的方法，通过引入无偏深度（unbiaseddepth）来解决这一问题，从而显著提高了表面重建的几何质量。具体来说，研究者们分析了光泽表面反射不连续性的问题，并提出了一种新的深度收敛损失（depthconvergenceloss）来确保深度连续性。此外，改进后的深度标准能够更准确地确定实际表面，从而有效填补高光区域的孔洞，最终实现更完整和准确的表面重建。
本文的方法主要由两个核心组件构成。首先，研究者们引入了一种新的深度收敛损失，该损失函数旨在促进高光区域的边缘生长，从而增强光泽区域的表面几何重建。具体来说，该损失函数通过约束高斯点的深度连续性，确保它们在优化过程中向真实表面靠拢。其次，本文提出了一种深度修正标准，该标准在确定实际表面时考虑了沿光线的交叉高斯点数量及其累积不透明度。这种新的标准能够有效识别出实际的表面位置，从而显著提升重建的准确性。这两项创新结合，使得2DGS在处理光泽表面时能够克服传统方法的不足，提供更为精细的重建结果。
本文在多个数据集上进行了广泛的实验，以验证所提出方法的有效性。实验结果表明，采用无偏深度的2DGS相比于传统的2DGS在几何重建质量上有显著提升。具体而言，在DTU数据集上，所提出的方法在Chamfer距离测量中表现出最高的几何准确性，超越了其他基线方法。此外，针对Mip-NeRF360和TanksandTemples数据集的实验也显示出本方法在处理复杂场景时的鲁棒性，能够有效消除由高光反射引起的伪影，并成功重建出更为细致的几何细节。通过定量与定性评估，研究者们证实了新方法在高光区域的孔洞填补和表面重建的优越性。
在这项研究中，研究者们发现传统的2D高斯点云方法在处理光滑和反光的表面时表现不佳，容易出现孔洞。为了解决这个问题，他们提出了两种新方法。首先，他们设计了一种新的损失函数，帮助高斯点在优化过程中更好地靠近实际表面，尤其是在光泽区域。这个过程就像是在填补空洞，让表面更加完整。其次，他们还引入了一种新的标准，帮助更准确地判断哪里是实际的表面。通过结合这两种方法，研究者们能够在重建过程中更好地捕捉到光滑表面的细节，最终使得重建的结果更加精确和完整。简单来说，这项研究让计算机在处理反光物体时表现得更聪明，能够更好地“看”到真实的表面。
2025-03-09｜UTS
http ://arxiv.org/abs/2503.06462v1
在3D重建领域，StructGS框架的提出代表了3D高斯点云渲染技术的显著进步。该方法结合了3D高斯散射（3DGS）与神经渲染技术，通过对一系列3D高斯体的精细调整，能够有效地重建具有体积连续性的真实场景。尽管现有的3DGS模型在渲染质量上取得了进展，但在捕捉非局部结构信息的随机特性和细节分辨率方面仍存在不足。StructGS通过引入补丁基础的结构相似性损失（P-SSIM）、动态球谐函数初始化和多尺度残差网络（MSRN），旨在克服这些限制。实验结果表明，StructGS在渲染质量和细节捕捉上均显著优于现有的最先进模型，且在较低分辨率输入下能实现高分辨率的渲染效果。
StructGS的核心方法包括三个主要创新点。首先，采用补丁基础的结构相似性损失（P-SSIM），该方法通过随机选取图像补丁对比渲染图像与真实图像的非局部结构信息，从而提升训练效果。其次，动态调整球谐函数的初始化，根据高斯体的透明度和相邻点的距离，优化球谐函数的系数初始化。这种方法能够有效减少高阶项的计算冗余，提升训练效率。最后，集成预训练的多尺度残差网络（MSRN），它通过对低分辨率输入进行特征提取，直接生成高分辨率图像，避免了传统方法中高分辨率训练所带来的内存消耗和时间延长。这些方法的结合使得StructGS在3D重建中实现了更高的渲染质量和较低的计算成本。
在实验部分，StructGS框架的性能通过与多个最先进的3D重建模型进行比较来评估。使用了来自多个公开数据集的18个场景进行测试，评估指标包括峰值信噪比（PSNR）、结构相似性指数（SSIM）和学习感知图像补丁相似性（LPIPS）。实验结果显示，StructGS在各个数据集上均优于对比模型，尤其在细节表现和清晰度方面表现突出。具体而言，StructGS在处理复杂场景时，能够有效减少模糊和伪影现象，保持图像的真实感和细节。通过对比不同模型的结果，StructGS在高频细节的保留上表现尤为出色，显示出其在实际应用中的潜力和优势。
StructGS的工作原理可以简单理解为通过“聪明”的方式来处理和渲染3D图像。首先，它分成许多小块来分析图像，这样可以更好地捕捉到图像中细微的结构变化。然后，它会根据每个点的透明度和与其他点的距离来调整颜色和细节，这样即使是远处的物体也能清晰可见。最后，StructGS使用一个强大的网络来把这些低分辨率的图像转换成高分辨率的图像，确保最终效果既清晰又真实。通过这样的处理，StructGS不仅提高了渲染的质量，还减少了计算所需的时间和资源，能够在各种场景中表现出色。
2025-03-08｜PKU,PKU(SZ)｜ICLR2025
http ://arxiv.org/abs/2503.06118v1
在近年来，3D高斯点渲染（3DGaussianSplatting,3DGS）因其出色的实时渲染能力和高质量输出而成为主流的3D表示方法。然而，随着3D资产隐私保护需求的增加，传统的神经辐射场（NeRF）隐写方法无法有效应对3DGS的透明性问题。因此，现有的3DGS隐写解决方案在渲染保真度、计算需求和安全性方面仍存在不足。为此，本文提出了一种名为SecureGS的安全高效的3DGS隐写框架，采用了混合解耦高斯加密机制，能够在锚点特征中嵌入隐藏的3D高斯点属性，且仅能由授权用户通过隐私保护的神经网络进行检索。此外，本文还提出了一种密度区域感知的锚点生长和修剪策略，以自适应地定位最佳隐藏区域，从而增强隐私安全。实验结果表明，SecureGS在渲染保真度、速度和安全性方面显著优于现有的3DGS隐写方法。
SecureGS框架的设计主要包括以下几个方面：
混合解耦高斯加密机制：该机制通过锚点特征嵌入隐藏3D高斯点的偏移量、缩放、旋转和RGB属性。此过程确保了原始场景和隐藏信息的高效渲染。
锚点特征设计：采用稀疏点云作为输入，锚点被赋予局部上下文特征和可学习的偏移量，以生成表示原始场景和隐藏对象的密集高斯点。
区域感知密度优化：通过计算锚点的梯度，动态调整高斯点的生长和修剪策略，以确保隐藏对象的几何结构不暴露。
训练细节：使用像素级损失、结构相似性（SSIM）和体积正则化等多种损失函数进行优化，确保隐写信息的高保真度和安全性。这些方法的结合使SecureGS在隐写过程中有效地提升了渲染的质量和安全性。
在实验部分，SecureGS框架被应用于隐藏3D对象、2D图像和位信息，评估其在不同场景下的性能。实验首先在多个3D场景中进行，使用Mip-NeRF360数据集作为基础场景，并从Blender数据集中提取隐藏的3D对象。对于位信息的隐藏，SecureGS能够实现100%的解码准确率，且在渲染质量方面，SecureGS在PSNR和SSIM等指标上均优于对比方法GS-Hider和3DGS+StegaNeRF。此外，SecureGS在存储空间和渲染速度上也表现出显著优势，减少了存储需求并提高了渲染效率。通过对比实验，SecureGS展示了其在隐写保真度、速度和安全性方面的卓越性能，验证了其在实际应用中的潜力。
在SecureGS方法中，我们主要通过几种简单的方式来确保信息的安全和隐蔽。首先，我们使用一种特殊的加密技术，将隐藏的信息（比如3D物体或图片）嵌入到一个叫做“锚点”的特征中。这些锚点可以想象成一个个小标记，它们帮助我们在场景中找到合适的位置来放置隐藏的信息。其次，我们还设计了一种智能的生长策略，确保这些锚点在需要的地方生长得更密集，而在其他地方保持稀疏，这样可以有效遮掩隐藏的信息，防止被发现。最后，我们在训练过程中使用了一系列的损失函数，确保渲染出来的图片看起来真实且不会泄露隐藏的信息。通过这些方法，SecureGS能够在保持高质量的同时，有效地保护隐私。
2025-03-10｜NTU
http ://arxiv.org/abs/2503.07191v1https ://github.com/RY-Paper/KeySS
本研究提出了一种新颖的端到端3D隐写框架，称为KeySS，旨在通过3D高斯点云在3D场景中隐藏秘密信息。该框架的核心在于结合3D高斯点云的特性与密钥保护机制，以实现高保真度的秘密重建，同时确保隐写过程的不可察觉性。当前的3D隐写方法在隐蔽性和重建质量之间存在显著的折中，容易受到检测。KeySS通过优化3D高斯模型和密钥保护解码器，克服了这些局限性。研究表明，不同的高斯特征对隐写效果的贡献并不均衡，因此，KeySS系统地探索了最优特征组合，以平衡隐写的安全性和重建的保真度。此外，提出了3D-Sinkhorn距离作为新颖的安全评估指标，超越了传统的2D评估方法，为3D隐写的研究提供了新的方向。
KeySS框架的设计包括三个主要部分：3D高斯模型的构建、密钥保护解码器的设计以及特征贡献的探索。首先，3D高斯模型用于表示覆盖场景和隐藏信息，通过对N个各向异性高斯的优化，学习其中心位置、透明度、旋转、缩放和颜色等特征。其次，密钥保护解码器通过用户特定的密钥来控制解码过程，确保只有授权用户能重建秘密场景。该解码器由共享分支和特征特定分支组成，前者负责提取全面的3D高斯特征，而后者则专注于不同高斯属性的系统化量化。最后，特征贡献探索通过结合不同的高斯特征，分析其对隐写效果的影响，确保在保证隐写安全性的同时，最大限度地提高重建质量。
在实验部分，研究使用了来自Mip-NeRF360和DeepBlending数据集的多个场景进行验证。通过对比KeySS与现有方法（如GS-Hider和WaterGS），评估其在单一秘密和多重秘密隐写中的表现。实验结果表明，KeySS在视觉质量和重建保真度上均超过了现有技术，且在安全性方面表现出色。特别是，KeySS在多重秘密隐写中能够有效地保持覆盖场景的完整性，同时确保每个秘密信息的准确恢复。此外，研究还通过3D-Sinkhorn距离分析了隐写的不可察觉性，验证了该方法在面对不同检测攻击时的鲁棒性。实验结果展示了KeySS在3D隐写领域的前沿性能，为未来的研究提供了坚实的基础。
KeySS框架的工作原理可以简单理解为一个“安全的秘密盒子”，它能在一个3D场景中隐藏多个秘密信息。首先，框架会创建一个3D模型，这个模型就像一个多层的立体图像，其中每一层都包含不同的信息。然后，框架会使用一个特殊的“钥匙”来控制谁可以打开这个秘密盒子。只有拥有正确钥匙的人才能看到隐藏的信息，而其他人即使看到这个3D场景，也无法发现里面的秘密。为了确保这个秘密盒子不会被轻易打开，KeySS还会分析不同的特征，比如透明度和位置，以找到最佳的隐藏方法，从而让这些秘密更加安全。最终，KeySS不仅能有效隐藏秘密，还能在需要时准确恢复这些秘密，确保信息的安全和隐私。
