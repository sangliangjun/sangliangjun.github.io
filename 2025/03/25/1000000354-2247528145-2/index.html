2025-03-09｜ECNU,Xiaohongshu
http ://arxiv.org/abs/2503.06749v1https ://github.com/Osilly/Vision-R1
本研究提出了一种新型多模态大型语言模型，名为Vision-R1，旨在提升模型的推理能力。通过构建高质量的多模态链式思维（CoT）数据集，研究者利用现有的大型语言模型（MLLM）和DeepSeek-R1技术生成了200K条无人工标注的推理数据，作为模型的冷启动初始化数据。Vision-R1结合了冷启动初始化和强化学习（RL）训练，以应对推理能力不足的问题。尽管直接应用RL训练存在挑战，Vision-R1通过渐进思维抑制训练（PTST）策略，成功引导模型克服过度思考的优化问题，最终实现了与更大参数模型相媲美的推理性能。
Vision-R1的训练方法包括几个关键步骤。首先，通过模态桥接技术，利用现有的MLLM生成“伪CoT”推理文本，结合图像-文本对，形成高质量的多模态CoT数据集。其次，采用渐进思维抑制训练（PTST）策略，初期限制推理过程的长度，以引导模型学习正确的推理方法。随着训练的进行，逐步放宽这些限制，鼓励模型生成更复杂的推理过程。最后，使用群体相对策略优化（GRPO）与硬格式结果奖励函数，进一步增强模型的推理能力。该方法不仅提升了模型的推理质量，还有效解决了在冷启动初始化后出现的过度思考问题。
在实验部分，研究者使用多种数学推理基准测试对Vision-R1进行评估，包括MathVista、MathVerse和MM-Math等。实验结果显示，Vision-R1-7B模型在这些基准上表现出色，尤其是在数学推理任务中，其准确率达到73.5%，与领先的推理模型相差仅0.4%。此外，通过与多种现有模型的比较，Vision-R1在多个测试中均表现出优越的推理能力，证明了其在多模态推理任务上的有效性。最终，研究者还计划公开数据集和代码，以促进后续研究。
Vision-R1的训练方法可以简单理解为三个步骤。首先，研究者通过现有的语言模型生成了一种叫“伪CoT”的推理文本，结合图像和文字，制作出一个高质量的数据集。接下来，为了让模型更好地学习思考，研究者在训练初期限制了模型的思考长度，这样可以帮助它集中注意力，掌握基本的推理方法。随着训练的深入，研究者逐步放宽这些限制，让模型能够处理更复杂的问题。最后，研究者使用一种特殊的奖励机制，鼓励模型在推理时更具创造性和准确性。这种方法有效提升了模型的推理能力，并解决了在训练过程中可能出现的过度思考问题。
2025-03-09｜WHU,DIIS,ShanghaiAILab,USTC,NKU,HZAU
http ://arxiv.org/abs/2503.06553v1https ://github.com/jiaxin-ai/ProJudge
在科学问题求解中，多模态大语言模型（MLLMs）经常出现错误，因此评估其推理过程的有效性至关重要。这不仅可以确保模型的可靠性，还能揭示其潜在的弱点。由于人工评估既费时又昂贵，利用MLLMs作为自动化过程评估者已成为一种普遍做法。然而，这种基于模型的评估者的可靠性仍然存在不确定性。为了解决这一问题，本文引入了ProJudgeBench，这是首个专门设计用于评估MLLMs作为过程评估者能力的综合基准。该基准包含2400个测试案例和50118个步骤级标签，涵盖四个科学学科，具有多样的难度和多模态内容。每个步骤都经过专家的细致标注，确保了对模型检测、分类和诊断错误能力的系统评估。此外，研究还揭示了开源模型与专有模型之间存在显著的性能差距。
为了解决MLLMs在过程评估中的能力不足，本文提出了ProJudge-173k，这是一个大规模的指令调优数据集，并引入了动态双阶段微调策略。方法的主要步骤包括：
数据构建：从多个公开基准中收集问题，确保涵盖多种科学学科和难度水平。生成的解决方案来自多种不同的MLLMs，以确保数据的多样性。
错误类型定义：对模型在推理过程中常见的错误进行分类，定义了七种错误类型，包括推理错误、视觉解释错误、数值计算错误等。
动态双阶段微调：该策略包含两个训练阶段，第一阶段直接评估学生模型的解答，第二阶段鼓励模型在评估之前先进行问题推理。这种方法旨在增强模型的推理能力及其对解决方案的理解，从而提高评估的准确性。
为了全面评估不同模型在ProJudgeBench上的表现，研究选择了包括开源和专有MLLMs在内的多样化模型进行测试。实验结果显示，模型规模对过程评估能力具有显著影响。较大的模型在步骤正确性和错误类型分类的准确性上表现更佳。例如，InternVL系列模型的步骤正确性准确率随着模型规模的增加而显著提升。研究还发现，经过微调的开源模型在性能上接近专有模型，表明通过针对性的微调可以显著改善模型的评估能力。此外，实验揭示了不同模型在处理不同类型错误时的独特优势，为未来的模型改进提供了重要的见解。
在这项研究中，研究者们想要解决的问题是：如何让大型语言模型更好地检查和评估自己的推理过程。为了做到这一点，他们提出了一种新的方法，首先收集了大量科学问题和模型的解答，然后对这些解答进行仔细的错误分类，比如推理错误或计算错误。接着，他们设计了一种训练方式，让模型在评估之前先进行思考和推理，这样模型就能更好地理解问题，进而做出更准确的评估。通过这种方法，研究者们希望提高模型的能力，让它们在判断其他模型的解答时更加可靠。最终的实验结果表明，经过这种训练的模型在各个方面的表现都得到了显著提升。
2025-03-10｜THU,Microsoft
http ://arxiv.org/abs/2503.07523v1https ://github.com/zhangquanchen/VisRL
本文提出了一种新颖的学习框架VisRL，旨在通过强化学习（RL）优化视觉感知过程，使其更具人类意图驱动的特征。当前的视觉理解方法往往忽视了人类在观察场景时的目标导向性，导致模型在处理视觉信息时缺乏灵活性和有效性。VisRL通过奖励信号来指导模型的学习，而不是依赖于密集的标注数据，从而解决了数据标注的可扩展性问题。该框架在多个基准测试中表现出色，超越了许多现有的强基线，展示了其在多模态模型中的广泛适用性。
VisRL的核心方法包括两个主要阶段：数据生成和模型优化。具体步骤如下：
数据生成阶段：
利用多次采样生成候选的边界框（boundingboxes），并通过多样性控制器确保生成的框覆盖广泛的关注区域。
通过对生成的边界框进行过滤，选择适合当前模型能力的问题，以确保模型能够有效学习。
模型优化阶段：
第一步，模型生成一个代表关注区域的边界框，并从原始图像中裁剪出相应区域。
第二步，将裁剪后的区域与原始图像和问题一起输入到多模态模型中，生成最终的响应。
应用逐步偏好优化（DPO）算法来优化视觉推理过程，具体分为两个步骤：
通过奖励信号来指导每一步的学习，确保模型能够逐步完善其推理能力。
在实验部分，VisRL在多个基准测试上与当前最先进的方法进行了比较。结果显示，VisRL在不同类型的任务中表现优异，尤其是在视觉问答（VQA）和图像描述等领域。实验通过对比不同训练阶段（如SFT、RL1和RL2）展示了模型性能的逐步提升。通过逐步优化边界框和最终响应，VisRL在实现意图驱动的视觉感知方面取得了显著的进展。此外，实验还表明，VisRL在使用较少的标注数据（仅30k）时，依然能够达到或超过其他方法的性能，验证了其在数据稀缺环境中的有效性。
VisRL的工作方式可以简单理解为一个两步走的学习过程。首先，模型会从图像中生成多个可能的“关注区域”，就像我们在看一张照片时，可能会特别注意某些部分。为了确保这些区域的多样性，模型会进行多次尝试。接下来，模型会根据每个区域的表现来学习如何更好地理解图像。它会先选择一个区域，然后裁剪出这部分图像，再结合问题一起输入到模型中，生成最终的答案。通过不断的尝试和反馈，模型逐渐学会了如何像人类一样，依据不同的任务来选择最重要的视觉信息。这样一来，VisRL不仅提高了视觉理解的效果，还减少了对大量标注数据的依赖。
2025-03-10｜TRI,ColumbiaU,Stanford,MIT｜ICLR2025
http ://arxiv.org/abs/2503.07603v1
本文探讨了视觉语言模型（VLM）预训练过程中引入图像数据的必要性和效果。研究表明，经过图像数据的进一步训练的预训练大型语言模型（LLMs）在视觉语言任务上表现优异。然而，如何在预训练过程中有效地整合图像数据仍然存在争议。本文通过对不同数据集、模型规模、图像文本比例及预训练阶段图像数据引入时机的实验，评估了这些因素对模型下游性能的影响。结果显示，适当的图像数据引入策略能显著提升视觉语言任务的表现，同时保持文本任务的强劲性能。尤其是，在预训练过程中，约80%完成时引入视觉标记的模型在多个任务上相比于完全预训练的模型有2%的平均性能提升。
本文的研究方法包括三个主要步骤：首先进行部分文本预训练，接着将图像文本对与文本数据混合继续预训练，最后进行多任务微调。具体步骤如下：
文本预训练：使用现有的文本数据集进行初步训练，确保模型在语言理解上有良好的基础。
图像文本混合训练：在文本预训练的后期引入图像数据，特别是在文本预训练达到80%时进行图像数据的引入，以此打破传统的完全预训练后再引入图像的模式。
多任务微调：在图像文本混合训练后，进行针对特定任务的微调，以提升模型在视觉语言任务上的表现。
此外，实验中还对图像数据的比例、训练步骤的设计以及微调阶段的策略进行了详细分析，以优化模型的整体性能。
实验设计旨在评估不同图像文本数据引入策略对模型性能的影响。研究团队训练了300个不同参数规模的模型，涵盖了多种文本和图像数据组合。在实验中，首先评估了不同阶段引入图像数据的效果，发现最优的时机是在文本预训练完成80%时引入图像数据，这样可以显著提升视觉任务的表现。其次，研究了图像数据比例对模型性能的影响，发现10%到20%的图像数据比例能够达到最佳效果，而在较小模型中则需要更多的图像数据。最后，微调阶段的设计也被纳入考量，发现适当的微调策略能够进一步提升视觉语言任务的性能。通过这些实验，本文明确了图像数据在VLM预训练中的重要性及最佳实践。
在这项研究中，科学家们试图找出在训练计算机模型时，何时引入图像数据最有效。首先，他们让模型学习大量文本，然后在模型学习到80%时，突然加入图像数据。这个时机非常关键，因为它能帮助模型更好地理解图像和文字之间的关系。接下来，研究团队还测试了图像数据和文本数据的比例，发现加入10%到20%的图像数据能让模型表现最佳。最后，他们还进行了微调，这就像是在模型学习后给它一些额外的指导，以确保它在处理视觉和语言任务时更聪明。通过这些方法，研究者们发现了如何更好地训练模型，使其在理解图像和文本方面都能表现得更出色。
2025-03-10｜HKUST,Dartmouth
http ://arxiv.org/abs/2503.07503v1https ://cse.hkust.edu.hk/
本研究提出了一种名为ThinkFirst的新型推理分割框架，旨在提高复杂场景下的图像分割质量。传统的图像分割方法在处理模糊边界、遮挡物以及颜色相似性较高的对象时，往往表现不佳。ThinkFirst利用GPT-4o的链式思维（CoT）能力，首先对输入图像进行深入分析，生成详细的描述，然后将该描述与用户的查询结合，以指导分割模型进行更精确的分割。通过这种方式，ThinkFirst能够处理隐式查询和复杂背景下的对象分割，显著提升了分割的准确性和鲁棒性。
ThinkFirst框架的设计分为几个关键步骤。首先，预备知识部分介绍了推理分割代理的基本构成，包括视觉编码器、掩膜解码器和视觉语言模型。其次，利用GPT-4o的链式思维，框架通过以下步骤进行处理：
输入图像和用户查询被送入GPT-4o进行链式思维分析。
GPT-4o生成一系列问题和答案，以形成对图像的全面理解。
生成的总结信息与用户查询合并，形成最终的分割提示。
最后，将综合提示输入到分割代理中，生成目标对象的二进制掩膜。这一过程确保了模型在处理复杂场景时能够更好地理解图像内容。
在实验部分，研究者对ThinkFirst框架的有效性进行了全面评估。实验设置包括使用COD10K和MVK数据集，重点考察隐式和显式查询下的分割表现。结果显示，ThinkFirst在处理复杂的伪装图像时，gIoU和cIoU指标均显著高于基线模型LISA-13B-Llama2，尤其在隐式查询场景中，性能提升达2.2%-3.0%。此外，研究还展示了ThinkFirst在“找沃尔多”这一经典游戏中的应用，成功定位了沃尔多，证明了其在图像理解和分割任务中的强大能力。这些结果表明，ThinkFirst不仅提高了分割质量，也增强了用户交互的灵活性。
ThinkFirst的方法可以简单理解为一个智能助手，它帮助我们更好地理解和分割图像。首先，当我们给这个助手一张图片和一个问题时，它会仔细观察图片，像是在思考一样，逐步回答与图像相关的问题。这些问题可能涉及图像中的物体、颜色、位置等信息。然后，助手会把这些回答整理成一个总结，再结合我们的原始问题，生成一个详细的指令，告诉分割工具应该如何进行分割。通过这种方法，ThinkFirst能够更准确地识别复杂场景中的物体，尤其是那些难以辨认的部分，比如伪装的动物或模糊的边界。这样，我们就能更轻松地找到我们想要的东西，而不必担心被背景干扰。
2025-03-09｜CUHK,HKUST,RUC
http ://arxiv.org/abs/2503.06520v1https ://github.com/dvlab-research/Seg-Zero
Seg-Zero是一种新颖的框架，旨在通过认知强化学习实现推理链引导的分割。传统的分割方法通常依赖于监督微调和简单的标签描述，限制了其在不同领域的泛化能力。Seg-Zero通过引入一个解耦的架构，将推理模型与分割模型分开，前者负责理解用户意图并生成推理链，后者则利用这些推理信息生成精确的分割掩码。该框架的核心在于其强化学习策略，能够在没有明确推理数据的情况下，激活模型的推理能力，从而实现强大的零-shot泛化能力。实验结果表明，Seg-Zero在多个基准测试中表现优异，尤其是在处理复杂的推理任务时，相较于传统方法具有显著的性能提升。
Seg-Zero的设计包括几个关键步骤。首先，推理模型使用强化学习（RL）来激活其推理能力，生成区域级别的边界框和位置点，确保能够精确地定位目标对象。接下来，分割模型利用推理模型提供的边界框和点作为提示，生成细粒度的分割掩码。在训练过程中，Seg-Zero采用了复杂的奖励机制，包括格式奖励和准确性奖励，以有效引导模型的优化方向。具体来说：
推理模型：利用多模态大语言模型（MLLM）进行推理，生成结构化的思维过程和最终答案。
奖励机制：设计多种奖励函数，确保模型的输出符合预定格式，并通过评估预测的边界框与真实值的交并比（IoU）来促进准确性。
训练过程：采用GRPO算法对模型进行训练，不依赖于任何显式的推理数据，从零开始学习推理能力。
在实验部分，Seg-Zero的性能通过与传统的监督微调（SFT）方法进行对比。训练数据仅使用9,000个样本，测试数据包括RefCOCO和ReasonSeg等基准。实验结果显示，Seg-Zero在领域内和领域外的任务中均表现优异，尤其是在处理复杂的推理指令时，模型能够准确识别并分割目标对象。此外，通过对比不同的奖励机制与模型规模，研究发现强化学习结合推理链的方式显著提高了模型的泛化能力和准确性。实验还表明，相较于传统方法，Seg-Zero在处理多样化的输入时，能够更好地理解用户意图并生成符合要求的分割结果。
Seg-Zero的工作原理可以简单理解为两个部分：推理和分割。首先，推理模型就像一个聪明的助手，它通过分析用户的问题，生成一个“思考链”，这就像是它在做一个复杂的数学题时的步骤。这个思考链帮助它确定图像中需要关注的区域。然后，分割模型就像是一个画家，利用推理模型提供的信息，准确地在图像上标记出目标对象的边界。为了让这个过程更有效，Seg-Zero使用了一种奖励机制，确保助手的思考过程和画家的绘画都符合特定的格式和准确性要求。通过这种方式，Seg-Zero能够在没有太多训练数据的情况下，依然表现出色，处理各种各样的图像分割任务。
2025-03-10｜SJTU,ZJU,PolyU,EIT
http ://arxiv.org/abs/2503.07413v1https ://github.com/MacavityT/REF-VLM
REF-VLM是一种新颖的端到端框架，旨在统一多任务训练，以增强视觉解码的能力。该框架通过引入三元组引用范式（Triplet-BasedReferringParadigm,TRP），有效地处理复杂的视觉解码任务。REF-VLM的设计目标是解决现有多模态大型语言模型（MLLMs）在视觉任务中的局限性，尤其是在语义分割和关键点检测等密集预测任务中。通过构建Visual-TaskInstructionFollowingDataset（VT-Instruct），REF-VLM支持超过25种视觉任务的多模态对话样本，涵盖了100多万条样本。该框架的创新之处在于其灵活的任务适应性和高效的语义一致性，显著提升了模型在各种标准基准测试中的表现。
REF-VLM的核心方法包括以下几个关键组件：
三元组引用范式（TRP）：TRP将视觉概念、解码类型和引用标记整合为结构化的输出，增强了模型的可解释性。
视觉解码链（VD-CoT）：该方法要求模型在生成响应之前，首先对输入图像进行概述和任务相关信息的总结，从而提高生成的准确性。
掩码引导聚合（Mask-GuidedAggregation）：通过无参数的聚合方法，将视觉提示（如点、框、涂鸦和掩码）与图像特征相结合，增强了对复杂视觉场景的理解。
潜在嵌入路由器（LatentEmbeddingsRouter）：该组件用于有效处理多任务和多粒度解码场景，确保输出的灵活性和准确性。
并行分组匈牙利匹配：此技术确保在多目标任务中精确对齐预测与真实标注，提升了模型在复杂场景中的表现。
在实验部分，REF-VLM在多个视觉任务上进行了全面评估，包括视觉理解、引用表达、互动对话生成和开放词汇识别等。通过与其他先进的多模态模型进行比较，REF-VLM在图像字幕生成和视觉问答（VQA）等任务中表现出色。例如，在Flickr30k数据集上，REF-VLM的CIDEr得分达到了96.0，显示出其在图像描述生成方面的优势。此外，在VQA任务中，REF-VLM的准确率为81.6%，同样超越了其他模型。实验还包括对不同组件的消融研究，以验证三元组引用范式和视觉解码链的有效性，结果表明这些设计显著提高了模型的性能和适应性。
REF-VLM的工作方式可以简单理解为一个高效的视觉助手，它通过三种主要的“信息”来理解和处理图像。首先，模型会识别图像中的“视觉概念”，比如“狗”或“车”。接着，它会确定如何处理这些概念，比如是要框出物体、标记关键点还是进行图像分割。最后，模型会用引用标记来连接这些概念和具体的视觉实例，这样它就能在复杂的场景中做出准确的反应。通过这种方式，REF-VLM能够更好地理解用户的指令，生成更准确的结果，像是回答问题或描述图像中的细节。
2025-03-09｜PKU,XJTU
http ://arxiv.org/abs/2503.06472v1https ://github.com/LoYuXr/CalliReader
CalliReader是一种创新的视觉语言模型（VLM），旨在解决中文书法的上下文化问题。书法作为联合国教科文组织认定的非物质文化遗产，具有复杂的视觉特征和文化背景，给计算机视觉和自然语言处理带来了挑战。现有的AI系统在理解书法作品时常常面临数据不足和视觉语义对齐不佳的问题。为了解决这些问题，CalliReader通过三项关键创新来提升其性能：字符级切片技术，CalliAlign模块和嵌入指令调优（e-IT）。这些技术的应用使得CalliReader在全页面和区域级识别、跨语言解释及意图识别等任务中表现优异，超越了现有的最先进模型，并在真实场景和文档基准测试中展现出强大的泛化能力。
CalliReader的核心方法包括以下三个方面：
字符级切片：该方法通过检测和排序模块，将高分辨率图像切分为单个字符，简化了页面级分析，保留了字符的语义信息。这种方式提高了识别的准确性，并减少了后续处理的复杂性。
CalliAlign模块：该模块负责处理从视觉特征提取的字符，并将其映射到伪文本嵌入。通过这种方式，CalliAlign能够有效地压缩字符的视觉信息，并与标准化的文本嵌入对齐，从而提升了模型的推理能力和效率。
嵌入指令调优（e-IT）：此方法通过将真实和合成数据统一在同一嵌入空间中，增强了模型的训练效果。e-IT帮助CalliReader在有限的页面级数据下，提升了对上下文信息的挖掘能力，使其在识别和理解方面表现更加出色。
在实验部分，CalliReader的性能通过多个基准测试进行评估，包括全页面识别、区域幻觉检测和上下文问答（VQA）等任务。实验结果表明，CalliReader在精确度、召回率和F1分数等指标上均超过了现有的OCR工具和其他VLM模型。特别是在复杂的书法样本上，CalliReader展现出较强的抗幻觉能力，能够有效识别和理解书法作品的内容、风格和作者信息。此外，研究还进行了用户研究，结果显示CalliReader的表现超越了普通用户和书法专家的平均水平，进一步验证了其在真实场景中的应用潜力。
CalliReader的工作原理可以简单理解为三个步骤。首先，它使用一种叫做“字符级切片”的技术，把书法作品的图像分解成一个个单独的字符，这样可以更容易地识别每一个字。其次，CalliAlign模块会将这些字符的图像特征转换成一种可以理解的文本格式，帮助模型更好地理解书法的内容。最后，嵌入指令调优（e-IT）则通过结合真实和虚拟的数据，帮助模型在缺乏足够数据的情况下仍然能够学习和提高识别能力。这样，CalliReader就能在处理复杂的书法作品时，准确地识别出文字、风格和作者，甚至理解作品的文化背景。
