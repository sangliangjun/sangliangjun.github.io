2025-03-10｜UTS(Sydney),HFUT
http ://arxiv.org/abs/2503.07047v1
本研究提出了一种新颖的图像补全方法，旨在恢复部分损坏的主要物体。该方法利用三模态输入（损坏图像、文本提示和草图）来增强生成模型的控制能力，特别是在物体的姿态和结构细节的恢复上。传统的图像补全方法往往依赖于文本提示，但在处理部分遮挡的物体时效果不佳，无法精确恢复细节。为了解决这一问题，本文引入了一种名为视觉草图自适应（VSSA）的模块，将草图信息嵌入到扩散模型的每个迭代步骤中，从而提升生成的图像在语义和结构上的一致性。通过对比实验，结果表明该方法在恢复部分损坏物体的任务中优于现有的多种主流方法。
本研究的方法主要包括三个核心组件：掩码图像编码器、草图条件编码器和视觉草图自适应（VSSA）模块。首先，掩码图像编码器提取损坏图像的多尺度特征，明确区分损坏区域和未损坏区域。这一过程为后续的草图信息注入提供了空间定位。其次，草图条件编码器从草图中提取特征，确保生成过程中能够准确反映草图所描绘的物体姿态。最后，VSSA模块将草图和损坏图像的特征在每个扩散步骤中融合，使得生成的图像不仅保持语义一致性，还能细致地恢复物体的结构特征。通过这种方式，研究者能够在生成过程中实现更高的控制精度，确保最终图像的高质量和一致性。
为验证所提出方法的有效性，研究者在两个新构建的数据集（CUB-sketch和MSCOCO-sketch）上进行了广泛的实验。这些数据集包含了图像、草图和文本的配对数据，支持三模态图像补全任务。实验结果通过多种评估指标进行量化，包括美学评分（AS）、弗雷歇特距离（FID）和CLIP评分等。与多种先进的图像补全方法（如SD-Inpainting、BrushNet、MaGIC等）进行比较后，结果显示本方法在恢复部分损坏物体的任务中表现出色，尤其在细节控制和语义一致性方面。通过消融实验，研究者进一步分析了各个组件对整体性能的贡献，证实了VSSA模块和掩码图像编码器在提升生成质量中的重要性。
本研究的方法可以简单理解为一种智能的图像修复工具，它通过结合图像、文字和草图来帮助我们修复那些部分损坏的图片。首先，工具会识别出图像中哪些部分是损坏的，哪些是好的。接着，它会利用一张草图，这个草图是我们自己画的，来告诉工具我们希望修复的物体应该是什么样子。这样，工具就能在修复过程中更好地理解我们的意图。最后，通过不断地调整和优化，工具会生成一张既美观又与原始图像相符的完整图像。这个过程确保了我们能够得到高质量的结果，尤其是在物体的姿态和细节方面，比起传统方法，这种方式更加灵活和准确。
2025-03-09｜FDU,ShanghaiAILab,CUHK,HIT
http ://arxiv.org/abs/2503.06676v1
随着基于变换器的模型和预训练-微调范式的普及，多个任务上单个微调模型的高存储和部署成本成为了一个关键挑战。为了解决这个问题，研究者们提出了“增量压缩”技术，旨在通过减少增量参数的冗余（即微调模型权重与预训练模型权重之间的差异）来降低成本。然而，现有方法往往面临数据可获取性和训练要求等问题。为此，本文提出了“Delta-DCT”，一种灵感来源于经典JPEG图像压缩的首个无数据增量压缩方法。Delta-DCT通过将增量参数分组为补丁，评估每个补丁的重要性并分配不同的量化位宽，进而在DCT域中进行量化，最终实现高效的增量压缩，且无需训练或数据校准。
Delta-DCT的核心方法包括几个关键步骤：首先，将增量参数按照层进行分组并划分为小块（补丁），以便进行后续处理；其次，计算每个补丁的L2范数，作为其重要性分数的评估依据；接着，根据这些重要性分数，为每个补丁分配不同的量化位宽；随后，应用离散余弦变换（DCT）到每个补丁，并基于分配的位宽进行混合精度量化；最后，在推理阶段，通过逆DCT（IDCT）和重标定过程重建增量参数。该方法的创新之处在于它实现了无数据的增量压缩，且在多个模型和任务上达到了与微调模型相媲美的性能。
为了验证Delta-DCT的有效性，研究者们进行了全面的实验，涵盖了不同类型的模型和任务，包括大型语言模型（LLM）、小型语言模型以及视觉模型等。实验结果显示，Delta-DCT在多个基准测试中表现出色，尤其在代码生成和安全性任务上，其性能甚至超过了微调模型。此外，与其他增量压缩方法（如BitDelta和Delta-CoMe）相比，Delta-DCT在保持高精度的同时，显著降低了性能退化。通过不同的超参数设置，Delta-DCT在各类模型和任务上均表现出稳定性和优越性，证明了其广泛的适用性和有效性。
Delta-DCT的方法可以简单理解为一个分步的增量压缩过程。首先，想象我们将一个大文件分成很多小块，方便处理。接着，我们评估每块的重要性，重要的部分我们给它更多的存储空间，而不那么重要的部分则可以用更少的空间来存储。然后，我们使用一种叫做离散余弦变换（DCT）的技术，把每块数据转换成一种更容易压缩的形式。最后，在需要使用这些数据时，我们再把它们还原回来。这个过程的好处在于，我们不需要额外的数据或训练，就能实现高效的数据压缩，同时保证压缩后的数据在很多情况下与原始数据几乎没有差别。
2025-03-10｜NTHU,Aeolus,AcademiaSinica
http ://arxiv.org/abs/2503.07446v1
EigenGS是一种新颖的方法，旨在将经典的主成分分析（PCA）与二维高斯表示相结合，以提高图像重建的效率和质量。该方法通过在特征空间和图像空间之间建立有效的转换管道，允许在不需从头开始优化每个图像的情况下，快速初始化新的图像的高斯参数。EigenGS引入了一种频率感知学习机制，使得高斯能够适应不同的空间频率，从而有效捕捉图像的细节并避免高分辨率图像中的伪影。实验结果表明，EigenGS在重建质量上优于直接的二维高斯拟合，同时减少了所需参数数量和训练时间，展现出出色的效果和广泛的适用性，适合实时应用。
EigenGS的方法主要包括以下几个关键步骤：
特征空间与高斯图像空间的桥接：通过PCA提取图像的特征，生成一组特征向量（特征基），并利用这些特征向量来快速初始化图像的高斯表示。
高斯参数的高效初始化：在处理新图像时，EigenGS能够利用预训练的高斯模型直接估计图像空间的高斯表示，而无需逐个图像进行优化。
频率感知学习机制：该机制通过将高斯分为低频和高频两组，分别针对图像的不同频率特征进行建模，确保在重建过程中有效捕捉到图像的细节，防止出现“便士圆砖”伪影。
分阶段学习：初始阶段使用少量高斯表示低频成分，随后利用剩余的高斯表示高频成分，逐步提高重建的细节与质量。
在实验部分，研究者使用多个广泛应用的数据集进行验证，包括CelebA、FFHQ、Cats和StanfordCars等。通过与基线方法GaussianImage的对比，EigenGS展示了在不同分辨率和类别的图像上的优越性能。实验结果表明，EigenGS在重建质量和收敛速度上均显著优于传统方法，特别是在早期阶段的表现更为突出。此外，研究者还进行了跨数据集的实验，以评估EigenGS的迁移学习能力，结果显示即使在不同图像类别之间，EigenGS也能保持良好的表现，证明了其在多样化图像处理中的有效性。
EigenGS的工作原理可以简单理解为一种高效的图像重建方法，它结合了传统的图像处理技术和现代的高斯表示方式。首先，它利用主成分分析（PCA）来提取图像的特征，就像为图像创建一个“简化版本”。然后，当需要重建新图像时，EigenGS可以直接使用这些特征来快速生成图像的高斯表示，而不必从头开始处理每一幅图像。更有趣的是，EigenGS引入了一种聪明的学习方式，能够根据图像的不同细节来调整高斯的大小，这样就能在重建时更好地保留图像的细节，避免出现不自然的伪影。总之，EigenGS使得图像重建变得更快、更高效，同时保持了高质量的视觉效果。
2025-03-10｜XJTU,ETHZurich,NWPU
http ://arxiv.org/abs/2503.07235v1
在多曝光图像融合（MEIF）领域，传统方法往往面临低动态范围图像合成高动态范围（HDR）图像的挑战。研究者们利用Retinex理论，通过将图像的照明与场景反射分离来增强图像的表现力。然而，现有的Retinex模型在处理过度曝光时，无法有效地模拟眩光效应，导致图像信息的损失。为了解决这一问题，本文提出了一种新的无监督多曝光图像融合方法，称为Retinex-MEF。该方法通过将多曝光图像分解为独立的照明成分和共享的反射成分，能够更好地处理过度曝光引起的眩光效应。此外，Retinex-MEF还引入了一种可控的曝光融合标准，以便在保持对比度的同时，实现全球曝光的调整，从而克服固定级别融合的局限性。
Retinex-MEF方法的核心在于三个主要模块：照明估计模块、共享反射提取模块和眩光估计模块。具体步骤如下：
照明估计：通过一个五层卷积网络，从输入的多曝光图像中估计单通道的照明成分L。
共享反射提取：利用Restormer块，从过度曝光和欠曝光的图像中提取共享反射成分Rˆ，确保提取的反射成分不受眩光影响。
眩光估计：在估计的反射成分基础上，计算由于过度曝光引起的眩光成分G，形成最终的图像重建公式I=L·(Rˆ+G)。此外，模型通过一系列损失函数进行训练，以确保反射成分的一致性和图像的重构精度，从而增强图像的质量和细节保留。
在实验部分，Retinex-MEF方法在多个数据集上进行了验证，包括欠曝光与过曝光的融合、曝光控制融合和均匀极端曝光融合。实验结果表明，该方法在图像质量、结构保留和信息保存方面均优于现有的多曝光图像融合技术。具体而言，使用不同的评价指标（如视觉感知Q、归一化互信息NMI、结构相似性SSIM等）对比了Retinex-MEF与其他先进方法的性能。结果显示，Retinex-MEF在处理复杂光照条件下表现出色，能够有效地捕捉场景细节，避免色彩失真，并在不同亮度水平下保持一致的视觉质量。
Retinex-MEF方法的工作原理可以简单理解为一个“分解与重组”的过程。首先，系统会从不同曝光程度的图像中提取出“光照成分”，就像分析照片的亮度来源。接着，它会找出一个“共享反射成分”，这部分代表了场景的真实色彩和细节。最后，系统会估计由于过度曝光而产生的“眩光成分”，这部分会影响图像的清晰度。通过将这些成分结合在一起，Retinex-MEF能够生成更清晰、更真实的图像，尤其是在光照变化剧烈的情况下。这种方法不仅可以在处理图像时保持细节，还可以根据需要调整图像的亮度，使用户能够获得最佳的视觉效果。
2025-03-10｜Opt-AI,SJU,UWurzburg
http ://arxiv.org/abs/2503.06898v1https ://github.com/sharif-apu/LSD-TFFormer
在低光照条件下，数字相机往往难以拍摄出质量可接受的图像。为了解决这一问题，本文提出了一个大规模的高分辨率单张低光照增强(SLLIE)数据集，包含6,425对独特的聚焦对齐图像，这些图像在动态环境下拍摄，光照条件范围从0.1到200lux。该数据集的构建旨在克服现有数据集在多样性和真实场景捕捉方面的不足，提供更丰富的低光照场景样本，以支持更有效的深度学习模型训练。此外，本文还提出了一种新型的深度学习网络TFFormer，旨在通过独立学习亮度和色度特征，提升低光照图像的质量，克服噪声放大和过度增强等问题，展示了其在实际应用中的有效性。
本文的方法主要包括以下几个关键部分：首先，构建了一个名为LSD的数据集，包含6,425对高分辨率低光照图像，并从中提取了约180,000个非重叠的训练补丁。其次，提出了TFFormer模型，该模型采用了调谐叉形结构，能够有效学习亮度和色度特征。具体方法包括：
亮度-色度映射：通过加权组合提取亮度和色度特征，确保在生成图像时保持自然色彩。
亮度-色度交叉注意力块（LCCAB）：用于增强亮度和色度特征之间的相关性，以便更好地捕捉全局信息。
亮度-色度引导重建块（LCGRB）：对中间图像进行进一步精细化重建。
亮度-色度感知损失（LCPL）：通过计算生成图像与真实图像之间的亮度和色度距离，确保生成图像的视觉一致性。
在实验部分，研究者对现有的SLLIE方法进行了全面评估，比较了TFFormer与多种最先进的模型的性能。通过在LSD数据集上进行定量和定性评估，TFFormer在多个低光照场景下表现出色，尤其是在去噪和减少视觉伪影方面。具体而言，TFFormer在PSNR、SSIM和LPIPS等指标上均显著优于其他方法。此外，研究还进行了跨数据集的比较，以验证模型的实际应用能力和用户偏好。用户研究表明，参与者普遍更倾向于TFFormer生成的图像，证明了其在真实世界低光照条件下的有效性。
在这项研究中，研究者们想要解决一个常见的问题：在光线不足的环境中拍照时，手机和相机拍出来的照片往往模糊不清、颜色失真。为了改善这一点，他们收集了大量的低光照图像，并创建了一个新的数据集。这个数据集中包含了很多不同场景的照片，帮助他们训练一个新的智能模型，叫做TFFormer。这个模型的工作原理是，它会分别处理照片的亮度和颜色，这样可以更好地还原出真实的图像。简单来说，TFFormer就像一个聪明的助手，能够在光线很暗的情况下，让照片看起来更清晰、更真实。这项研究的结果显示，TFFormer不仅能提升照片质量，还能在不同的设备和场景下有效工作，受到了用户的广泛欢迎。
2025-03-09｜MoonshotAI,KAUST,NVIDIA,Paradigm
http ://arxiv.org/abs/2503.06526v1https ://github.com/sming256/TimeLoc.
在视频理解领域，时间戳定位是一项重要的任务，旨在准确识别未剪辑视频中的特定时间点。尽管已有多种方法针对这一任务的不同子任务（如时间动作定位、视频文本基础定位、瞬间检索和事件边界检测）进行了研究，但这些方法通常是为特定任务设计的，缺乏跨领域的通用性。为了解决这一问题，本文提出了TimeLoc，一个统一的端到端框架，能够高效处理多种时间戳定位任务。TimeLoc通过简化的单阶段定位模型、联合训练视频编码器与定位模型，以及引入时间分块策略来应对长视频的处理，显著提升了模型的性能。实验结果表明，TimeLoc在多个基准测试中均达到了最先进的性能，展示了其在视频理解领域的广泛应用潜力。
TimeLoc框架的核心在于其统一的时间戳定位设计，支持多种任务。具体方法包括以下几个方面：
单阶段定位框架：该框架包含视频编码器、文本编码器、融合模块和定位头，能够处理多种时间定位任务，支持多尺度动作的预测。
视频与文本特征提取：使用预训练的VideoMAE模型提取视频特征，并通过文本编码器提取文本特征，随后进行融合以增强模型的多模态理解能力。
时间分块策略：为了解决长视频的处理问题，TimeLoc将视频分成小的时间块，逐块处理并聚合特征，降低了内存需求并提高了训练效率。
端到端训练：通过联合训练视频编码器、融合模块和定位头，TimeLoc能够适应任务特定的数据分布，从而提升模型的准确性。
多阶段训练策略：在训练过程中，采用逐步解冻的方式，初期冻结视频编码器，仅微调文本编码器，后期再微调视频编码器，以优化模型性能。
在实验部分，TimeLoc的表现通过多种数据集进行评估，包括THUMOS14、EPIC-Kitchens100、TACoS、Charades-STA和QVHighlights等。实验结果显示，TimeLoc在时间动作定位任务上超越了现有的最先进方法，尤其在THUMOS14和EPIC-Kitchens100上分别提升了1.3%和1.9%的平均精度（mAP）。在时间视频基础定位和瞬间检索任务中，TimeLoc也表现出显著的性能提升，特别是在TACoS和Charades-STA数据集上，分别提高了11.5%和6.7%的召回率。此外，TimeLoc在通用事件边界检测任务中同样取得了优异成绩，展示了其在多种视频理解任务中的广泛适用性和优越性。
TimeLoc的核心方法可以简单理解为一个高效的视频分析系统，能够自动找到视频中重要的时间点。首先，它使用一种单一的模型来处理不同的任务，比如识别视频中的动作或根据文本描述找到特定时刻。其次，为了应对长视频，TimeLoc将视频分成小块，逐块处理，这样可以减少计算负担，避免内存不足的问题。再者，它结合了视频和文本的特征，使得模型在理解内容时更加全面。最后，TimeLoc通过逐步训练的方式，先训练文本部分，再训练视频部分，从而提高整体性能。这种方法让TimeLoc在处理各种视频任务时都能表现得非常出色。
2025-03-08｜Microsoft,NJU,BSTC-BJ
http ://arxiv.org/abs/2503.06220v1
随着人机交互应用的兴起，流媒体视频对话的需求日益增加。为了解决这一需求，本文提出了STREAMMIND框架，旨在实现超高帧率（100fps）的视频处理，并支持实时、主动的对话响应，而无需用户的明确干预。当前技术在视频流处理上面临着线性速度与二次计算成本之间的矛盾，STREAMMIND通过引入一种新颖的“事件门控LLM调用”机制，优化了这一过程。该机制在视频编码器与大语言模型（LLM）之间引入了认知门控网络，仅在相关事件发生时激活LLM，从而实现高效的实时响应。通过此框架，用户可以在观看视频时主动提出问题，系统能够实时提供反馈，显著提升了人机交互的流畅性和智能化程度。
STREAMMIND的核心在于其事件门控LLM调用的感知-认知交替机制。该方法包括以下几个关键步骤：
感知阶段：视频流中的每一帧通过CLIP编码器提取空间特征，然后输入到事件保持特征提取器（EPFE），生成一个单一的感知标记，更新其内部状态。
认知门控判断：引入认知门控，基于当前的感知标记和用户查询，决定是否触发LLM的认知阶段。
认知阶段：若认知门控发出触发信号，系统将从感知记忆中抽取感知标记，输入到LLM以生成响应。通过这种设计，STREAMMIND实现了在每帧处理过程中保持恒定的计算成本，进而支持实时视频对话的高效执行。
在Ego4D和SoccerNet数据集上进行的实验表明，STREAMMIND在模型能力和实时效率上均达到了最先进的水平。与现有的流媒体对话模型相比，STREAMMIND在响应触发准确性和时间有效性上表现优异。实验结果显示，该框架能够以100fps的速度处理视频流，显著优于传统方法的10fps处理能力。此外，STREAMMIND在各类基准测试中表现出色，包括短期和长期活动识别，验证了其在真实世界场景中的适用性和高效性。通过对比不同模型的训练成本和性能，STREAMMIND展现了其在流媒体视频对话中的潜力，为未来的AI助手和互动媒体应用奠定了基础。
STREAMMIND的工作原理可以简单理解为一个智能的“视频助手”。首先，它会不断观察视频中的每一帧，提取出重要的信息，就像我们在看视频时会注意到重要的画面一样。接着，系统会判断这些信息是否与用户提出的问题相关。如果相关，它就会激活一个强大的语言模型，快速生成回应。比如，当你在看足球比赛时问：“这个球员表现如何？”系统会在你提问的同时，分析比赛画面，给出实时的分析和评论。通过这种方式，STREAMMIND能够在观看视频的同时，主动提供你需要的信息，而不是等你问了才反应，极大地提升了观看体验。
