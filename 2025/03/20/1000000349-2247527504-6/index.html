2025-03-06｜TianjinU,BIBMS
http ://arxiv.org/abs/2503.04207v1https ://github.com/HaitaoWuTJU/Uncertainty-aware-Blur-Prior
本研究提出了一种新的方法——不确定性感知模糊先验（Uncertainty-awareBlurPrior，UBP），旨在解决视觉信号与大脑信号之间的系统差距（SystemGAP）和随机差距（RandomGAP）。人类的视觉系统在处理视觉刺激时，常常会丢失一些信息，尤其是在高频细节方面，从而导致大脑信号无法准确反映原始视觉刺激。以往的研究在对大脑信号进行解码时，未能充分考虑这些差距，导致模型在新数据上的泛化能力较差。UBP通过动态模糊处理原始图像的高频细节，来减少信息不匹配，从而提升大脑信号与视觉刺激之间的对齐效果。实验结果表明，UBP在零样本大脑到图像检索任务中，达到了50.9%的Top-1准确率和79.7%的Top-5准确率，显著优于现有的最佳方法。
本研究的核心方法UBP由两个主要部分组成：模糊先验和不确定性量化。具体步骤如下：
模糊先验（BlurPrior）：通过对原始图像施加高斯模糊，降低图像的高频细节，旨在减小视觉刺激与大脑信号之间的系统差距。模糊程度根据视觉信号与大脑信号的相似度动态调整。
不确定性量化（UncertaintyQuantification）：通过分析大脑信号与视觉刺激之间的相似度分布，识别出异常样本，并据此调整模糊程度。具体而言，基于相似度的置信区间，动态改变模糊半径，以减轻信息不匹配的影响。
对比学习：将经过模糊处理的图像和相应的大脑信号进行对齐，使用对称交叉熵损失函数进行训练，以提升模型的检索性能。
在实验中，研究团队使用了THINGS-EEG和THINGS-MEG两个数据集进行验证。THINGS-EEG数据集包含10名受试者的16540个训练样本和200个测试样本，采用快速序列视觉呈现（RSVP）范式。通过对比多种基线方法，UBP在零样本检索任务中表现出色，Top-1和Top-5准确率均显著提高。此外，研究还探讨了模糊先验的有效性，发现动态模糊处理能够有效提升检索性能，特别是在处理高频细节丢失的情况下。通过对不同模糊半径的敏感性分析，确定了最佳模糊程度，从而进一步优化了模型的表现。
为了让大家更容易理解我们的方法，UBP可以看作是一个智能的模糊工具。首先，当我们看到一幅图像时，我们的大脑会处理这些视觉信息，但有些细节可能会丢失。UBP通过对图像进行模糊处理，减少这些丢失的信息，使得大脑信号与图像之间的匹配更好。具体来说，它会根据大脑信号与图像之间的相似度来决定模糊的程度。相似度越低，模糊程度就越高，这样可以帮助模型更好地理解和匹配这些信号。此外，UBP还会监测信号的质量，识别出那些可能存在问题的信号，从而进一步调整模糊效果。这种方法的目的是让机器能够更准确地“理解”我们看到的东西，提升大脑信号到图像的检索能力。
2025-03-06｜SNU
http ://arxiv.org/abs/2503.04333v1
随着视频分享平台的崛起，对视频处理的需求日益增加，传统视频编码方法在处理大规模视频数据时面临着速度慢和内存消耗高的问题。为了解决这些问题，本文提出了一种基于二维高斯分布的高效视频表示和压缩方法——GaussianVideo。该方法通过动态调整每帧的二维高斯分布，显著降低了内存消耗，并提高了视频编码和解码的速度。实验结果表明，GaussianVideo在渲染速度和重建质量上均优于现有的NeRV方法，且训练时间大幅缩短，展示了其在视频处理领域的潜力。
GaussianVideo的方法主要包括以下几个关键部分：
动态二维高斯分布：该方法采用可变形的二维高斯分布，能够在每个时间步动态调整高斯的均值、颜色和形状，从而更灵活地捕捉视频帧间的变化。
多平面编码器-解码器结构：通过引入多平面表示，该结构将空间和时间特征分离，允许更高效地处理高维视频数据。每个平面学习不同的空间或时空模式，减少了计算复杂度。
基于时间梯度的初始化：在初始化阶段，利用时间梯度来识别视频中变化较大的区域，从而在这些区域分配更多的高斯，提高模型对动态内容的表示能力。
损失函数设计：通过结合重建损失和总变差损失，确保重建图像的质量和时间上的平滑性。
在实验部分，GaussianVideo在多个高分辨率数据集（如Bunny、DAVIS和UVG）上进行了评估。实验结果显示，GaussianVideo在渲染速度（每秒帧数，FPS）和视频重建质量（峰值信噪比，PSNR）上均表现出色。与其他NeRV变体相比，GaussianVideo在相同模型大小下实现了71.7%的加速，且在GPU内存使用方面也显著降低。此外，通过对比不同初始化方法的效果，发现基于时间梯度的初始化方法在训练效率和渲染质量上均优于随机初始化和光流方法。
GaussianVideo的核心方法可以简单理解为将视频看作由许多小的“气球”组成，每个气球代表一部分画面。我们通过调整这些气球的形状和颜色，使它们适应每一帧的变化。为了更高效地处理视频，我们将视频信息分成几个部分，分别处理每个部分，就像把一幅复杂的画拆成几块来画。这样做不仅速度更快，而且能更好地捕捉到快速变化的画面。此外，我们还采用了一种聪明的方式来决定哪些地方需要更多的气球，这样可以确保在重要的场景中不会丢失细节。通过这些方法，GaussianVideo能够在保持高质量的同时，大幅提高视频处理的速度和效率。
2025-03-06｜FDU,ByteDanceInc.
http ://arxiv.org/abs/2503.04058v1
随着视频成为主流媒体格式，视频字幕提取逐渐成为研究的重点。字幕作为视频中的重要信息，能够显著提升视觉语言模型（VLMs）对视频内容的理解。传统的视频字幕提取方法多采用多阶段框架，逐帧处理，难以有效利用视频的时间信息。尽管一些大规模视觉语言模型具备强大的光学字符识别（OCR）能力，但准确预测字幕的时间戳仍然面临挑战。为此，本文提出了一种名为EVE的端到端视频字幕提取方法，旨在通过整合视觉编码器、适配模块和大型语言模型，提升字幕提取的整体性能。EVE通过引入新的适配模块InterleavedVT，结合视觉和文本区域的压缩方法，充分利用视频的时间信息，以实现更准确的字幕提取。
EVE的整体架构由三个主要模块组成：视觉编码器、适配模块和大型语言模型。具体方法如下：
视觉编码器：使用Swin-Transformer作为视觉编码器，提取每帧的视觉特征。
适配模块InterleavedVT：
视觉压缩器：采用平均池化操作，保留全局视觉特征，以便于提取字幕相关信息。
文本区域压缩器：使用Q-Former提取文本相关的标记，确保字幕的细粒度信息得到保留。
滑动窗口机制：增强相邻帧之间的文本相关标记提取的鲁棒性。
大型语言模型：将压缩后的视觉和文本标记输入到大型语言模型中，生成对应的字幕文本。
通过这种方式，EVE不仅能够处理字幕文本，还能为每个字幕提供准确的时间戳。
为评估EVE的性能，研究团队构建了一个名为ViSa的大型视频字幕提取数据集，包括2.5M个视频用于预训练，230K个视频用于监督微调，1.6K个视频用于测试。实验结果表明，EVE在多个基准测试中均超越了现有的开源工具和视觉语言模型。在短视频和长视频的测试集中，EVE-7B模型在标准化编辑距离（NED）和字幕编辑率（SubER）指标上表现优异，显示出其在字幕提取任务中的有效性。此外，通过对比不同的模型和方法，EVE的设计理念和模块组合显示出其在处理字幕提取时的优势，尤其是在准确性和效率方面。
EVE的工作原理可以简单理解为一个三步走的过程。首先，EVE使用一种叫做Swin-Transformer的技术来分析视频中的每一帧，提取出画面中的视觉信息。接着，EVE将这些视觉信息和字幕文本信息结合在一起，通过一个聪明的适配器模块，确保在处理过程中不会丢失重要的细节。这个适配器就像是一个过滤器，它能有效地压缩和整合信息，从而帮助模型更好地理解字幕的位置和内容。最后，EVE将处理后的信息输入到一个大型语言模型中，生成完整的字幕文本，并准确标记出每句字幕出现的时间。通过这种方式，EVE能够快速、准确地从视频中提取字幕，帮助观众更好地理解视频内容。
2025-03-06｜ChinaUofPetroleum(EastChina),HIT｜CVPR2025
http ://arxiv.org/abs/2503.04314v1https ://jeasco.github.io/S2Gaussian/
在本文中，我们提出了一种新颖的框架，称为S2Gaussian，旨在从稀疏和低分辨率视图中重建高质量的3D场景。随着虚拟现实和元宇宙技术的快速发展，对真实3D场景重建的需求日益增加。现有的方法大多只关注稀疏视图或低分辨率观察，未能有效处理这两者的复杂组合。S2Gaussian通过两阶段的优化流程，首先优化低分辨率的高斯表示，然后通过超分辨率图像对其进行精细化处理。我们的框架在多个基准测试中表现出色，显示出更一致的几何结构和更细致的纹理，具有更好的实际应用潜力。
S2Gaussian的实现分为两个主要阶段：高分辨率高斯初始化阶段和高分辨率高斯优化阶段。在初始化阶段，首先利用稀疏视图和预测的深度信息来优化低分辨率高斯表示。接着，通过定制的高斯ShuffleSplit操作将其细化为高分辨率高斯表示。在优化阶段，使用从稀疏视图和低分辨率高斯渲染的伪视图生成的超分辨率图像来进一步优化高分辨率高斯。在此过程中，我们引入了自定义的无模糊不一致性建模方案和3D鲁棒优化策略，以减轻多视图不一致性和错误更新的影响，从而重建更细致且高质量的3D场景。
为了验证S2Gaussian的有效性，我们在Blender、LLFF和Mip-NeRF360等多个基准数据集上进行了一系列实验。通过与现有方法（如NeRF-SR、3DGS等）的比较，我们的框架在峰值信噪比（PSNR）、结构相似性指数（SSIM）、LPIPS和FID等多个指标上均表现出显著的性能提升。定量评估结果表明，S2Gaussian在重建高质量3D场景方面的能力超越了所有竞争对手。此外，我们还进行了消融实验，分析了S2Gaussian各个组件对整体性能的贡献，进一步验证了我们方法的有效性和鲁棒性。
S2Gaussian的工作原理可以简单理解为两个步骤。首先，我们从一些稀疏且模糊的视图开始，利用这些视图和一些深度信息来创建一个初步的3D模型。这个初步模型就像是一个草图，显示了场景的基本结构。接下来，我们使用一些技巧将这个草图变得更加详细和清晰。具体来说，我们通过一个叫做高斯ShuffleSplit的过程，将草图中的每个部分细分成更多的小部分，从而更好地捕捉细节。然后，我们结合其他从不同角度生成的伪视图，对这个初步模型进行进一步的优化，确保最终的3D场景既真实又生动。这样，即使只有少量的模糊视图，我们也能重建出高质量的3D场景。
2025-03-06｜BIT
http ://arxiv.org/abs/2503.04034v1https ://wangxihan-bit.github.io/GaussianGraph
在开放世界的3D场景理解中，3D高斯点云（3DGaussianSplatting,3DGS）技术的进步为语义场景理解提供了重要支持，尤其是在通过自然语言查询定位场景中的物体方面。然而，现有的方法主要依赖压缩的CLIP特征嵌入到3D高斯中，导致物体分割准确性较低和空间推理能力不足。为了解决这些问题，本文提出了GaussianGraph框架，它通过结合自适应语义聚类和场景图生成，增强了基于3DGS的场景理解能力。该方法引入了一种“Control-Follow”聚类策略，能够根据场景规模和特征分布动态调整聚类数量，从而避免特征压缩带来的信息损失。此外，GaussianGraph还通过整合从2D基础模型中提取的物体属性和空间关系，丰富了场景表示，提升了对复杂场景的理解和交互能力。
GaussianGraph的方法主要包括三个核心组成部分：2D特征提取、3D高斯聚类和3D修正模块。首先，在2D特征提取阶段，利用SegmentAnythingModel（SAM）和CLIP模型获取图像的全分割和特征。接着，生成前景物体的边界框，并使用LLaVA模型生成物体类别和描述。第二步是3D高斯聚类，使用3DGS算法生成高斯分布以重建场景，并通过“Control-Follow”聚类策略将高斯分配到相应的聚类中。此策略根据特征分布和场景规模动态调整聚类数量，确保高斯的准确性。最后，3D修正模块通过空间一致性验证，剔除不合理的空间关系，确保生成的场景图的可靠性和准确性。这种方法不仅提高了物体分割的精度，还增强了模型的空间推理能力。
为验证GaussianGraph的有效性，本文在三个数据集上进行了广泛的实验，包括LERF、Replica和ScanNet。实验的目标是比较GaussianGraph与现有的开放词汇场景理解方法在3D语义分割和物体定位任务上的表现。通过使用准确率（Acc）和平均交并比（mIoU）等指标，评估模型在不同查询条件下的性能。此外，还通过消融研究探讨了“Control-Follow”聚类策略和3D修正模块对模型性能的影响。实验结果显示，GaussianGraph在语义分割和物体定位任务中均优于现有的最先进方法，证明了其在复杂场景理解和交互中的鲁棒性和有效性。
GaussianGraph的工作原理可以简单理解为一个三步走的过程。首先，模型从图像中提取出物体的特征和描述，就像我们在看一张图片时，首先注意到哪些物体，然后思考它们是什么。接下来，模型会根据这些特征将物体分成不同的类别，就像把水果和蔬菜分开。这个过程使用了一个叫“Control-Follow”的方法，能够根据场景的大小和复杂度自动调整分类的数量，避免了手动设置带来的麻烦。最后，模型会检查这些物体之间的空间关系，确保它们的相对位置是合理的。这样，GaussianGraph不仅能识别物体，还能理解它们之间的关系，从而更好地理解整个场景。
2025-03-06｜Unknown｜CVPR2025
http ://arxiv.org/abs/2503.04030v1
本文提出了一种新的自监督点云补全方法，旨在恢复因遮挡和表面缺失而导致的不完整点云，特别适用于大型考古遗址的复原。当前的自监督方法在处理大规模对象的表面缺失和点云分布不均时面临挑战。为此，研究者们采用了一种多中心投影（MCOP）图像表示法，将原始点云投影到一个五通道的MCOP图像中（包括RGB、深度和旋转信息），将补全任务转化为对MCOP图像缺失像素的填充。通过引入特定的损失函数，增强了补全图像的规律性和一致性，最终将其映射回三维空间，以实现高质量的点云复原。实验结果表明，该方法在完成600多个不完整的考古结构方面表现优越，特别是在处理具有显著缺失表面的复杂场景时。
本研究的方法主要分为以下几个步骤：
MCOP图像构建：首先，将目标点云投影到一个MCOP图像中，该图像通过多个视点捕捉到的点云数据形成，具有五个通道（RGB、深度和旋转）。
自监督补全：利用自监督学习机制，通过随机采样MCOP图像中的局部窗口，与完整补丁进行对抗训练，优化缺失像素的填充。
损失函数设计：在训练过程中，引入多个损失项，包括对抗损失、规律性损失和一致性损失，以确保补全结果的质量和一致性。
3D重建：完成的MCOP图像再通过重投影技术转化为三维点云，并应用泊松重建技术生成水密网格，以便于后续的可视化和分析。通过上述步骤，研究者能够有效地处理缺失表面和不均匀分布的问题，实现高质量的点云补全。
在实验部分，研究者对来自两个真实考古遗址（MawchuLlacta和Huamanmarca）的数百个点云数据集进行了定量和定性评估。首先，构建了一个包含大量点云的MawchuIncomplete数据集，通过随机去除点来模拟缺失情况。其次，使用多种评估指标（如均方误差、结构相似性、峰值信噪比等）对补全效果进行量化比较。实验结果显示，该方法在几乎所有指标上均优于现有的自监督点云补全方法，尤其在处理复杂场景和高缺失率的情况下表现突出。此外，研究者还进行了消融实验，验证了MCOP表示法和补丁级自监督的重要性，进一步证明了方法的有效性。
我们的方法可以简单理解为将一个不完整的三维物体“拍成照片”，然后通过这些照片来“修复”这个物体。首先，我们将物体的点云数据转化为一种特殊的图像格式，称为MCOP图像，这种图像包含了物体的颜色、深度和视角信息。接着，我们利用这些图像进行自我学习，通过比较缺失部分与完整部分的相似性，智能地填补这些缺失的像素。为了确保修复后的效果自然且一致，我们设计了多个“规则”，让机器在学习过程中遵循这些规则，避免出现不协调的地方。最后，我们将修复好的图像再转回三维形式，形成一个完整的物体，并通过一种叫做泊松重建的技术，使其看起来更光滑、真实。这种方法不仅适用于考古遗址的复原，也可以应用于其他需要修复的复杂场景。

本研究提出了一种新的训练方法，旨在利用伪细节填充现有的3D重建场景，以提升其细节和一致性。随着3D高斯点云（3D-GS）技术的发展，3D重建在多个应用场景中取得了显著进展，但在缩放视图时常常出现失真和伪影。现有方法主要聚焦于模型结构优化，忽视了训练采样的不足，导致在放大视图时高频细节的缺失。为了解决这一问题，本文引入了一种将扩散模型与多尺度训练相结合的新方法，通过生成伪地面真相数据来增强训练过程，从而显著减少了3D-GS在缩放视图中产生的伪影，并丰富了重建场景的细节。实验证明，该方法在多个基准测试中达到了最先进的性能，扩展了3D重建的能力，超越了原始训练数据集的限制。
本研究提出的训练方法主要包括以下几个步骤：
扩散模型的应用：利用扩散模型生成的图像作为伪地面真相数据，帮助改善训练过程中的细节生成。
多尺度训练：通过调整训练样本的尺度，确保高频细节和低频数据之间的一致性，增强模型对不同视图的适应能力。
自举技术：结合自举方法，利用已有的3D-GS训练结果生成新的视图，从而增强训练数据集的多样性。
损失函数设计：设计混合损失函数，将原始3D-GS损失、引入的自举损失和上采样损失结合在一起，以优化模型的整体性能。
过滤与插值：在渲染过程中应用频率过滤器，以平滑高频细节并减少伪影，同时采用加权插值方法提高细节的恢复效果。该方法的设计旨在提高训练效率并减少时间消耗，确保生成的细节能够与原始场景保持一致。
为了验证所提出方法的有效性，研究团队在多个公开数据集上进行了全面评估，包括复杂的室内环境和大规模城市景观。实验分为两个阶段：第一阶段采用上采样扩散模型，观察到性能没有下降，甚至在评估指标上略有提升；第二阶段结合自举方法，显著提高了每个场景的整体性能。具体而言，实验结果显示，采用新方法的重建在PSNR、SSIM和LPIPS等标准视觉质量指标上均优于传统方法。此外，针对不同的场景类型，研究团队还调整了训练策略，以确保在不同环境下均能实现最佳重建效果。实验表明，所提出的方法不仅减少了缩放视图中的伪影，而且在细节生成方面有显著提升。
在这项研究中，研究人员提出了一种新方法，目的是让计算机在重建3D场景时，能够更好地填补细节，尤其是在我们放大查看时。首先，他们利用一种叫做扩散模型的技术，生成一些“伪真相”的图像，这些图像帮助计算机更准确地学习如何重建细节。其次，他们通过不同的缩放层次进行训练，确保计算机在处理不同大小的图像时，能够保持细节的一致性。此外，他们还使用了一种自举方法，从已有的训练结果中生成新的视图，进一步丰富训练数据。最后，他们设计了一个混合损失函数，帮助计算机更好地优化重建效果。整个过程旨在提高训练的效率，并确保生成的细节与原始场景相匹配，使得重建的3D场景在放大时依然清晰可见。
