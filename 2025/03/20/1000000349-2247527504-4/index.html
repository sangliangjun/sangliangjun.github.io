315-A3｜室内场景合成，3D场景物体移除与修补，视频3D流体重建与预测，为各种物体生成3D动作

315-A3｜室内场景合成，3D场景物体移除与修补，视频3D流体重建与预测，为各种物体生成3D动作

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月7日｜周五
2025-03-06｜VisionSystemsInc.,BrownU,SFU｜⭐️🟡
http ://arxiv.org/abs/2503.04496v1
本研究提出了一种新的室内场景合成方法，旨在解决当前数据驱动和自回归系统在物体放置时的不足。现有系统往往会产生不完整的物体位置分布，限制了最终场景的多样性和实用性。为此，研究者们引入了一种基于领域特定语言（DSL）的程序生成方法，通过自动推导程序来预测物体的最佳放置位置。该系统不仅可以生成与人类标注者一致的物体位置分布，还能在数据稀疏的情况下保持性能稳定。通过迭代自训练的机制，系统能够不断优化程序，从而提升场景合成的质量和准确性。
该方法主要包括以下几个步骤：
输入处理：系统接收一个部分场景和待放置的物体作为输入。
程序生成：利用生成模型自动生成DSL程序，程序结构采用构造性固体几何（CSG）树的形式，输出可能的物体放置位置的二进制掩码。
约束条件：程序中的约束条件分为位置约束和方向约束，确保物体的放置符合人类的直觉和实际场景需求。
自训练算法：通过引入程序自训练机制，系统可以从已有的场景数据中自动提取和改进程序，进而提升物体位置的预测精度。
评估机制：最后，系统通过比较生成的物体位置分布与人类标注者的标注，评估模型的有效性和准确性。
在实验中，研究者们使用了多个室内场景类型进行评估，包括卧室、书房、客厅和餐厅。通过与现有的自回归合成系统（如Fastsynth和ATISS）进行比较，评估了所提方法在物体位置分布建模和场景合成方面的表现。实验结果表明，提出的方法在生成的场景质量和物体位置分布的完整性上均优于其他系统。即使在训练数据稀疏的情况下，本系统依然能够保持较高的性能，证明了其在实际应用中的潜力和有效性。
这项研究的核心在于如何让计算机更好地理解和生成室内空间。首先，计算机会接收一个房间的部分信息和需要放置的家具。然后，它会使用一种特殊的“语言”来生成放置家具的规则，这些规则就像是给家具放置提供的说明书。接着，计算机会考虑家具之间的关系，比如一个床应该靠墙放，或者一个夜stand要放在床旁边。通过不断学习和改进这些规则，计算机能够在没有太多数据的情况下，依然准确地预测家具的最佳放置位置。这种方法让生成的房间看起来更自然，也更符合人们的生活习惯。
2025-03-06｜HuaweiCanada,UAlberta,McMasterU,ConcordiaU｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.04501v1https ://xinxinzuo2353.github.io/imfine/
本研究提出了一种新颖的3D修复方法，旨在从3D场景中去除目标物体，并无缝填补被遮挡区域。该方法在处理各种场景时，确保生成合理的几何形状和一致的纹理。现有的3D修复和物体去除方法多局限于前视场景，在面对多样化的“无约束”场景时，表现出显著的挑战。为了弥补这一差距，我们的方法集成了几何先验和多视角细化网络，通过测试时适应性训练，提升了在复杂场景下的表现。我们还开发了一种新的修复掩码检测技术，能够从物体掩码中提取目标修复区域，进而增强了在无约束场景中的处理能力。通过广泛的实验验证，我们的方法在多项基准测试中显著超越了现有的最先进技术。
本方法的整体流程包括两个主要阶段：物体去除和3D场景修复。具体步骤如下：
物体去除：通过为每个高斯原始体添加可学习参数，确定其是否属于待去除的物体。使用渲染技术生成物体掩码，并通过优化步骤更新该参数，以识别需要进行修复的区域。
几何重建：从修剪后的场景中渲染深度图，利用截断符号距离函数（TSDF）重建3D网格。该网格用于后续的内容变形和视角调整。
多视角细化：采用多视角细化模型，针对不同视角的扭曲内容进行优化，确保生成一致的高保真图像。该模型通过测试时适应性训练，结合空间时间注意机制，增强了跨视角的一致性。
修复掩码检测：提出了一种新方法，通过扩展物体掩码以包括邻近像素，来识别真正需要修复的区域，从而提升了修复的准确性。
为了验证所提出方法的有效性，我们构建了一个包含20个多样化场景的数据集，涵盖室内和室外环境，并进行了广泛的实验。首先，我们评估了方法在不同场景下的性能，特别是在360°视角和复杂几何结构的情况下。其次，通过与现有最先进方法进行对比，使用峰值信噪比（PSNR）、结构相似性（LPIPS）和弗雷歇距离（FID）等指标进行定量评估。实验结果显示，我们的方法在各种基准测试中都表现出色，尤其是在无约束场景中，明显优于其他方法。此外，我们还进行了消融实验，验证了几何引导扭曲和多视角细化模型对最终效果的重要性。
我们的3D修复方法可以简单地理解为一个两步走的过程。首先，我们需要从一幅3D场景中去掉某个物体，比如一棵树或一辆车。为此，我们会给每个物体打上标签，告诉系统哪些是需要去掉的。接着，我们会用一些数学工具来帮助我们理解这些物体在空间中的位置，并把它们从场景中移除。第二步是填补那些被去掉物体留下的空白区域。我们会通过分析周围的环境，生成新的图像部分，确保它们看起来和周围的背景相匹配。为了做到这一点，我们还会参考多个视角的图像，这样可以确保最终的结果既清晰又一致。通过这些步骤，我们的技术能够在复杂的场景中表现得很好，确保生成的图像既真实又美观。
2025-03-06｜Stanford,Microsoft,GeorgiaTech｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.04720v1https ://yuegao.me/FluidNexus
本文提出了一种名为FluidNexus的新框架，旨在从单一视频中重建和预测三维流体的外观和速度。现有的方法通常依赖多视角视频进行流体重建，而FluidNexus则通过合成多个新视角视频来克服这一限制。该框架的核心在于将视频生成与物理模拟相结合，通过生成多个同步的新视角视频，作为重建的参考。FluidNexus包括两个主要组件：一是新视角视频合成器，通过逐帧视角合成与视频扩散精炼来生成逼真的视频；二是物理集成的粒子表示，结合可微分的流体模拟和渲染，以同时促进三维流体的重建和预测。为了评估该方法，研究团队收集了两个新的真实流体数据集，展示了其在动态新视角合成、未来预测和交互模拟方面的能力。
FluidNexus的实现主要分为两个部分。首先是新视角视频合成器，其目标是从单一视角视频生成多个新视角视频。具体步骤包括：
逐帧新视角合成：利用条件图像扩散模型，根据输入帧和摄像机变换矩阵生成新视角的图像。
视频生成精炼：通过生成性视频精炼模型，确保生成的各帧在时间上保持一致性，避免因独立生成而导致的时间不连贯。其次是物理集成的两层粒子表示，旨在重建流体的速度场和外观。该方法包括：
物理粒子模拟：使用基于位置的流体模拟，计算流体的速度和密度。
视觉粒子渲染：通过三维高斯点云渲染生成流体的外观，并与物理粒子相结合，以确保流体动态的物理合理性。通过这两个部分，FluidNexus能够从单一视频输入中有效重建和预测三维流体的行为。
为验证FluidNexus的有效性，研究团队在两个新收集的流体数据集上进行了实验，分别为FluidNexus-Smoke和FluidNexus-Ball。每个数据集包含120个场景，涵盖了不同行为的流体动态。实验评估包括新视角合成、未来预测和重建的准确性。通过与现有的最先进方法（如PINF和HyFluid）进行比较，FluidNexus在多个指标上表现出色。在新视角合成方面，FluidNexus的PSNR和SSIM指标显著高于其他方法，表明其在视觉质量上的优势。在未来预测任务中，FluidNexus能够更准确地模拟流体动态，尤其是在复杂背景和物体交互的场景中。此外，研究还进行了消融实验，以评估各个组件对整体性能的贡献，结果表明，物理约束和视频生成的结合是实现高质量重建和预测的关键。
FluidNexus的工作原理可以简单理解为：从一段普通的视频中提取流体的三维信息和未来动向。首先，它能将单一视角的视频转变为多个不同角度的新视频，就像从不同位置拍摄同一场景一样。这是通过一种叫做“视频合成”的技术来实现的，确保每个新视频都看起来连贯且真实。接下来，FluidNexus会利用这些新视频来重建流体的运动和外观。它使用一种能模拟流体物理特性的计算方法，确保生成的流体动态不仅看起来真实，还符合物理规律。这样，无论是流体的形状、颜色还是运动方式，FluidNexus都能提供一个非常逼真的模拟，甚至可以预测未来流体的变化。这种技术在电影特效、科学可视化等领域都有广泛应用。
2025-03-06｜SNU,KRAFTON,NVIDIA｜⭐️⭐️
http ://arxiv.org/abs/2503.04257v1
本研究提出了一种新的框架，旨在实现基于文本的运动合成，能够为各种物体生成高质量的3D运动，特别是针对具有不同骨骼结构的对象，如动物、恐龙和幻想生物。传统的运动合成方法通常依赖于固定的骨骼模板，限制了其在多样化物体上的应用。为了克服这一挑战，研究者们扩展了TruebonesZoo数据集，并引入了详细的文本描述，以便为多样化的骨骼结构提供丰富的运动数据。此外，本研究还创新性地提出了骨骼增强技术，以生成多样化的运动数据，同时保持运动的动态一致性。通过重新设计现有的运动扩散模型，使其能够动态适应任意骨骼模板，从而实现对不同物体的运动合成，研究结果显示，该方法能够生成高保真的运动，奠定了在多样物体类别和骨骼模板上进行运动合成的坚实基础。
本研究的方法包括几个关键步骤。首先，构建了一个高质量的文本到运动合成框架，主要由以下几个部分组成：
数据增强：通过对TruebonesZoo数据集进行注释，生成包含详细文本描述的高质量运动数据集，涵盖70种动物。
骨骼增强技术：引入三种骨骼增强方法，包括骨长调整、关节数量调整和休息姿势重置，以模拟不同的骨骼配置，同时保持原始运动的动态特性。
运动扩散模型：通过引入树状位置编码和休息姿势编码，扩展了现有的运动扩散模型，使其能够动态适应不同的骨骼结构。
多级文本注释：为运动数据提供多层次的文本描述，确保合成的运动与输入文本准确对齐。这些方法的结合使得模型能够在多样化的骨骼结构上生成高保真的运动，展示了其在实际应用中的广泛潜力。
在实验部分，研究者通过多个步骤评估了所提出方法的有效性。首先，使用TruebonesZoo数据集进行运动合成的评估，模型在不同对象类别上生成运动并进行聚类分析，确保生成的运动符合输入条件。其次，进行消融实验，验证了位置编码和骨骼增强技术在提升模型适应性和生成质量方面的重要性。实验结果表明，结合位置编码和骨骼增强的模型在生成运动的准确性和多样性上显著优于不使用这些技术的基线模型。此外，模型还展示了对未见对象的良好泛化能力，能够合成与训练数据不同的运动动态。最后，通过长序列生成实验，展示了模型在故事级运动合成中的潜力，进一步验证了其在动画和虚拟现实等领域的应用前景。
在这项研究中，科学家们开发了一种新方法，可以通过简单的文字描述来创造各种生物或物体的运动。想象一下，你只需要说出“龙飞起来了”或“恐龙在攻击”，模型就能生成相应的3D动画。这种方法有几个步骤：首先，研究人员收集了大量的动物运动数据，并为每个动作写下详细的描述。接着，他们设计了一些技术，可以调整动物的骨架，使它们看起来更真实，比如改变骨头的长度或数量。然后，他们创建了一种新的计算模型，能够理解这些骨架的不同结构，并根据文字描述生成流畅的运动。最后，他们进行了多次测试，确保这种方法可以在不同的生物和动作上都能有效工作。通过这种方式，研究人员不仅可以生成生动的动画，还能为游戏和虚拟现实带来更多可能性。
