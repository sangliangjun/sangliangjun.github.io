2025-03-06｜HKUST(GZ),HKUST,SYSU,CUHK｜⭐️⭐️
http ://arxiv.org/abs/2503.04641v1https ://github.com/ALEEEHU/World-Simulator
在人工通用智能（AGI）研究中，理解和模拟现实世界是一个重要的挑战。现有的许多方法，如世界模型，旨在捕捉物理世界的基本原则，从而实现更准确的模拟和有意义的交互。然而，当前的方法往往将不同的模态（如2D图像、视频、3D和4D表示）视为独立的领域，忽视了它们之间的相互依赖性。此外，这些方法通常专注于现实的孤立维度，而没有系统地整合它们之间的联系。本调查研究提供了一种统一的多模态生成模型的综述，探讨了在现实世界模拟中数据维度的增长，涵盖了从2D生成到视频、3D和最终的4D生成的进展。通过提供数据集、评估指标和未来研究方向的全面回顾，本调查为新手提供了宝贵的资源，并为经验丰富的研究人员提供了批判性分析的基础。
本研究的核心方法是从数据维度增长的视角出发，系统地审视现实世界的模拟。具体步骤如下：
2D生成：首先关注仅涉及外观的2D生成，探讨如何通过生成对抗网络（GANs）和变分自编码器（VAEs）等技术生成高质量的图像。
视频生成：接下来，扩展到视频生成，结合外观和动态特征，利用自回归模型和扩散模型等先进技术处理时间维度。
3D生成：进一步引入几何维度，探讨如何通过显式和隐式表示方法（如点云、网格和神经辐射场）生成真实感的3D内容。
4D生成：最终，集成所有维度，研究如何在生成中结合时间和空间，以实现更复杂的现实世界模拟。
通过这些方法，研究旨在提供一个全面的框架，促进多模态生成模型的发展。
在实验部分，研究者们通过对比不同生成模型的性能，验证了所提出方法的有效性。首先，针对2D生成，使用多个数据集评估GANs和VAEs在图像质量和生成速度上的表现。接着，在视频生成阶段，采用了多种评估指标，如帧间一致性和动态真实性，来比较基于VAE和GAN的模型。3D生成实验则着重于模型在几何细节和外观一致性上的表现，利用不同的3D表示方法进行评估。最后，4D生成的实验通过结合视频和3D模型的评估标准，验证了集成模型在复杂场景中的表现。整体实验结果表明，随着数据维度的增加，生成模型的表现得到了显著提升，为未来的研究指明了方向。
在这项研究中，科学家们想要更好地模拟现实世界，使用了一种分步骤的方法。首先，他们从2D图像开始，利用一些智能算法（比如GANs和VAEs）生成高质量的图片。接着，他们将注意力转向视频，结合图像的外观和运动，使用一些新技术来处理时间变化。然后，他们进一步探索3D图像的生成，使用不同的方式（如点云和网格）来创建更真实的三维物体。最后，他们把所有这些技术结合起来，创建4D模型，这样可以同时模拟时间和空间的变化。通过这种方法，研究者们希望能更好地理解和再现现实世界，为未来的技术进步铺平道路。

本研究提出了一种基于能量引导优化的个性化图像编辑方法，利用预训练的文本到图像扩散模型。随着个性化内容编辑需求的增加，尤其在处理任意对象和复杂场景时，现有方法面临诸多挑战。传统方法往往将掩码视为对象形状的先验，导致无法实现无缝整合。为了解决这些问题，我们提出了一种新的无训练框架，将个性化内容编辑形式化为在潜在空间中优化编辑图像的过程。该方法利用扩散模型作为能量函数的引导，结合参考文本-图像对进行优化。通过采用粗到细的策略，我们在特征层面上进行图像能量引导，以实现与目标对象的细致外观对齐，并引入潜在空间内容组合以增强整体身份一致性。实验结果表明，该方法在对象替换任务中表现优异，展示了高质量个性化图像编辑的潜力。
本方法的核心是将个性化图像编辑视为条件优化任务，利用扩散模型作为能量函数。具体步骤包括：
条件能量函数构建：使用参考文本和图像作为查询，扩散模型提供条件能量函数，优化目标是最小化编辑图像与参考图像之间的特征差异。
内容组合操作：在优化过程中，设计替换内容组合操作，将目标信息整合到潜在变量中，以提高目标对象的一致性。
优化策略：采用粗到细的时间步调度策略，确保在早期阶段强调文本能量以缩小语义差距，而在后期阶段则逐渐过渡到图像能量，以细化视觉细节。
背景梯度平滑：为了实现自然的过渡，采用背景梯度平滑技术，确保编辑区域与周围背景的和谐融合。
为了验证所提方法的有效性，我们进行了多项实验，使用标准基准数据集进行对象交换任务的评估。实验中，我们选择了DreamEditBench和PIE-Bench数据集，分别进行了跨类别和同类别的对象替换。通过与现有方法的比较，定量评估了我们的EGO-Edit方法在DINO和CLIP评分上的表现，结果显示出明显的优势。此外，我们还进行了消融实验，以分析不同组件对整体性能的贡献。实验结果表明，结合图像能量引导和内容组合操作的优化策略显著提高了编辑图像的质量和一致性，尤其在处理复杂场景时，能够有效避免模糊和不一致的问题。
我们的研究提出了一种新的图像编辑方法，简单来说，就是让计算机更好地理解我们想要的图像。首先，我们使用了一种叫做扩散模型的技术，这种技术可以通过文本和图像之间的关系来优化图像。具体步骤是：首先，我们输入一张源图像和一张参考图像，系统会分析这两张图像的不同之处。接下来，我们通过优化过程，逐步调整源图像，使其更接近参考图像。为了确保编辑的效果自然，我们采取了一种分阶段的方法，早期主要关注文本描述的内容，而在后期则更加注重图像的细节。最后，我们通过大量实验验证了这种方法的有效性，发现它在处理复杂图像时表现得更好，能够更好地融合不同对象，产生更高质量的编辑结果。

http ://arxiv.org/abs/2503.04353v1https ://github.com/chandagrover/ObjMST
本文提出了一种名为ObjMST（Object-focusedMultimodalStyleTransfer）的框架，旨在解决现有图像风格迁移方法中的对齐问题和内容不匹配问题。传统的多模态风格迁移方法在处理显著对象和其周围元素时，常常会出现风格表示不一致或不对齐的情况。ObjMST通过引入风格特定的MaskedDirectionalCLIPLoss，确保显著对象和其环境之间的风格表示一致性。同时，采用Salient-To-Key注意机制，专注于显著对象的风格化，确保风格迁移过程中的内容保持不变。通过对比实验，ObjMST在量化和定性评估上均优于相关基线方法，展现了其在多模态图像风格迁移中的有效性。
ObjMST框架主要分为两步：生成风格表示和执行对象专注的风格迁移。首先，通过交叉模态GAN反演，生成前景和背景的独特风格表示。具体步骤如下：
生成风格表示：使用风格文本和输入图像对，计算前景和背景的潜在向量，以获得显著和周围的风格表示。这一过程引入了MaskedDirectionalCLIPLoss，确保风格表示的一致性。
对象风格化：提取掩模内容图像中的内容特征，并将显著风格表示应用于前景对象，同时使用周围风格表示对背景进行风格化。通过图像和风格的和谐化，确保最终输出的自然融合。
为了验证ObjMST的有效性，研究者进行了多项实验，包括单文本和双文本条件下的多模态图像风格迁移。实验通过定量指标（如NIMA、LPIPS等）和定性评估（用户研究）来评估生成的风格化图像。结果表明，ObjMST在保持内容一致性和风格准确性方面，显著优于传统方法，如MMIST、CS等。用户研究中，参与者更倾向于选择ObjMST生成的图像，认为其在风格一致性和内容保留上表现更佳。这些实验结果进一步验证了ObjMST在多模态图像风格迁移中的潜力。
ObjMST的工作原理可以简单理解为两步走：首先，它会根据输入的风格文本和图片，生成两种风格表示，一种用于前景（显著对象），另一种用于背景。这个过程使用了一种特殊的损失函数，确保风格在显著对象和背景之间的一致性。接着，在风格化时，ObjMST会只对显著对象进行风格转化，而背景则保持原貌，避免了风格混淆。这样，最终生成的图像既能体现所需的风格，又能保留原图的内容，使得风格化效果更加自然和和谐。
2025-03-06｜ZJU｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.04119v1https ://github.com/scn-00/SCSA
在风格迁移领域，基于注意力的任意风格迁移方法（Attn-AST）近年来取得了显著进展，能够生成高质量的风格化图像。然而，这些方法在处理具有相同语义的内容和风格图像时表现不佳，导致生成的图像在语义区域的风格不一致。本文提出了一种名为SCSA（语义连续稀疏注意力）的新方法，旨在解决这一问题。SCSA通过考虑局部区域与语义区域之间的关系，确保每个查询点能够关注到其对应语义区域中的关键点，从而实现更一致的风格迁移。通过结合语义连续注意力和语义稀疏注意力，SCSA能够同时对整体风格和细节纹理进行有效捕捉，从而生成高质量的语义风格化图像。
SCSA方法包含两个核心模块：语义连续注意力（SCA）和语义稀疏注意力（SSA）。首先，SCA确保每个查询点能够充分关注同一语义区域内的所有连续关键点，利用内容和风格语义图特征作为查询和关键点。这种机制有助于捕捉区域的整体风格特征，确保相邻区域的风格连续性。其次，SSA使每个查询点能够专注于同一语义区域内最相似的稀疏关键点，从而提取特定的风格纹理。通过将这两个模块结合，SCSA不仅能够保持语义一致性，还能在细节上实现生动的风格迁移。值得注意的是，SCSA设计为“即插即用”解决方案，可以轻松集成到现有的Attn-AST框架中，提升其在任意语义风格迁移中的表现。
为验证SCSA的有效性，本文进行了广泛的实验，比较了三种基于Attn-AST的方法（CNN、Transformer和Diffusion）与五种最新的语义风格迁移方法。实验包括定性和定量分析，定量指标包括语义风格损失、FrechetInceptionDistance（FID）和内容特征结构距离（CFSD）。结果表明，结合SCSA的Attn-AST方法在生成的风格化图像质量上显著优于传统方法，尤其在保持内容和风格一致性方面表现突出。用户研究也表明，使用SCSA的风格化图像更受欢迎，且在视觉效果上超越了现有的最先进方法。这些结果验证了SCSA在提高语义风格迁移效果方面的有效性和普适性。
SCSA是一种新颖的图像风格迁移方法，旨在让计算机在处理图像时更好地理解和应用风格。想象一下，你有一幅画和一张风景照片，SCSA帮助计算机将风景照片的内容与画的风格结合在一起。这个方法分为两个部分：第一部分确保每个图像的特定区域都可以关注到与之相关的风格特征，像是把同一块区域的颜色和纹理都考虑进去；第二部分则专注于提取那些最符合该区域风格的细节。这就像是在画画时，不仅要注意整体的色调，还要确保每个细节都与风格相匹配。通过这样的方式，SCSA能够生成更美观、更自然的图像，而不是简单地将一种风格生硬地贴到另一幅图像上。
2025-03-06｜JIIOV,NJU,NKU｜⭐️🟡
http ://arxiv.org/abs/2503.04344v1https ://shenzhang2145.github.io/ledit/
在本研究中，提出了一种新的扩展性图像生成模型，称为长度可扩展扩散变换器（LEDiT）。传统的扩散变换器在生成高分辨率图像时面临着显著的限制，主要是因为它们依赖于显式的位置信息编码（PE），这在推理分辨率与训练分辨率不匹配时会导致性能下降。LEDiT通过去除显式的PE，采用因果注意机制来隐式捕捉位置信息，从而克服了这一限制。该模型在256×256分辨率上进行训练，能够生成高质量的图像，并且在推理时扩展到512×512和1024×1024等更高分辨率，展示了出色的图像质量和细节保留能力。
LEDiT的核心创新在于它不依赖于显式的位置信息编码，而是通过因果注意机制和卷积操作来实现位置信息的隐式提取。具体方法包括：
因果注意机制：通过限制注意力只关注前面的输入，LEDiT能够在序列中自动学习位置信息。
卷积增强局部感知：在图像块处理后添加卷积层，以提高局部特征的提取能力，改善图像细节。
多膨胀训练策略：在训练期间随机调整卷积的膨胀率和填充大小，以提高模型对不同分辨率的适应能力。
LEDiT块设计：交替使用自注意力和因果注意力的块结构，以增强模型的表现力。
通过这些设计，LEDiT能够在不增加显式编码复杂性的情况下，有效生成高分辨率图像。
在实验部分，研究人员对LEDiT进行了广泛的定性和定量评估。实验使用了ImageNet数据集，分别在256×256和512×512分辨率上训练模型，并评估其在更高分辨率（如384×384、512×512、768×768等）下的表现。结果显示，LEDiT在多种分辨率下均显著优于现有的长度外推方法。具体而言，LEDiT在FID（FréchetInceptionDistance）和IS（InceptionScore）等指标上都取得了较低的分数和较高的评分，表明其生成的图像在结构和细节上均表现出色。此外，LEDiT还展示了在不同长宽比下生成图像的能力，进一步验证了其灵活性和实用性。
LEDiT模型的核心思想是通过一种聪明的方式来处理图像生成，让它在生成高分辨率图像时不需要复杂的位置信息。传统模型通常依赖于明确的位置信息，这在处理不同分辨率时会出现问题。LEDiT则采用了一种叫做因果注意的技术，这种技术允许模型在处理图像时，自动学习图像中每个部分的位置，而不需要显式的位置信息。此外，LEDiT还使用了卷积技术来增强模型对局部细节的感知，使得生成的图像更加清晰和真实。通过这些创新，LEDiT能够在更高的分辨率下生成高质量的图像，同时保持细节的丰富性。
2025-03-05｜UPenn｜⭐️
http ://arxiv.org/abs/2503.03944v1
随着扩散模型的普及，图像编辑技术得到了革命性的发展，但也引发了对未经授权修改的严重担忧，如虚假信息传播和抄袭等问题。传统的防护措施主要依赖于对抗扰动，旨在干扰扩散模型的输出，但这些方法容易受到简单图像预处理技术（如压缩和噪声添加）的中和。为了解决这一局限性，我们提出了GuardDoor，这是一种新颖且稳健的保护机制，旨在促进图像所有者与模型提供者之间的合作。GuardDoor通过在图像中嵌入不可察觉的触发器，确保在未经授权的用户尝试编辑受保护图像时，扩散模型会输出无意义的结果，从而降低恶意图像编辑的风险。我们的研究展示了该方法在抵御图像预处理操作方面的增强鲁棒性，并强调了在生成性AI时代，图像所有者与模型提供者合作的重要性。
GuardDoor的核心在于三大关键组件：首先，设计保护触发器模式，以确保其在视觉上对人类观察者不可察觉，并且能够抵抗常见的图像预处理技术。其次，通过微调扩散模型的图像编码器，将保护后门嵌入模型中，使其能够识别和响应嵌入在受保护图像中的不可察觉触发器。最后，将这些保护触发器应用于需要保护的图像，确保在未经授权的编辑尝试中，模型能够产生预定义的无意义输出。具体而言，微调过程包括两个损失项：一是确保未保护图像保持正常功能，二是确保触发器嵌入图像后，模型输出无效内容（如黑色图像）。这种设计使得GuardDoor在保持模型对未保护图像正常功能的同时，有效地防止了未经授权的编辑。
为了评估GuardDoor的有效性，我们在两个开源数据集（CelebA和WikiArt）上进行了一系列实验，涵盖了自然图像和艺术作品。我们随机选择了1800张图像用于训练，200张用于测试。实验中，我们将GuardDoor与两种最先进的图像保护方法（PhotoGuard和EditShield）进行了比较。结果显示，GuardDoor在面对各种攻击场景时，表现出更强的鲁棒性，能够有效抵御对抗性净化和图像预处理攻击。具体而言，GuardDoor在处理高斯模糊、JPEG压缩等攻击时，始终能够生成无意义输出，而其他方法则在这些攻击下逐渐失效。此外，GuardDoor的保护过程在计算时间上表现出显著优势，能够快速为图像添加保护，适合大规模应用。
GuardDoor的工作原理可以简单理解为一个合作保护机制，旨在防止未经授权的图像编辑。首先，图像所有者会向模型提供者请求保护他们的图像，模型提供者则会在图像中嵌入一种特殊的、几乎不可见的“触发器”。这种触发器就像一个隐形的保护标志。当有人试图编辑这些受保护的图像时，模型会识别出这个触发器，并故意生成一些没有意义的内容，比如一张黑色的图像。这种方法不仅能有效防止恶意编辑，还能保持未受保护图像的正常编辑功能。通过这种方式，GuardDoor确保了创作者的作品不会被不当使用，同时也让图像保护变得更加高效和实用。
