主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月6日｜周四
2025-03-05｜SenseTimeResearch,BUPT,CUHKMMLab｜CVPR2025
http ://arxiv.org/abs/2503.03196v1
SpiritSight是一种先进的图形用户界面（GUI）代理，旨在提升人机交互的效率和准确性。该代理通过视觉语言模型（VLMs）实现高效的GUI导航，克服了传统方法在元素定位和准确性方面的不足。研究团队开发了一个名为GUI-Lasagne的大规模高质量数据集，包含573万样本，涵盖了多种GUI平台。此数据集不仅提高了SpiritSight在GUI理解和定位能力方面的表现，还通过引入通用块解析（UBP）方法，解决了动态高分辨率输入中的模糊性问题。SpiritSight在多个基准测试中表现出色，展示了其在不同GUI平台上的强大能力和兼容性。
SpiritSight的开发方法包括三个主要步骤。首先，构建了GUI-Lasagne数据集，分为三个层次：视觉-文本对齐、视觉-功能对齐和GUI导航。每个层次的数据通过高效的收集策略获得，确保数据质量和多样性。其次，引入了通用块解析（UBP）方法，以解决动态高分辨率输入中的定位模糊问题。UBP通过使用块特定坐标表示，增强了模型对GUI元素的定位能力。最后，SpiritSight在多个GUI基准上进行了评估，展示了其在准确性和效率方面的优势。模型的训练采用了逐步预训练和微调策略，以确保其在多种场景中的应用能力。
在实验阶段，SpiritSight的性能通过与其他先进方法的比较进行评估。研究团队在多个基准测试上验证了SpiritSight的效果，包括Multimodal-Mind2Web、ScreenSpot和AMEX等。实验结果显示，SpiritSight在不同的任务和平台上均超越了现有的最先进方法，特别是在元素准确性（Ele.Acc）和步骤成功率（StepSR）指标上表现突出。此外，SpiritSight在跨平台兼容性方面也表现良好，能够在多种GUI环境中实现高效的导航。通过对不同模型大小的评估，研究团队还发现，SpiritSight在数据集规模和模型尺寸上具有良好的扩展性。
SpiritSight的工作原理可以简单理解为，它通过分析和理解图形界面中的元素（如按钮和文本框），帮助用户更好地与电脑互动。首先，研究团队收集了大量的界面数据，确保这些数据的质量和多样性。接着，他们设计了一种新方法，称为通用块解析（UBP），用来解决在处理高分辨率图像时可能出现的定位问题。这就像在一个复杂的拼图中，确保每一块都能准确放置。最后，通过在多个测试中验证，SpiritSight展示了它的强大能力，不仅能快速识别界面元素，还能在不同的设备和平台上流畅运行，极大地提升了用户体验。
2025-03-05｜YonseiU｜ICLR2025
http ://arxiv.org/abs/2503.03321v1
本研究探讨了大型多模态模型（LMMs）在处理图像时的视觉注意机制，发现这些模型倾向于将高注意权重分配给与文本无关的视觉标记，形成所谓的“视觉注意沉没”。这种现象源于某些隐藏状态维度的强烈激活，类似于语言模型中的注意沉没。研究表明，移除这些无关的视觉标记不会显著影响模型性能，因此提出了一种新的方法——视觉注意重分配（VAR），旨在优化模型的注意资源分配。VAR通过回收这些无关视觉标记的注意预算，将其重新分配给图像相关的视觉信息，从而提升模型在视觉语言任务上的表现。实验结果表明，VAR不仅能够有效提高模型的视觉信息处理能力，还能在多个任务上实现性能的普遍提升。
本研究提出的视觉注意重分配（VAR）方法包括两个主要步骤。第一步是选择图像中心头。通过分析每个注意头对视觉标记的注意权重，识别出那些在处理图像时表现出高视觉非沉没比率的头。具体而言，低于特定阈值的注意权重的头将被排除。第二步是重新分配注意权重。将从视觉沉没标记中回收的注意预算分配给在选定图像中心头中与图像相关的视觉非沉没标记。这一过程确保了注意权重的总和保持不变，同时增强了模型对相关视觉信息的关注。通过这两个步骤，VAR能够有效地优化模型的注意资源，使其在处理视觉信息时更加高效。
为了验证VAR的有效性，研究团队在多个视觉语言基准上进行了广泛的实验，包括一般视觉语言任务、视觉幻觉任务和视觉中心任务。实验结果显示，VAR在所有测试中均显著提升了模型的性能。例如，在一般视觉语言任务中，应用VAR的模型在多个基准上均取得了更高的准确率。此外，VAR还有效降低了视觉幻觉的发生率，增强了模型在复杂视觉理解任务中的表现。通过对比实验，研究人员发现，VAR不仅能够改善模型对图像的注意力分配，还能在不增加模型参数的情况下，提升其多模态能力。这些结果表明，VAR为现有的多模态模型提供了一个有效的性能提升策略。
在这项研究中，科学家们发现大型多模态模型在“看”图像时，常常会把注意力集中在一些与文本无关的部分，这就像在阅读时总是被一些不相关的内容吸引。为了改善这一点，他们提出了一种叫做视觉注意重分配的方法。这个方法分为两个简单的步骤：首先，找出那些真正关注图像重要部分的“注意头”；其次，把那些无关部分的注意力转移到更重要的图像信息上。这样，模型就能更好地理解图像和文本之间的关系。通过实验，研究人员发现这种方法不仅能让模型更专注于图像，还能在许多任务中表现得更好，证明了这种方法的有效性。
2025-03-05｜UCBerkeley,MetaAI
http ://arxiv.org/abs/2503.03734v1https ://ottervla.github.io/
OTTER是一种新型的视觉-语言-动作（VLA）模型，旨在通过视觉观察和语言指令预测机器人行为。与现有模型不同，OTTER通过显式的文本感知视觉特征提取，避免了对预训练视觉-语言模型（VLMs）的微调，从而保持了其丰富的语义理解能力。该模型选择性地提取与任务相关的视觉特征，并将其与语言指令进行语义对齐，确保信息的有效传递。这种方法使得OTTER在面对新对象和环境时，展现出强大的零-shot泛化能力。在模拟和实际实验中，OTTER在机器人抓取和放置任务上显著超越了现有的VLA模型，展示了其在多种新任务中的高成功率和强劲的泛化能力。
OTTER的核心方法在于文本感知视觉特征提取。具体而言，OTTER利用预训练的CLIP模型来提取视觉和语言特征。其步骤包括：
文本感知特征提取：OTTER从CLIP的最后一个视觉注意层提取特征，这些特征与语言指令语义相关。通过计算视觉特征与语言特征的相似度，OTTER选择最相关的视觉特征用于任务执行。
模型架构：提取的视觉特征和语言特征通过交叉注意力池化操作进行压缩，生成单一的特征表示。这些特征与机器人的本体状态一起输入到一个基于变换器的政策网络中。
动作参数化：模型通过预测相对运动变化来生成机器人的动作指令，确保机器人能够在动态环境中执行复杂任务。
OTTER的实验分为两大类：语言条件的多任务学习和在未见环境中的零-shot泛化。实验设置包括：
模拟环境：使用LIBERO基准进行评估，涵盖多种任务和对象布局。通过提供多种语言指令，测试OTTER在不同场景下的表现。
真实机器人环境：在实际环境中，OTTER被测试在四种基本操作（如抓取、倾倒等）上的表现。通过与现有模型的比较，评估OTTER在训练和未见任务上的成功率。
基准测试：OTTER与多种基线模型进行比较，验证其在新任务上的泛化能力和性能稳定性。
OTTER的工作原理可以简单理解为“聪明的特征选择”。首先，它使用一个已经训练好的模型（CLIP）来帮助机器人理解视觉信息和语言指令。然后，OTTER会从这个模型中挑选出与当前任务最相关的视觉信息，而不是把所有的信息都传递给机器人。接着，这些选中的信息会与语言指令结合，形成一个清晰的指令，让机器人知道该如何行动。最后，机器人根据这些指令来执行任务，比如抓取或放置物体。通过这种方式，OTTER不仅能在训练中表现出色，还能在面对新任务时，依然保持高效的工作能力。
2025-03-05｜HIT-SZ｜CVPR2025
http ://arxiv.org/abs/2503.03663v2https ://github.com/JiuTian-VL/LION-FS
LION-FS（Fast&SlowVideo-LanguageThinker）是一种新型的在线视频助手，旨在通过实时视频对话提升用户体验。现有的在线视频助手往往在效率与效能之间存在妥协，通常采用低帧率视频和粗糙的视觉特征处理，从而降低了助手的响应准确性。为了解决这一问题，LION-FS采用了双阶段优化策略，结合快速路径和慢速路径，能够提供实时、主动、时间准确且上下文精确的响应。快速路径通过基于路由的响应确定来评估每一帧是否需要立即响应，而慢速路径则通过多粒度关键帧增强来优化生成的响应，使其更为详细和精确。通过这种创新的方法，LION-FS在多个在线视频任务中展现了卓越的效能和效率，超越了现有的技术。
LION-FS的核心方法包括两个主要路径：快速路径和慢速路径。1）快速路径：采用路由基础的响应确定策略，动态评估每一帧的响应需求。为提高响应确定的准确性，LION-FS引入了TokenAggregationRouting，能够在不增加token数量的情况下，动态融合来自高帧率视频的时空特征和低帧率视频的视觉特征，同时使用TokenDroppingRouting来去除冗余特征。2）慢速路径：通过多粒度关键帧增强技术优化关键帧的处理，提取细粒度的空间特征和人机交互特征。这些特征经过精心设计的多模态思维模板整合，指导更精确的响应生成。整体框架通过这种分离的方式，有效提升了在线视频助手的响应效率和准确性。
为验证LION-FS的有效性，研究者在Ego4D和Ego-Exo4D数据集上进行了全面评估。实验结果表明，LION-FS在多个指标上优于现有的在线视频对话方法，尤其是在语言建模能力和响应准确性方面。具体而言，LION-FS的语言建模困惑度（LM-PPL）和语言模型正确性（LM-Correctness）均显示出显著的改进。此外，LION-FS通过快速路径实现了四倍于现有方法的视频流处理效率，能够实时响应用户的查询。实验还通过消融研究分析了不同组件对整体性能的贡献，结果显示，TokenAggregationRouting和TokenDroppingRouting在提升模型的视觉信息整合和响应精度方面发挥了关键作用。
LION-FS是一种智能视频助手，它能够通过观看视频实时回答用户的问题。这个助手的工作原理可以分为两个部分：快速反应和慢速思考。快速反应部分会快速判断每一帧画面是否需要立即回答用户的问题，而慢速思考部分则会在需要详细回答时，分析关键画面并提取更多细节信息。为了确保助手的反应既快又准确，LION-FS会动态选择和整合不同来源的视频信息，并去掉那些多余的、重复的信息。这样一来，用户不仅可以得到及时的回答，还能获得更准确、更详细的信息。通过这种聪明的设计，LION-FS在处理复杂的实时视频对话时，表现得更加出色。
2025-03-05｜MurdochU,UWA,FSU
http ://arxiv.org/abs/2503.03132v1https ://4d-dsns.github.io/DSNS/
本研究提出了一种新颖的动态神经表面（DynamicSphericalNeuralSurfaces,D-SNS）框架，用于统计分析和表示变形的四维（4D）形状。传统上，4D形状通常被视为离散的点云或三角网格，这种离散化方式在进行时空配准时存在局限性，尤其是当物体以不同速率变形和生长时。本文通过将4D形状视为在空间和时间上连续的函数，避免了传统方法中所需的离散化步骤，从而实现更为精准的时空配准、几何计算和形状统计分析。我们专注于零基数（genus-0）表面，例如人体和身体部位，这些表面在自然界中非常常见。通过将神经网络与经典的黎曼几何和统计形状分析技术相结合，我们为4D形状的功能性分析奠定了基础。
本研究的方法主要包括以下几个步骤：
动态球面神经表面（D-SNS）表示：将4D形状表示为从单位球面及时间到三维空间的连续映射。使用多层感知器（MLP）对这种映射进行参数化，避免了离散化带来的问题。
时空配准：将配准问题转化为寻找神经表示的最优空间和时间重参数化。通过物理动机的黎曼弹性度量来度量最优性，从而实现有效的配准。
几何计算和统计分析：在SRNF（平方根法线场）空间中高效计算神经4D表面之间的几何距离、最短路径（测地线）以及平均4D形状的估计。通过将这些计算映射回原始空间，实现可视化。
联合优化：在计算平均4D表面时，同时进行配准，以确保得到的平均表面与所有输入表面尽可能接近。
在实验部分，我们评估了所提出的D-SNS框架在多个4D人脸和人体数据集上的表现。具体实验包括：
表示质量评估：通过计算D-SNS表示与真实离散4D表面之间的平均L2距离，验证了该表示的准确性，结果显示误差在10^-6数量级。
时空配准：展示了4D人类和面部数据集的时空配准能力，结果表明该方法能够有效地将不同速率变形的4D形状对齐。
测地线计算：在时空配准后的4D表面上计算测地线，结果显示该方法能够找到最短路径，表现出良好的几何特性。
平均4D形状计算：通过联合优化实现对输入4D表面的平均化，结果展示了对不同动作的表面进行有效的统计分析。
在这项研究中，我们提出了一种新方法来处理和分析四维形状，比如人类的运动。我们不再把这些形状分成小块，而是把它们看作连续的、流动的表面，就像是一张随时间变化的动态画布。我们使用一种叫做“多层感知器”的神经网络来学习这些形状，确保它们在空间和时间上都能流畅地变化。接下来，我们通过优化算法来调整这些形状，使它们能够更好地对齐和匹配。最后，我们验证了这种方法的有效性，通过多个实验数据集展示了它在实际应用中的表现，包括如何准确地描绘出不同人类动作的平均形状。这种方法不仅提高了分析的精度，还简化了处理过程，使得对复杂动态形状的理解变得更加直观。
