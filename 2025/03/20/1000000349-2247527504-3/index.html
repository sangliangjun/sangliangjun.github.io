主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月7日｜周五
2025-03-06｜GoogleDeepMind｜⭐️🟡
http ://arxiv.org/abs/2503.04666v1https ://github.com/google-deepmind/wyd-benchmark
随着视频生成技术的迅速发展，评估其能力的标准化数据集显得尤为重要。当前，虽然有多个视频生成数据集，但在可控的人类视频生成领域，缺乏一个全面且多样化的基准。为此，研究团队推出了“WhatAreYouDoing?”（WYD）数据集，专注于人类行为的细粒度评估。WYD数据集包含1544个经过精心收集和标注的视频，涵盖56个细分类别，能够系统地评估人类生成的9个关键方面，如动作、互动和运动等。此外，该研究还提出了新的自动评估指标，以更准确地捕捉人类评估的细微差别。通过对七种最先进的可控图像到视频生成模型的深入分析，WYD展现了这些模型的能力与局限，为人类视频生成的未来发展提供了新的见解。
为构建WYD数据集，研究团队采用了一个七步的数据过滤和分类流程。首先，筛选出包含人类演员的视频，确保其主要行为与人类相关。其次，移除场景切换频繁的视频，以保持动作的连贯性。第三，确保视频中人类在第一帧及大部分帧中可见。接着，限制视频时长在1.5至15秒之间，以适应现有模型的处理能力。第五步，确保视频描述与主要演员匹配，以提高文本控制的可行性。第六步，过滤掉分辨率低于360像素的低质量视频。最后，通过人工审核确保视频的光照、稳定性和动作的清晰度。完成这些步骤后，WYD数据集不仅具有高质量的多样性，还包含56个细分类别，以便于针对不同方面进行系统评估。
在实验部分，研究团队对七个主流的可控视频生成模型进行了评估，包括基于姿态、深度和边缘引导的模型。通过使用WYD数据集，团队能够识别出当前模型在生成复杂人类行为时的多种局限性。评估采用了多种指标，包括视频质量、逐帧相似度和运动相似度等。此外，研究还引入了人类中心的评估指标，如基于姿态的平均精度误差（pAPE），以更好地量化生成视频中人类动作的准确性。实验结果表明，WYD数据集对于揭示现有模型的缺陷具有重要意义，尤其是在生成跨镜头或非典型动作时，模型的表现明显不足。
在构建WYD数据集的过程中，研究人员采取了一系列简单易懂的步骤。首先，他们从互联网上收集了大量视频，确保这些视频中有人的参与。接下来，研究人员仔细检查视频，确保没有频繁的场景切换，以便观众能清楚地看到人类的动作。为了确保视频质量，他们还规定视频的时长不能太长或太短，确保适合大多数视频生成模型的处理能力。此外，研究人员还确保视频的描述与视频内容相符，以便生成模型能够准确理解并生成相关内容。最后，他们通过人工审核确保视频的清晰度和稳定性。这一系列步骤使得WYD数据集不仅丰富多样，还能有效评估生成模型在表现人类行为方面的能力。
2025-03-06｜ZJU,MoonshotAI｜⭐️🟡
http ://arxiv.org/abs/2503.04606v1https ://landiff.github.io/
在文本到视频生成（T2V）领域，近年来的研究主要围绕两大主流范式：自回归语言模型和扩散模型。然而，这两种方法各有其局限性：语言模型在视觉质量和误差累积方面表现不佳，而扩散模型则缺乏语义理解和因果建模能力。为了解决这些问题，本文提出了一种名为LanDiff的混合框架，旨在通过粗到细的生成过程结合这两种模型的优势。该框架引入了三项关键创新：首先，采用语义标记器将三维视觉特征压缩为紧凑的一维离散表示，达到约14,000倍的压缩比；其次，利用语言模型生成语义标记，并结合高层次的语义关系；最后，设计了一个流式扩散模型，通过逐步去噪来细化粗语义，最终生成高保真视频。实验结果表明，LanDiff在VBenchT2V基准测试中得分85.43，超越了其他开源模型，展现出在短视频和长视频生成上的卓越性能。
LanDiff框架的设计旨在有效整合自回归建模和扩散过程的优点，主要由四个部分组成：第一，视频语义标记器负责将三维视觉特征转换为紧凑的一维离散表示，同时保留和增强语义信息；第二，语言模型通过时间序列建模生成代表视频蓝图的语义标记；第三，流式扩散模型逐步细化粗语义视频，添加精细化细节以生成高质量的VAE特征；第四，VAE解码器将这些特征重建为最终的视频帧。具体来说，视频语义标记器利用查询令牌压缩视频的语义特征，采用基于MP4的视频帧分组策略以减少时间冗余。语言模型则基于文本生成语义标记，并引入控制条件以增强生成结果的可控性。扩散模型则以条件方式生成视频，支持长视频生成的流式推理策略，确保高效的生成过程。
在实验设置中，研究团队使用了包含2亿对视频-文本的内部数据集进行训练，所有视频的时长均少于6秒，并经过中心裁剪处理。为了评估LanDiff在文本到视频生成任务中的表现，使用了广泛认可的VBench基准。实验结果表明，LanDiff在多个维度上超越了其他开源模型，特别是在动态内容和语义一致性方面表现突出。此外，LanDiff在长视频生成任务中也取得了最高得分，显示出其在保持时间一致性和视觉质量方面的能力。实验还通过对比分析了不同模型的生成效果，强调了LanDiff在理解复杂动态场景和保持视觉一致性方面的优势。
在这个研究中，LanDiff方法可以简单理解为一个“视频生成组合拳”。首先，它使用一种叫做“语义标记器”的工具，将视频的复杂视觉信息压缩成简单的符号，就像把一个长篇小说的内容浓缩成一句话。接下来，利用一个强大的语言模型，LanDiff根据这些符号生成视频的蓝图，确保视频的内容符合用户的文本描述。然后，使用一个流式扩散模型，这个模型像画家一样，逐步为视频添加细节，直到画出一幅完整的高质量画面。最后，所有生成的图像通过一个解码器转变为最终的视频帧。这样的设计不仅提高了生成速度，还确保了生成视频的质量和连贯性，让用户能够看到更生动、更真实的场景。
2025-03-06｜SEU｜⭐️
http ://arxiv.org/abs/2503.04067v1
在音频驱动的动态肖像合成领域，确保高保真度与精准的唇语同步一直是个重大挑战。现有的一些研究尝试通过多阶段管道或扩散模型来实现高质量的动态肖像，但这些方法往往伴随着过高的计算成本。虽然某些方法在特定个体上取得了显著进展且资源需求较低，但仍然存在唇部动作不匹配的问题。本文提出了一种新的框架——FREAK（频率调制高保真实时音频驱动的动态肖像合成），旨在通过频率域分析来提升合成肖像的自然度与真实感。FREAK结合了音频与视觉信息，通过引入视觉编码频率调制器（VEFM）和音频视觉频率调制器（AVFM），从而在频率域内进行特征的调制与整合，显著减少合成视频与自然视频之间的频率差异。
FREAK框架的设计包括两个主要模块。首先，视觉编码频率调制器（VEFM）通过对参考图像和掩码图像的多尺度视觉特征进行频率域调制，确保合成图像中保留了丰富的频率信息。这一过程通过对参考图像进行快速傅里叶变换（FFT），提取其频率特征，并通过1×1卷积学习自适应的通道权重，从而形成频率滤波器。其次，音频视觉频率调制器（AVFM）利用音频特征对视觉特征进行调制，确保生成的视频与输入音频高度同步。AVFM首先对音频特征进行处理，然后与掩码图像的频率特征进行逐元素乘法，最终将调制后的特征转换回空间域，输出与音频同步的动态肖像。
为验证FREAK的有效性，研究团队进行了广泛的实验。实验数据来源于多种视频集，涵盖不同个体的动态肖像。对比实验中，FREAK与多种现有的先进方法（如Wav2Lip、DINet等）进行了性能评估，主要考量指标包括视觉质量、唇语同步精度和运行速度。实验结果表明，FREAK在FID、FVD和LMD等多个指标上均表现优异，尤其在唇语同步方面显著优于其他方法。此外，通过消融实验，团队进一步分析了VEFM和AVFM对整体性能的贡献，结果显示这两个模块对提高合成质量和实时性至关重要。
FREAK的工作原理可以想象成一个音乐会，音频和视觉信息就像乐器和指挥。首先，FREAK会将参考图像（比如一张人的照片）和掩码图像（只显示嘴部的图像）转化为频率信息，这就像把乐器的声音转换为乐谱。然后，系统会根据音频（比如说话的声音）来调整这些频率信息，使得嘴巴的动作与声音完美匹配。这个过程就像指挥根据乐谱来调整乐队的演奏。最后，FREAK将这些调制后的频率信息转换回我们可以看到的图像，生成一个与音频同步的动态肖像。通过这种方式，FREAK能够在实时合成中保持高质量的视觉效果和精准的唇语同步。
