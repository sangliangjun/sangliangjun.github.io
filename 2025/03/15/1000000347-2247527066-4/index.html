2025-03-02｜Tongji,ShanghaiAILab,NJUST｜ICLR2025｜⭐️🟡
http ://arxiv.org/abs/2503.00740v1https ://faceshot2024.github.io/faceshot/
本文提出了一种新颖的无训练肖像动画框架FaceShot，旨在通过任何驱动视频使任意角色栩栩如生。与传统方法依赖于精确的面部特征点识别不同，FaceShot利用外观引导的特征点匹配模块和基于坐标的特征点重新定向模块，提供了稳健的重定位特征点序列。这些组件结合潜在扩散模型的语义对应关系，能够为各种角色生成面部运动序列。随后，这些运动序列被输入到预训练的基于特征点驱动的动画模型中，以生成动画视频。FaceShot不仅突破了对现实肖像特征点检测的局限性，还兼容任何基于特征点的动画模型，显著提升了整体性能。通过在新构建的角色基准CharacBench上进行广泛实验，FaceShot在各个角色领域的表现均优于现有最先进的方法。
FaceShot框架由三个关键组件构成：外观引导的特征点匹配模块、基于坐标的特征点重新定向模块和角色动画模型。首先，在外观引导的特征点匹配模块中，利用潜在扩散模型的特征进行特征点的匹配，通过注入外观先验知识来减少不同域之间的外观差异。其次，基于坐标的特征点重新定向模块通过捕捉驱动视频中的细微面部运动，生成与驱动视频对齐的特征点序列。最后，生成的特征点序列被输入到预训练的动画模型中，以实现角色的动画效果。这种方法的优越性在于它能够精确捕捉非人类角色的细微动作，并且可以作为插件与任何特征点驱动的动画模型兼容使用，从而提升其在非人类角色上的表现。
为了验证FaceShot的有效性，研究团队构建了一个包含46个不同角色的基准CharacBench，并进行了定性和定量评估。实验结果显示，FaceShot在多项指标上均优于其他最先进的方法，包括身份相似度、图像质量和表情准确性等。通过与现有方法的比较，FaceShot不仅能够在驱动视频中保持角色的身份特征，还能有效捕捉到细微的面部运动，如眼睛闭合和嘴巴张合。此外，FaceShot在用户偏好调查中也取得了最高的得分，表明其在动画效果和用户满意度方面的出色表现。最终，FaceShot展示了其在开放领域肖像动画中的广泛应用潜力，尤其是在处理非人类角色时的能力。
FaceShot的工作原理可以简单理解为三个步骤。首先，它使用一种智能算法来识别和匹配角色面部的关键点，这些关键点就像是角色面部的“坐标”，帮助我们知道每个表情的具体位置。这个过程利用了大量的图像数据来确保匹配更准确。接着，FaceShot会根据驱动视频中的动作变化，调整这些关键点的位置，捕捉到细微的表情变化，比如眼睛的眨动或嘴巴的开合。最后，这些调整后的关键点会输入到一个动画模型中，生成流畅的动画效果。通过这种方法，FaceShot能够让任何角色，无论是人类还是非人类，都能生动地表现出各种表情和动作，而不需要进行额外的训练或调整。
2025-03-03｜UTS(Sydney),ZJU｜⭐️⭐️
http ://arxiv.org/abs/2503.01739v1
本文提出了VideoUFO，这是一个专门针对用户需求的文本到视频生成数据集，包含超过109万段视频剪辑，覆盖1291个用户关注的主题。当前的文本到视频模型在实际应用中常常未能满足用户期望，主要原因在于这些模型缺乏对用户感兴趣主题的训练。VideoUFO的独特之处在于其极少与现有数据集重叠（仅0.29%），并且所有视频均通过YouTube的官方API在遵循知识共享协议的基础上进行搜索和获取。该数据集的生成和验证过程严格遵循用户需求，旨在提升文本到视频生成模型的实际表现和用户满意度。
VideoUFO的数据集构建过程包括以下几个步骤：
用户主题分析：通过对VidProM数据集中1.67百万个用户提供的提示进行嵌入和聚类，识别出1291个用户关注的主题。使用K-means聚类算法，结合GPT-4o生成相应主题。
视频检索：利用YouTube的API，根据识别出的主题搜索相关视频，确保视频长度不超过4分钟，分辨率为720p，且遵循知识共享协议。
视频剪辑：将检索到的视频分割成多个语义一致的短片，并为每个短片生成简要和详细的描述。
质量评估：对每个视频剪辑进行质量评分，包括主体一致性、背景一致性、运动平滑度等六个维度，以确保视频质量符合人类感知标准。
数据验证：通过对生成的详细描述进行主题验证，确保每个视频剪辑确实与其标识的主题相关。
在实验部分，研究者对当前16种文本到视频生成模型在用户关注主题上的表现进行了评估。通过构建名为BenchUFO的新基准，研究者首先从1291个主题中随机选择10个提示，并为每个提示生成对应的视频。然后，利用多模态大语言模型对生成的视频进行描述，并计算生成描述与原始提示之间的相似度。实验结果表明，当前的模型在不同主题上的表现不均衡，且在一些用户关注的主题上表现较差。相较之下，基于VideoUFO训练的简单模型在表现不佳的主题上取得了显著的提升，验证了VideoUFO数据集在提升文本到视频生成模型性能方面的有效性。
VideoUFO的构建方法可以简单理解为几个步骤：首先，研究团队分析了用户在生成视频时最感兴趣的主题，这些主题是通过对大量用户输入的文本进行分类得到的。接着，他们使用YouTube的官方工具搜索与这些主题相关的视频，确保这些视频质量高且符合使用规定。然后，他们将这些视频切割成短片，并为每个短片写上简单和详细的描述，以便于后续使用。最后，他们还对视频的质量进行评估，确保每个视频在内容和视觉效果上都能达到用户的期待。这种方法确保了VideoUFO数据集不仅丰富多样，而且能够真正满足用户的需求。
2025-03-03｜KAIST,AdobeResearch｜⭐️🟡
http ://arxiv.org/abs/2503.01107v1https ://videohandles.github.io/
本文提出了一种名为VideoHandles的新方法，旨在使用视频生成先验（generativepriors）对静态场景中的视频进行3D对象组合编辑。与传统的视频编辑方法不同，VideoHandles专注于在保持视频整体一致性的前提下，编辑视频中3D对象的位置。这种方法解决了现有技术在视频编辑中难以处理对象空间组合的问题，尤其是在相机运动的情况下。通过利用生成模型的中间特征，VideoHandles能够在所有视频帧中实现3D对象位置的时间一致性编辑。该方法不仅适用于生成视频，也能处理真实视频，展示了其广泛的应用潜力。
VideoHandles的核心方法包括以下几个步骤：
3D重建与特征提取：首先，从输入视频中重建3D场景，并提取每一帧的中间特征。这些特征代表了视频中对象的身份。
3D变换：通过定义一个共享的3D坐标系统，对选定的3D对象进行平移或旋转变换。这一变换在所有帧中保持一致，确保时间一致性。
特征投影：将编辑后的3D特征投影回各帧中，以生成新的编辑视频。此过程涉及对生成模型的引导，以确保生成视频的真实性和物理效果（如阴影和反射）。
编辑控制：用户可以通过提供目标位置来控制编辑过程，确保生成结果符合预期。此外，方法中还引入了基于自注意力机制的加权和无文本预测，以提高编辑质量。
为了验证VideoHandles的有效性，研究团队进行了多项实验，包括生成视频和真实视频的编辑任务。实验使用27个输入视频，分辨率为320×320，包含51帧。通过与现有的图像编辑基线（如DiffusionHandles）进行比较，评估了VideoHandles在可行性、时间一致性、身份保持和编辑一致性方面的表现。实验结果表明，VideoHandles在时间一致性和编辑质量上显著优于基线方法。用户研究也显示，参与者在可行性、身份保持和编辑一致性方面更倾向于选择VideoHandles生成的视频。定量评估结果进一步确认了该方法在视频编辑领域的优势。
VideoHandles的工作原理可以简单理解为一个智能的视频编辑工具。首先，它会从你提供的视频中提取出所有对象的3D信息，就像为每个对象建立一个数字模型。接下来，你可以选择想要移动的对象，并告诉工具你希望它移动到哪里。VideoHandles会计算出这个对象在每一帧中应该如何移动，并在所有帧中保持一致，确保视频看起来自然。最后，它会将这些信息转换为一个新的视频，并确保新生成的视频中阴影和反射等细节也会随着对象的移动而变化。这样，你就可以轻松地编辑视频而不需要复杂的操作，得到一个流畅且真实的效果。
2025-03-02｜HUST,CUHK｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.00948v1https ://github.com/Chuge0335/EDG
本文提出了一种新的图像到视频生成（I2V）框架，旨在通过给定的图像和文本条件生成视频。现有的I2V扩散模型在生成自然运动的同时，往往难以保持图像的原始外观，导致生成的视频存在运动度不足或与文本条件不一致的问题。为了解决这些问题，本文引入了“外推与解耦”框架，该框架通过模型合并技术，首次应用于I2V领域。该框架分为三个主要阶段：首先，通过轻量级的可学习适配器将文本条件显式注入到时间模块，以提高运动可控性；其次，引入无训练的外推策略，增强运动动态范围；最后，解耦与运动能力相关的参数，并根据不同的去噪时间步骤动态注入到基础I2V模型中。通过这一系列创新，本文的框架在运动可控性和运动度方面表现出显著的优势。
本研究的方法分为三个阶段，以实现更高效的图像到视频生成。第一阶段是运动可控性适应。在这一阶段，文本条件被显式注入到I2V扩散模型的时间注意力模块中，利用轻量级适配器来增强运动的可控性。第二阶段是运动度外推。通过无训练的外推策略，本文提出了一种增强运动动态的方式，旨在逆转模型的微调过程，从而显著提高运动度。第三阶段是解耦注入与采样。在这一阶段，将与运动可控性和运动度相关的参数进行解耦，并在去噪过程中灵活注入。这种方法允许在不同的去噪步骤中动态调整运动相关参数，从而生成自然运动与时间一致性兼具的视频。整体框架的设计旨在克服现有方法的局限性，改善生成视频的质量。
为了验证所提框架的有效性，本文在VBenchI2V基准和UCF101数据集上进行了广泛的定性和定量实验。实验结果显示，所提出的“外推与解耦”框架在多个指标上均优于现有的I2V方法。具体而言，评估指标包括I2V主题一致性、背景一致性、视频质量、运动度和运动控制等。通过与其他先进方法的对比，本文的方法在运动度和运动控制方面表现出色，且在视频质量上保持了可接受的水平。此外，实验还探讨了不同设计选择的影响，进一步分析了模型的性能表现。结果表明，本文框架在生成高质量视频方面的潜力，以及在运动建模中的创新性。
在这项研究中，作者提出了一种新的方法来生成视频，方法分为三个步骤。首先，他们将文本信息（比如描述视频中应该发生什么）直接嵌入到处理运动的部分，这样可以更好地控制视频中的动作。接下来，他们开发了一种新策略，可以在不需要重新训练模型的情况下，增强视频中的动作表现。最后，他们将与运动相关的参数分开处理，这样可以在生成视频时，根据不同的时间点灵活地调整这些参数，以确保视频中的动作自然且连贯。通过这三个步骤，研究者们成功地提高了生成视频的质量，使得视频不仅看起来更生动，而且与给定的文本描述更一致。
2025-03-01｜NTU,ByteDance,NUS｜⭐️🟡
http ://arxiv.org/abs/2503.00276v1https ://lihaoxin05.github.io/human_action_animation/
本研究探讨了一种新颖的方法，即FLASH（Few-shotLearningtoAnimateandSteerHumans），旨在从少量视频中学习动画人类动作。这一任务的挑战在于，传统的视频生成模型通常需要大量的训练数据，而在一些特定的人类动作（如平衡木跳跃或足球射门）中，数据稀缺使得这些模型难以有效学习。FLASH通过对视频中的运动特征和帧间对应关系进行对齐，提升了运动的泛化能力，从而能够在仅有16个或更少视频的情况下，生成流畅且具有真实感的动画。这一方法不仅减少了对大量视频数据的需求，还扩展了视频生成模型的应用领域，特别是在电影和视频制作等实际应用中具有重要价值。
FLASH的核心组成部分包括运动对齐模块和细节增强解码器。具体方法如下：
运动对齐模块：该模块通过对比具有相同运动但外观不同的视频，强制模型学习一致的运动特征。通过强数据增强技术生成增强视频，确保运动信息保持一致，同时改变外观。
帧间对应关系对齐：该过程引导模型学习在原始视频和增强视频之间的跨帧运动。这种对齐有助于模型在不同外观下保持相同的运动模式，增强运动的泛化能力。
细节增强解码器：此解码器通过层级细节传播，将参考图像的多尺度细节传递到生成的帧中。这一过程旨在提高生成视频的细节质量和过渡的平滑性，确保最终动画的视觉效果更为真实。
为验证FLASH的有效性，研究团队在HAA500数据集上进行了实验，选取了12种人类动作进行测试。通过与多种基线模型的比较，FLASH在文本对齐、图像对齐和时间一致性等多个指标上表现优越。实验结果显示，FLASH不仅能够生成与真实视频相似的动作，还能在不同的参考图像下保持流畅的过渡。此外，用户研究表明，参与者对FLASH生成的视频质量给予了高度评价，显示出其在实际应用中的潜力。通过定量和定性的评估，FLASH在动画生成任务中展现了强大的性能。
FLASH的方法可以简单理解为一个聪明的动画生成系统。首先，它学习如何从一小段视频中提取动作，比如一个人跳舞或踢足球。然后，它通过比较不同外观但动作相同的视频，帮助自己记住这些动作的“本质”，而不是仅仅记住某个人的样子。接着，FLASH还会把从参考图像中学到的细节，比如衣服的颜色或背景的样子，运用到生成的新视频中，以确保动画不仅动作真实，而且看起来也很自然。这样，即使只有少量的视频，FLASH依然能够创造出流畅且真实的动画效果，适用于电影和视频制作等多个领域。
