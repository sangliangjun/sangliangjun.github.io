2025-03-03｜ZJU,AlibabaGroup｜⭐️🟡
http ://arxiv.org/abs/2503.01298v1
MINT（Multi-modalChainofThought）是一种创新的统一生成模型，旨在提升图像生成的质量和复杂性。尽管现有的统一生成模型在文本和图像生成方面表现出色，但在处理复杂图像时往往存在不足，尤其是当图像中包含多种交织条件时。为了解决这一挑战，MINT引入了多模态链式思维（MCoT），通过逐步的思维和推理过程，增强了模型对文本与图像关系的理解能力。MINT的设计包括混合变换器专家（MTXpert）结构，能有效支持自然语言生成和视觉能力，同时避免不同模态之间的潜在冲突。通过这一创新，MINT在多个基准测试中展现出卓越的性能，证明了其在文本到图像（T2I）和图像到文本（I2T）任务中的有效性。
MINT的实现分为几个关键部分。首先，输入表示阶段将文本和图像转换为适合模型处理的嵌入形式。文本通过Qwen2的分词器进行处理，而图像则利用变分自编码器（VAE）进行编码。其次，MINT的核心是混合变换器专家（MTXpert），该模块由多个专家组成，各自专注于不同的任务，如语言理解和视觉生成。MTXpert通过多模态注意力机制实现深度交互与对齐，确保文本与视觉信息的有效结合。接下来，MCoT训练范式将生成过程分为四个步骤：规划、行动、反思和修正。这一过程通过逐步推理和自我反省，提升了生成图像的质量和细节。最后，模型通过反思和修正步骤自我评估生成的图像，进一步优化结果，确保生成的图像与文本描述之间的高度一致性。
在实验部分，MINT的性能通过多个基准测试进行评估，重点关注图像生成和理解能力。使用MS-COCO数据集进行图像生成评估，MINT在FrechetInceptionDistance（FID）和CLIPScore等指标上表现优异，显示出其生成高质量图像的能力。此外，MINT还在GenEval基准上进行测试，整体得分达到0.73，超越了所有其他模型。在图像理解方面，MINT在MME和VQA-v2等评估中也表现出色，证明了其在多模态任务中的强大能力。实验结果表明，MINT在处理复杂图像生成任务时，能够有效应对多种条件交织的挑战，展现出显著的优势。
MINT模型的工作原理可以简单理解为一个分步骤的过程。首先，它会把输入的文字和图像转化为计算机能够理解的格式。接着，MINT利用多个“专家”模块来分别处理语言和视觉信息，这样可以避免不同类型信息之间的冲突。在生成图像时，MINT会先进行规划，制定一个详细的图像描述和布局，然后根据这个计划生成图像。生成后，它会自我检查，找出图像中的问题和不一致之处，并进行修正。这样的步骤可以确保生成的图像不仅美观，而且与输入的描述高度一致，能够更好地满足用户的期望。通过这种方式，MINT能够处理复杂的图像生成任务，提供高质量的视觉内容。
2025-03-03｜USTC,SJTU,TencentWeChat,CAS,Galbot｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.01115v1https ://github.com/hzphzp/WeGen
在本研究中，我们提出了WeGen，一个统一的交互式多模态生成模型，旨在通过自然对话实现图像和文本的生成与理解。现有的多模态生成模型在处理复杂的设计任务时常常表现不佳，尤其是在用户指令不够详细的情况下。WeGen通过结合多模态大语言模型（MLLM）与扩散模型，能够生成具有高创意和多样性的结果。该模型不仅可以逐步优化用户满意的生成结果，还能在用户交互中整合具体的内容。为此，我们构建了一个大规模的数据集，从互联网视频中提取，包含丰富的对象动态和自动标注的描述。这一创新使得WeGen在多个视觉生成基准测试中表现出色，展现出作为用户友好设计助手的潜力。
WeGen的架构主要由三部分组成：CLIP编码器、语言模型（LLM）和SDXL解码器。CLIP编码器将参考图像转化为视觉特征，LLM处理交替的文本和视觉输入，而SDXL解码器则将生成的特征转换为最终图像。为了训练模型，我们采用了两阶段的训练流程：首先训练SDXL解码器重建图像，然后对LLM进行微调以处理多种生成任务。此外，为了应对实例身份一致性的问题，我们引入了动态实例身份一致性（DIIC）数据管道，利用视频序列捕捉自然变换，同时保持身份一致性。为了提高生成的多样性，我们还提出了提示自重写（PSR）机制，通过引入随机性来探索不同的解释，从而增强生成结果的丰富性。
为了验证WeGen的有效性，我们进行了多项实验，评估其在多种视觉生成任务中的表现。首先，我们对其文本到图像生成能力进行了定量评估，结果表明WeGen在生成质量和多样性方面均达到了最先进的水平。我们还分析了DIIC数据管道和PSR机制对实例身份一致性和生成多样性的影响。通过与现有最先进方法的比较，WeGen在保持关键身份特征的同时，能够自然地融入用户所需的变化。实验结果显示，WeGen在多个基准测试中均表现出色，特别是在生成多样性和一致性方面，展现了其作为设计助手的强大潜力。
WeGen的工作原理可以简单理解为一个智能助手，能够根据用户的描述生成图像。它的核心是三个部分：一个负责理解图像的“眼睛”（CLIP编码器），一个负责处理文本的“大脑”（语言模型），还有一个将想法转化为图像的“画家”（SDXL解码器）。在训练时，WeGen首先学习如何从图像中提取信息，然后再学习如何根据这些信息生成新的图像。为了确保生成的图像与用户的要求一致，WeGen会不断进行调整，同时保持图像中的重要元素不变。此外，它还会在生成过程中加入一些随机因素，使得每次生成的图像都有所不同，从而给用户提供更多的选择。这样，用户可以在与WeGen的对话中，轻松地实现他们的设计想法。
2025-03-02｜KAIST｜⭐️🟡
http ://arxiv.org/abs/2503.00861v1
随着社交网络和媒体对面部编辑需求的增长，头部交换技术的应用变得愈加重要。这项技术允许用户将一个人的整个头部（包括面部特征、头发样式和面部形状）无缝地与另一个人的身体进行整合。然而，传统的头部交换方法大多依赖于面部中心裁剪的数据集，主要针对正面视图，导致在复杂的现实场景中表现不佳。为了克服这些局限性，本文提出了一种新颖的头部交换方法——头部注入扩散（HID），该方法能够处理包括完整头部和上半身的图像，并在从正面到侧面的视角中自动生成上下文感知的掩膜，从而实现更自然的头部融合。
HID方法的核心在于其创新的掩膜生成和头发注入模块。具体来说，该方法包括以下几个步骤：
输入图像处理：接收一张身体图像和一张头部图像，输出融合的头部图像。
逆正交掩膜（IOMask）生成：通过提取与头部相关的区域，生成精确的掩膜，以指导头部的整合。该掩膜基于图像的潜在特征进行计算，以确保头部与身体的无缝融合。
头发注入模块：专门用于提取和转移头发细节，确保头发样式的准确性和自然性。
扩散模型：结合以上两个模块，进行图像的去噪和重建，输出最终的头部交换图像。
实验部分使用SHHQ-1.0数据集进行训练和评估，该数据集包含40,000张图像，经过预处理后，聚焦于头部交换任务。研究中将HID方法与现有的REFace方法进行比较，评估指标包括学习感知图像补丁相似度（LPIPS）、CLIP-I和FrechetInceptionDistance（FID）。结果显示，HID在头部和头发区域的保留度上优于REFace，且在FID评分上也表现出更高的图像质量。此外，消融实验验证了IOMask和头发注入模块在实现高质量头部交换中的重要性，证明了HID方法在复杂场景下的有效性和实用性。
HID方法的工作原理可以简单理解为将一个人的头部与另一个人的身体进行自然的结合。首先，系统会接收两张图片：一张是身体图像，另一张是头部图像。接着，它会创建一个特殊的掩膜，像是一个剪贴画，确保只有需要的头部区域被替换，而身体的其他部分保持不变。这个掩膜通过分析图像中的特征来生成，确保头部和身体的融合看起来非常自然。此外，HID还会特别处理头发，确保头发的细节和样式也能准确地转移到新的图像中。最后，通过去噪和重建的过程，HID输出一张看起来非常真实的头部交换图像，适用于各种不同的场景。
2025-03-02｜PKU,Tencent｜⭐️🟡
http ://arxiv.org/abs/2503.00811v1https ://github.com/TheRoadQaQ/Predicting-Distortion
随着文本到图像（T2I）模型的迅速发展，生成高质量图像的能力得到了显著提升。然而，生成解剖结构准确的人体图像仍然面临许多挑战。AI生成的图像常常出现各种畸形现象，如多余的肢体、缺失的手指、变形的四肢等，这些问题在现有的评估指标中难以被有效检测。为了解决这一问题，研究者们首次建立了识别AI生成图像中人体畸形的标准，并推出了一个名为Distortion-5K的综合数据集，包含4700张正常和畸形的人体图像。基于此数据集，提出了一种新的视觉变换器模型ViT-HD，专门用于检测这些畸形，取得了优于现有模型的性能，显示出当前T2I模型在生成解剖结构准确的人体图像方面的关键局限性。
本研究提出了一种新颖的ViT-HD模型，旨在检测AI生成图像中的人体畸形。该模型的设计方法包括以下几个关键步骤：
数据集构建：通过收集和标注4700张图像，Distortion-5K数据集涵盖了各种风格和畸形类型，确保了数据的多样性。
模型架构：ViT-HD基于视觉变换器（VisionTransformer）架构，采用了两阶段训练策略。首先进行补丁级别的掩膜训练，以帮助模型从宏观上理解图像中的潜在畸形；接着进行像素级掩膜微调，以提高模型对局部细节的捕捉能力。
评估指标：使用精确度、召回率、F1分数和交并比（IoU）等多种标准来评估模型的性能，确保其在检测畸形方面的有效性。
在实验部分，研究者对ViT-HD模型进行了广泛的性能评估。模型在4700张图像的训练集上进行训练，并在300张验证集和400张测试集上进行调优和评估。实验结果显示，ViT-HD在多个评估指标上均优于其他基线模型（如U-Net和DeeplabV3），证明了其在检测人体畸形方面的有效性。此外，通过对比不同模型的预测结果，发现ViT-HD能够更准确地识别和定位畸形区域，展现了其在处理复杂人体图像时的潜力。最终，研究者还利用HumanDistortionBenchmark对流行的T2I模型进行了评估，结果表明，当前的T2I模型在生成解剖结构准确的人体图像方面仍存在显著不足。
在这项研究中，科学家们开发了一种新模型，名为ViT-HD，专门用来识别人工智能生成图像中的人体畸形。为了训练这个模型，他们首先创建了一个包含4700张图像的数据集，这些图像既有正常的人体，也有不同类型的畸形。接下来，研究团队采用了两步训练方法：第一步是让模型学习如何从整体上识别可能的畸形，第二步则是让模型更精细地关注每个细节，确保能准确地找到畸形的具体位置。最后，他们通过一系列测试来评估模型的表现，结果显示ViT-HD在识别畸形方面的准确率远超其他模型。这意味着未来的AI生成图像将能更好地表现出人体的真实形态，减少畸形的出现。
2025-03-03｜BJUT｜⭐️🟡
http ://arxiv.org/abs/2503.01667v1https ://github.com/misaka12435/ToLo
在布局到图像生成领域，现有的训练自由模型在处理高度重叠的布局时面临显著挑战，常常导致属性泄漏和缺失实体的问题。为了解决这些问题，本文提出了一种名为ToLo的两阶段框架，旨在提高高重叠布局的图像生成质量。ToLo框架由两个主要阶段组成：聚合阶段和分离阶段。每个阶段都有其独特的损失函数，旨在改善注意力图的处理，以确保模型能够更准确地生成与输入布局相符的图像。通过对HRS数据集的重新划分，本文为不同重叠程度的布局生成创建了新的数据集，并通过大量实验验证了ToLo的有效性和鲁棒性，显示出其在处理高重叠布局时的显著性能提升。
ToLo的核心在于其两阶段的生成过程，具体包括聚合阶段和分离阶段。首先，在聚合阶段，使用损失函数L_agg来确保每个概念的注意力图集中在其对应的边界框内。这一过程通过对每个概念的注意力图进行计算，使其能够更精确地控制生成图像中各个对象的位置。其次，在分离阶段，损失函数L_sep的引入确保了不同概念之间的注意力图分离，从而减少重叠带来的问题。具体实现中，模型通过对输入的文本提示和边界框进行交叉注意力计算，生成注意力图，并根据定义的损失函数更新潜在表示。通过这种方式，ToLo不仅提高了生成图像的质量，还有效缓解了属性泄漏和缺失实体的问题，确保了更好的空间控制。
在实验部分，本文使用HRS-Bench数据集进行评估，分为三类：空间关系、大小和颜色。通过计算输入布局的最大交并比（IoU），将数据集划分为不同重叠程度的子集。实验结果表明，ToLo在高重叠布局的处理上明显优于现有方法，尤其是在空间类别中，ToLo相较于R&B方法在IoU>0.1的子集上提升了6.46%。此外，通过对比定量结果和定性结果，ToLo在生成图像的准确性和完整性方面表现出色，成功减少了属性泄漏和缺失实体的现象。实验还显示，尽管分离阶段可能影响对象大小的控制，但通过引入IoU阈值可以有效解决这一问题，进一步提升了模型的适应性和性能。
ToLo是一种新颖的图像生成方法，主要分为两个步骤。首先，在聚合阶段，系统会确保生成的图像中每个对象都能准确地落在它应该在的位置上。想象一下，如果你想在一张图片中放置一个红色苹果和一个黄色时钟，聚合阶段会确保红色苹果的注意力集中在它的边界框内，而不是与时钟重叠。接下来是分离阶段，这一步的目的是确保不同对象之间的注意力不会混淆，避免像“红色苹果”和“黄色时钟”这样两个对象的注意力图重叠在一起。通过这两个步骤，ToLo能够更好地生成清晰且准确的图像，尤其是在输入布局中有很多重叠的情况下。这样，无论是颜色、大小还是空间关系，生成的图像都能更好地反映出用户的期望。
2025-03-03｜AntGroup｜⭐️🟡
http ://arxiv.org/abs/2503.01122v1
在图像个性化领域，文本到图像生成模型（T2I）逐渐受到关注，尤其是在仅用少量参考图像进行定制化生成时。然而，个性化过程中面临的一个主要挑战是“概念耦合”，即由于参考图像数量有限，模型容易在个性化目标与其他概念之间形成不必要的关联。这种耦合影响了模型对文本提示的准确控制，导致生成的图像与预期不符。为了解决这一问题，本文提出了一种名为ACCORD的方法，通过统计分析直接解决概念耦合问题，并提出了两种互补的损失函数：去噪解耦损失（DDLoss）和先验解耦损失（PDLoss），旨在分别最小化不同类型的依赖差异。实验结果表明，该方法在文本控制与个性化保真度之间取得了更优的平衡。
ACCORD方法的核心在于通过去噪解耦损失（DDLoss）和先验解耦损失（PDLoss）来解决概念耦合问题。具体步骤包括：
去噪解耦损失（DDLoss）：该损失通过限制去噪过程中的条件依赖来减少去噪依赖差异。它通过汇总连续去噪步骤之间的依赖差异来实现，并利用扩散模型作为隐式分类器来最小化这些差异。
先验解耦损失（PDLoss）：该损失用于减少先验依赖差异，尤其是在个性化目标的表示训练时。它利用CLIP模型的条件分类能力，确保个性化目标与一般文本条件之间的依赖关系保持一致。
统计分析：通过将概念耦合重新表述为统计问题，识别出去噪依赖差异和先验依赖差异两个独特来源，从而为损失函数的设计提供理论基础。
在实验部分，研究者通过多种图像个性化任务验证了ACCORD方法的有效性，包括主体驱动的个性化、风格个性化和零-shot面部个性化。实验使用了DreamBench、StyleBench和FFHQ数据集，评估了文本对齐和图像保真度等指标。结果显示，ACCORD在不同方法中均显著提高了性能，尤其是在文本对齐和主体保真度方面。同时，研究者还进行了消融实验，探讨了不同T2I模型和损失函数对个性化效果的影响，结果表明DDLoss和PDLoss的结合能够有效提升个性化效果，且对T2I骨干模型的选择并不敏感。
ACCORD方法的核心是帮助生成图像时避免模型产生错误的关联。想象一下，如果我们想生成一张背包的图片，但参考图像中总是有一个女孩和背包在一起，模型可能会错误地把女孩也加到生成的图片中。为了防止这种情况，ACCORD引入了两种损失函数。首先，去噪解耦损失（DDLoss）帮助模型在生成图像的过程中，确保背包和女孩的关系不会被错误地放大。其次，先验解耦损失（PDLoss）确保模型在学习背包的特征时，不会偏离其本身的定义。这两种方法结合起来，使得生成的图像更符合我们的期望，既能准确反映文本提示，又能保留个性化特征。
