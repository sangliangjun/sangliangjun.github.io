主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月4日｜周二
2025-03-02｜UAmsterdam,RadboudU｜⭐️🟡
http ://arxiv.org/abs/2503.00897v2
本研究提出了一种简单有效的强化学习方法，名为“Leave-One-OutPPO（LOOP）”，用于文本到图像扩散模型的微调。随着扩散模型在生成复杂数据分布方面的能力不断增强，强化学习（RL）已经成为对抗黑箱目标的强大工具。尽管当前的主流方法ProximalPolicyOptimization（PPO）在性能上表现出色，但其对超参数的敏感性和高计算开销使得实际应用中存在挑战。相比之下，REINFORCE方法虽然在计算复杂性上有所减轻，但由于高方差和样本效率低下，导致其性能不佳。为了解决这些问题，LOOP结合了REINFORCE的优点和PPO的稳定性，旨在提高扩散模型在多个黑箱目标上的表现。
LOOP方法将反向扩散过程视为马尔可夫决策过程（MDP），其中输入提示成为状态的一部分，每个时间步生成的图像对应一个动作，并通过固定的奖励模型进行评估。LOOP的核心在于通过以下几个关键步骤来优化模型：
多动作采样：针对每个输入提示，LOOP通过采样多个动作来降低方差，确保生成的样本更加稳定。
基线校正：使用REINFORCE的基线校正技术，进一步减少样本效率低下的问题。
结合PPO的优势：利用PPO的剪切和重要性采样技术来增强模型的鲁棒性和样本效率。通过这些设计，LOOP能够在保持计算效率的同时，显著提高模型的最终性能。
在实验部分，研究团队在T2I-CompBench基准上评估了LOOP方法的有效性。该基准专注于图像生成模型的属性绑定能力，特别是模型生成的图像在颜色、形状和纹理等属性上的一致性。实验结果表明，LOOP在各项任务中均优于传统的PPO和REINFORCE方法。具体而言，LOOP在生成图像的美学质量和图像-文本语义对齐任务中也表现出色，显示出其在多种黑箱目标上的适应性。此外，研究还探讨了不同独立轨迹数量对训练效果的影响，发现增加轨迹数量可以有效降低训练过程中的方差，进一步提升模型性能。
LOOP方法的核心思想是通过让模型在生成图像时考虑多个选择，从而提高其生成的图像质量。首先，模型会根据输入的提示生成多个可能的图像，然后选择最符合提示的那一张。这就像在考试时，学生在多个答案中选择最佳答案一样。此外，LOOP还引入了一个“基线”概念，帮助模型判断生成的图像好坏，从而减少错误的影响。最后，LOOP结合了其他方法的优点，使得整个过程更加高效和稳定。通过这些步骤，LOOP能够帮助模型更好地理解和生成图像，使其在各种任务中都能表现出色。
2025-03-03｜NVIDIA,THU,UTAustin｜⭐️
http ://arxiv.org/abs/2503.01103v1https ://research.nvidia.com/labs/dir/ddo/
在现代视觉生成模型的研究中，基于似然的生成模型（如扩散模型和自回归模型）取得了显著的成功。然而，这些模型在最大似然估计（MLE）过程中存在模式覆盖倾向，限制了生成质量。为了解决这一问题，本文提出了一种新的框架——直接判别优化（DirectDiscriminativeOptimization,DDO），旨在将基于似然的生成训练与生成对抗网络（GAN）目标结合起来，从而克服MLE的固有限制。DDO通过隐式参数化判别器，利用目标模型与参考模型之间的似然比，提供了一种更为高效的训练方式。实验结果表明，DDO在多个标准数据集上显著提高了生成模型的性能，尤其在FID分数上达到了新的记录。
DDO的核心思想是通过隐式参数化判别器，利用目标生成模型与固定参考模型之间的似然比来优化生成质量。具体方法如下：
判别器参数化：将判别器定义为目标生成模型的似然比，使得优化过程不再需要单独的判别器网络。
优化目标：通过最小化基于GAN的损失函数，使得生成样本更接近真实数据分布，从而提升生成质量。
迭代自我提升：DDO采用多轮自我对抗的方式，每轮优化都使用上轮的生成结果作为新的参考，逐步提高生成模型的性能。
高效训练：每轮优化所需的预训练时间少于1%，有效提升了训练效率并降低了计算成本。
实验部分主要评估DDO在视觉生成模型上的有效性和效率。我们在CIFAR-10和ImageNet-64等标准数据集上进行了一系列实验，验证了DDO的性能。实验结果显示，使用DDO的模型在FID分数上达到了前所未有的记录，分别为1.38和0.97，显著优于其他指导方法和GAN方法。此外，我们还考察了超参数α和β对模型性能的影响，发现适当的超参数设置可以进一步提升生成质量。通过多轮优化，DDO在减少生成模糊度和提高样本多样性方面表现出色，展示了其在实际应用中的潜力。
直接判别优化（DDO）是一种新方法，旨在提高生成模型的效果。它的工作原理可以简单理解为：首先，我们有一个生成模型，它负责创造图像；同时，还有一个参考模型，它提供标准。DDO通过比较这两个模型的输出，来调整生成模型，使其生成的图像更接近真实的样本。这个过程就像是一个比赛，生成模型不断学习如何赢过参考模型。DDO的好处在于，它不需要额外的复杂网络，也不需要大量的计算资源，每次调整只需很少的时间。这种方法有效地提高了图像的质量，并且在多个测试中表现出色，显示出它在未来的应用潜力。
2025-03-03｜CASIA,UCAS｜CVPR2025｜⭐️
http ://arxiv.org/abs/2503.01323v1https ://github.com/BienLuky/CacheQuant
CacheQuant是一种新提出的训练无关范式，旨在全面加速扩散模型的推理过程。近年来，扩散模型在图像合成领域表现出色，但其推理速度慢和网络复杂性高，限制了其在实际应用中的低延迟使用。现有的加速方法通常分别从时间和结构两个层面进行优化，但这种独立优化往往导致性能下降。CacheQuant通过联合优化模型缓存和量化技术，克服了这些问题，确保在加速的同时保持模型性能。研究表明，CacheQuant在MS-COCO数据集上实现了5.18倍的速度提升和4倍的压缩，同时CLIP分数仅下降了0.02，显示出其在扩散模型加速方面的显著优势。
CacheQuant的方法主要包括两个关键技术：动态编程调度（DPS）和解耦误差校正（DEC）。首先，DPS通过分析特征图在各个步骤的表现，优化缓存调度，旨在减少因缓存和量化引入的误差。具体而言，DPS将扩散模型的推理步骤分为多个组，并为每个组分配缓存特征，以确保时间一致性。其次，DEC通过解耦缓存和量化引入的误差，分别进行校正。该方法采用最小二乘法计算校正参数，以减少累积误差。DEC的设计使得在推理过程中仅需增加少量计算，从而有效提升了模型性能。这种联合优化策略确保了在加速的同时，模型的生成质量得以维持。
为了验证CacheQuant的有效性，研究者在多个数据集上进行了广泛的实验，包括CIFAR-10、LSUN-Bedroom、ImageNet等。实验结果表明，CacheQuant在加速和性能方面均优于传统的加速方法。具体来说，CacheQuant在不同的框架（如UNet和DiT）上均表现出色，尤其是在MS-COCO数据集上，成功地实现了显著的速度提升和压缩效果。此外，实验还展示了CacheQuant在多种硬件平台上的鲁棒性，尤其是在GPU上，速度提升尤为显著。通过与其他加速技术的对比，CacheQuant展现了其在实际应用中的潜力，提供了新的思路来推动扩散模型的加速研究。
CacheQuant的核心理念是通过优化模型的缓存和量化来加速图像生成模型。首先，它使用一种叫做动态编程的技术，这就像为每个步骤安排最佳的工作计划，确保在生成图像时只计算必要的部分，避免重复工作。其次，CacheQuant会对模型的错误进行分开处理，确保每次生成的图像都尽可能接近真实的结果。这就像在做一道复杂的菜时，先准备好所有的材料，再一步一步地进行烹饪，确保每个步骤都尽量完美。通过这种方法，CacheQuant不仅让生成模型的速度大幅提升，还保持了图像质量，适合在很多实际场景中使用。
2025-03-03｜Caltech,OpenAI｜⭐️
http ://arxiv.org/abs/2503.01161v1
在这项研究中，我们探讨了在离散状态空间中使用离散扩散模型进行后验采样的问题。尽管针对连续扩散模型的后验采样方法已取得显著进展，但在离散扩散模型中，后验采样仍然面临着挑战，主要是由于缺乏明确的梯度定义。为了解决这一问题，我们提出了一种基于分裂吉布斯采样的后验采样算法，称为SG-DPS。该算法能够实现奖励引导的生成，并有效解决离散状态空间中的逆问题。实验结果表明，SG-DPS在合成基准测试上收敛到了真实的后验分布，并在多项离散数据基准测试中表现出色，性能提升可达2倍，超越了现有基线方法。
我们的方法基于分裂吉布斯采样的原则，通过引入辅助变量来放宽后验采样问题。具体步骤包括：1)引入辅助变量：在后验分布中引入一个辅助变量z，以便从扩展的分布中进行采样；2)定义正则化势函数：通过正则化势函数D(x,z;ρ)来控制x和z之间的距离，确保在正则化强度增大时，两个变量均收敛到目标后验分布；3)交替采样步骤：算法交替执行两个采样步骤：一是基于似然的采样步骤，二是基于先验的采样步骤。在每次迭代中，使用预训练的离散扩散模型进行高效采样，以实现收敛。通过这种方式，SG-DPS能够在不依赖于梯度信息的情况下，逐步逼近真实的后验分布。
我们在多个领域进行了实验，以验证SG-DPS的有效性，包括合成数据、离散图像、单声道音乐和DNA序列的生成。实验中，我们首先在合成任务上评估了SG-DPS的准确性，结果显示其在后验采样中明显优于基线方法。接着，在离散图像逆问题上，SG-DPS也展现了卓越的性能，通过生成的样本与真实信号的一致性得到了高峰信噪比（PSNR）。在单声道音乐的填充任务中，SG-DPS能够生成与给定条件一致的音乐序列，且在奖励引导的DNA生成任务中，其生成的序列在活动值上达到了最高水平。综合这些结果，SG-DPS在逆问题解决和序列生成任务中均表现出色，证明了其在离散扩散模型中的应用潜力。
我们的方法SG-DPS可以想象成一种“智能采样器”，它通过引入一个“辅助助手”来帮助我们更好地从复杂的数据分布中抽取样本。首先，我们让这个助手z和我们想要的样本x一起工作，确保它们之间的关系能够逐渐变得紧密。接下来，我们设定一个“规则”，这个规则就像是一条距离的标准，帮助我们判断助手和样本之间的关系是否足够好。然后，我们通过两步来进行采样：第一步，我们根据已有的数据来决定样本x应该是什么样的；第二步，我们再根据助手z来进行调整。通过不断重复这两个步骤，助手和样本会逐渐接近真实的样本分布。这样的过程就像是在逐渐磨合，最终帮助我们得到高质量的样本，适用于许多实际问题，比如图像修复和基因序列生成等。
2025-03-03｜DukeU,JHU｜⭐️
http ://arxiv.org/abs/2503.01839v1
本文研究了如何利用大型语言模型（LLM）来突破文本到图像生成模型的安全防护机制，特别是针对不安全提示的生成。随着文本到图像模型的广泛应用，其潜在的滥用风险也日益显现。为应对这些风险，通常会在模型上添加安全过滤器或通过对模型进行对齐来减少有害输出。然而，这些防护措施仍然存在被攻击者通过精心设计的对抗性提示绕过的风险。本研究提出了一种名为PromptTune的方法，旨在通过微调的LLM生成对抗性提示，从而有效绕过这些安全防护。通过在三个不安全提示数据集上进行评估，结果表明该方法在绕过安全防护方面表现优异，超越了现有的无盒攻击，并能有效辅助其他基于查询的攻击。
PromptTune方法的核心在于其三大组成部分：AttackLLM、安全的文本到图像模型和Judge。首先，给定一个不安全的提示，AttackLLM将其重写为一个可能绕过模型防护的对抗性提示。生成的图像与原始不安全提示一起输入Judge，Judge计算匹配分数以指导AttackLLM的微调。为了微调AttackLLM，研究者构建了一个包含不安全提示及其对应的对抗性提示的参考数据集。微调过程中采用监督微调（SFT）和直接偏好优化（DPO）方法，确保AttackLLM能够将不安全提示转化为能够生成有害图像的对抗性提示。该方法的创新之处在于无需对目标模型进行多次查询，显著提高了效率。
实验部分主要评估PromptTune在不同数据集和安全防护机制下的有效性。研究者使用了三个包含色情内容的提示数据集，分别进行测试，涉及五种安全防护机制，包括三种安全过滤器和两种对齐方法。通过对PromptTune的三个变体进行比较，结果显示其在绕过安全过滤器时的成功率显著高于其他现有攻击方法。此外，PromptTune还在生成图像的质量上表现出色，生成的图像与无防护模型生成的图像在语义上保持更高的一致性。实验结果验证了PromptTune在无盒和黑盒设置下的有效性，并展示了其在提升攻击效果和效率方面的潜力。
在这项研究中，科学家们想出了一个聪明的方法，叫做PromptTune，来帮助绕过一些保护措施，防止生成不当内容的图像。这个方法使用了一个大型语言模型（LLM），首先它会接收一个不安全的提示，比如“生成一幅裸体的图像”，然后它会把这个提示转换成一个更隐蔽的提示，这样就能逃过模型的检测。整个过程分为几个步骤：首先，LLM会重写提示；接着，生成的图像和原来的提示会被送到一个评估系统，看看生成的图像是否符合预期的内容。为了让这个模型更聪明，研究人员还收集了大量的例子来训练它，使得它能够更好地理解哪些提示能够成功绕过保护。这种方法的好处是，它不需要反复地向目标模型询问，只需一次就能生成有效的提示，大大提高了效率。
2025-02-23｜Meta,DukeU,UNCC｜⭐️
http ://arxiv.org/abs/2503.00020v1
本文探讨了文本到图像（T2I）生成AI模型的安全性，特别是依赖于公开可用数据集进行训练和评估的挑战。尽管这些模型能够根据简单的文本提示生成引人注目的图像，但它们的开放性使得确保生成内容符合伦理、文化和社会规范变得复杂。模型可能被滥用以生成有害或不当内容，因其训练数据往往来自互联网，可能包含偏见和刻板印象。本文的目标是系统性地回顾用于T2I安全性研究的关键开放数据集，分析其收集方法、组成、语义和句法多样性，以及数据集中有害类型的覆盖和分布。通过识别数据集的优缺点，本文为研究人员提供了选择合适数据集的依据，并强调了未来研究需要填补的覆盖和质量空白。
本研究采用了多种方法来分析T2I安全性相关的数据集。首先，进行了提示注释以分类有害内容，使用了AIR分类法的二级（L2）标签。其次，分析了提示的多样性，包括概念的多样性和语法的多样性。通过计算n-gram的独特性，评估了数据集中提示的语言结构多样性。此外，还调查了提示的长度分布和情感分布，以了解数据集的整体特征。为确保标签的一致性，采用了贝叶斯方法评估标签的准确性，并将生成的标签与原始标签进行比较。最后，使用了语义多样性分析，通过比较提示的嵌入向量来测量提示之间的语义关系，以识别潜在的有害或不当内容。
在实验部分，研究团队汇总了多个重要的开放数据集，并描述了每个数据集的提示数量和来源。具体包括“SafeGenExplicit56k”、“LatentGuardCoPro”、“ART”、“P4D”等数据集，分别涵盖了数千个提示，主要集中在性内容、暴力和其他有害主题。通过对提示的情感和语法多样性进行分析，发现数据集中存在不均衡的语言使用模式。使用文本情感分析工具，发现尽管数据集主要包含不安全的提示，但整体情感分布偏向积极，挑战了以情感为基础的过滤效果。此外，实验还揭示了不同数据集中有害内容的覆盖情况，强调了在生成AI模型中对多样性和偏见的关注。
在这项研究中，我们使用了一些简单的方法来分析文本到图像生成模型的安全性。首先，我们对生成的提示进行了分类，看看哪些可能会产生有害的内容。我们还检查了这些提示的多样性，确保它们在语言和主题上有足够的变化。为了了解提示的长度和情感，我们计算了每个提示的单词数量，并分析它们传达的情感是积极还是消极。我们还使用了一种统计方法来检查我们的标签有多准确，确保我们标记的内容与原始数据一致。最后，我们通过比较提示的含义，查看它们之间的相似性，从而识别潜在的有害内容。这些方法帮助我们更好地理解和改进生成模型的安全性。
