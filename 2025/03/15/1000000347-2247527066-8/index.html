2025-03-01｜GMU,LightThought｜⭐️⭐️
http ://arxiv.org/abs/2503.00260v1
本研究提出了一种快照成像技术，旨在从微型场景中恢复三维周围视图。微型场景中的物体通常尺寸在毫米级，这使得三维重建变得复杂且具有挑战性，尽管这些微型物体在日常生活中非常普遍。为了实现高质量的三维数字化，我们设计了一种带有单个相机和八对平面镜子的类折射成像系统，能够在一次拍摄中获取全方位的图像。该镜子设计可以根据场景的大小进行定制，以优化视图覆盖率。我们采用了三维高斯点云（3DGaussianSplatting,3DGS）作为场景重建和新视图合成的基础，结合视觉外壳深度约束，克服了稀疏视图输入带来的挑战。通过对合成和真实微型场景进行验证，我们的方法展示了在多种场景下的先进性能。
我们的方法包括几个关键步骤。首先，我们设计了一种平面类折射镜头，允许同步全方位多视图图像采集。该镜头由八对平面镜子组成，镜子被安排在两个嵌套的八角金字塔的侧面，以捕捉场景的周围视图。接下来，我们分析了光线在镜头中的传播，并推导出优化镜头配置的封闭形式公式，以确保最佳的视图覆盖。我们通过展开反射光路径，将镜子反射映射到虚拟相机视图。然后，利用3DGS框架进行场景重建，并通过视觉外壳生成的深度图进行深度约束，以改善新视图合成的质量。最后，我们建立了原型系统并在真实微型场景中进行了验证，证明了该方法的有效性。
我们在合成数据和真实数据上对所提方法进行了评估。合成实验中，我们使用Autodesk3dsMax生成模拟图像，并对各种复杂程度的场景进行渲染。我们还使用AgisoftMetashape生成初始点云，并将其转换为3DGS格式，以便进行比较。真实数据实验中，我们定制了镜头原型，并使用5百万像素的FLIR相机拍摄了多个微型场景。结果表明，我们的方法在定量和定性比较中均优于其他最新的3DGS算法。尤其是在处理缺乏特征的微型场景时，我们的视觉外壳深度约束显著提高了重建质量，展示了该方法的广泛适用性和优越性。
我们的研究旨在帮助我们更好地捕捉和重建微小场景的三维图像。我们使用了一种特殊的镜头系统，这个系统就像一个装满镜子的盒子，可以在一次拍摄中捕捉到周围的所有细节。首先，我们设计了一个由多个镜子组成的镜头，这些镜子可以反射光线，使得相机能够看到不同的角度。然后，我们通过分析光线的路径，找到最佳的镜子排列方式，以确保捕捉到的图像尽可能全面。接着，我们利用一种叫做3D高斯点云的技术，将捕捉到的图像转换为三维模型，并通过额外的深度信息来提高图像的质量。最后，我们进行了大量的实验，结果显示我们的方法在重建微型物体时表现出色，能够清晰地呈现出它们的细节。
2025-03-02｜HUST,NTUS-Lab｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.00746v1https ://dof-gaussian.github.io/
在3D场景重建和实时新视图生成的领域，3D高斯点云（3DGaussianSplatting,3D-GS）技术取得了显著的进展。然而，现有方法通常假设输入图像为针孔成像且完全聚焦，这限制了其在实际应用中的有效性。为了解决这一问题，本文提出了一种名为DoF-Gaussian的可控景深方法，旨在利用几何光学原理，通过镜头成像模型来控制景深效果。该方法不仅能够处理浅景深输入图像，还能在新视图合成中实现可控的景深效果，增强结果的电影质感。此外，研究还引入了场景特定的深度先验和模糊到聚焦的适应策略，以提高场景几何的准确性和深度关系的精确度。
DoF-Gaussian的整体流程包括三个主要步骤。首先，开发一个基于镜头的景深模型，以几何光学原理为基础控制景深效果。该模型通过学习每个输入图像的镜头参数（如光圈大小和焦距）来实现。其次，采用场景特定的深度先验，通过使用稀疏重建的深度图来指导场景几何的准确重建，确保在处理模糊图像时能够更好地捕捉细节。最后，应用模糊到聚焦的适应策略，以最小化模糊圈（CoC）中的不一致性，从而提升去模糊效果的性能。这些方法结合起来，使得DoF-Gaussian能够在新视图合成中实现更高的图像质量和更丰富的景深控制。
在实验部分，研究团队通过广泛的定量和定性评估来验证DoF-Gaussian的有效性。使用多种数据集，包括Deblur-NeRF和自建的合成数据集，比较了该方法与现有最先进的去模糊和景深处理方法的性能。实验结果表明，DoF-Gaussian在图像清晰度、结构保留和细节再现方面均优于其他方法，特别是在处理浅景深输入时。此外，进行了一系列消融实验，以评估每个组成部分对最终性能的贡献，结果显示，镜头模型和深度先验的结合显著提高了重建质量。最终，研究展示了DoF-Gaussian在动态场景和交互式应用中的潜力，强调了其在电影风格效果生成中的应用价值。
DoF-Gaussian的工作原理可以简单理解为利用镜头的特性来控制照片中的清晰区域。首先，我们需要调整镜头的光圈大小和焦距，这样就可以决定哪些部分是清晰的，哪些部分是模糊的。接下来，我们会根据每个场景的特点，利用一些已知的深度信息来帮助我们更准确地重建图像。最后，我们会根据模糊的程度进行调整，使得模糊的部分看起来更自然。通过这样的步骤，我们不仅可以从模糊的照片中恢复出清晰的图像，还能在不同的视角下生成新的图像，甚至可以让用户在拍摄时实时调整景深效果，创造出更具电影感的画面。
2025-03-03｜NVIDIA,NUS,UToronto,VectorInstitute｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.01774v1https ://research.nvidia.com/labs/toronto-ai/difix3d
DIFIX3D+是一种新颖的3D重建和新视角合成管道，旨在通过单步扩散模型提高重建质量。尽管神经辐射场（NeRF）和3D高斯喷溅（3DGS）等技术在3D重建和新视角合成方面取得了显著进展，但在极端新视角下实现照片级真实感仍然面临挑战，尤其是在稀疏输入场景中，常常会出现伪几何和缺失区域等伪影。DIFIX3D+的核心是DIFIX，一个经过训练的单步图像扩散模型，旨在增强和去除由于3D表示的约束不足而导致的渲染伪影。该方法不仅在重建阶段清理伪训练视图，还在推理阶段作为神经增强器，进一步提升渲染质量。DIFIX3D+兼容NeRF和3DGS表示，展现了在FID分数上平均提升2倍的能力，同时保持了3D一致性。
DIFIX3D+的工作流程包括几个关键步骤。首先，给定一个预训练的3D表示，系统会渲染出新视图并将其输入DIFIX，以去除伪影并提高渲染质量。此阶段的相机位姿通过插值获得，逐步接近目标位姿。接下来，经过清理的新视图会被重新蒸馏回3D表示中，以提升其质量。为了确保对扩散模型的强条件，步骤一和步骤二会在多个迭代中反复进行，逐步扩展重建的空间范围。最后，在推理时，DIFIX还会作为实时神经增强器，进一步改善渲染的新视图质量。这种方法的创新在于通过单步扩散模型快速去除伪影，同时在3D更新中保持多视图一致性。
在实验部分，DIFIX3D+在多个数据集上进行评估，包括DL3DV和Nerfbusters等，验证其在增强NeRF和3DGS管道方面的能力。实验结果表明，DIFIX3D+在所有性能指标上均优于对比方法，显示出显著的提升效果，特别是在LPIPS和FID评分上，分别减少了0.1和几乎3倍的相对值。此外，DIFIX3D+也在PSNR上实现了约1dB的提升，表明其在保持与原始视图的高保真度的同时，显著改善了感知质量。通过定量和定性分析，DIFIX3D+展示了其在去除伪影和提高渲染质量方面的有效性，特别是在复杂的现实场景中。
DIFIX3D+是一个用于提升3D图像质量的智能工具。它的工作原理可以分为几个步骤。首先，系统会生成一些新视图图像，然后使用DIFIX这个模型来清理这些图像中的瑕疵，比如模糊或不自然的区域。接着，经过处理的图像会被用来改进3D模型的整体质量，确保不同视角下的图像看起来一致。这个过程会重复多次，以不断提升图像的质量。最后，当你查看这些图像时，DIFIX会在后台运行，进一步优化图像，确保它们看起来尽可能真实。这种方法的好处是，它不仅能快速修复图像中的问题，还能在不同的视角下保持一致的视觉效果。
2025-03-02｜Stanford,NVIDIA｜Eurographics2025｜⭐️🟡
http ://arxiv.org/abs/2503.01016v1
在现代计算机图形学中，关键帧是描述运动的标准方式。然而，动画师在创建关键帧时，往往面临着时间不准确的问题，这导致生成的运动质量下降。为了解决这一挑战，本文提出了一种新的生成运动插值方法，能够从不精确时间的关键帧中生成高质量的动画。该方法的核心在于引入了一种时间扭曲函数，能够在生成过程中自动调整关键帧的时序，同时添加空间细节以丰富运动表现。通过这一创新，系统能够将大致时间的关键帧转换为多样且逼真的运动序列，确保运动的合理性和细节的丰富性，极大地简化了动画师的工作流程。
本文的方法包括两个主要创新，首先是数据生成过程，其次是模型架构。具体而言：
数据生成：我们从高质量的运动片段中合成出不精确时间的关键帧。通过随机选择运动的极值姿势，并对其进行时间上的轻微偏移，形成一组新的关键帧。这些关键帧可以稀疏或密集地分布在运动序列中，模拟现实中的不一致性。
模型架构：我们设计了一种双头的扩散模型，能够同时预测一个全局时间扭曲函数和局部姿态残差。全局时间扭曲函数用于调整关键帧的时间，而局部姿态残差则用于补充运动细节。这种模型能够在生成过程中保持运动的连贯性和丰富性，即使在关键帧时间不精确的情况下。
实验部分通过对比不同的基线方法，验证了所提出模型的有效性。我们设置了多个实验条件，包括不进行时间扭曲的基线（NoTime）、只预测姿态残差的基线（NoWarp）以及当前最先进的模型（CondMDI）。通过对生成运动的重建精度、姿态保留误差和运动多样性等多个指标进行评估，结果显示，本文的方法在运动的时间重建和细节保留上均优于其他基线。具体而言，模型在处理大规模的运动数据集时，能够生成更自然且多样化的运动序列，充分展示了其在动画制作中的潜力。
在我们的研究中，我们提出了一种新方法，可以帮助动画师从大致时间的关键帧生成流畅的运动。想象一下，如果你在制作一段舞蹈动画，但你并不确定每个动作的确切时间，我们的方法可以自动调整这些动作的时间，使得整体看起来更自然。我们首先从已有的高质量舞蹈视频中提取出一些关键的动作，然后稍微改变这些动作的时间，形成新的关键帧。接下来，我们使用一种智能算法，能够同时调整这些关键帧的时间，并补充动作之间的细节。这就像在拼图时，不仅要把每块拼图放在正确的位置，还要确保每块拼图之间的连接自然顺畅。通过这样的方式，动画师可以节省大量时间，同时也能创造出更生动的动画效果。
2025-03-03｜ShanghaiTech,CUHK｜CVPR2025｜⭐️
http ://arxiv.org/abs/2503.01291v1https ://4dvlab.github.io/project_page/semgeomo/
本文提出了一种新颖的方法SemGeoMo，旨在生成动态环境中合理且高质量的人类交互动作。随着人机交互的日益重要，传统的人类动作生成方法往往局限于静态场景，缺乏对动态环境的适应能力。SemGeoMo通过结合语义信息和几何引导，有效地构建了一个互动导向的世界模型，使得生成的人类动作不仅符合常识，还能在几何上准确。该方法的核心在于利用文本、可接触性和关节信息等多层次引导，确保生成动作的语义一致性和几何精确性。实验结果表明，SemGeoMo在多个数据集上达到了最先进的性能，并展示了其在多样化交互场景中的优越泛化能力。
SemGeoMo的方法分为两个主要阶段。首先，使用大语言模型（LLM）作为注释器生成初步的文本描述，提供语义引导。在此基础上，进行SemGeo分层引导生成，提取交互对象的几何特征和关节信息。具体步骤包括：
数据表示：将人类动作表示为时间序列的特征向量，并使用点云表示交互目标的几何信息。
LLM注释器：该模块从点云中提取交互目标的边界框，并生成粗略文本描述，随后细化为更精确的描述。
分层引导生成：在第一阶段生成可接触性和关节级别的引导信息后，第二阶段利用这些信息指导详细的人类动作生成，从而提高生成结果的语义合理性和几何准确性。
运动控制网络：通过条件扩散模型生成高保真动作，确保生成的动作与给定的条件相一致。
实验部分主要通过在多个数据集上的评估来验证SemGeoMo的有效性。使用的主要数据集包括FullBodyManipulation、BEHAVE、IMHD2和HoDome，涵盖了不同的交互场景。评估指标包括手部和关节位置误差、接触精度、生成多样性等。实验结果显示，SemGeoMo在各项指标上均优于现有的基准方法，尤其在接触精度和生成质量方面表现突出。此外，通过消融实验，进一步分析了语义和几何引导对生成结果的影响，证实了多层次引导的有效性。最终，SemGeoMo展示了在未见对象和复杂交互场景中的良好泛化能力，证明了其在动态人机交互中的应用潜力。
SemGeoMo的核心思想是帮助机器人或虚拟角色在动态环境中做出合理的运动。它的工作流程可以简单理解为两个步骤。首先，系统会分析周围的环境，比如人和物体的形状和位置，并使用一种智能语言模型来生成关于如何与这些物体互动的描述。接着，系统会根据这些描述生成具体的动作，比如一个人如何弯腰去拿一个物体。在这个过程中，系统不仅考虑了动作的意义，还确保这些动作在空间上是合理的，比如避免碰撞或保持平衡。通过这种方式，SemGeoMo能够生成既符合逻辑又准确的动作，适用于各种交互场景，如机器人助手、游戏角色等。
2025-03-01｜Unknown｜⭐️
http ://arxiv.org/abs/2503.00382v2
本研究提出了一种新颖的文本驱动人机交互合成系统EigenActor，旨在从文本指令中推断出3D人类与物体的交互（HOIs）。现有的文本到HOI合成方法主要依赖于将文本直接映射到特定对象的3D运动，这可能导致性能瓶颈。我们观察到，具有相同交互意图的HOI样本在不同目标之间展现出相似的动作模式，因此，学习有效的动作特定运动先验和对象特定交互先验对模型的文本-HOI语义一致性和身体-对象交互的现实性至关重要。EigenActor通过分阶段推理策略，首先推断对象无关的典型身体动作，然后丰富对象特定的交互风格，从而有效提升合成的语义一致性和交互现实性。
EigenActor的核心方法包括两个主要组件：BodyNet和ObjectNet。BodyNet负责从给定的文本和对象形状条件中合成3D全身姿势，采用分阶段推理策略。具体步骤如下：
动作特定运动推断：利用去噪扩散模型学习从文本到其对应的典型运动的映射。通过逐步去噪，生成与文本指令一致的动作特定姿势。
对象特定交互推断：在推断出的动作基础上，进一步学习与指定对象形状相协调的交互风格。ObjectNet则负责规划3D对象的运动，结合推断出的身体姿势和文本条件，确保身体与对象的交互自然合理。该模块包含接触部位推断、对象运动扩散和交互优化三部分，确保生成的HOI样本在物理上合理且符合语义。
我们在三个大规模数据集上对EigenActor进行了广泛的实验评估，包括HIMO、FullBodyManipulation和GRAB。实验分为多个阶段，涉及动作特定运动和对象特定交互的去噪训练。通过定量和定性评估，我们比较了EigenActor与其他先进方法的表现。评估指标包括文本-HOI语义一致性、交互现实性、生成多样性等。结果显示，EigenActor在这些指标上均显著优于现有方法，验证了其在少样本学习和生成复杂人机交互方面的强大能力。此外，实验还展示了EigenActor在生成的HOI样本中更好的手-对象接触精度和物理合理性。
EigenActor的工作原理可以简单理解为分两步走。首先，它通过分析用户给出的文本指令，生成一个基础的身体动作。这一步就像是画出一个人要做的动作的轮廓，比如“举起一个杯子”。接着，EigenActor会根据这个动作，考虑到与对象的关系，进一步调整这个动作，使其看起来更自然、真实。这就像是在给这个动作加上细节，比如手如何握住杯子、身体如何倾斜等。通过这种方式，EigenActor不仅能确保动作符合文本指令，还能让人和物体的互动显得更加真实和合理。这样，用户就能看到一个既符合他们要求又生动自然的3D动画效果。

本研究提出了一种名为“共同理解您的命令与意图”的方法，旨在实现场景感知的3D人类动作合成与分析的相互协作。通过引入一个名为“共演化合成-分析（CESA）”的管道，该方法将文本描述、3D场景以及3D人体动作的生成与分析结合起来，形成一个双向学习的系统。具体而言，基于给定的文本描述，场景感知的文本到人类合成能够生成多样化的室内动作样本，从而提升人类-场景交互的多样性，并为构建稳健的人类动作分析系统提供支持。同时，动作分析也为合成过程提供语义一致性的检验，从而提升生成动作的真实感。该方法的核心在于将人类动作的合成与分析视为两个相互促进的任务，通过共同学习实现更高效的结果。
在方法部分，我们提出了一种层次化的生成策略，将文本驱动的场景特定人类动作合成分为三个阶段：目标推断、路径规划和姿态合成。具体步骤如下：
目标推断：根据给定的文本指令和3D场景，推断出动作目标。使用多层感知机（MLP）来生成目标的高斯分布参数，从而实现非确定性的目标推断。
路径规划：基于推断出的目标，规划移动路径。路径解码器会生成一个N帧的3D运动路径，模型通过结合目标和场景条件来探索路径的不确定性。
姿态合成：在确定的路径上合成具体的3D姿态，解码器会生成符合路径的多样化动作样本。通过学习姿态的后验概率分布，确保生成的动作在风格和自然性上具有多样性。
在实验部分，我们使用了多个数据集来评估所提出方法的有效性，包括HUMANISE和TRUMANS等。通过定量和定性的评估指标，如FrechetInceptionDistance（FID）、多样性（DIV）、运动现实性评分（MRS）和文本-动作一致性评分（TMCS），我们验证了CESA在运动合成和分析上的优越性。实验结果显示，与现有的最先进方法相比，CESA在动作的真实感、多样性和文本一致性方面均有显著提升。此外，用户感知研究也表明，参与者对CESA生成的动作质量和语义一致性给予了较高评价。综合分析表明，结合动作合成与分析的共演化策略显著增强了模型的性能。
在这个研究中，我们开发了一种新方法来让计算机更好地理解人类的动作和意图。我们把这个方法叫做“共同理解”，它可以通过一句话和一个3D场景来生成和分析人类的动作。这个过程分为三个简单的步骤：首先，我们让计算机根据描述的内容猜测人要做什么，比如“走到椅子旁边”。接下来，它会计划出一个移动的路径，告诉人应该如何走。最后，计算机会根据这个路径生成具体的动作，比如“坐下”或“站起来”。通过这种方法，我们不仅可以生成多样的动作，还能确保这些动作在场景中看起来很自然。我们的实验结果显示，这种方法比传统的方式更有效，能生成更真实、更符合文本描述的动作。
