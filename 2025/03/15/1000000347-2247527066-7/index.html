312-A3.2｜单图生VDM；自动化角色扮演游戏角色创造，人类几何生成，语音驱动3D说话头

主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月4日｜周二
2025-03-01｜UTAustin,SFU,AdobeResearch｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.00605v1
本研究提出了一种名为GenVDM的新方法，旨在从单张图像生成向量位移图（VDM），这是一种在3D建模中常用的详细几何图案。与现有的3D生成模型不同，GenVDM专注于生成可以无缝附加到3D形状表面的几何部分。该方法首先通过多视角法生成法线图，然后通过创新的重建流程将法线图转换为VDM。此外，研究还开发了一种高效的算法，用于从3D对象中提取VDM，并建立了第一个学术VDM数据集。实验结果表明，GenVDM在生成细节方面的表现优于现有的基准方法，为艺术家在3D形状中添加几何细节提供了丰富的控制选项。
GenVDM的方法包括三个主要步骤。首先，输入图像通过图像扩散模型生成多视角法线图，这些法线图提供了目标对象在不同视角下的几何信息。其次，使用这些法线图重建一个3D网格，这一过程采用了神经隐式表面（SDF）表示法，以确保生成的网格能够准确反映目标形状。最后，应用一个神经变形模型，将生成的3D形状参数化为VDM图像。在这个过程中，研究者们还设计了数据处理管道，以从Objaverse3D模型中提取形状补丁，进而创建VDM的训练数据集。此外，为了解决缺乏训练数据的问题，研究者们开发了一个交互式工具，以从3D对象中分割出有趣的几何区域，从而生成VDM。
在实验部分，研究者们对GenVDM进行了广泛的评估，旨在验证其有效性，并与多种最先进的方法进行比较。实验中使用了一个包含50张RGB图像的数据集，这些图像展示了常见的VDM类别。通过计算CLIP相似度和3D-FID分数，研究者们定量评估了生成结果的视觉保真度和语义相似性。结果显示，GenVDM在多个指标上均优于其他基准模型，尤其是在生成细节和处理小型VDM区域方面。此外，研究还通过消融实验验证了不同设计选择的有效性，进一步证明了该方法在3D建模中的潜力。
GenVDM的工作流程可以简单理解为三个步骤。首先，我们输入一张图片，系统会利用一种叫做图像扩散模型的技术，从不同的视角生成该物体的法线图，这些法线图就像是物体表面的“深度地图”。接着，系统会根据这些法线图重建出一个3D网格，确保这个网格能够准确地代表物体的形状。最后，系统会将这个3D形状转换成一个VDM图像，这个图像可以用来在3D模型上添加细致的几何细节。为了让系统更好地学习，研究者们还开发了一种工具，可以从现有的3D模型中提取出有趣的部分，形成用于训练的数据集。这样，艺术家就可以通过简单的图片，轻松地为他们的3D模型添加独特的细节。
2025-03-03｜NeteaseFuxiAILab,UQueensland｜⭐️🟡
http ://arxiv.org/abs/2503.01158v1
EasyCraft是一种创新的框架，旨在自动化角色扮演游戏（RPG）中的角色定制过程。该方法的核心在于支持文本和图像输入的统一处理，解决了现有方法在不同游戏引擎间的通用性问题。传统的自动化角色创建方法通常只支持特定类型的输入，导致在多样化的游戏环境中表现不佳。EasyCraft通过构建一个自我监督学习的视觉变换器（ViT）编码器，创建了一个统一的特征分布，使得来自任何风格的面部图像都可以转换为特定的游戏引擎参数。此外，EasyCraft还结合了文本到图像的技术，使得用户可以通过文本描述直接生成个性化的3D角色，极大地提升了角色创建的灵活性和准确性。
EasyCraft的方法主要包含以下几个部分：首先，通过自我监督学习，构建一个通用的视觉变换器（ViT）编码器，能够从多种风格的面部图像中提取特征。其次，利用该编码器，设计一个翻译器，将输入图像转换为游戏引擎所需的角色参数。这一过程包括生成面部结构参数、化妆纹理参数和化妆属性参数。翻译器的训练依赖于游戏引擎数据，采用两阶段训练策略：第一阶段为ViT编码器的预训练，第二阶段为特定引擎的翻译器训练。此外，EasyCraft还结合了稳定扩散（SD）模型，用于文本到图像的生成，确保生成的图像与游戏引擎的风格一致，从而实现更准确的文本基础角色创建。
在实验部分，EasyCraft在两个复杂的角色定制引擎上进行了广泛的测试，评估其性能与现有方法的比较。首先，构建了一个包含510万张多样化面部图像的大规模数据集，以训练ViT编码器。随后，使用2000组测试样本进行评估，比较EasyCraft与其他两种自动化角色创建方法在身份相似度、FID评分和生成速度等指标上的表现。结果表明，EasyCraft在身份相似度和生成速度上均优于传统方法，且在FID评分上表现出色，显示出其在多样化输入和引擎适应性方面的优势。
EasyCraft的工作原理可以简单理解为一个智能助手，它能够根据你的照片或文字描述，快速生成一个3D角色。首先，EasyCraft会分析你的照片，提取出面部特征，比如眼睛的大小、鼻子的形状等。接着，它会将这些特征转化为游戏中可以使用的参数，就像给角色穿衣服一样，确保角色看起来既真实又符合游戏风格。如果你输入的是文字描述，EasyCraft会利用一个强大的模型，生成符合描述的角色图像，然后再将这些图像转化为角色参数。这种方法不仅提高了角色创建的效率，还让玩家能够更加自由地定制自己的游戏角色，无论是通过照片还是文字描述。
2025-03-03｜KAUST｜⭐️🟡
http ://arxiv.org/abs/2503.01448v1
本研究提出了一种新颖的3D人类几何生成框架，旨在通过建模人类几何分布来生成高保真度的人物模型。该方法的核心在于利用扩散去噪模型，首次实现了对人类几何分布的学习。通过两阶段的生成流程，首先生成几何分布，然后从中采样以合成高质量的人物模型。研究中验证了该方法在两个主要任务上的有效性：基于姿态的3D人类生成和单视角新姿态生成。实验结果表明，与现有的最先进生成方法相比，所提方法在真实感和几何保真度方面均取得了显著的提升。
该方法的实现可分为几个关键步骤：
几何分布建模：首先，利用SMPL模型生成与人类几何相近的分布，并将其编码为紧凑的特征图。这一过程通过流匹配算法实现，旨在减少建模复杂性。
特征图生成：通过自编码器架构，将几何分布映射为特征图，以便于后续的生成任务。特征图中包含了丰富的几何信息，能够有效捕捉服装的细节。
条件生成任务：该方法设计了两种生成任务：姿态条件下的随机生成和新姿态生成。前者依赖于给定的SMPL姿态模板，后者则通过单视角法生成新姿态模型，确保生成的人物模型在不同姿态下保持自然的服装变形。
在实验部分，研究者进行了多项评估以验证所提方法的有效性。首先，通过消融实验分析了训练样本对生成质量的影响，比较了不同几何分布的训练效率。其次，使用THuman2数据集进行了姿态条件生成和新姿态生成的对比实验，结果表明该方法在生成几何质量方面显著优于其他现有方法。在姿态条件生成中，所提方法的几何质量提高了57%，而在新姿态生成中，生成的服装细节与人类姿态高度一致，展示了优越的鲁棒性。
我们的方法可以想象成一个制作人类3D模型的工厂。首先，我们通过一个叫SMPL的模型来理解人类的形状和姿态。接着，我们将这些形状转化为一种“特征图”，就像把复杂的设计图简化成易于理解的草图。然后，我们有两个主要的任务：一个是根据给定的姿势生成相应的人物模型，另一个是从一张普通的图片中生成新姿势的人物模型。我们的方法通过这种分阶段的方式，能够生成非常细致和自然的人物形象，尤其是在服装的细节处理上，表现得尤为出色。
2025-03-01｜SJTU,XueshenAI｜⭐️
http ://arxiv.org/abs/2503.00495v1https ://xuanchenli.github.io/TexTalk/
在当前的3D面部动画研究中，语音驱动的技术已经取得了显著的进展。然而，许多现有方法主要关注面部运动的生成，而忽视了动态纹理在渲染高保真谈话头像中的重要性。本研究提出了一种新颖的高分辨率4D数据集TexTalk4D，包含100分钟的音频同步扫描级网格及其详细的8K动态纹理。基于此数据集，我们引入了TexTalker框架，使得面部运动和动态纹理可以同时从语音中生成，显著提升了个性化纹理面部动画的真实感和一致性。此外，我们提出了一种新的样式注入策略，以便更好地捕捉不同的面部运动和皱纹风格，从而实现对动画风格的解耦控制。
本研究的核心方法包括三个主要步骤：首先，我们将面部运动和纹理变化表示为运动图和皱纹图，并将其压缩到两个独立的低维潜在空间中。其次，基于学习到的低维动画原理，我们训练了一个潜在扩散模型（LDM），以联合捕捉语音驱动下的长时运动和皱纹相关性。最后，为了实现风格的解耦控制，我们提出了一种新的基于枢轴的样式注入方法，从训练的潜在空间中提取各种风格的枢轴，并将其注入到输出中。这些步骤的结合使得TexTalker能够生成与语音同步的高质量面部动画和动态纹理。
我们在TexTalk4D数据集上进行了大量实验，以评估TexTalker在生成面部运动和动态纹理方面的性能。实验分为定量比较和定性分析两部分。在定量比较中，我们使用了多个指标，如唇部顶点误差（LVE）、面部整体顶点误差（MVE）和上半脸动态偏差（FDD）来评估生成动画的质量。结果表明，TexTalker在各项指标上均优于现有的最先进方法。在定性分析中，我们通过用户研究评估了生成动画的真实感和几何-纹理一致性。结果显示，TexTalker生成的动画在真实感和一致性方面表现最佳，进一步验证了其有效性。
在我们的研究中，我们提出了一种新方法来创建3D面部动画，这种动画能够根据人们说话时的声音和表情变化而变化。首先，我们把面部的运动和纹理变化分别记录下来，就像拍照时记录每一个动作和表情一样。接着，我们使用一种叫做潜在扩散模型的技术，这种技术可以帮助我们更好地理解声音和表情之间的关系。最后，我们引入了一种新的方法，允许我们在生成动画时灵活地调整不同的风格，比如说话的方式和面部的皱纹。这种创新使得我们能够生成更加自然和个性化的面部动画，适用于各种应用场景，比如电影和游戏。
