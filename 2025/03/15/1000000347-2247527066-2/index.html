主编｜庄才林（CailinZhuang）技术支持｜胡耀淇（YaoqiHu）｜编辑支持｜张奇佳（QijiaZhang）发布日期｜2025年3月4日｜周二
2025-03-03｜USTC,MSRA｜CVPR2025｜⭐️⭐️
http ://arxiv.org/abs/2503.01645v1
本文介绍了DesignDiffusion，这是一种基于扩散模型的框架，旨在从文本描述中合成高质量的设计图像。设计图像在图形设计、广告和科学可视化等多个领域中具有重要作用。传统的文本生成方法往往面临生成的图像与文本元素之间的不一致性，限制了创造力。为了解决这一问题，DesignDiffusion采用了一种端到端的一阶段扩散框架，旨在直接从用户提示中合成图像和文本元素。该框架利用独特的字符嵌入来增强输入提示，并引入字符定位损失以提高文本生成的监督效果。通过一系列实验，DesignDiffusion展示了其在设计图像生成方面的卓越性能，超越了现有的最先进技术。
DesignDiffusion的核心方法包括几个关键技术。首先，采用字符级别的单词分解，提高了CLIP文本编码器对每个字符的渲染能力。其次，引入字符定位损失，迫使模型关注每个字符在图像中的具体区域，从而确保文本的准确视觉表现。最后，采用自我对抗直接偏好优化（SP-DPO）策略进行模型的微调，以增强生成的文本质量和图像的整体表现。这一微调过程不需要人工标注数据，而是通过生成的样本与真实样本的比较来优化模型。通过这些方法，DesignDiffusion能够更好地处理复杂的文本和视觉内容生成，确保生成图像的高质量与一致性。
在实验部分，研究团队首先收集了包含一百万个高质量设计图像的数据集，并对其进行了训练。采用了多种评估指标来衡量生成图像的质量，包括FID（FréchetInceptionDistance）、文本精度、召回率、F1分数及准确率等。实验结果表明，DesignDiffusion在生成设计图像的各项指标上均优于现有的最先进技术。此外，通过用户研究，参与者对生成图像的美学、文本质量、布局美学及文本-图像匹配进行了评估，结果显示DesignDiffusion在这些方面表现突出。实验结果证明了该方法在设计图像生成中的有效性和鲁棒性。
DesignDiffusion的工作原理可以简单理解为三大步骤。首先，为了让计算机更好地理解文本，研究人员将每个字母单独处理，而不是把整个单词当作一个整体。这种方法使得生成的图像中的文字更加准确。其次，研究人员设计了一种特殊的损失函数，帮助计算机在生成图像时，能够关注每个字母在图像中的位置，从而确保文字和图像的完美结合。最后，通过一种叫做自我对抗优化的技术，研究人员不断调整和改进模型，使其生成的图像和文字更加符合人们的期望。通过这三种方法的结合，DesignDiffusion能够创造出既美观又符合要求的设计图像。
2025-03-01｜Adobe｜CVPR2025｜⭐️🟡
http ://arxiv.org/abs/2503.00591v1
本研究提出了一种名为Aesthetic-AwarePreferenceAlignment（AAPA）的新方法，旨在通过提升多模态大语言模型（MLLMs）的美学感知来改善图形布局设计。当前的生成模型在内容感知布局生成方面取得了一定进展，但它们往往未能充分理解布局设计的美学要求，且与人类偏好之间存在差距。AAPA通过引入美学偏好对布局预测进行训练，利用MLLM的美学能力进行直接的偏好优化。研究中还提出了一种数据过滤协议，以确保训练数据的高质量，并引入了一种新的评估指标，通过另一种MLLM来计算生成布局在美学标准下相较于真实布局的胜率。通过对多个挑战性基准的定性和定量分析，验证了AAPA在不同规模的MLLM（从1B到8B参数）中的有效性，显示出相较于现有最先进方法有显著提升。
本研究的布局预测方法以生成美学一致的设计为目标，使用一个多模态大语言模型（MLLM）作为布局质量的评估者。具体方法分为以下几个步骤：
输入表示：将设计元素（如图像和文本）转化为多模态嵌入，便于后续处理。
位置感知布局指令调优：将连续的边界框坐标离散化为固定的离散位置令牌，简化布局预测任务。
美学偏好对齐：引入另一个MLLM作为评判者，进行图形布局的成对比较，判断哪一布局更具美学优势。
训练过程：使用AAPA损失函数，优化布局模型，使其生成的布局更符合美学偏好。
数据过滤：通过质量指标筛选高质量布局，以增强训练数据的有效性。
研究在两个基准数据集——Crello和WebUI上评估了AesthetiQ的性能。Crello数据集包含多种格式的设计模板，而WebUI数据集则涵盖了大量网页用户界面。实验中，采用了多种评估指标，包括平均交并比（MeanIoU）和判别模型赢率（JudgeWinRate），以全面评估模型在布局生成中的表现。结果显示，AesthetiQ在不同规模的模型中均表现出色，尤其是8B参数的模型在各项指标上均优于现有基准，展示了其在处理复杂布局和保持视觉一致性方面的能力。此外，预训练和质量过滤的引入显著提升了模型的美学敏感度和布局理解能力。
在这项研究中，我们提出了一种新的方法来帮助计算机更好地设计图形布局。我们使用了一种叫做多模态大语言模型的技术，它可以同时理解文字和图像。我们的步骤如下：首先，我们把设计元素（比如图片和文字）转化为计算机可以理解的格式。接着，我们将元素的位置数据简化为几个离散的选项，这样计算机可以更容易地进行布局预测。然后，我们引入另一个模型来比较不同的布局，看看哪一个更好看。最后，我们通过只使用高质量的布局数据来训练我们的模型，确保它学习到的都是最优的设计。实验结果表明，我们的方法在设计美观的图形布局方面表现优异，能够生成更符合人类审美的设计。
2025-02-26｜UPitt｜⭐️🟡
http ://arxiv.org/abs/2503.00046v1https ://github.com/JoeyHou/ads_creativity
在创意评估领域，尤其是广告创意的评估中，研究者们面临着主观性和复杂认知过程的挑战。本文旨在将视觉广告创意分解为“非典型性”和“原创性”两个维度，并通过细致的人类标注来建立一个新的评估基准。研究者们提出了一系列专门针对这一主观问题的任务，并评估了当前最先进的视觉语言模型（VLM）在这一基准上的表现。这一研究不仅展示了VLM在自动化创意评估中的潜力，同时也揭示了其面临的挑战，为未来的研究奠定了基础。
本研究采用了多种方法来评估广告的创意表现，主要包括以下几个方面：
维度分解：将创意分解为“非典型性”和“原创性”，这两个维度被认为是广告创意最重要的组成部分。
数据收集：通过AmazonMechanicalTurk收集高质量的人类标注，确保每个广告在这两个维度上的评分都有充分的依据。
分布建模：采用分布建模方法，模拟人类群体在创意评分上的行为，而不是简单的单一标签预测。这种方法通过多次提示VLM，收集多个评分来近似真实的评分分布。
评估指标：使用Spearman相关系数和Kullback-Leibler散度（KL散度）来评估VLM输出与人类评分之间的相关性和分布差异。
实验部分主要集中在评估VLM在创意评估任务中的表现。使用了PittAds数据集，其中包括两个子集：Creative-100和Atypical-300。研究者们对100个广告进行了详细分析，收集了关于创意、非典型性和原创性的评分。同时，针对300个广告的非典型性进行了二元标注。实验中，研究者们使用了多种开源和闭源的VLM，如LLaVA和GPT-4v，并进行了零-shot提示推理。实验结果显示，VLM在创意评估任务中与人类评分存在显著的相关性，但在捕捉评分的主观性方面仍面临一定挑战。
在这项研究中，科学家们想要找出如何用计算机来评估广告的创意。他们将创意分成两个部分：一个是“非典型性”，也就是广告是否与众不同；另一个是“原创性”，即广告是否有新意。为了做到这一点，他们请人类评审来给广告打分，并收集了很多不同的意见。接着，他们使用了一种叫做分布建模的方法，模拟人们在给广告评分时的不同意见，而不是只给出一个简单的分数。最后，他们还用一些数学方法来比较计算机的评分和人类的评分，看看它们之间有多接近。这种方法帮助他们理解计算机在评估广告创意时的表现和局限性。
2025-03-03｜ZJGSU,Inceptio,ZJU｜⭐️
http ://arxiv.org/abs/2503.01294v1
本文提出了一种新颖的服装中心外扩（GCO）框架，旨在生成可控的服装展示图像。该框架基于潜在扩散模型（LDM），通过两阶段的流程实现对时尚模型外观的精细控制。第一阶段引入了一种服装自适应姿态预测模型，该模型能够根据给定服装生成多样的姿态图。第二阶段则利用预测的姿态、服装图像、文本提示和面部图像生成服装展示图像。与现有的服装生成方法不同，GCO框架消除了对服装变形的需求，从而确保了服装细节的真实保留。通过大规模实验，本文验证了GCO在生成质量和可控性方面优于现有的最先进方法。
GCO框架的设计分为两个主要阶段。第一阶段是服装自适应姿态预测模型，针对输入的服装图像，生成适合该服装的姿态图。具体步骤如下：
输入服装图像，通过预训练的服装分割模型和DensePose模型，获取变形的服装图像和相应的姿态图。
利用反向去噪过程，训练UNet去噪器，以生成与服装匹配的姿态图。第二阶段为服装中心外扩模型，使用LDM生成展示图像，步骤包括：
将服装图像、预测的姿态图、文本提示和面部图像作为输入，进行图像生成。
设计多尺度外观定制模块（MS-ACM），实现对生成模型外观的整体和细致控制。
应用轻量特征融合操作，将多个条件有效整合，提高生成效率。
为验证GCO框架的有效性，本文进行了广泛的实验。实验数据来自VITON-HD数据集，该数据集包含14221张训练图像和2032张测试图像。通过与多种基线方法进行比较，评估了GCO在生成质量和可控性方面的表现。实验结果表明，GCO在多项指标上均优于现有方法，包括生成的图像质量、面部特征保留和服装细节的真实度。此外，用户研究也显示，GCO在用户偏好度方面表现优异，能够更好地满足用户对服装展示图像的期望。
GCO框架的工作原理可以简单理解为两步走。第一步是预测服装穿着者的姿势。系统会根据给定的服装图像，生成一个适合该服装的姿势图，就像为模特挑选合适的站姿一样。第二步是生成最终的服装展示图像。系统会结合服装图像、姿势图、文本描述和面部图像，创造出一个穿着该服装的模特的图像。这个过程不仅确保了服装的细节得到保留，还允许用户通过简单的文本描述来调整模特的外观，比如肤色或发型等。这种方法比传统的服装生成方式更加高效和灵活。
